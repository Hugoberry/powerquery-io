# Number.Random


Returns a random number.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Returns a random number between 0 and 1.


## Examples

### Example #1 
Get a random number.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
