# Number.RandomBetween


Returns a random number between two numbers.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Remarks

Returns a random number between <code>bottom</code> and <code>top</code>.


## Examples

### Example #1 
Get a random number between 1 and 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
