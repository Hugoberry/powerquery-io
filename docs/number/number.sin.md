# Number.Sin


Returns the sine of the number.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

Returns the sine of <code>number</code>.


## Examples

### Example #1 
Find the sine of the angle 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
