# Cube.ApplyParameter


Returns a cube after applying a parameter to it.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

Returns a cube after applying <code>parameter</code> with <code>arguments</code> to <code>cube</code>.



## Category
Cube
