# DateTimeZone.UtcNow


Returns the current date and time in UTC (the GMT timezone).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

Returns the current date and time in UTC (the GMT timezone).


## Examples

### Example #1 
Get the current date &amp; time in UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```

DateTimeZone
