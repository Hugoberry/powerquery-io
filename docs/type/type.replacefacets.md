# Type.ReplaceFacets


Replaces the facets of a type.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Remarks

Replaces the facets of <code>type</code> with the facets contained in the record <code>facets</code>.

Type
