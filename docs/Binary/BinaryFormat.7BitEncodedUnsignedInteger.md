# BinaryFormat.7BitEncodedUnsignedInteger


## Description

A binary format that reads a 64-bit unsigned integer that was encoded using a 7-bit variable-length encoding.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details
