# Text.Length


Returns the number of characters.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

Returns the number of characters in the text <code>text</code>.


## Examples

### Example #1 
Find how many characters are in the text &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
