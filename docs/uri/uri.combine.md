# Uri.Combine


Returns an absolute URI that is the combination of the input base URI and relative URI.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Remarks

Returns an absolute URI that is the combination of the input <code>baseUri</code> and <code>relativeUri</code>.
