# Number.Ln


Returns the natural logarithm of the number.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Remarks

Returns the natural logarithm of a number, <code>number</code>. If <code>number</code> is null <code>Number.Ln</code> returns null.


## Examples

### Example #1 
Get the natural logarithm of 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
