# Number.BitwiseShiftLeft


## Description

Shifts the bits set to the left.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Returns the result of performing a bitwise shift to the left on <code>number1</code>, by the specified number of bits <code>number2</code>.



## Category
Number.Bytes
