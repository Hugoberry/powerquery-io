# Number.Factorial


Returns the factorial of the number.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

Returns the factorial of the number <code>number</code>.


## Examples

### Example #1 
Find the factorial of 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
