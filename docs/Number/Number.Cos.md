# Number.Cos


## Description

Returns the cosine of the number.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Returns the cosine of <code>number</code>.


## Examples

### Example #1 
Find the cosine of the angle 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
