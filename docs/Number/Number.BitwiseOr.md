# Number.BitwiseOr


Returns the result of performing a bitwise &#34;Or&#34; between the two inputs.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Returns the result of performing a bitwise "Or" between <code>number1</code> and <code>number2</code>.



## Category
Number.Bytes
