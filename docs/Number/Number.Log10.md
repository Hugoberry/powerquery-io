# Number.Log10


## Description

Returns the base 10 logarithm of the number.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Returns the base 10 logarithm of a number, <code>number</code>. If <code>number</code> is null <code>Number.Log10</code> returns null.


## Examples

### Example #1 
Get the base 10 logarithm of 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
