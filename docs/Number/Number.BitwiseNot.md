# Number.BitwiseNot


## Description

Returns a byte where each bit is the opposite of the input.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Returns the result of performing a bitwise "Not" operation on <code>number</code>.



## Category
Number.Bytes
