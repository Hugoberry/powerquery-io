# Number.IsEven


## Description

Indicates if the value is even.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

Indicates if the value, <code>number</code>, is even by returning <code>true</code> if it is even, <code>false</code> otherwise.


## Examples

### Example #1 
Check if 625 is an even number.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Check if 82 is an even number.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
