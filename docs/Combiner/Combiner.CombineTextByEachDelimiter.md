# Combiner.CombineTextByEachDelimiter


## Description

Returns a function that combines a list of text into a single text using each specified delimiter in sequence.


## Syntax

```powerquery
Combiner.CombineTextByEachDelimiter(
    delimiters as list,
    optional quoteStyle as QuoteStyle.Type
) as function
```


## Details

Returns a function that combines a list of text into a single text using each specified delimiter in sequence.
