# Combiner.CombineTextByLengths


## Description

Returns a function that combines a list of text into a single text using the specified lengths.


## Syntax

```powerquery
Combiner.CombineTextByLengths(
    lengths as list,
    optional template as text
) as function
```


## Details

Returns a function that combines a list of text into a single text using the specified lengths.
