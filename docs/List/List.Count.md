# List.Count
Returns the number of items in the list.
> _function (<code>list</code> as list) as number_

# Description 
Returns the number of items in the list <code>list</code>.
# Category 
List.Information
# Examples 
Find the number of values in the list {1, 2, 3}.
```
List.Count({1, 2, 3})
```
> 3
