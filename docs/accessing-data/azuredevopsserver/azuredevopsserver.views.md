# AzureDevOpsServer.Views


Enter organization and project names.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Remarks

Returns a table of OData feeds offered by Azure DevOps.
