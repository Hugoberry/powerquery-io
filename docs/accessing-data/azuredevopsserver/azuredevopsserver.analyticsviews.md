# AzureDevOpsServer.AnalyticsViews


Enter organization and project names.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Returns a table of Analytics views offered by Azure DevOps.
