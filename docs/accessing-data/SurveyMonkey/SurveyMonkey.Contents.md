# SurveyMonkey.Contents



## Syntax

```powerquery
SurveyMonkey.Contents(

) as table
```


## Remarks

A Navigation table showing all the surveys in the account related to the input access token.


## Examples

### Example #1 
Returns the navigation table.
```powerquery
SurveyMonkey.Contents()
```
