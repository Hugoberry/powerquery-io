# BinaryFormat.7BitEncodedUnsignedInteger


Ett binärformat som läser ett 64-bitars heltal utan tecken som kodats med 7-bitars kodning med variabel längd.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks
