# Cube.AttributeMemberProperty


Returnerar en egenskap för ett dimensionsattribut.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

Returnerar egenskapen <code>propertyName</code> för dimensionsattributet <code>attribute</code>.



## Category
Cube
