# ActiveDirectory.Domains


Returnerar en lista över Active Directory-domäner i samma skog som den angivna domänen eller i den aktuella datorns domän om ingen anges.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Remarks

Returnerar en lista över Active Directory-domäner i samma skog som den angivna domänen eller i den aktuella datorns domän om ingen anges.
