# DateTime.FixedLocalNow


Returnerar aktuellt datum och tid i den lokala tidszonen. Det här värdet är fast och ändras inte i samband med efterföljande anrop.


## Syntax

```powerquery
DateTime.FixedLocalNow(

) as datetime
```


## Remarks

Returnerar ett <code>datetime</code>-värde för systemets aktuella datum och tid. Det här värdet är fast och ändras inte i samband med efterföljande anrop, till skillnad mot DateTime.LocalNow, som kan returnera olika värden när ett uttryck körs.
