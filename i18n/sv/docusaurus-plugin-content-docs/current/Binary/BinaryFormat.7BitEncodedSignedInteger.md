# BinaryFormat.7BitEncodedSignedInteger


## Description

Ett binärformat som läser ett 64-bitars heltal med tecken som kodats med 7-bitars kodning med variabel längd.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
