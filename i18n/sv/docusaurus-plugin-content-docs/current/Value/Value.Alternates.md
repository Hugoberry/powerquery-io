# Value.Alternates


## Description

Uttrycker alternativa frågeplaner.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Uttrycker alternativa frågeplaner inom ett frågeplansuttryck som uppnås via Value.Expression(Value.Optimize(...)). Inte avsett för annan användning.



## Category
Expression
