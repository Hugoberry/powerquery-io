# Value.Equals


## Description

Kontrollerar om två värden är lika.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Returnerar true om värdet <code>value1</code> är lika med värdet <code>value2</code>. Annars returneras false.



## Category
Values
