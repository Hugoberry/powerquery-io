# DateTimeZone.FixedUtcNow


## Description

Returnerar aktuellt datum och tid i UTC (GMT-tidszonen). Det här värdet är fast och ändras inte i samband med efterföljande anrop.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
