# Value.Compare


Returnerar -1, 0 eller 1 baserat på om det första textvärdet är mindre än, lika med eller större än det andra.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```


## Remarks

Returnerar -1, 0 eller 1 baserat på om det första textvärdet är mindre än, lika med eller större än det andra.
