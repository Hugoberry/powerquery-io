# Lines.ToText


## Description

Konverterar en lista med text till en enskild text. Angiven lineSeparator läggs till för varje rad. Om den inte anges används retur- och radmatningstecknen.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```
