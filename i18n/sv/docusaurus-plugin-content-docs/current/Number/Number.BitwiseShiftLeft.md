# Number.BitwiseShiftLeft


## Description

Skiftar bitarna till vänster.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnerar resultatet från ett bitvis skifte till vänster för <code>number1</code> med angivet antal bitar, <code>number2</code>.



## Category
Number.Bytes
