# Number.Atan2


## Description

Returnerar arctangens för divisionen av de två talen.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Returnerar arctangens för divisionen av de två talen, <code>y</code> och <code>x</code>. Divisionen skapas som <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
