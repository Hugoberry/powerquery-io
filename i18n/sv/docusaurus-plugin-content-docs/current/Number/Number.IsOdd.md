# Number.IsOdd


Anger om värdet är udda.


## Syntax

```powerquery
Number.IsOdd(
    number as number
) as logical
```


## Remarks

Anger om värdet är udda. Returnerar <code>true</code> om <code>number</code> är ett udda tal och annars <code>false</code>.


## Examples

### Example #1 
Kontrollera om 625 är ett udda tal.
```powerquery
Number.IsOdd(625)
```

Result: 
```powerquery
true
```


### Example #2 
Kontrollera om 82 är ett udda tal.
```powerquery
Number.IsOdd(82)
```

Result: 
```powerquery
false
```
