# Number.Random


Returnerar ett slumptal.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Returnerar ett slumpmässigt värde mellan 0 och 1.


## Examples

### Example #1 
Hämta ett slumptal.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
