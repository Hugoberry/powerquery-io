# Number.BitwiseNot


Returnerar en byte där varje bit är motsatsen till indatavärdet.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Remarks

Returnerar resultatet av en bitvis "Not"-åtgärd för <code>number</code>.



## Category
Number.Bytes
