# Number.Sin


Returnerar sinus för talet.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

Returnerar sinus för <code>number</code>.


## Examples

### Example #1 
Hitta sinus för vinkeln 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
