# Number.Ln


## Description

Returnerar den naturliga logaritmen för talet.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Returnerar den naturliga logaritmen för ett tal, <code>number</code>. Om <code>number</code> är null returnerar <code>Number.Ln</code> null.


## Examples

### Example #1 
Hämta den naturliga logaritmen för 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
