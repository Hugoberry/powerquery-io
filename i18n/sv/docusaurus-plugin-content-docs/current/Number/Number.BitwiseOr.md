# Number.BitwiseOr


## Description

Returnerar resultatet av en bitvis &#34;Or&#34;-åtgärd mellan de två indatavärdena.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnerar resultatet av en bitvis "Or"-åtgärd mellan <code>number1</code> och <code>number2</code>.



## Category
Number.Bytes
