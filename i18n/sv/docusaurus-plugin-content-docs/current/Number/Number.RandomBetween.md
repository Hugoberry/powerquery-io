# Number.RandomBetween


## Description

Returnerar ett slumptal mellan två tal.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Returnerar ett slumptal mellan <code>bottom</code> och <code>top</code>.


## Examples

### Example #1 
Hämta ett slumptal mellan 1 och 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
