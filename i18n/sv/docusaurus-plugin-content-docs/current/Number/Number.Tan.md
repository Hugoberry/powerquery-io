# Number.Tan


Returnerar tangens för talet.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

Returnerar tangens för <code>number</code>.


## Examples

### Example #1 
Hitta tangens för vinkeln 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
