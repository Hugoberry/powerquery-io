# Geography.ToWellKnownText


Översätter ett värde för strukturerad geografisk punkt till WKT-motsvarighet (well-known text).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Remarks

Översätter en strukturerad geografisk punkt till dess WKT-motsvarighet (well-known text) enligt definitionen i OGC (Open Geospatial Consortium), det serialiseringsformat som används av många databaser, däribland SQL Server.
