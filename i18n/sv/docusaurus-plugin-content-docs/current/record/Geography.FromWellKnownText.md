# Geography.FromWellKnownText


Översätter text som motsvarar ett geografiskt värde i WKT-format (well-known text) till en strukturerad post.


## Syntax

```powerquery
Geography.FromWellKnownText(
    input as text
) as record
```


## Remarks

Översätter text som motsvarar ett geografiskt värde i WKT-format (well-known text) till en strukturerad post. WKT är ett standardformat som definieras av OGC (Open Geospatial Consortium) och är det vanliga serialiseringsformatet som används av databaser, till exempel SQL Server.
