# List.NonNullCount


Returnerar antalet objekt i listan som inte är noll.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Returnerar antalet objekt i listan <code>list</code> som inte är noll.



## Category
List.Information
