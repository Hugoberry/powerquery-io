# Exchange.Contents


## Description

Returnerar en innehållsförteckning från ett Microsoft Exchange-konto.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Details

Returnerar en tabell med innehållet från Microsoft Exchange-kontot <code>mailboxAddress</code>. Om <code>mailboxAddress</code> inte har angetts används standardkontot för de autentiseringsuppgifter som används.
