# DynamicsNav.Contents


Ange URL:en för slutpunkten för din Dynamics NAV OData-tjänstslutpunkt.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

Returnerar en tabell med relevanta Dynamics NAV-data.
