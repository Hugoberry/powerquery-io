# Web.Page


## Description

Returnerar innehållet i HTML-dokumentet i strukturen med dokumentets beståndsdelar, samt en representation av det fullständiga dokumentet och texten när taggarna har tagits bort.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
