# Cube.DisplayFolders


## Description

Returnerar ett kapslat träd av tabeller som representerar mapphierarkin med objekt (till exempel dimensioner och mått).


## Syntax

```powerquery
Cube.DisplayFolders(
    cube as table
) as table
```


## Details

Returnerar ett kapslat träd av tabeller som representerar mapphierarkin med objekt (till exempel dimensioner och mått) som är tillgängliga att användas i <code>cube</code>.

Cube
