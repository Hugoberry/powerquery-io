# Cube.Transform


## Description

Tillämpar en lista med kubfunktioner.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Tillämpar en lista med kubfunktioner (<code>transforms</code>) på <code>cube</code>.
