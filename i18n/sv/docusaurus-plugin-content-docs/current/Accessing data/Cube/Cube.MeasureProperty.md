# Cube.MeasureProperty


## Description

Returnerar en egenskap för ett mått (cellegenskap).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Returnerar egenskapen <code>propertyName</code> för dimensionsattributet <code>measure</code>.



## Category
Cube
