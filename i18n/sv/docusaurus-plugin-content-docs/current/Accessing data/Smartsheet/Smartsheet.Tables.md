# Smartsheet.Tables


Returnerar en tabell med blad, rapporter, mappar och arbetsytor från Smartsheet-API:et


## Syntax

```powerquery
Smartsheet.Tables(

) as table
```


## Remarks

Returnerar en kapslad tabell med tillgängliga blad, rapporter, mappar och arbetsytor från Smartsheet-API:et.


## Examples

### Example #1 
Returnerar blad, mappar, rapporter och arbetsytor på toppnivå i Smartsheet-hierarkin
```powerquery
Smartsheet.Tables()
```

Result: 
```powerquery
En tabell med blad, mappar, rapporter och arbetsytor på toppnivå i Smartsheet-hierarkin
```
