# VSTS.Contents


Ange URL för din Azure DevOps Analytics Service.


## Syntax

```powerquery
VSTS.Contents(
    url as text,
    optional options as record
) as binary
```


## Remarks

Returnerar det innehåll som har laddats ner från Azure DevOps Analytics Service-URL:en som ett binärt värde. Funktionen använder globala autentiseringsuppgifter för alla åtkomliga organisationer.
