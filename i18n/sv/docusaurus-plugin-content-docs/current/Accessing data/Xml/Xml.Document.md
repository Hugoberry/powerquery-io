# Xml.Document


Returnerar innehållet i XML-dokumentet som en hierarkisk tabell.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Remarks

Returnerar innehållet i XML-dokumentet som en hierarkisk tabell.
