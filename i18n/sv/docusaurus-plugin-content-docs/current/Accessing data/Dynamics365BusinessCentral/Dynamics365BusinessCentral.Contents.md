# Dynamics365BusinessCentral.Contents


## Description

Ange Dynamics 365 Business Central-miljö och -företag.


## Syntax

```powerquery
Dynamics365BusinessCentral.Contents(
    optional company as text
) as table
```


## Details

Returnerar en tabell med relevanta Dynamics 365 Business Central-data.
