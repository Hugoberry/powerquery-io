# GoogleAnalytics.Accounts


## Description

Returnerar Google Analytics-konton.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

Returnerar Google Analytics-konton som är tillgängliga med de aktuella autentiseringsuppgifterna.
