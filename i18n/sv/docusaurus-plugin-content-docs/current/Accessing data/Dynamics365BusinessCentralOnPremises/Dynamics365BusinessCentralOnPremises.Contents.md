# Dynamics365BusinessCentralOnPremises.Contents


## Description

Ange URL för din Dynamics 365 Business Central (lokal) OData-tjänstens slutpunkt.


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Returnerar en tabell med relevanta Dynamics 365 Business Central (lokal)-data.
