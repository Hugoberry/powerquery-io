# Number.Factorial


Returnerar fakulteten av talet.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

Returnerar fakulteten av talet <code>number</code>.


## Examples

### Example #1 
Hitta fakulteten av 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
