# Number.Log10


Returnerar 10-logaritmen för talet.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Remarks

Returnerar 10-logaritmen för ett tal, <code>number</code>. Om <code>number</code> är null så returnerar <code>Number.Log10</code> null.


## Examples

### Example #1 
Hämta 10-logaritmen av 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
