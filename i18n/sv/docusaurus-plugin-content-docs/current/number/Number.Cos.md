# Number.Cos


Returnerar cosinus för talet.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Returnerar cosinus för <code>number</code>.


## Examples

### Example #1 
Hitta cosinus för vinkeln 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
