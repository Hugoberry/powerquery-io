# Number.IsEven


Anger om värdet är jämnt.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Remarks

Anger om värdet, <code>number</code>, är jämnt genom att returnera <code>true</code> om det är jämnt och annars <code>false</code>.


## Examples

### Example #1 
Kontrollera om 625 är ett jämnt tal.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Kontrollera om 82 är ett jämnt tal.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
