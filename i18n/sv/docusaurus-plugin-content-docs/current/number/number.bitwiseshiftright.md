# Number.BitwiseShiftRight


Skiftar bitarna till höger.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Returnerar resultatet från ett bitvis skifte till höger för <code>number1</code> med angivet antal bitar, <code>number2</code>.



## Category
Number.Bytes
