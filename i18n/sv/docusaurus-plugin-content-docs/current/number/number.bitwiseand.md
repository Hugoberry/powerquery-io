# Number.BitwiseAnd


Returnerar resultatet av en bitvis &#34;And&#34;-åtgärd mellan de två indatavärdena.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Returnerar resultatet av en bitvis "And"-åtgärd mellan <code>number1</code> och <code>number2</code>.



## Category
Number.Bytes
