# Number.BitwiseXor


Returnerar resultatet av en bitvis &#34;XOR&#34;-åtgärd (Exclusive-OR) mellan de två indatavärdena.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Returnerar resultatet av en bitvis "XOR"-åtgärd (Exclusive-OR) mellan <code>number1</code> och <code>number2</code>.



## Category
Number.Bytes
