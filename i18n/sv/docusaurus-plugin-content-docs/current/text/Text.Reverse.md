# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

Kastar om det angivna <code>text</code>.


## Examples

### Example #1 
Skriv texten ”123” omvänt.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
