# Text.FromBinary


Avkodar data från ett binärformat till textformat.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

Avkodar data, <code>binary</code>, från ett binärvärde till ett textvärde med typen <code>encoding</code>.
