# Type.TableColumn


## Description

Returnerar typen för en kolumn i en tabell.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Returnerar typen för kolumnen <code>column</code> i tabelltypen <code>tableType</code>.



## Category
Type
