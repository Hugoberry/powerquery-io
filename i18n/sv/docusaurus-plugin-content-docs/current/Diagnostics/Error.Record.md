# Error.Record


## Description

Returnerar en felpost från angivna textvärden för orsak, meddelande och detaljinformation.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details

Returnerar en felpost från angivna textvärden för orsak, meddelande och detaljinformation.
