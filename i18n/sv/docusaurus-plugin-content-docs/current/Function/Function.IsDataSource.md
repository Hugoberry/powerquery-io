# Function.IsDataSource


## Description

Returnerar om en viss funktion anses vara en datakälla eller inte.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Returnerar om en <code>function</code> anses vara en datakälla eller inte.
