# DateTimeZone.FixedUtcNow


## Description

UTC(GMT 표준 시간대)의 현재 날짜 및 시간을 반환합니다. 이 값은 고정되므로 이후 호출에서도 변경되지 않습니다.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```


## Details
