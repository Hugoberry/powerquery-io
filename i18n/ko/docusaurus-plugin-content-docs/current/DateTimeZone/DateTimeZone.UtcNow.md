# DateTimeZone.UtcNow


## Description

UTC(GMT 표준 시간대)의 현재 날짜 및 시간을 반환합니다.


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

UTC(GMT 표준 시간대)의 현재 날짜 및 시간을 반환합니다.


## Examples

### Example #1 
UTC의 현재 날짜 및 시간을 가져옵니다.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
