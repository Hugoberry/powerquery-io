# Value.Alternates


## Description

대체 쿼리 계획을 나타냅니다.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Value.Expression(Value.Optimize(...))를 통해 얻은 쿼리 계획 식 내에서 대체 쿼리 계획을 나타냅니다. 다른 용도로는 사용할 수 없습니다.



## Category
Expression
