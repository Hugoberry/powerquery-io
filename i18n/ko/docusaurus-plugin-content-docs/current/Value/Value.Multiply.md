# Value.Multiply


## Description

두 값의 곱을 반환합니다.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

<code>value1</code>을(를) <code>value2</code>(으)로 곱한 값을 반환합니다. 선택적으로 <code>precision</code> 매개 변수를 지정할 수 있습니다. 기본적으로 <code>Precision.Double</code>이 사용됩니다.



## Category
Values.Arithmetic operations
