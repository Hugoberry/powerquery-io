# Cube.AttributeMemberProperty


## Description

차원 특성의 속성을 반환합니다.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

차원 특성 <code>attribute</code>의 속성 <code>propertyName</code>을(를) 반환합니다.



## Category
Cube
