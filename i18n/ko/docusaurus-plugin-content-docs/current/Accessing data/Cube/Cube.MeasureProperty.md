# Cube.MeasureProperty


## Description

측정값(셀 속성)의 속성을 반환합니다.


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

측정값 <code>measure</code>의 속성 <code>propertyName</code>을(를) 반환합니다.



## Category
Cube
