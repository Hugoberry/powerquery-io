# Cube.Dimensions


## Description

사용 가능한 차원 집합이 포함된 테이블을 반환합니다.


## Syntax

```powerquery
Cube.Dimensions(
    cube as table
) as table
```


## Details

<code>cube</code> 내에서 사용 가능한 차원 집합이 포함된 테이블을 반환합니다. 각 차원은 차원 특성 집합을 포함하는 테이블이고 각 차원 특성은 차원 테이블에서 열로 표시됩니다. 차원은 Cube.AddAndExpandDimensionColumn을 사용해서 큐브에서 확장할 수 있습니다.
