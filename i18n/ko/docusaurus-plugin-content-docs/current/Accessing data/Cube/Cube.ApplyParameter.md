# Cube.ApplyParameter


## Description

큐브에 매개 변수를 적용한 후 큐브를 반환합니다.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

<code>parameter</code>과(와) <code>arguments</code>을(를) <code>cube</code>에 적용한 후 큐브를 반환합니다.



## Category
Cube
