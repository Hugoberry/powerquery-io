# Cube.ReplaceDimensions


## Description

Cube.Dimensions에서 반환된 차원 집합을 바꿉니다.


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Details

<code>Cube.Dimensions</code>에서 반환된 차원 집합을 대체합니다.     예를 들어, 이 함수를 사용하여 차원 속성에 ID 열을 추가하여 데이터 원본을 그룹화할 수 있습니다. 표시된 값이 아닌 ID입니다.
