# Hdfs.Files


## Description

Hadoop 파일 시스템의 지정된 폴더 및 하위 폴더에 있는 파일의 속성과 내용을 포함하는 테이블을 반환합니다.


## Syntax

```powerquery
Hdfs.Files(
    url as text
) as table
```


## Details

Hadoop 파일 시스템의 폴더 URL <code>url</code> 및 하위 폴더에 있는 각 파일에 대한 행을 포함하는 테이블을 반환합니다. 각 행에는 파일의 속성과 해당 내용에 대한 링크가 포함됩니다.
