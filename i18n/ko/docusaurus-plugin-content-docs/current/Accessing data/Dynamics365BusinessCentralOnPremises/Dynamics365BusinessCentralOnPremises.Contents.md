# Dynamics365BusinessCentralOnPremises.Contents


## Description

Dynamics 365 Business Central(온-프레미스) OData Service 엔드포인트의 URL을 입력합니다.


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Details

관련 Dynamics 365 Business Central(온-프레미스) 데이터가 포함된 테이블을 반환합니다.
