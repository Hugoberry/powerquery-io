# ActiveDirectory.Domains


## Description

지정된 도메인과 동일한 포리스트에 있는 Active Directory 도메인의 목록을 반환하거나, 도메인이 지정되지 않은 경우 현재 컴퓨터의 도메인 목록을 반환합니다.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Details

지정된 도메인과 동일한 포리스트에 있는 Active Directory 도메인의 목록을 반환하거나, 도메인이 지정되지 않은 경우 현재 컴퓨터의 도메인 목록을 반환합니다.
