# Excel.CurrentWorkbook


## Description

현재 Excel 통합 문서의 내용을 반환합니다.


## Syntax

```powerquery
Excel.CurrentWorkbook(

) as table
```


## Details

현재 Excel 통합 문서의 내용을 반환합니다. 테이블, 명명된 범위 및 동적 배열을 반환합니다. <code>Excel.Workbook</code>과 달리 시트를 반환하지 않습니다.
