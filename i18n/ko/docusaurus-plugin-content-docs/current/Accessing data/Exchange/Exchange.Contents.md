# Exchange.Contents


## Description

Microsoft Exchange 계정에서 목차를 반환합니다.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Details

Microsoft Exchange 계정 <code>mailboxAddress</code>에서 콘텐츠 테이블을 반환합니다. <code>mailboxAddress</code>이(가) 지정되지 않은 경우 자격 증명의 기본 계정이 사용됩니다.
