# Type.TableSchema


## Description

지정한 테이블 유형의 열 설명(예: 스키마)을 포함하는 테이블을 반환합니다.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

<code>tableType</code> 열을 설명하는 테이블을 반환합니다.<br />결과 테이블의 설명은 <code>Table.Schema</code>에 대한 문서를 참조하세요.<br />
