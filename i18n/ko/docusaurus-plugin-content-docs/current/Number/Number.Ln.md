# Number.Ln


## Description

숫자의 자연 로그를 반환합니다.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

숫자 <code>number</code>의 자연 로그를 반환합니다. <code>number</code>이(가) null인 경우 <code>Number.Ln</code>은 null을 반환합니다.


## Examples

### Example #1 
15의 자연 로그를 가져옵니다.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
