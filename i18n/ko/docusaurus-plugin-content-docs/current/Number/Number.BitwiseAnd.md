# Number.BitwiseAnd


## Description

두 입력 간의 비트 &#34;And&#34; 연산을 수행한 결과를 반환합니다.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

<code>number1</code>과(와) <code>number2</code> 간의 비트 "And" 연산을 수행한 결과를 반환합니다.



## Category
Number.Bytes
