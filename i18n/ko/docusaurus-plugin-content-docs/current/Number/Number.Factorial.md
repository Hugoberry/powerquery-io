# Number.Factorial


## Description

숫자의 계승값을 반환합니다.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

숫자 <code>number</code>의 계승값을 반환합니다.


## Examples

### Example #1 
10의 계승값을 구합니다.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
