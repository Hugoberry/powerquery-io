# Number.Cos


## Description

숫자의 코사인을 반환합니다.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

<code>number</code>의 코사인을 반환합니다.


## Examples

### Example #1 
각도 0의 코사인을 구합니다.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
