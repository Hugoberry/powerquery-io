# Number.Log10


## Description

숫자의 상용 로그를 반환합니다.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

숫자 <code>number</code>의 상용 로그를 반환합니다. <code>number</code>이(가) Null인 경우 <code>Number.Log10</code>은 Null을 반환합니다.


## Examples

### Example #1 
2의 상용 로그를 가져옵니다.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
