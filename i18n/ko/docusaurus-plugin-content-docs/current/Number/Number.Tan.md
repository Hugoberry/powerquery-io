# Number.Tan


## Description

숫자의 탄젠트를 반환합니다.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

<code>number</code>의 탄젠트를 반환합니다.


## Examples

### Example #1 
각도 1의 탄젠트를 구합니다.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
