# Number.Sin


## Description

숫자의 사인을 반환합니다.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

<code>number</code>의 사인을 반환합니다.


## Examples

### Example #1 
각도 0의 사인을 구합니다.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
