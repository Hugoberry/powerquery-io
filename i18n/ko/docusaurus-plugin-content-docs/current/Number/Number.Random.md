# Number.Random


## Description

난수를 반환합니다.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

0과 1 사이의 난수를 반환합니다.


## Examples

### Example #1 
난수를 가져옵니다.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
