# BinaryFormat.7BitEncodedUnsignedInteger


## Description

7비트 가변 길이 인코딩을 사용하여 인코딩된 부호 없는 64비트 정수를 읽는 이진 형식입니다.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details

7비트 가변 길이 인코딩을 사용하여 인코딩된 부호 없는 64비트 정수를 읽는 이진 형식입니다.
