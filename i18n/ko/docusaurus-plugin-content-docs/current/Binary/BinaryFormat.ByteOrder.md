# BinaryFormat.ByteOrder


## Description

함수에 지정된 바이트 순서로 이진 형식을 반환합니다.


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

<code>binaryFormat</code>에 지정된 바이트 순서로 이진 형식을 반환합니다. 기본 바이트 순서는 <code>ByteOrder.BigEndian</code>입니다.
