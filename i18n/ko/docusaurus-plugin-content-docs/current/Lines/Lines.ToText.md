# Lines.ToText


## Description

텍스트 목록을 단일 텍스트로 변환합니다. 지정된 lineSeparator는 각 줄의 끝에 추가됩니다. lineSeparator가 지정되지 않은 경우에는 캐리지 리턴 및 줄 바꿈 문자가 사용됩니다.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```


## Details

텍스트 목록을 단일 텍스트로 변환합니다. 지정된 lineSeparator는 각 줄의 끝에 추가됩니다. lineSeparator가 지정되지 않은 경우에는 캐리지 리턴 및 줄 바꿈 문자가 사용됩니다.
