# Error.Record


## Description

지정된 이유, 메시지 및 세부 정보에 대한 텍스트 값에서 오류 레코드를 반환합니다.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
