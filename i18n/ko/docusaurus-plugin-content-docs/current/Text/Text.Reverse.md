# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

제공한 <code>text</code>을(를) 반전시킵니다.


## Examples

### Example #1 
텍스트 &#34;123&#34;을 반전시킵니다.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
