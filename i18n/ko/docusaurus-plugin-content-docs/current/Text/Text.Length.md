# Text.Length


## Description

문자 수를 반환합니다.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

<code>text</code> 텍스트의 문자 수를 반환합니다.


## Examples

### Example #1 
&#34;Hello World&#34; 텍스트의 문자 수를 찾습니다.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
