# Table.PartitionValues


## Description

테이블이 분할되는 방식에 대한 정보를 반환합니다.


## Syntax

```powerquery
Table.PartitionValues(
    table as table
) as table
```


## Details

테이블이 분할되는 방식에 대한 정보를 반환합니다. 각 열이 원래 테이블의 파티션 열이고 각 행이 원래 테이블의 파티션에 해당하는 테이블이 반환됩니다.
