# Table.ViewError


## Description

Table.View를 통해 뷰에 정의된 핸들러에서 throw될 때 대체를 트리거하지 않는 수정된 오류 레코드를 만듭니다.


## Syntax

```powerquery
Table.ViewError(
    errorRecord as record
) as record
```


## Details

Table.View를 통해 뷰에 정의된 핸들러에서 throw될 때 대체를 트리거하지 않는 <code>errorRecord</code>의 수정된 오류 레코드를 만듭니다.
