# Error.Record


## Description

Retorna un registre d&#39;error a partir dels valors de text proporcionat per a raó, missatge i detall.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
