# Text.Length


## Description

Retorna el nombre de caràcters.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Retorna el nombre de caràcters del text <code>text</code>.


## Examples

### Example #1 
Troba quants caràcters hi ha al text &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
