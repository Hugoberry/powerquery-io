# Type.ListItem


## Description

Retorna un tipus d&#39;element d&#39;un tipus de llista.


## Syntax

```powerquery
Type.ListItem(
    type as type
) as type
```


## Details

Retorna un tipus d'element d'una llista <code>type</code>.


## Examples

### Example #1 
Cerqueu el tipus d&#39;element de la llista &lt;code&gt;\{number}&lt;/code&gt;.
```powerquery
Type.ListItem(type {number})
```

Result: 
```powerquery
type number
```
