# Type.TableColumn


## Description

Retorna el tipus d&#39;una columna d&#39;una taula.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Retorna el tipus de la columna <code>column</code> al tipus de taula <code>tableType</code>.

Type
