# DateTimeZone.FixedUtcNow


## Description

Retorna la data i hora actuals en UTC (el fus horari GMT). Aquest valor és fix i no canviarà amb crides successives.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
