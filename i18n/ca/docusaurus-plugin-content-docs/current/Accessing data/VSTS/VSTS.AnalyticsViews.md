# VSTS.AnalyticsViews


## Description

Introduïu el nom de l&#39;organització i el del projecte.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Retorna una taula de visualitzacions de l'Analytics que ofereix l'Azure DevOps.
