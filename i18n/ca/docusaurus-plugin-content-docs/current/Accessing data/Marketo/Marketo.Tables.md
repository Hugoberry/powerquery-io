# Marketo.Tables


## Description

Introdueix l&#39;URL de l&#39;extrem de l&#39;API de REST del Marketo associat al compte.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details
