# Cube.AttributeMemberProperty


## Description

Retorna una propietat d&#39;un atribut de dimensió.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Retorna la propietat <code>propertyName</code> de l'atribut de dimensió <code>attribute</code>.

Cube
