# Cube.ApplyParameter


## Description

Retorna un cub després d&#39;aplicar-li un paràmetre.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details
