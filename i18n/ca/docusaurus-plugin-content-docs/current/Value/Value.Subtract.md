# Value.Subtract


## Description

Retorna la diferència dels dos valors.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Retorna la diferència de <code>value1</code> i <code>value2</code>. Es pot especificar un paràmetre opcional <code>precision</code>, s'utilitza <code>Precision.Double</code> per defecte.
