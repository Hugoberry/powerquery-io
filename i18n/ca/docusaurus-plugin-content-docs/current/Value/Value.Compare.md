# Value.Compare


## Description

Retorna -1, 0 o 1 en funció de si el primer valor és més petit, igual o més gran que el segon.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```


## Details

Retorna -1, 0 o 1 en funció de si el primer valor és més petit, igual o més gran que el segon.
