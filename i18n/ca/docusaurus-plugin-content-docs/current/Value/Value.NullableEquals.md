# Value.NullableEquals


## Description

Retorna si dos valors són iguals.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Retorna el valor nul si qualsevol dels arguments "<code>value1</code>", "<code>value2</code>" és nul, altrament, l'equivalent a Value.Equals.



## Category
Values
