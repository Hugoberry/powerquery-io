# Value.Equals


## Description

Retorna si dos valors són iguals.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Retorna el valor <code>value1</code> és igual que el valor <code>value2</code> i el valor false en cas contrari.



## Category
Values
