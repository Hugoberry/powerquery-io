# Number.Sin


## Description

Retorna el sinus del nombre.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Retorna el sinus de <code>number</code>.


## Examples

### Example #1 
Troba el sinus de l&#39;angle 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
