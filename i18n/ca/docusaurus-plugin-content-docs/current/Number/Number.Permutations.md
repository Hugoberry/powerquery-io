# Number.Permutations


## Description

Retorna el nombre de permutacions.


## Syntax

```powerquery
Number.Permutations(
    setSize as number,
    permutationSize as number
) as number
```


## Details

Retorna el nombre de permutacions que es poden generar a partir d'un nombre d'elements, <code>setSize</code>, amb una mida de permutaci√≥ especificada, <code>permutationSize</code>.


## Examples

### Example #1 
Troba el nombre de permutacions d&#39;un total de 5 elements en grups de 3.
```powerquery
Number.Permutations(5, 3)
```

Result: 
```powerquery
60
```
