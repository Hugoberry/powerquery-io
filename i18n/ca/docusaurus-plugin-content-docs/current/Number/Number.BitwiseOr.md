# Number.BitwiseOr


## Description

Retorna el resultat de fer un bit a bit &#34;Or&#34; entre les dues entrades.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Retorna el resultat de fer un bit a bit "Or" entre <code>number1</code> i <code>number2</code>.



## Category
Number.Bytes
