# Number.RandomBetween


## Description

Retorna un nombre aleatori entre dos nombres.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Retorna un nombre aleatori entre <code>bottom</code> i <code>top</code>.
