# Number.Atan2


## Description

Retorna l&#39;arctangent de la divisió dels dos nombres.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Retorna l'arctangent de la divisió dels dos nombres, <code>y</code> i <code>x</code>. La divisió es construirà com a <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
