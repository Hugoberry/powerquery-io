# Number.BitwiseXor


## Description

Retorna el resultat de fer un bit a bit &#34;XOR&#34; (OR exclusiu) entre les dues entrades.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Retorna el resultat de fer un bit a bit "XOR" (OR exclusiu) entre <code>number1</code> i <code>number2</code>.



## Category
Number.Bytes
