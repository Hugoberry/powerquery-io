# Number.Factorial


## Description

Retorna el factorial del nombre.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Retorna el factorial del nombre <code>number</code>.


## Examples

### Example #1 
Troba el factorial de 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
