# Number.Tan


## Description

Retorna la tangent del nombre.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Retorna la tangent de <code>number</code>.


## Examples

### Example #1 
Troba la tangent de l&#39;angle 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
