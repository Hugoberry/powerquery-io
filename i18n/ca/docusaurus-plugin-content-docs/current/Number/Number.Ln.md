# Number.Ln


## Description

Retorna el logaritme natural del nombre.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Retorna el logaritme natural d'un nombre, <code>number</code>. Si <code>number</code> és nul, <code>Number.Ln</code> retorna el valor nul.


## Examples

### Example #1 
Obté el logaritme natural de 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
