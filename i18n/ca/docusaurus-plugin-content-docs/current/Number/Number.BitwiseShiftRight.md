# Number.BitwiseShiftRight


## Description

Desplaça el conjunt de bits a la dreta.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Retorna el resultat de fer un desplaçament bit a bit a la dreta a <code>number1</code>, en el nombre especificat de bits <code>number2</code>.



## Category
Number.Bytes
