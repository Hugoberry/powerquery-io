# WorkplaceAnalytics.Data



## Syntax

```powerquery
WorkplaceAnalytics.Data(
    scopeId as text,
    optional jobName as text,
    optional jobId as text,
    optional schemaType as text
) as table
```


## Details

Importa mètriques setmanals i dades d'atribut del Workplace Analytics.


## Examples

### Example #1 
Retorna una taula amb atributs visibles i mètriques per setmana per a cada empleat mesurat a la partició.
```powerquery
VivaInsights.Data("5f142ea5-c08a-eb02-de68-41dccdefb141")
```

Result: 
```powerquery
#table({"Query1"}, {{"PersonId""}, {"PID"}})
```
