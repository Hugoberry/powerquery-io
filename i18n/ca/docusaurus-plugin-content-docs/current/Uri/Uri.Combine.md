# Uri.Combine


## Description

Retorna un URI absolut que és la combinació de l&#39;URI base d&#39;entrada i l&#39;URI relatiu.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Retorna un URI absolut que és la combinació de l'<code>baseUri</code> d'entrada i <code>relativeUri</code>.
