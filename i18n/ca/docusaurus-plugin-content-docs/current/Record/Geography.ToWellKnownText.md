# Geography.ToWellKnownText


## Description

Tradueix un valor de punt geogràfic estructurat en la seva representació en format Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Tradueix un valor de punt geogràfic estructurat en la seva representació en format Well-Known Text (WKT) tal com el defineix l'Open Geospatial Consortium (OGC), així com el format de serialització que utilitzen moltes bases de dades, com ara l'SQL Server.



## Category
Record.Serialization
