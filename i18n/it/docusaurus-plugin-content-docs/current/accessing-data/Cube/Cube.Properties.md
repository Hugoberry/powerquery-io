# Cube.Properties


Restituisce una tabella che contiene il set di proprietà disponibili per le dimensioni espanse nel cubo.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```


## Remarks

Restituisce una tabella che contiene il set di proprietà disponibili per le dimensioni espanse nel cubo.
