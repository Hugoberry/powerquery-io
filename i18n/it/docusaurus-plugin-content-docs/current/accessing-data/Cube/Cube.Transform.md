# Cube.Transform


Applica un elenco di funzioni del cubo.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Remarks

Applica le funzioni del cubo dell'elenco <code>transforms</code> in <code>cube</code>.

Cube
