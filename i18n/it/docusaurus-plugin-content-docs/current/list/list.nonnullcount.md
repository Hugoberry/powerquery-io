# List.NonNullCount


Restituisce il numero di elementi non Null dell&#39;elenco.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Restituisce il numero di elementi non Null dell'elenco <code>list</code>.
