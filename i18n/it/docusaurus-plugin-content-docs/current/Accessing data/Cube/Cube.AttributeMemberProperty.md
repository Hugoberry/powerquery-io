# Cube.AttributeMemberProperty


## Description

Restituisce una proprietà di un attributo dimensione.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Restituisce la proprietà <code>propertyName</code> dell'attributo dimensione <code>attribute</code>.



## Category
Cube
