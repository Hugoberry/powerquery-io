# Cube.MeasureProperties


## Description

Restituisce una tabella che contiene il set di proprietà disponibili per le misure espanse nel cubo.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Restituisce una tabella che contiene il set di proprietà disponibili per le misure espanse nel cubo.
