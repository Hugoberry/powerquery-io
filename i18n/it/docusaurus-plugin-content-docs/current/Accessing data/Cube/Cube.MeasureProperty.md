# Cube.MeasureProperty


## Description

Restituisce una proprietà di una misura (proprietà della cella).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Restituisce la proprietà <code>propertyName</code> della misura <code>measure</code>.



## Category
Cube
