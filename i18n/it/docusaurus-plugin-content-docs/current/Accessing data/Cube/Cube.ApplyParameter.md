# Cube.ApplyParameter


## Description

Restituisce un cubo dopo avervi applicato un parametro.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Restituisce un cubo dopo aver applicato <code>parameter</code> con <code>arguments</code> a <code>cube</code>.



## Category
Cube
