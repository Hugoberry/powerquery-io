# Xml.Document


Restituisce il contenuto del documento XML come tabella gerarchica.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Remarks

Restituisce il contenuto del documento XML come tabella gerarchica.
