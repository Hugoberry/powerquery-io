# DynamicsNav.Contents


## Description

Immettere l&#39;URL dell&#39;endpoint di servizio OData di Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Restituisce una tabella con i dati pertinenti di Dynamics NAV.
