# AzureDevOpsServer.Views


## Description

Consente di immettere i nomi di progetto e organizzazione.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Restituisce una tabella di feed OData offerti da Azure DevOps.
