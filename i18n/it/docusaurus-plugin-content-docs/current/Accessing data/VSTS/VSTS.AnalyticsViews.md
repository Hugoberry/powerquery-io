# VSTS.AnalyticsViews


Consente di immettere i nomi di progetto e organizzazione.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Restituisce una tabella di visualizzazioni del Servizio analisi offerta da Azure DevOps.
