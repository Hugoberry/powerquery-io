# Value.Alternates


Esprime piani di query alternativi.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Remarks

Esprime piani di query alternativi all'interno di un'espressione del piano di query ottenuta tramite Value.Expression(Value.Optimize(...)). Non destinata ad altri usi.

Expression
