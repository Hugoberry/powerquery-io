# Splitter.SplitByNothing


Restituisce una funzione che non effettua la suddivisione e restituisce il relativo argomento come elenco di un singolo elemento.


## Syntax

```powerquery
Splitter.SplitByNothing(

) as function
```


## Remarks

Restituisce una funzione che non effettua la suddivisione e restituisce il relativo argomento come elenco di un singolo elemento.
