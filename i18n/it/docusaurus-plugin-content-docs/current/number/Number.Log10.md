# Number.Log10


Restituisce il logaritmo in base 10 del numero.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Remarks

Restituisce il logaritmo in base 10 di un numero, <code>number</code>. Se <code>number</code> Ã¨ Null, <code>Number.Log10</code> restituisce Null.


## Examples

### Example #1 
Ottenere il logaritmo in base 10 di 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
