# Number.Abs


Restituisce il valore assoluto del numero.


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Remarks

Restituisce il valore assoluto di <code>number</code>. Se <code>number</code> Ã¨ Null, <code>Number.Abs</code> restituisce Null.    <ul>        <li><code>number</code>: <code>number</code> per cui deve essere calcolato il valore assoluto.</li>      </ul>


## Examples

### Example #1 
Valore assoluto di -3.
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
