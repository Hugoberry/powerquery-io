# Number.Random


Restituisce un numero casuale.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Restituisce un numero casuale compreso tra 0 e 1.


## Examples

### Example #1 
Ottenere un numero casuale.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
