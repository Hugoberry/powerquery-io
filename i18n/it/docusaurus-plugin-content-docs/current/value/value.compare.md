# Value.Compare


Restituisce -1, 0 o 1 a seconda che il primo valore sia minore, uguale o maggiore del secondo.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```


## Remarks

Restituisce -1, 0 o 1 a seconda che il primo valore sia minore, uguale o maggiore del secondo.
