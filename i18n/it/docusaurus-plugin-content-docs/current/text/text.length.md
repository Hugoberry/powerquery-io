# Text.Length


Restituisce il numero di caratteri.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

Restituisce il numero di caratteri nel testo <code>text</code>.


## Examples

### Example #1 
Trovare il numero di caratteri presenti nel testo &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
