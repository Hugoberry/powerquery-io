# Type.TableColumn


## Description

Restituisce il tipo di colonna in una tabella.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Restituisce il tipo della colonna <code>column</code> nel tipo di tabella <code>tableType</code>.



## Category
Type
