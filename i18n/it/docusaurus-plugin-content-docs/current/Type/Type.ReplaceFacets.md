# Type.ReplaceFacets


Sostituisce i facet di un tipo.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Remarks

Sostituisce i facet di <code>type</code> con i facet contenuti nel record <code>facets</code>.
