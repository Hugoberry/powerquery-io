# BinaryFormat.7BitEncodedSignedInteger


## Description

Formato binario che legge un intero con segno a 64 bit che è stato codificato tramite la codifica di lunghezza variabile a 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
