# List.Count


Restituisce il numero di elementi dell&#39;elenco.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Restituisce il numero di elementi dell'elenco <code>list</code>.


## Examples

### Example #1 
Trovare il numero di valori dell&#39;elenco \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
