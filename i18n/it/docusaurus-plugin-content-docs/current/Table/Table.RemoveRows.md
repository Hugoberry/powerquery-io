# Table.RemoveRows


## Description

Rimuove il numero di righe specificato.


## Syntax

```powerquery
Table.RemoveRows(
    table as table,
    offset as number,
    optional count as number
) as table
```


## Details

Rimuove <code>count</code> di righe dall'inizio di <code>table</code> a partire da <code>offset</code>. Se non viene specificato il parametro <code>count</code>, verr√† utilizzato il conteggio predefinito di 1. 


## Examples

### Example #1 
Rimuovere la prima riga dalla tabella.
```powerquery
Table.RemoveRows(
    Table.FromRecords({
        [CustomerID = 1, Name = "Bob", Phone = "123-4567"],
        [CustomerID = 2, Name = "Jim", Phone = "987-6543"],
        [CustomerID = 3, Name = "Paul", Phone = "543-7890"],
        [CustomerID = 4, Name = "Ringo", Phone = "232-1550"]
    }),
    0
)
```

Result: 
```powerquery
Table.FromRecords({
    [CustomerID = 2, Name = "Jim", Phone = "987-6543"],
    [CustomerID = 3, Name = "Paul", Phone = "543-7890"],
    [CustomerID = 4, Name = "Ringo", Phone = "232-1550"]
})
```


### Example #2 
Rimuovere la riga nella posizione 1 dalla tabella.
```powerquery

        [CustomerID = 4, Name = "Ringo", Phone = "232-1550"]
    }),
    1

```

Result: 
```powerquery
Table.FromRecords({

    [CustomerID = 3, Name = "Paul", Phone = "543-7890"],
    [CustomerID = 4, Name = "Ringo", Phone = "232-1550"]
})
```


### Example #3 
Rimuovere due righe a partire dalla posizione 1 dalla tabella.
```powerquery
Table.RemoveRows(
    Table.FromRecords({
```

Result:
