# Number.Cos


## Description

Restituisce il coseno del numero.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Restituisce il coseno di <code>number</code>.


## Examples

### Example #1 
Trovare il coseno dell&#39;angolo 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
