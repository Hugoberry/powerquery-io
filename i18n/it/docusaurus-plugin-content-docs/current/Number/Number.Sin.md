# Number.Sin


## Description

Restituisce il seno del numero.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Restituisce il seno di <code>number</code>.


## Examples

### Example #1 
Trovare il seno dell&#39;angolo 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
