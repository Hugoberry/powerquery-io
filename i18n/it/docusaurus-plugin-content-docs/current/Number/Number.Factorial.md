# Number.Factorial


## Description

Restituisce il fattoriale del numero.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Restituisce il fattoriale del numero <code>number</code>.


## Examples

### Example #1 
Trovare il fattoriale di 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
