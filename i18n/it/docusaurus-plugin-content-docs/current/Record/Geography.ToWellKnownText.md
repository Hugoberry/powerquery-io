# Geography.ToWellKnownText


## Description

Converte un valore di punto geografico strutturato nella relativa rappresentazione WKT (well-known text).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```
