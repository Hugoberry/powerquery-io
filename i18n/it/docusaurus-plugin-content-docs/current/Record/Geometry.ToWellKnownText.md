# Geometry.ToWellKnownText


## Description

Converte un valore di punto geometrico strutturato nella relativa rappresentazione WKT (well-known text).


## Syntax

```powerquery
Geometry.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```
