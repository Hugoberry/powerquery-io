# Error.Record


## Description

Restituisce un record di errore dai valori di testo specificati per motivo, messaggio e dettagli.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
