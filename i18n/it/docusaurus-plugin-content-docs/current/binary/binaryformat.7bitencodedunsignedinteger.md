# BinaryFormat.7BitEncodedUnsignedInteger


Formato binario che legge un intero senza segno a 64 bit che è stato codificato tramite la codifica di lunghezza variabile a 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks
