# List.Count


## Description

Vrne število elementov na seznamu.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Vrne število elementov na seznamu <code>list</code>.


## Examples

### Example #1 
Poiščite število vrednosti na seznamu \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
