# Uri.Combine


## Description

Vrne absolutni URI, ki je kombinacija vhodnega osnovnega URI-ja in relativnega URI-ja.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Vrne absolutni URI, ki je kombinacija vnosa <code>baseUri</code> in <code>relativeUri</code>.
