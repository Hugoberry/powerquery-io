# Value.Optimize


## Description

Pošlje signal funkciji Value.Expression, naj vrne optimiziran izraz za vrednost.


## Syntax

```powerquery
Value.Optimize(
    value as any
) as any
```


## Details

Če je uporabljen v funkciji Value.Expression in če <code>value</code> predstavlja poizvedbo, ki jo je mogoče optimizirati, ta funkcija označuje, da je treba vrniti optimiziran izraz. V nasprotnem primeru bo <code>value</code> naveden brez vpliva.



## Category
Expression
