# Value.Divide


## Description

Vrne rezultat deljenja prve vrednosti z drugo vrednostjo.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Vrne rezultat deljenja vrednosti <code>value1</code> z vrednostjo <code>value2</code>. Doloƒçite lahko izbirni parameter <code>precision</code>; privzeto je uporabljen <code>Precision.Double</code>.
