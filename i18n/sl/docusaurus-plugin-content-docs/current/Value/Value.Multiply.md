# Value.Multiply


## Description

Vrne zmnožek dveh vrednosti.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Vrne zmnožek vrednosti <code>value1</code> in <code>value2</code>. Določite lahko izbirni parameter <code>precision</code>; privzeto je uporabljen <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
