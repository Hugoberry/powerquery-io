# Value.NullableEquals


## Description

Vrne podatek, ali sta dve vrednosti enaki.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Vrne "null", če je argument "<code>value1</code>" ali "<code>value2</code>" ničeln, sicer je enako "Value.Equals".



## Category
Values
