# Value.Alternates


## Description

Prikaže nadomestne načrte poizvedb.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Prikaže nadomestne načrte poizvedb v izrazu načrta poizvedbe, pridobljenem z ukazom Value.Expression(Value.Optimize(...)). Ni namenjeno za druge načine uporabe.



## Category
Expression
