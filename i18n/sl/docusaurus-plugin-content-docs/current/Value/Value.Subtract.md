# Value.Subtract


## Description

Vrne razliko dveh vrednosti.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Vrne razliko vrednosti <code>value1</code> in <code>value2</code>. Doloƒçite lahko izbirni parameter <code>precision</code>; privzeto je uporabljen <code>Precision.Double</code>.
