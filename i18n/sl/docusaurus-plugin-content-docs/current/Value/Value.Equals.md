# Value.Equals


## Description

Vrne podatek, ali sta dve vrednosti enaki.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Vrne "true", ƒçe je vrednost <code>value1</code> enaka kot vrednost <code>value2</code>, sicer vrne "false".
