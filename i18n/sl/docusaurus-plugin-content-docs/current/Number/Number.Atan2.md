# Number.Atan2


## Description

Vrne arkus tangens deljenja dveh števil.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Vrne arkus tangens deljenja dveh števil, <code>y</code> in <code>x</code>. Deljenje bo v obliki <code>y</code>/<code>x</code>.

Number.Trigonometry
