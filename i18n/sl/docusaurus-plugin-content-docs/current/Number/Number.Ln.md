# Number.Ln


## Description

Vrne naravni logaritem števila.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Vrne naravni logaritem števila <code>number</code>. Če ima <code>number</code> ničelno vrednost, <code>Number.Ln</code> vrne "null".


## Examples

### Example #1 
Poiščite naravni logaritem števila 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
