# Number.Exp


## Description

Potencira &#34;e&#34; na dani eksponent.


## Syntax

```powerquery
Number.Exp(
    number as number
) as number
```


## Details

Vrne rezultat potenciranja "e" na eksponent <code>number</code> (eksponentna funkcija).      <ul>        <li><code>number</code>: <code>number</code>, za katero bo izračunana eksponentna funkcija. Če ima <code>number</code> ničelno vrednost, <code>Number.Exp</code> vrne "null". </li>      </ul>


## Examples

### Example #1 
Potencirajte &#34;e&#34; na eksponent 3.
```powerquery
Number.Exp(3)
```

Result: 
```powerquery
20.085536923187668
```
