# Number.Random


## Description

Vrne naključno število.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Vrne naključno število med 0 in 1.


## Examples

### Example #1 
Poiščite naključno število.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
