# Number.Tan


## Description

Vrne tangens števila.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Vrne tangens števila <code>number</code>.


## Examples

### Example #1 
Poiščite tangens kota 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
