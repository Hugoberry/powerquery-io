# Number.BitwiseOr


## Description

Vrne rezultat izvajanja bitne operacije &#34;Or&#34; med dvema vnosoma.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Vrne rezultat izvajanja bitne operacije "Or" med <code>number1</code> in <code>number2</code>.

Number.Bytes
