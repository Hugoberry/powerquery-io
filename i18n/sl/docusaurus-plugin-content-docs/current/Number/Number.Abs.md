# Number.Abs


## Description

Vrne absolutno vrednost števila.


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Details

Vrne absolutno vrednost števila <code>number</code>. Če ima <code>number</code> ničelno vrednost, <code>Number.Abs</code> vrne "null".    <ul>        <li><code>number</code>: <code>number</code>, za katero bo izračunana absolutna vrednost.</li>      </ul>


## Examples

### Example #1 
Absolutna vrednost števila –3.
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
