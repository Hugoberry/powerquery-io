# Number.Factorial


## Description

Vrne fakulteto števila.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Vrne fakulteto števila <code>number</code>.


## Examples

### Example #1 
Poiščite fakulteto števila 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
