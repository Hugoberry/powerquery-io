# Number.Cos


## Description

Vrne kosinus števila.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Vrne kosinus števila <code>number</code>.


## Examples

### Example #1 
Poiščite kosinus kota 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
