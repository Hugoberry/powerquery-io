# Number.RandomBetween


## Description

Vrne naključno število med dvema številoma.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Vrne naključno število med <code>bottom</code> in <code>top</code>.


## Examples

### Example #1 
Poiščite naključno število med 1 in 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
