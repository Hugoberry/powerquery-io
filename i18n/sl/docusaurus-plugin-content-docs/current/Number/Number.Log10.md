# Number.Log10


## Description

Vrne logaritem števila z osnovo 10.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Vrne logaritem števila <code>number</code> z osnovo 10. Če ima <code>number</code> ničelno vrednost, <code>Number.Log10</code> vrne "null".


## Examples

### Example #1 
Poiščite logaritem števila 2 z osnovo 10.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
