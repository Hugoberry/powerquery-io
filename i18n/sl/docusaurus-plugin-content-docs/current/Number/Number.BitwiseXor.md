# Number.BitwiseXor


## Description

Vrne rezultat izvajanja bitne operacije &#34;XOR&#34; (Exclusive-OR) med dvema vnosoma.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Vrne rezultat izvajanja bitne operacije "XOR" (Exclusive-OR) med <code>number1</code> in <code>number2</code>.



## Category
Number.Bytes
