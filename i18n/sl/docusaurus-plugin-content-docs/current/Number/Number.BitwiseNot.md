# Number.BitwiseNot


## Description

Vrne bajt, kjer je vsak bit nasprotje vnosa.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Vrne rezultat izvajanja bitne operacije "Not" za <code>number</code>.



## Category
Number.Bytes
