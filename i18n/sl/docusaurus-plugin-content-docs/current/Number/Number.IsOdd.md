# Number.IsOdd


## Description

Označuje, ali je vrednost liho število.


## Syntax

```powerquery
Number.IsOdd(
    number as number
) as logical
```


## Details

Označuje, ali je vrednost liho število. Vrne <code>true</code>, če je <code>number</code> liho število, sicer vrne <code>false</code>.


## Examples

### Example #1 
Preverite, ali je 625 liho število.
```powerquery
Number.IsOdd(625)
```

Result: 
```powerquery
true
```


### Example #2 
Preverite, ali je 82 liho število.
```powerquery
Number.IsOdd(82)
```

Result: 
```powerquery
false
```
