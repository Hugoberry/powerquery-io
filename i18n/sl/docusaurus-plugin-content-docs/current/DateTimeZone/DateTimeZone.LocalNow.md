# DateTimeZone.LocalNow


## Description

Vrne trenutni datum in uro v lokalnem časovnem pasu.


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

Vrne vrednost <code>datetimezone</code>, nastavljeno na trenutni datum in uro v sistemu.    Vrnjena vrednost vsebuje podatke o časovnem pasu, ki predstavljajo lokalni časovni pas.
