# Geography.ToWellKnownText


## Description

Prevede vrednost strukturirane geografske točke v svojo upodobitev Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Prevede vrednost strukturirane geografske točke v svojo upodobitev Well-Known Text (WKT), ki ga opredeljuje Open Geospatial Consortium (OGC), vključno z obliko serializacije, ki jo uporablja veliko zbirk podatkov, vključno s strežnikom SQL Server.
