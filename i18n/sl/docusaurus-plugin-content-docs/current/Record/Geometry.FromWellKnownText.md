# Geometry.FromWellKnownText


## Description

Prevede besedilo, ki predstavlja geometrijsko vrednost v obliki zapisa Well-Known Text (WKT), v strukturiran zapis.


## Syntax

```powerquery
Geometry.FromWellKnownText(
    input as text
) as record
```


## Details

Prevede besedilo, ki predstavlja geometrijsko vrednost v obliki zapisa Well-Known Text (WKT), v strukturiran zapis. WKT je standardna oblika zapisa, ki jo opredeljuje Open Geospatial Consortium (OGC) in je običajna oblika zapisa serializacije, ki jo uporabljajo zbirke podatkov, vključno s strežnikom SQL Server.
