# Lakehouse.Contents


## Description

Uvozite podatke iz skladišča Lakehouse


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Details

Ustvarite povezavo do vseh skladišč Lakehouse, do katerih lahko dostopate, in izberite entitete, ki se jih želite ogledati.
