# Folder.Contents


## Description

Vrne tabelo, ki vsebuje lastnosti in vsebino datotek in map, najdenih v doloƒçeni mapi.


## Syntax

```powerquery
Folder.Contents(
    path as text,
    optional options as record
) as table
```


## Details

Vrne tabelo, ki vsebuje vrstico za vsako mapo in datoteko, najdeno v poti mape, <code>path</code>. Vsaka vrstica vsebuje lastnosti mape ali datoteke in povezavo do vsebine. Parameter <code>options</code> je trenutno namenjen samo notranji uporabi.
