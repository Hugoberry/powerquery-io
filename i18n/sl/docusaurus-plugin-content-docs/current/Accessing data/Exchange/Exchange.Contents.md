# Exchange.Contents


## Description

Vrne kazalo vsebine iz računa Microsoft Exchange.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Details

Vrne kazalo vsebine iz Microsoft Exchangeevega računa <code>mailboxAddress</code>. Če parametra <code>mailboxAddress</code> ne navedete, bo uporabljen privzeti račun za poverilnice.
