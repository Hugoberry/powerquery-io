# HdInsight.Contents


## Description

Vrne tabelo za krmarjenje, ki vsebuje vsebnike, najdene v določenem računu iz trezorja shrambe Azure.


## Syntax

```powerquery
HdInsight.Contents(
    account as text
) as table
```


## Details

Vrne tabelo za krmarjenje, ki vsebuje vrstico vsakega vsebnika, najdenega na URL-ju računa (<code>account</code>), iz trezorja shrambe Azure. Vsaka vrstica vsebuje povezavo na zbirke dvojiških podatkov vsebnikov.
