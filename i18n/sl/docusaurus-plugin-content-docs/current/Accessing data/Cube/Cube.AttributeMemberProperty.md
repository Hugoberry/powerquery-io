# Cube.AttributeMemberProperty


## Description

Vrne lastnost atributa dimenzije.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Vrne lastnost <code>propertyName</code> za atribut dimenzije <code>attribute</code>.



## Category
Cube
