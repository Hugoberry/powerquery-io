# Cube.AttributeMemberId


## Description

Vrne enolični identifikator člana iz vrednosti lastnosti članov.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Vrne enolični identifikator člana iz vrednosti lastnosti člana. <code>attribute</code>. Vrne "null" za vse druge vrednosti.

Cube
