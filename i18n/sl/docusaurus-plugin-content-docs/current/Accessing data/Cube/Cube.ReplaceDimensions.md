# Cube.ReplaceDimensions


## Description

Zamenja nabor dimenzij, ki jih vrne Cube.Dimensions.


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Details

Zamenja nabor dimenzij, ki jih vrne <code>Cube.Dimensions</code>. To funkcijo lahko na primer uporabite za dodajanje stolpca ID atributu razsežnosti, tako da lahko vir podatkov združuje na ID in ne na prikazano vrednost.
