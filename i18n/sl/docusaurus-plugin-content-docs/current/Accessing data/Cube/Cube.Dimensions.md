# Cube.Dimensions


## Description

Vrne tabelo z naborom razpoložljivih dimenzij.


## Syntax

```powerquery
Cube.Dimensions(
    cube as table
) as table
```


## Details

Vrne tabelo z naborom razpoložljivih dimenzij v <code>cube</code>. Vsaka dimenzija je tabela, ki vsebuje nabor dimenzijskih atributov, in vsak dimenzijski atribut je predstavljen kot stolpec v tabeli dimenzij. Dimenzije lahko v kocki razširite z Cube.AddAndExpandDimensionColumn.
