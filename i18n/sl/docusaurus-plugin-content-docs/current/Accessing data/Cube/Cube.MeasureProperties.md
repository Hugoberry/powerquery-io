# Cube.MeasureProperties


## Description

Vrne tabelo z naborom razpoložljivih lastnosti mere, ki so razširjene v kocki.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Vrne tabelo z naborom razpoložljivih lastnosti, ki so na voljo za mere, razširjene v kocki.
