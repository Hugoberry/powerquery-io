# Cube.CollapseAndRemoveColumns


## Description

Spremeni dimenzionalno granularnost konteksta filtra kocke s strnjevanjem atributov, preslikanih na navedene stolpce.


## Syntax

```powerquery
Cube.CollapseAndRemoveColumns(
    cube as table,
    columnNames as list
) as table
```


## Details

Spremeni dimenzionalno granularnost konteksta filtra za <code>cube</code> s strnjevanjem atributov, preslikanih na navedene stolpce <code>columnNames</code>. Stolpci se tudi odstranijo iz tabelarnega pogleda kocke.
