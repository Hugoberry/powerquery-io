# Cube.Measures


## Description

Vrne tabelo z naborom razpoložljivih meritev.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

Vrne tabelo z naborom razpoložljivih meritev v <code>cube</code>.    Vsaka meritev je predstavljena kot funkcija. Meritve se lahko za kocko uporabijo s Cube.AddMeasureColumn.

Cube
