# Cube.ApplyParameter


## Description

Vrne kocko, ko zanjo uporabite parameter.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Vrne kocko, ko uporabite <code>parameter</code> z vrednostjo <code>arguments</code> za <code>cube</code>.



## Category
Cube
