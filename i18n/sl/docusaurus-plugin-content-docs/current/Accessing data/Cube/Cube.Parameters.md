# Cube.Parameters


## Description

Vrne tabelo z naborom parametrov, ki jih lahko uporabite v kocki.


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Details

Vrne tabelo z naborom parametrov, ki jih lahko uporabite v kocki <code>cube</code>. Vsak parameter je funkcija, ki jo lahko prikličete, da dobite <code>cube</code> z uporabljenim parametrom in njegovimi argumenti.
