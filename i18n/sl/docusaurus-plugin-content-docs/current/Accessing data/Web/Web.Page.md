# Web.Page


## Description

Vrne vsebino dokumenta HTML, razdeljeno na sestavne dele, ter predstavitev celotnega dokumenta in njegovega besedila po odstranitvi oznak.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
