# Excel.CurrentWorkbook


## Description

Vrne vsebino trenutnega Excelovega delovnega zvezka.


## Syntax

```powerquery
Excel.CurrentWorkbook(

) as table
```


## Details

Vrne vsebino trenutnega Excelovega delovnega zvezka. Vrne tabele, imenovane obsege in dinamična polja strežnikov. Za razliko od <code>Excel.Workbook</code> ne vrne listov.
