# Marketo.Tables


## Description

Vnesite URL končne točke vmesnika API REST za Marketo, ki je povezan z vašim računom.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Vrne tabelo z ustreznimi podatki programa Marketo.
