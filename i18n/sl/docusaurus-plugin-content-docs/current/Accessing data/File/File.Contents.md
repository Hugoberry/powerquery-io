# File.Contents


## Description

Vrne vsebino določene datoteke kot dvojiško vrednost.


## Syntax

```powerquery
File.Contents(
    path as text,
    optional options as record
) as binary
```


## Details

Vrne vsebino datoteke, <code>path</code>, kot dvojiško vrednost. Parameter <code>options</code> je trenutno namenjen samo za notranjo uporabo.

Accessing data
