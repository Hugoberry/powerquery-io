# Soda.Feed


## Description

Vrne tabelo iz vsebine na navedenem URL-ju, formatirano skladno z API-jem SODA 2.0.


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Details

Vrne tabelo iz vsebine na navedenem URL-ju <code>url</code>, formatirano skladno z API-jem SODA 2.0. URL mora kazati na veljaven vir, skladen s SODA, ki ima pripono .csv.
