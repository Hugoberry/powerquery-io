# ActiveDirectory.Domains


## Description

Vrne seznam domen storitve Active Directory v istem gozdu, kot je določena domena, ali iz trenutne domene računalnika, če nobena ni določena.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```
