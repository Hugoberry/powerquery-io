# VSTS.AnalyticsViews


## Description

Vnesite organizacijo in imena projektov.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Vrne tabelo pogledov storitve Analytics, ki jih zagotavlja storitev Azure DevOps.
