# Webtrends.Tables


## Description

Vnesite ID profila, ki je povezan z vašim računom za Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Vrne tabelo z ustreznimi podatki storitve Webtrends.
