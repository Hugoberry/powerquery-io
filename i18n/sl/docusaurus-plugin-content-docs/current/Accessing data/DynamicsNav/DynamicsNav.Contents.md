# DynamicsNav.Contents


## Description

Vnesite URL končne točke storitve Dynamics NAV OData.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Vrne tabelo z ustreznimi podatki storitve Dynamics NAV.
