# AzureDevOpsServer.AccountContents


## Description

Vnesite URL storitve Azure DevOps Analytics.


## Syntax

```powerquery
AzureDevOpsServer.AccountContents(
    url as text,
    optional options as record
) as table
```


## Details

Vrne vsebino, preneseno z naslova URL storitve Azure DevOps Analytics, v obliki dvojiške vrednosti. Ta funkcija uporablja različne poverilnice za različne organizacije.
