# Dynamics365BusinessCentralOnPremises.Contents


## Description

Vnesite URL končne točke storitve OData za Dynamics 365 Business Central (na mestu uporabe).


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Vrne tabelo z ustreznimi podatki storitve Dynamics 365 Business Central (na mestu uporabe).
