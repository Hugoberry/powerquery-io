# Twilio.Tables


## Description

Vnesite število mesecev, za katere želite pridobiti zgodovinske podatke storitve Twilio.


## Syntax

```powerquery
Twilio.Tables(
    historyMonths as number
) as table
```


## Details

Vrne tabelo z ustreznimi podatki storitve Twilio.
