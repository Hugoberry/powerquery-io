# Hdfs.Contents


## Description

Vrne tabelo, ki vsebuje lastnosti in vsebino datotek in map, najdenih v določeni mapi iz datotečnega sistema Hadoop.


## Syntax

```powerquery
Hdfs.Contents(
    url as text
) as table
```


## Details

Vrne tabelo, ki vsebuje vrstico za vsako mapo in datoteko, najdeno na URL-ju mape, <code>url</code>, iz datotečnega sistema Hadoop. Vsaka vrstica vsebuje lastnosti mape ali datoteke in povezavo do vsebine.
