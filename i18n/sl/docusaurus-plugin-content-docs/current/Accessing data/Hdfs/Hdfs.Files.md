# Hdfs.Files


## Description

Vrne tabelo, ki vsebuje lastnosti in vsebino datotek, najdenih v določeni mapi in podmapah iz datotečnega sistema Hadoop.


## Syntax

```powerquery
Hdfs.Files(
    url as text
) as table
```


## Details

Vrne tabelo, ki vsebuje vrstico za vsako datoteko, najdeno na URL-ju mape, <code>url</code>, in podmape iz datotečnega sistema Hadoop. Vsaka vrstica vsebuje lastnosti datoteke in povezavo do vsebine.
