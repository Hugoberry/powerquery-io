# Error.Record


## Description

Vrne zapis o napaki iz podanih besedilnih vrednosti za razlog, sporočilo in podrobnosti.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
