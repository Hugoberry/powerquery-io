# Type.ReplaceFacets


## Description

Nadomesti fasete neke vrste.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Nadomesti fasete za <code>type</code> s fasetami, ki so vsebovane v zapisu <code>facets</code>.
