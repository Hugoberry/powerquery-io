# Type.TableColumn


## Description

Vrne vrsto stolpca v tabeli.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Vrne vrsto stolpca <code>column</code> v vrsti tabele <code>tableType</code>.



## Category
Type
