# Tables.GetRelationships


## Description

Pridobi relacije med naborom tabel.


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Details

Pridobi relacije med naborom tabel. Za nabor <code>tables</code> se predvideva, da ima strukturo, ki je podobna strukturi tabele za krmarjenje. Stolpec, doloƒçen s/z <code>dataColumn</code>, vsebuje dejanske podatkovne tabele.
