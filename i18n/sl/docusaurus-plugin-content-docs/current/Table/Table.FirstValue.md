# Table.FirstValue


## Description

Vrne prvi stolpec prve vrstice tabele ali določeno privzeto vrednost.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

Vrne prvi stolpec prve vrstice tabele <code>table</code> ali določeno privzeto vrednost.
