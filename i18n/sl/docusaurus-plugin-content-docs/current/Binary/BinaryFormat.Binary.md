# BinaryFormat.Binary


## Description

Vrne dvojiško obliko, ki bere dvojiško vrednost.


## Syntax

```powerquery
BinaryFormat.Binary(
    optional length as any
) as function
```


## Details

Vrne dvojiško obliko, ki bere dvojiško vrednost. Če je določen parameter <code>length</code>, bo dvojiška vrednost vsebovala toliko bajtov. Če parameter <code>length</code> ni določen, bo dvojiška vrednost vsebovala preostanek bajtov. <code>length</code> je lahko določen kot število ali kot dvojiška oblika dolžine pred dvojiškimi podatki.
