# BinaryFormat.7BitEncodedSignedInteger


## Description

Dvojiška oblika, ki bere 64-bitno celo število s predznakom, ki je bilo kodirano s 7-bitnim kodiranjem spremenljive dolžine.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
