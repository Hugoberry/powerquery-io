# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Dvojiška oblika, ki bere 64-bitno celo število brez predznaka, ki je bilo kodirano s 7-bitnim kodiranjem spremenljive dolžine.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
