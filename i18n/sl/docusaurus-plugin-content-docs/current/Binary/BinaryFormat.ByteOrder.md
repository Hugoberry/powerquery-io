# BinaryFormat.ByteOrder


## Description

Vrne dvojiško obliko z vrstnim redom bajtov, ki ga določa funkcija.


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

Vrne dvojiško obliko z vrstnim redom bajtov, ki ga določa <code>binaryFormat</code>.  Privzeti vrstni red bajtov je <code>ByteOrder.BigEndian</code>.
