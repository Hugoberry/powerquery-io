# Text.ToBinary


## Description

Kodira besedilo v dvojiško obliko.


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

Kodira dano besedilno vrednost <code>text</code> v dvojiško vrednost z navedenim <code>encoding</code>.



## Category
Text.Conversions from and to text
