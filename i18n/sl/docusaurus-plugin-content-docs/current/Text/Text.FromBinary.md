# Text.FromBinary


## Description

Dekodira podatke iz dvojiške oblike v besedilo.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

Dekodira podatke <code>binary</code> iz dvojiške vrednosti v besedilno vrednost z vrsto <code>encoding</code>.



## Category
Text.Conversions from and to text
