# Text.Length


## Description

Vrne število znakov.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Vrne število znakov v besedilu <code>text</code>.


## Examples

### Example #1 
Ugotovite, koliko znakov je v besedilu &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
