# Lines.ToText


## Description

Pretvori seznam besedil v eno besedilo.  Določeni lineSeparator je priložen vsaki vrstici.  Če ni določen, se uporabijo vrnitev prevoza in znaki za pomike v novo vrstico.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```


## Details
