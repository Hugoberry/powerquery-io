# BinaryFormat.7BitEncodedSignedInteger


## Description

Et binært format, der læser et 64-bit heltal med fortegn, som er kodet ved hjælp af en 7-bit kodning med variabel længde.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details
