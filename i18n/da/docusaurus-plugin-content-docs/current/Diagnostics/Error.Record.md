# Error.Record


## Description

Returnerer en fejlpost fra de angivne tekstværdier for årsag, meddelelse og oplysninger.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
