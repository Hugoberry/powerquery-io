# Geography.ToWellKnownText


## Description

Oversætter en struktureret geografisk punktværdi til den formatet Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Oversætter en struktureret geografisk punktværdi til formatet Well-Known Text (WKT) som defineret af Open Geospatial Consortium (OGC), også det serialiseringsfunktionen, der bruges af mange databaser, herunder SQL Server.



## Category
Record.Serialization
