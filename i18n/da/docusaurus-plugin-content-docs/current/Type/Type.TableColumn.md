# Type.TableColumn


Returnerer typen af en kolonne i en tabel.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Remarks

Returnerer typen af kolonnen <code>column</code> i tabeltypen <code>tableType</code>.



## Category
Type
