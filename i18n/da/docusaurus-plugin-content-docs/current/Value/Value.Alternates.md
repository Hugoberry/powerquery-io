# Value.Alternates


## Description

Udtrykker alternative forespørgselsplaner.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Udtrykker alternative forespørgselsplaner i et forespørgselsplanudtryk, der er opnået via Value.Expression(Value.Optimize(...)). Ikke beregnet til andre formål.



## Category
Expression
