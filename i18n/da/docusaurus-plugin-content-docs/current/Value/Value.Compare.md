# Value.Compare


## Description

Returnerer -1, 0 eller 1 baseret på, om den første værdi er mindre end, lig med eller større end den anden.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```


## Details
