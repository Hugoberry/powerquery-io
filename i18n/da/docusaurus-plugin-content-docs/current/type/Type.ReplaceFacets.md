# Type.ReplaceFacets


Erstatter facetterne for en type.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Remarks

Erstatter facetterne for <code>type</code> med de facetter, der findes i posten <code>facets</code>.
