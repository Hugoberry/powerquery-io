# Lines.ToText


Konverterer en liste med tekst til en enkelt tekst.  Den angivne lineSeparator tilføjes på hver linje.  Hvis den ikke er angivet, bruges tegnene for vognretur og linjeskift.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```
