# DateTimeZone.FixedUtcNow


Returnerer den aktuelle dato og det aktuelle klokkeslæt i UTC (GMT-tidszonen). Denne værdi er fast og ændres ikke ved efterfølgende opkald.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```


## Remarks

Returnerer den aktuelle dato og det aktuelle klokkeslæt i UTC (GMT-tidszonen). Denne værdi er fast og ændres ikke ved efterfølgende opkald.
