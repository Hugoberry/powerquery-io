# DateTimeZone.UtcNow


Returnerer den aktuelle dato og det aktuelle klokkeslæt i UTC (GMT-tidszonen).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

Returnerer den aktuelle dato og det aktuelle klokkeslæt i UTC (GMT-tidszonen).


## Examples

### Example #1 
Hent den aktuelle dato og det aktuelle klokkeslæt i UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
