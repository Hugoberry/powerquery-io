# Number.Cos


## Description

Returnerer cosinus til tallet.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Returnerer cosinus til <code>number</code>.


## Examples

### Example #1 
Find cosinus af vinklen 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
