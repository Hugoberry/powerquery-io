# Number.Tan


Returnerer tangensen til tallet.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

Returnerer tangensen til <code>number</code>.


## Examples

### Example #1 
Find tangensen af vinklen 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
