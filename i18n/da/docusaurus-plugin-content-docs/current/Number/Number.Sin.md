# Number.Sin


Returnerer sinus til tallet.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

Returnerer sinus til <code>number</code>.


## Examples

### Example #1 
Find sinus af vinklen 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
