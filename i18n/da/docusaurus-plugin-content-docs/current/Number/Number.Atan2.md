# Number.Atan2


## Description

Returnerer arcus tangens til divisionen af de to tal.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Returnerer arcus tangens til divisionen af de to tal, <code>y</code> og <code>x</code>. Divisionen oprettes som <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
