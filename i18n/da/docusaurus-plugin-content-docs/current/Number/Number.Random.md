# Number.Random


Returnerer et tilfældigt tal.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Returnerer et vilkårligt tal mellem 0 og 1.


## Examples

### Example #1 
Hent et tilfældigt tal.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
