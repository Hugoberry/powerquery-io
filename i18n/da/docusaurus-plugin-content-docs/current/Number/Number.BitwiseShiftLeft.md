# Number.BitwiseShiftLeft


## Description

Skifter bitsættet til venstre.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnerer resultatet af at udføre et bitvis skift til venstre på <code>number1</code> med det angivne antal bit <code>number2</code>.



## Category
Number.Bytes
