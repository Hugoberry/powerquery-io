# Number.RandomBetween


## Description

Returnerer et tilfældigt tal mellem de to tal.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Returnerer et tilfældigt tal mellem <code>bottom</code> og <code>top</code>.


## Examples

### Example #1 
Hent et tilfældigt tal mellem 1 og 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
