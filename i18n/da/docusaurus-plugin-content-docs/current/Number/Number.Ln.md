# Number.Ln


## Description

Returnerer tallets naturlige logaritme.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Returnerer et tals naturlige logaritme, <code>number</code>. Hvis <code>number</code> er null, returnerer <code>Number.Ln</code> null.


## Examples

### Example #1 
Hent den naturlige logaritme for 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
