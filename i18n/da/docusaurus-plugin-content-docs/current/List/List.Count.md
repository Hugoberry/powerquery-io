# List.Count


## Description

Returnerer antallet af elementer på listen.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Returnerer antallet af elementer på listen <code>list</code>.


## Examples

### Example #1 
Find antallet af værdier på listen \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
