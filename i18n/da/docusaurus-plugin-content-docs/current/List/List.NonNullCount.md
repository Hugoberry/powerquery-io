# List.NonNullCount


Returnerer antallet af elementer på listen, der ikke er null.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Returnerer antallet af elementer på listen <code>list</code>, der ikke er null.
