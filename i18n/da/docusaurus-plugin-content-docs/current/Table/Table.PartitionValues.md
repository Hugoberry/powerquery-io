# Table.PartitionValues


Returnerer oplysninger om, hvordan en tabel er partitioneret.


## Syntax

```powerquery
Table.PartitionValues(
    table as table
) as table
```


## Remarks

Returnerer oplysninger om, hvordan en tabel er partitioneret.  Der returneres en tabel, hvor de enkelte kolonner er en partitionskolonne i den oprindelige tabel, og hvor de enkelte r√¶kker svarer til en partition i den oprindelige tabel.
