# Value.Equals


Returnerer, om de to værdier er ens.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Remarks

Returnerer true, hvis værdien <code>value1</code> er lig med værdien <code>value2</code>. Ellers returneres false.

Values
