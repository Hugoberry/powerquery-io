# Text.Length


## Description

Returnerer antallet af tegn.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Returnerer antallet af tegn i strengen <code>text</code>.


## Examples

### Example #1 
Find ud af, hvor mange tegn der er i teksten &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
