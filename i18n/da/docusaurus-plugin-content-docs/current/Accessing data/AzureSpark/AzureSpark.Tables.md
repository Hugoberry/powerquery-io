# AzureSpark.Tables


Vis tabellerne på en liste i en Azure Spark-instans.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Remarks

Vis tabellerne på en liste i en Azure Spark-instans.


## Examples

### Example #1 
Vis tabellerne på en liste i en Azure Spark-instans.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
