# Marketo.Tables


## Description

Angiv URL-adressen for det Marketo REST API-slutpunkt, der er knyttet til din konto.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Returnerer en tabel med relevante Marketo-data.
