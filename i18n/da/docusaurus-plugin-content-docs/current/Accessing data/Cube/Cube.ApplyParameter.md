# Cube.ApplyParameter


## Description

Returnerer en kube efter at have anvendt en parameter for den.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Returnerer en kube efter at have anvendt <code>parameter</code> med <code>arguments</code> for <code>cube</code>.



## Category
Cube
