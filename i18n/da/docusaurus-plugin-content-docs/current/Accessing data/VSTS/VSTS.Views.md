# VSTS.Views


## Description

Angiv organisation og projektnavne.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Returnerer en tabel med OData-feeds fra Azure DevOps.
