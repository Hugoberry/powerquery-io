# PowerPlatform.Dataflows


## Description

Importér data fra et dataflow


## Syntax

```powerquery
PowerPlatform.Dataflows(
    optional options as record
) as table
```


## Details

Opret forbindelse til alle de dataflow, du har adgang til, og vælg de enheder, du vil bruge
