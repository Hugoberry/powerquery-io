# Webtrends.Tables


## Description

Angiv det profil-id, der er knyttet til din Webtrends-konto.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Returnerer en tabel med relevante Webtrends-data.
