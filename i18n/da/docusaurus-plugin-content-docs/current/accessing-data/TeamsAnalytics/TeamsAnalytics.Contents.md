# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Remarks

Teams Analytics-forbindelseskomponenten gør det muligt for dig at få indsigt i din brug af Teams. Der er tilgængelige data til både din personlige aktivitet i Teams og aktiviteten i Teams, som du er medlem af.<br />Data er tilgængelige for de sidste 90 dage. <br /> Oplysninger om de tilgængelige typer data finder du i følgende hjælp-dokumentation: <br /><ul><li><b>Teams-brug:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>Teams-brugeraktivitet:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
