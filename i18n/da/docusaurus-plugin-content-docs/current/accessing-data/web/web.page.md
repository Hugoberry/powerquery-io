# Web.Page


Returnerer indholdet i HTML-dokumentet delt op i konstituentstrukturer samt en repræsentation af hele dokumentet og dokumentets tekst, efter at koder er fjernet.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Remarks

Returnerer indholdet i HTML-dokumentet delt op i konstituentstrukturer samt en repræsentation af hele dokumentet og dokumentets tekst, efter at koder er fjernet.
