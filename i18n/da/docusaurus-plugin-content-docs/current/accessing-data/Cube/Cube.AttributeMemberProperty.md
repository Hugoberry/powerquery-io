# Cube.AttributeMemberProperty


Returnerer en egenskab for en dimensionsattribut.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

Returnerer egenskaben <code>propertyName</code> for dimensionsattributten <code>attribute</code>.

Cube
