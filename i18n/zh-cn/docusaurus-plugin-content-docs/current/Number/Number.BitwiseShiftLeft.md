# Number.BitwiseShiftLeft


## Description

将位左移设置的位数。


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

返回对 <code>number1</code> 执行按位左移指定的位数 <code>number2</code> 所得的结果。



## Category
Number.Bytes
