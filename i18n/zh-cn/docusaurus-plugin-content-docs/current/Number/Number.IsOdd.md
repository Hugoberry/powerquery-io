# Number.IsOdd


## Description

指示值是否为奇数。


## Syntax

```powerquery
Number.IsOdd(
    number as number
) as logical
```


## Details

指示值是否为奇数。如果 <code>number</code> 为奇数，则返回 <code>true</code>；否则返回 <code>false</code>。


## Examples

### Example #1 
检查 625 是否为奇数。
```powerquery
Number.IsOdd(625)
```

Result: 
```powerquery
true
```


### Example #2 
检查 82 是否为奇数。
```powerquery
Number.IsOdd(82)
```

Result: 
```powerquery
false
```
