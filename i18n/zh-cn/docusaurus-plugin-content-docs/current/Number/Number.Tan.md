# Number.Tan


## Description

返回数的正切。


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

返回 <code>number</code> 的正切。


## Examples

### Example #1 
计算角 1 的正切。
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```




## Category
Number.Trigonometry
