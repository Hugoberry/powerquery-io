# HdInsight.Files


## Description

返回包含从 Azure 存储库的指定容器中找到的 Blob 属性和内容的表。


## Syntax

```powerquery
HdInsight.Files(
    account as text,
    containerName as text
) as table
```


## Details

返回包含从 Azure 存储库的容器 URL <code>account</code> 中找到的每个 Blob 文件行的表。

Accessing data
