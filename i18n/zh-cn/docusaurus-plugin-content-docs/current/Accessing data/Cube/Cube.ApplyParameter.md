# Cube.ApplyParameter


## Description

在将参数应用到多维数据集后将其返回。


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

通过 <code>arguments</code> 将 <code>parameter</code> 应用到 <code>cube</code> 后返回多维数据集。



## Category
Cube
