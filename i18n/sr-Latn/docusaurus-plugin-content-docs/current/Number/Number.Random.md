# Number.Random


## Description

Vraća nasumičan broj.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Vraća nasumični broj između 0 i 1.


## Examples

### Example #1 
Dobijanje nasumičnog broja.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
