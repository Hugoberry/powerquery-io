# Number.Log10


## Description

Vraća logaritam broja sa osnovom 10.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Vraća logaritam broja, <code>number</code>, sa osnovom 10. Ako je <code>number</code> bez vrednosti, <code>Number.Log10</code> vraća rezultat bez vrednosti.


## Examples

### Example #1 
Izračunavanje logaritma broja 2 sa osnovom 10.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
