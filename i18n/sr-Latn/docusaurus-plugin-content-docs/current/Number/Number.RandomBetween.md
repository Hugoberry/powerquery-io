# Number.RandomBetween


## Description

Vraća nasumičan broj između dva broja.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Vraća nasumičan broj između <code>bottom</code> i <code>top</code>.


## Examples

### Example #1 
Dobijanje nasumičnog broja između 1 i 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
