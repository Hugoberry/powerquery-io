# Number.Cos


## Description

Vraća kosinus broja.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Vraća kosinus za <code>number</code>.


## Examples

### Example #1 
Pronalaženje kosinusa ugla 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
