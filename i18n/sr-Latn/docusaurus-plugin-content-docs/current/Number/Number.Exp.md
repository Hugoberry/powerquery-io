# Number.Exp


## Description

Podiže e na dati stepen.


## Syntax

```powerquery
Number.Exp(
    number as number
) as number
```


## Details

Vraća rezultat podizanja e na stepen <code>number</code> (eksponencijalna funkcija).      <ul>        <li><code>number</code>: <code>number</code> za koji se izračunava eksponencijalna funkcija. Ako je <code>number</code> bez vrednosti, <code>Number.Exp</code> vraća rezultat bez vrednosti. </li>      </ul>


## Examples

### Example #1 
Podizanje e na treći stepen.
```powerquery
Number.Exp(3)
```

Result: 
```powerquery
20.085536923187668
```
