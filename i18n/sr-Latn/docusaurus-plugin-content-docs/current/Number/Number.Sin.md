# Number.Sin


## Description

Vraća sinus broja.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Vraća sinus za <code>number</code>.


## Examples

### Example #1 
Pronalaženje sinusa ugla 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
