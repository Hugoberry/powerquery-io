# Number.Tan


## Description

Vraća tangens broja.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Vraća tangens za <code>number</code>.


## Examples

### Example #1 
Pronalaženje tangensa ugla 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
