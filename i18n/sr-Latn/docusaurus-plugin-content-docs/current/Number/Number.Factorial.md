# Number.Factorial


## Description

Vraća faktorijel broja.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Vraća faktorijel broja <code>number</code>.


## Examples

### Example #1 
Pronalaženje faktorijela broja 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
