# Number.BitwiseShiftRight


## Description

Pomera skup bitova udesno.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Vraća rezultat izvršavanja pomeranja udesno na nivou bita na <code>number1</code>, za navedeni broj bitova <code>number2</code>.



## Category
Number.Bytes
