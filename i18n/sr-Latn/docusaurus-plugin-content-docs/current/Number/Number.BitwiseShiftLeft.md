# Number.BitwiseShiftLeft


## Description

Pomera skup bitova ulevo.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Vraća rezultat izvršavanja pomeranja ulevo na nivou bita na <code>number1</code>, za navedeni broj bitova <code>number2</code>.



## Category
Number.Bytes
