# Number.Ln


## Description

Vraća prirodni logaritam broja.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Vraća prirodni logaritam broja, <code>number</code>. Ako je <code>number</code> bez vrednosti, <code>Number.Ln</code> vraća rezultat bez vrednosti.


## Examples

### Example #1 
Izračunavanje prirodnog logaritma broja 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
