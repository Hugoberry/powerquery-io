# Number.BitwiseOr


## Description

Vraća rezultat izvršavanja operacije „Or“ na nivou bita između dva unosa.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Vraća rezultat izvršavanja operacije „Or“ na nivou bita između <code>number1</code> i <code>number2</code>.



## Category
Number.Bytes
