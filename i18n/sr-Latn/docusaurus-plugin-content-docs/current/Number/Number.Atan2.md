# Number.Atan2


## Description

Vraća arkus tangens za deljenje dva broja.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Vraća arkus tangens za deljenje dva broja, <code>y</code> i <code>x</code>. Deljenje će se konstruisati na sledeći način: <code>y</code>/<code>x</code>.

Number.Trigonometry
