# Number.BitwiseNot


## Description

Vraća bajt ako je svaki bit suprotan unosu.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Vraća rezultat izvršavanja operacije „Not“ na nivou bita na <code>number</code>.



## Category
Number.Bytes
