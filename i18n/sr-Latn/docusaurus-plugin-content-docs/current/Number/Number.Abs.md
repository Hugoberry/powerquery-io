# Number.Abs


## Description

Vraća apsolutnu vrednost broja.


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Details

Vraća apsolutnu vrednost za <code>number</code>. Ako je <code>number</code> bez vrednosti, <code>Number.Abs</code> vraća rezultat bez vrednosti.    <ul>        <li><code>number</code>: <code>number</code> za koji se izračunava apsolutna vrednost.</li>      </ul>


## Examples

### Example #1 
Apsolutna vrednost za -3.
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
