# Number.BitwiseXor


## Description

Vraća rezultat izvršavanja operacije „XOR“ (isključivo OR) na nivou bita između dva unosa.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Vraća rezultat izvršavanja operacije „XOR“ (isključivo OR) na nivou bita između <code>number1</code> i <code>number2</code>.



## Category
Number.Bytes
