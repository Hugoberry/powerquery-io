# Uri.Combine


## Description

Vraća apsolutni URI koji predstavlja kombinaciju ulaznog osnovnog URI identifikatora i relativnog URI identifikatora.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Vraća apsolutni URI koji predstavlja kombinaciju ulaznog <code>baseUri</code> i <code>relativeUri</code>.
