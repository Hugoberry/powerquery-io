# DateTimeZone.LocalNow


## Description

Vraća trenutni datum i vreme u lokalnoj vremenskoj zoni.


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

Vraća vrednost <code>datetimezone</code> podešenu na trenutni datum i vreme na sistemu.    Vraćena vrednost sadrži informacije o vremenskoj zoni koje se odnose na lokalnu vremensku zonu.
