# DateTimeZone.UtcNow


## Description

Vraća trenutni datum i vreme po UTC vremenu (GMT vremenska zona).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Vraća trenutni datum i vreme po UTC vremenu (GMT vremenska zona).


## Examples

### Example #1 
Dobijanje trenutnog datuma i vremena po UTC vremenu.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
