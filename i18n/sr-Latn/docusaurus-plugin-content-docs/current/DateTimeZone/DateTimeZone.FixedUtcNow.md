# DateTimeZone.FixedUtcNow


## Description

Vraća trenutni datum i vreme po UTC vremenu (GMT vremenska zona). Ova vrednost je fiksna i neće se promeniti uzastopnim pozivima.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
