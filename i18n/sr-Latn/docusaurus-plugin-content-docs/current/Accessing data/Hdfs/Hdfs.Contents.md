# Hdfs.Contents


## Description

Vraća tabelu koja sadrži svojstva i sadržaj datoteka i fascikli pronađenih u navedenoj fascikli sa Hadoop sistema datoteka.


## Syntax

```powerquery
Hdfs.Contents(
    url as text
) as table
```


## Details

Vraća tabelu koja sadrži red za svaku fasciklu i datoteku pronađenu na URL adresi fascikle, <code>url</code>, sa Hadoop sistema datoteka. Svaki red sadrži svojstva fascikle ili datoteke i vezu do njenog sadržaja.
