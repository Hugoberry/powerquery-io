# Web.Page


## Description

Vraća sadržaj HTML dokumenta razložen na sastavne strukture, kao i reprezentaciju celog dokumenta i odgovarajućeg teksta nakon uklanjanja oznaka.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
