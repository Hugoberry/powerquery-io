# ActiveDirectory.Domains


## Description

Vraća listu Active Directory domena u istoj šumi kao navedeni domen ili domena trenutnog računara ako nije naveden nijedan.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Details

Vraća listu Active Directory domena u istoj šumi kao navedeni domen ili domena trenutnog računara ako nije naveden nijedan.
