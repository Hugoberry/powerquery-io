# Exchange.Contents


## Description

Vraća sadržaj iz Microsoft Exchange naloga.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Details

Vraća tabelu sadržaja sa Microsoft Exchange naloga <code>mailboxAddress</code>. Ako se <code>mailboxAddress</code> ne navede, koristiće se podrazumevani nalog za akreditiv.
