# HdInsight.Containers


## Description

Vraća navigacionu tabelu koja sadrži kontejnere pronađene na navedenom nalogu iz Azure bezbednog skladišta.


## Syntax

```powerquery
HdInsight.Containers(
    account as text
) as table
```


## Details

Vraća navigacionu tabelu koja sadrži red za svaki kontejner pronađen na URL adresi naloga, <code>account</code>, iz Azure bezbednog skladišta. Svaki red sadrži vezu do blob objekata kontejnera.
