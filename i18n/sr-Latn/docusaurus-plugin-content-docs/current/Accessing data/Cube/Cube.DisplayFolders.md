# Cube.DisplayFolders


## Description

Vraća ugnežđeno stablo tabela koje predstavlja hijerarhiju objekata (na primer, dimenzije i mere) prikazane fascikle.


## Syntax

```powerquery
Cube.DisplayFolders(
    cube as table
) as table
```


## Details

Vraća ugnežđeno stablo tabela koje predstavlja hijerarhiju objekata (na primer, dimenzije i mere) prikazane fascikle koja se može koristiti u <code>cube</code>.
