# Cube.ReplaceDimensions


## Description

Zamenjuje skup dimenzija koje je vratila funkcija Cube.Dimensions.


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Details

Zamena skupa dimenzija koje vraća <code>Cube.Dimensions</code>.    Na primer, ova funkcija može da se koristi za dodavanje ID kolone atributu dimenzije kako bi izvor podataka mogao da se grupiše na ID-u, a ne na prikazanoj vrednosti.
