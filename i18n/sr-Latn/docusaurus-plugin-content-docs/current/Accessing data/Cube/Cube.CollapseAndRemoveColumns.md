# Cube.CollapseAndRemoveColumns


## Description

Menja dimenzionu granularnost konteksta filtera za kocku skupljanjem atributa mapiranih na navedene kolone.


## Syntax

```powerquery
Cube.CollapseAndRemoveColumns(
    cube as table,
    columnNames as list
) as table
```


## Details

Menja dimenzionu granularnost konteksta filtera za <code>cube</code> skupljanjem atributa mapiranih na navedene kolone <code>columnNames</code>. Kolone se takoÄ‘e uklanjaju iz tabelarnog prikaza kocke.
