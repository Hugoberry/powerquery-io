# Cube.AttributeMemberProperty


## Description

Vraća svojstvo atributa dimenzije.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Vraća svojstvo <code>propertyName</code> atributa dimenzije <code>attribute</code>.



## Category
Cube
