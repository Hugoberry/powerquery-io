# Cube.Measures


## Description

Vraća tabelu koja sadrži skup dostupnih mera.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

Vraća tabelu koja sadrži skup dostupnih mera u okviru <code>cube</code>.    Svaka mera je predstavljena kao funkcija. Mere mogu biti primenjene na kocku pomoću Cube.AddMeasureColumn.

Cube
