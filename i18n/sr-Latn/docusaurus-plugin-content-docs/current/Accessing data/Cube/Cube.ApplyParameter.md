# Cube.ApplyParameter


## Description

Vraća kocku nakon primene parametra.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Vraća kocku nakon primene <code>parameter</code> sa <code>arguments</code> na <code>cube</code>.



## Category
Cube
