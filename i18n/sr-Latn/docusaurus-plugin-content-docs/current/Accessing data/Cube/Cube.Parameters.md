# Cube.Parameters


## Description

Vraća tabelu sa skupom parametara koji mogu da se primene na kocku.


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Details

Vraća tabelu sa skupom parametara koji mogu da se primene na <code>cube</code>. Svaki parametar je funkcija koja može da se aktivira da bi se primenio <code>cube</code> sa parametrom i argumentima.

Cube
