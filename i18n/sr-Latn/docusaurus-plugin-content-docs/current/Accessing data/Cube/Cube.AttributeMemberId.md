# Cube.AttributeMemberId


## Description

Vraća jedinstveni identifikator člana iz vrednosti svojstava članova.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Vraća jedinstveni identifikator člana iz vrednosti svojstava članova. <code>attribute</code>. Vraća praznu vrednost za sve ostale vrednosti.

Cube
