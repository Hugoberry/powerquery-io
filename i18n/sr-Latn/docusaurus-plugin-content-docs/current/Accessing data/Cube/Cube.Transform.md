# Cube.Transform


## Description

Primenjuje listu funkcija kocke.


## Syntax

```powerquery

    cube as table,
    transforms as list
) as table
```


## Details

Primenjuje funkcije kocke sa liste, <code>transforms</code>, na <code>cube</code>.
