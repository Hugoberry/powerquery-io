# Cube.MeasureProperty


## Description

Vraća svojstvo mere (svojstvo ćelije).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Vraća svojstvo <code>propertyName</code> mere <code>measure</code>.



## Category
Cube
