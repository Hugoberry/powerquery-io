# Soda.Feed


## Description

Vraća tabelu na osnovu sadržaja sa navedene URL adrese oblikovanu u skladu sa SODA 2.0 API.


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Details

Vraća tabelu na osnovu sadržaja sa navedene URL adrese <code>url</code> oblikovanu u skladu sa SODA 2.0 API. URL mora da ukazuje da važeći izvor usaglašen sa SODA standardom koji ima oznaku tipa datoteke .csv.
