# Excel.CurrentWorkbook


## Description

Vraća sadržaj trenutne Excel radne sveske.


## Syntax

```powerquery
Excel.CurrentWorkbook(

) as table
```


## Details

Vraća sadržaj trenutne Excel radne sveske. Vraća tabele, imenovane opsege i dinamičke nizove. Za razliku od funkcije <code>Excel.Workbook</code>, ona ne vraća listove.
