# List.Count


## Description

Vraća broj stavki na listi.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Vraća broj stavki na listi <code>list</code>.


## Examples

### Example #1 
Pronalaženje broja vrednosti na listi \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
