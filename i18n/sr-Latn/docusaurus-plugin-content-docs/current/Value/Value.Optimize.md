# Value.Optimize


## Description

Signali Value.Expression za vraćanje optimizovanog izraza za vrednost.


## Syntax

```powerquery
Value.Optimize(
    value as any
) as any
```


## Details

Kada se koristi u okviru Value.Expression, ako <code>value</code> predstavlјa upit koji može da se optimizuje, ova funkcija ukazuje na to da treba da se vrati optimizovani izraz. U suprotnom, <code>value</code> se prosleđuje bez efekta.



## Category
Expression
