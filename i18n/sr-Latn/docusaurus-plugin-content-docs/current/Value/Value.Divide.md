# Value.Divide


## Description

Vraća rezultat deljenja prve vrednosti drugom.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Vraća rezultat deljenja vrednosti <code>value1</code> vrednošću <code>value2</code>. Može se navesti opcionalni parametar <code>precision</code>. Podrazumevano se koristi <code>Precision.Double</code>.
