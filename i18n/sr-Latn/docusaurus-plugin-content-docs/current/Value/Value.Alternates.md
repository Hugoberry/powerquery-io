# Value.Alternates


## Description

Izražava alternativne planove upita.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Izražava alternativne planove upita u okviru izraza plana upita dobijenog putem Value.Expression(Value.Optimize(...)). Nije namenjeno za druge upotrebe.



## Category
Expression
