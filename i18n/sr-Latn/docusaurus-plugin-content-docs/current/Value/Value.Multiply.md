# Value.Multiply


## Description

Vraća proizvod dve vrednosti.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Vraća proizvod množenja <code>value1</code> i <code>value2</code>. Može se navesti opcionalni parametar <code>precision</code>. Podrazumevano se koristi <code>Precision.Double</code>.

Values.Arithmetic operations
