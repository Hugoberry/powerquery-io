# Value.Equals


## Description

Navodi da li su dve vrednosti jednake.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Vraća „true“ ako je vrednost <code>value1</code> jednaka vrednosti <code>value2</code>. U suprotnom vraća „false“.

Values
