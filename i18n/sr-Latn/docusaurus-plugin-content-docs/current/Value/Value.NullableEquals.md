# Value.NullableEquals


## Description

Navodi da li su dve vrednosti jednake.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Vraća rezultat bez vrednosti ako je bilo koji argument „<code>value1</code>“, „<code>value2</code>“ bez vrednosti. U suprotnom je ekvivalentno vrednosti Value.Equals.
