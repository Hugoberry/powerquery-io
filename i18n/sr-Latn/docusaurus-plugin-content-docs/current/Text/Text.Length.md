# Text.Length


## Description

Vraća broj znakova.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Vraća broj znakova u tekstu <code>text</code>.


## Examples

### Example #1 
Otkrivanje koliko se znakova nalazi u tekstu „Hello World“.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
