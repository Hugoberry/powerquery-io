# Table.PartitionValues


## Description

Vraća informacije o tome kako je tabela podeljena.


## Syntax

```powerquery
Table.PartitionValues(
    table as table
) as table
```


## Details

Vraća informacije o tome kako je tabela podeljena.  Vraća se tabela u kojoj svaka kolona predstavlja podeljenu kolonu iz prvobitne tabele, a svaki red odgovara delu originalne tabele.
