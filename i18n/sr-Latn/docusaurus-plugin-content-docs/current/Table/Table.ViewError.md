# Table.ViewError


## Description

Kreira izmenjeni zapis o grešci koji neće aktivirati rezervnu postavku kada ga prijavi rukovalac definisan u prikazu (putem Table.View).


## Syntax

```powerquery
Table.ViewError(
    errorRecord as record
) as record
```


## Details

Kreira izmenjeni zapis o grešci sa <code>errorRecord</code> koji neće aktivirati rezervnu postavku kada ga prijavi rukovalac definisan u prikazu (putem Table.View).
