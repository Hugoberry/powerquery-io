# Table.FirstValue


## Description

Vraća prvu kolonu prvog reda tabele ili navedenu podrazumevanu vrednost.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

Vraća prvu kolonu prvog reda tabele <code>table</code> ili navedenu podrazumevanu vrednost.
