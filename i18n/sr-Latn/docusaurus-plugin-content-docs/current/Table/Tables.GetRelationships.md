# Tables.GetRelationships


## Description

Preuzima relacije u skupu tabela.


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Details

Preuzima relacije u skupu tabela. Za skup <code>tables</code> se pretpostavlja da ima sličnu strukturu kao navigaciona tabela. Kolona koju definiše <code>dataColumn</code> sadrži stvarne tabele sa podacima.
