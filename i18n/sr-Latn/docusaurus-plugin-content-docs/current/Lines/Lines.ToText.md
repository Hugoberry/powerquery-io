# Lines.ToText


## Description

Konvertuje listu tekstualnih vrednosti u pojedinačnu tekstualnu vrednost.  Navedeno svojstvo lineSeparator se dodaje na kraj svakog reda.  Ako nije navedeno, koriste se znakovi za kraj reda i znakovi za položaj u sledećem redu.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```
