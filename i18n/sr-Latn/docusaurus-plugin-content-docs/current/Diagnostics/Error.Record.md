# Error.Record


## Description

Vraća zapis o grešci od navedenih tekstualnih vrednosti za razlog, poruku i detalj.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details

Vraća zapis o grešci od navedenih tekstualnih vrednosti za razlog, poruku i detalj.
