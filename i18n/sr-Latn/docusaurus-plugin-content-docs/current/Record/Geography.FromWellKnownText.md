# Geography.FromWellKnownText


## Description

Prevodi tekst koji predstavlja geografsku vrednost u Well-Known Text (WKT) formatu u strukturirani zapis.


## Syntax

```powerquery
Geography.FromWellKnownText(
    input as text
) as record
```


## Details

Prevodi tekst koji predstavlja geografsku vrednost u Well-Known Text (WKT) formatu u strukturirani zapis. WKT je standardni format koji definiše Open Geospatial Consortium (OGC) i on je uobičajeni format serijalizacije koji koriste baze podataka, uključujući SQL Server.
