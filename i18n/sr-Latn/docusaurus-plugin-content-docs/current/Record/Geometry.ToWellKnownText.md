# Geometry.ToWellKnownText


## Description

Prevodi vrednost strukturirane geometrijske tačke u svoju Well-Known Text (WKT) reprezentaciju.


## Syntax

```powerquery
Geometry.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Prevodi vrednost strukturirane geometrijske tačke u svoju Well-Known Text (WKT) reprezentaciju na način koji je definisao Open Geospatial Consortium (OGC), takođe format serijalizacije koji koristi mnogo baza podataka, uključujući SQL Server.
