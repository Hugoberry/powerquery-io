# Type.ReplaceFacets


## Description

Zamenjuje aspekte tipa.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Zamenjuje aspekte za <code>type</code> aspektima koji se nalaze u zapisu <code>facets</code>.

Type
