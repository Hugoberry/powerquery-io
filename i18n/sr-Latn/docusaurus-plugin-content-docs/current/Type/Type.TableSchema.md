# Type.TableSchema


## Description

Vraća tabelu koja sadrži opis kolona (tj. šemu) navedenog tipa tabele.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

Vraća tabelu koja opisuje kolone za <code>tableType</code>.<br />Pogledajte dokumentaciju za <code>Table.Schema</code> da biste videli opis date tabele.<br />
