# BinaryFormat.ByteOrder


## Description

Vraća binarni format sa rasporedom bajtova koji navodi funkcija.


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

Vraća binarni format sa rasporedom bajtova koji navodi <code>binaryFormat</code>.  Podrazumevani raspored bajtova je <code>ByteOrder.BigEndian</code>.
