# BinaryFormat.Binary


## Description

Vraća binarni format koji čita binarnu vrednost.


## Syntax

```powerquery
BinaryFormat.Binary(
    optional length as any
) as function
```


## Details

Vraća binarni format koji čita binarnu vrednost.  Ako je navedena <code>length</code>, binarna vrednost će sadržati toliko bajtova.  Ako nije navedena <code>length</code>, binarna vrednost će sadržati preostale bajtove.  <code>length</code> može se navesti kao broj ili kao binarni format dužine koja prethodi binarnim podacima.
