# Binary.ViewError


## Description

Kreira izmenjeni zapis o grešci koji neće aktivirati vraćanje kada ga izbaci rukovalac definisan u prikazu (putem funkcije Binary.View).


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Details

Kreira izmenjeni zapis o grešci iz <code>errorRecord</code> koji neće aktivirati vraćanje kada ga izbaci rukovalac definisan u prikazu (putem funkcije Binary.View).



## Category
Binary
