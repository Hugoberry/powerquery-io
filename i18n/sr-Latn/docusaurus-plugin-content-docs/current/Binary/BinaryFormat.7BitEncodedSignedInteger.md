# BinaryFormat.7BitEncodedSignedInteger


## Description

Binarni format koji čita 64-bitni potpisani ceo broj koji je kodiran pomoću kodiranja promenljive dužine sa bazom od 7 bitova.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
