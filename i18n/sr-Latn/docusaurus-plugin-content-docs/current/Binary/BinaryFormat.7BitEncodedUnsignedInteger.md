# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Binarni format koji čita 64-bitni nepotpisani ceo broj koji je kodiran pomoću kodiranja promenljive dužine sa bazom od 7 bitova.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
