# DateTime.FixedLocalNow


## Description

Vraća trenutni datum i vreme u lokalnoj vremenskoj zoni. Ova vrednost je fiksna i neće se promeniti uzastopnim pozivima.


## Syntax

```powerquery
DateTime.FixedLocalNow(

) as datetime
```


## Details

Vraća vrednost <code>datetime</code> podešenu na trenutni datum i vreme na sistemu. Ova vrednost je fiksna i neće se promeniti uzastopnim pozivima, za razliku od DateTime.LocalNow, koja može vratiti različite vrednosti tokom izvršenja izraza.
