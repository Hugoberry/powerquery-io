# Function.InvokeAfter


## Description

Poziva datu funkciju nakon što je prosleđeno navedeno trajanje.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

Vraća rezultat <code>function</code> nakon što je trajanje <code>delay</code> prosleđeno.



## Category
Function
