# Lines.ToBinary


## Description

Мәтін тізімін көрсетілген кодтауды және жол бөлгішін пайдаланып екілік мәнге түрлендіреді. Көрсетілген жол бөлгіші әр жолға үстеледі.  Егер көрсетілмеген болса, онда жылжыма бөлікті қайтару және бір жолға түсу таңбалары пайдаланылады.


## Syntax

```powerquery
Lines.ToBinary(
    lines as list,
    optional lineSeparator as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```
