# Type.ReplaceFacets


## Description

Түр аспектілерін ауыстырады.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

<code>type</code> аспектілерін <code>facets</code> жазбасындағы аспектілермен ауыстырады.
