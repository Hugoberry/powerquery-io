# Type.TableColumn


## Description

Кестедегі бағанның түрін қайтарады.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

<code>tableType</code> кесте түріндегі <code>column</code> бағанының түрін қайтарады.
