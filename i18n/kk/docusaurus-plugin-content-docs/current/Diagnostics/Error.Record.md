# Error.Record


## Description

Себеп, хабар және мәліметтер үшін көрсетілген мәтіндік мәндерден қате жазбасын қайтарады.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
