# Table.Buffer


## Description

Кестені жадқа буферге сақтайды, сөйтіп оны бағалау кезінде сыртқы өзгертулерден оқшаулайды.


## Syntax

```powerquery
Table.Buffer(
    table as table,
    optional options as record
) as table
```


## Details

Есептеу кезінде сыртқы өзгерістерден оқшаулай отырып, жадтағы кестені буферге сақтайды.    Буферге сақтау толық емес. Ол кез келген скаляр ұяшық мәндерін есептеуге алып келеді, бірақ скаляр емес мәндерді (жазбалар, тізімдер, кестелер және т.б.) өзгеріссіз қалдырады.    <br />    <br />    Бұл функцияны пайдаланғанда сұрауларыңыз жылдамырақ орындалуы да, орындалмауы да мүмкін екенін ескеріңіз. Кейбір жағдайларда ол     барлық деректерді оқуға және оны жадта сақтауға кететін шығын қосылуына байланысты, сондай-ақ буферге сақтау төменгі ағындағы қысқартуды болдырмайтындықтан сұрауларыңыздың орындалуын баяулатуы мүмкін. Егер деректерді буферге сақтау қажет болмаса,     бірақ төменгі ағындағы қысқартудың алдын алғыңыз келсе, оның орнына <code>Table.StopFolding</code> пайдаланыңыз.


## Examples

### Example #1 
Кез келген алдағы болатын операциялар SQL серверіне сұрау жасай алмайтындай SQL кестесінің барлық жолдарын жадқа жүктеу.
```powerquery
let
    Source = Sql.Database("SomeSQLServer", "MyDb"),
    MyTable = Source{[Item="MyTable"]}[Data],
    BufferMyTable = Table.Buffer(dbo_MyTable)
in
    BufferMyTable
```

Result: 
```powerquery
table
```
