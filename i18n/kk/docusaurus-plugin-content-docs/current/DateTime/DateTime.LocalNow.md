# DateTime.LocalNow


## Description

Жергілікті уақыт белдеуіндегі ағымдағы күнді және уақытты қайтарады.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

<code>datetime</code> мәндер жиынын жүйедегі ағымдағы күнге және уақытқа қайтарады.



## Category
DateTime
