# Cube.AttributeMemberProperty


## Description

Өлшем атрибутының сипатын қайтарады.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

<code>attribute</code> өлшем атрибутының <code>propertyName</code> сипатын қайтарады.



## Category
Cube
