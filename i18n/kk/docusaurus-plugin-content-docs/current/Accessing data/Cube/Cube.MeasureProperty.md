# Cube.MeasureProperty


## Description

Өлшем сипатын (ұяшық сипатын) қайтарады.


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

<code>measure</code> өлшемінің <code>propertyName</code> сипатын қайтарады.



## Category
Cube
