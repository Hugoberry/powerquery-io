# Cube.ApplyParameter


## Description

Параметрді қолданғаннан кейін текшені қайтарады.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

<code>cube</code> үшін <code>arguments</code> бар <code>parameter</code> параметрін қолданғаннан кейін текшені қайтарады.



## Category
Cube
