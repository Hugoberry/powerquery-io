# Cube.AttributeMemberId


## Description

Бірегей мүше идентификаторын мүшелер сипаты мәнінен қайтарады.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Бірегей мүше идентификаторын мүше сипаты мәнінен қайтарады. <code>attribute</code>. Басқа мәндер үшін нөл мәнін қайтарады.

Cube
