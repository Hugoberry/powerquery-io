# Cube.MeasureProperties


## Description

Текшеде кеңейтілетін қолжетімді өлшем сипаттарының жиынтығы бар кестені қайтарады.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Текшеде кеңейтілетін өлшемдер үшін қолжетімді сипаттар жиынтығы бар кестені қайтарады.

Cube
