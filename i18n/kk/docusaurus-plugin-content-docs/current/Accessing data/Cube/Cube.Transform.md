# Cube.Transform


## Description

Текше функцияларының тізімін қолданады.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

<code>cube</code> ішінде <code>transforms</code> текше функцияларының тізімін қолданады.
