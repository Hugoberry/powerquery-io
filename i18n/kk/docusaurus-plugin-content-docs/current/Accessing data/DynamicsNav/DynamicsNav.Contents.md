# DynamicsNav.Contents


## Description

Dynamics NAV OData қызметінің соңғы нүктесінің URL мекенжайын енгізіңіз.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Тиісті Dynamics NAV деректері бар кестені қайтарады.
