# Marketo.Tables


## Description

Тіркелгіңізбен байланысты Marketo REST API соңғы нүктесінің URL мекенжайын енгізіңіз.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Тиісті Marketo деректері бар кестені қайтарады.
