# Identity.IsMemberOf


## Description

Куәліктің куәліктер жиынтығына кіретінін не кірмейтінін анықтайды.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details

Куәліктің куәліктер жиынтығына кіретінін не кірмейтінін анықтайды.
