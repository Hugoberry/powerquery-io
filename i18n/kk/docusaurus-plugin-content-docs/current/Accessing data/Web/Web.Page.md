# Web.Page


## Description

HTML құжатының құрайтын құрылымдарына бөлінген мазмұнын, сонымен бірге, тегтерді жоюдан кейінгі толық құжаттың оның мәтінінің көрінісін қайтарады.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Details
