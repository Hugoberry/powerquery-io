# Webtrends.Tables


## Description

Webtrends тіркелгісімен байланысты Профильдің идентфикаторын енгізіңіз.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Қатысты Webtrends деректері бар кестені қайтарады.
