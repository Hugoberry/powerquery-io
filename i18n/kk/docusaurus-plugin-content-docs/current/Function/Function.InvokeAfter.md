# Function.InvokeAfter


## Description

Көрсетілген ұзақтық өткен соң берілген функцияны шақырады.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

Ұзақтық <code>delay</code> өткен соң <code>function</code> шақыру нәтижесін қайтарады.



## Category
Function
