# Value.Divide


## Description

Бірінші мәнді екіншісіне бөлудің нәтижесін қайтарады.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

<code>value1</code> мәнін <code>value2</code> мәніне бөлудің нәтижесін қайтарады. <code>precision</code> міндетті емес параметрін көрсетуге болады, әдепкі мәні бойынша <code>Precision.Double</code> пайдаланылады.
