# Value.Subtract


## Description

Екі мәннің айырмашылығын қайтарады.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

<code>value1</code> және <code>value2</code> айырмашылығын қайтарады. <code>precision</code> міндетті емес параметрін көрсетуге болады, әдепкі мәні бойынша <code>Precision.Double</code> пайдаланылады.
