# Value.Equals


## Description

Тең екі мәнді қайтарады.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

<code>value1</code> мәні <code>value2</code> мәніне тең болса шын мәнін, олай болмаса өтіркі мәнін қайтарады.



## Category
Values
