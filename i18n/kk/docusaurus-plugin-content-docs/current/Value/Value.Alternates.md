# Value.Alternates


## Description

Басқа сұрау жоспарларын көрсетеді.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Value.Expression(Value.Optimize(...)) арқылы алынған сұрау жоспарының өрнегіндегі басқа сұрау жоспарларын көрсетеді. Басқа мақсатта пайдалануға арналмаған.



## Category
Expression
