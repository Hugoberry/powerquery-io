# Value.NullableEquals


## Description

Тең екі мәнді қайтарады.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

"<code>value1</code>", "<code>value2</code>" аргументттердің біреуінің мәні null болса, null мәнін қайтарады, олай болмаса, Value.Equals тең.

Values
