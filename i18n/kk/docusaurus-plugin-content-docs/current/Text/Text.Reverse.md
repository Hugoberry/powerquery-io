# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Берілген <code>text</code> кері өзгертеді.


## Examples

### Example #1 
&#34;123&#34; мәтінін кері өзгерту.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
