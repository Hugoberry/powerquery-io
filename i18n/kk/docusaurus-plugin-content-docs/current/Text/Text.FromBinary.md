# Text.FromBinary


## Description

Деректерді екілік түрден мәтінге кодсыздандырады.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

<code>encoding</code> түрін пайдаланып <code>binary</code> деректерін екілік мәннен мәтіндік мәнге түрлендіреді.
