# Text.InferNumberType


## Description

Мәтінде кодталған сан үшін нақтыланған сан түрін (Int64.Type, Double.Type және т.б.) алады.


## Syntax

```powerquery
Text.InferNumberType(
    text as text,
    optional culture as text
) as type
```


## Details

<code>text</code> нақтыланған сан түрін (Int64.Type, Double.Type және т.б.) алады. Егер <code>text</code> сан болмаса, қате шығарылады. Сонымен қатар міндетті емес <code>culture</code> мәні берілуі мүмкін (мысалы, "en-US").
