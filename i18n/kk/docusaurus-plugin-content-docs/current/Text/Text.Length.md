# Text.Length


## Description

Таңбалар санын қайтарады.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

<code>text</code> мәтініндегі таңбалар санын қайтарады.


## Examples

### Example #1 
&#34;Сәлем Әлем&#34; мәтініндегі таңбалар санын анықтау.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
