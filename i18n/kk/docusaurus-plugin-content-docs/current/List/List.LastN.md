# List.LastN


## Description

Тізімдегі соңғы мәнді қайтарады.  Таңдау бойынша, қанша мән қайтарылатынын көрсетуге немесе шартты нақтылауға болады.


## Syntax

```powerquery
List.LastN(
    list as list,
    optional countOrCondition as any
) as any
```


## Details

<code>list</code> тізімінің соңғы элементін қайтарады. Тізім бос болса, қиыс жағдай туралы хабар қайтарылады. Бұл функция бірнеше элементті жинауды немесе элементтерді сүзуді қолдау үшін <code>countOrCondition</code> міндетті емес параметрін қабылдайды. <code>countOrCondition</code> үш жолмен көрсетуге болады: <ul>  <li>Егер сан көрсетілген болса, соған дейінгі элемент қайтарылады. </li>  <li>Егер шарт көрсетілген болса, басында шартқа сай болған барлық элементтер тізімнің соңынан бастап қайтарылады. Бір элемент шартқа сай болмаса, басқа ешбір элементтер қарастырылмайды. </li>  <li>Егер бұл параметрдің мәні нөл болса, тізімдегі соңғы элемент қайтарылады.</li> </ul>


## Examples

### Example #1 
\{3, 4, 5, -1, 7, 8, 2} тізіміндегі соңғы мәнді анықтау.
```powerquery
List.LastN({3, 4, 5, -1, 7, 8, 2}, 1)
```

Result: 
```powerquery
{2}
```


### Example #2 
\{3, 4, 5, -1, 7, 8, 2} тізіміндегі 0-ден үлкенірек соңғы мәндерді анықтау.
```powerquery
List.LastN({3, 4, 5, -1, 7, 8, 2}, each _ > 0)
```

Result: 
```powerquery
{7, 8, 2}
```
