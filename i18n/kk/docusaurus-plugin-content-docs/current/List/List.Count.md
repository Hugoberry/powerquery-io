# List.Count


## Description

Тізімдегі элементтер санын қайтарады.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

<code>list</code> тізіміндегі элементтер санын қайтарады.


## Examples

### Example #1 
\{1, 2, 3} тізіміндегі мәндер санын анықтау.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
