# List.NonNullCount


## Description

Тізімдегі мәні нөл емес элементтер санын қайтарады.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

<code>list</code> тізіміндегі мәні нөл емес элементтер санын қайтарады.
