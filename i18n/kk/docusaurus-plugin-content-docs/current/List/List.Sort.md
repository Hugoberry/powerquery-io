# List.Sort


## Description

Деректер тізімін көрсетілген шарттарға сай сұрыптайды.


## Syntax

```powerquery
List.Sort(
    list as list,
    optional comparisonCriteria as any
) as list
```


## Details

Көрсетілген қосымша критерийге байланысты, <code>list</code>, деректер тізімін сұрыптайды.    <code>comparisonCriteria</code> қосымша параметрін салыстыру критерийі ретінде көрсетуге болады. Бұл келесі мәндерге ие бола алады:    <ul>    <li> Реттілікті басқару үшін, салыстыру критерийі “Реттілік” тізім мәні болуы мүмкін. (<code>Order.Descending</code>, <code>Order.Ascending</code>). </li>    <li> Сұрыптау үшін пайдаланылатын кілтті есептеу үшін, 1 аргумент функциясын пайдалануға болады. </li>    <li> Кілтті таңдау мен реттілікті басқарудың екеуі үшін де салыстыру критерийі кілт пен реттілікті қамтитын тізім болуы мүмкін (<code>\{each 1 / _, Order.Descending}</code>). </li>    <li> Салыстыруды толығымен басқару үшін 2 аргумент функциясын пайдалануға болады. Бұл функцияға тізімнен екі элемент жіберіледі (кез келген екі элемент, кез келген реттілікте). Функция келесі мәндердің біреуін қайтаруы тиіс:        <ul>            <li> <code>-1</code>: Бірінші элемент екінші элементтен кем.</li>            <li> <code>0</code>: Элементтер тең.</li>            <li> <code>1</code>: Бірінші элемент екінші элементтен артық.</li>        </ul>        Value.Compare осы логикаға өкілеттілік беру үшін пайдалануға болатын әдіс болып табылады. </li>    </ul>


## Examples

### Example #1 
\{2, 3, 1} тізімін сұрыптау.
```powerquery
List.Sort({2, 3, 1})
```

Result: 
```powerquery
{1, 2, 3}
```


### Example #2 
\{2, 3, 1} тізімін кему ретімен сұрыптау.
```powerquery
List.Sort({2, 3, 1}, Order.Descending)
```

Result: 
```powerquery
{3, 2, 1}
```


### Example #3 
Value.Compare әдісін пайдаланып \{2, 3, 1} тізімін кему ретімен сұрыптау.
```powerquery
List.Sort({2, 3, 1}, (x, y) => Value.Compare(1/x, 1/y))
```

Result: 
```powerquery
{3, 2, 1}
```
