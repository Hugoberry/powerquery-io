# BinaryFormat.7BitEncodedSignedInteger


## Description

7 биттік ұзындығы айнымалы кодтауды пайдаланып кодталған 64 биттік таңбалы бүтін санды оқитын екілік пішім.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
