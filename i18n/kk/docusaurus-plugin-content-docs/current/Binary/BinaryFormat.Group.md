# BinaryFormat.Group


## Description

Элементтер тобын оқитын екілік пішімді қайтарады.


## Syntax

```powerquery
BinaryFormat.Group(
    binaryFormat as function,
    group as list,
    optional extra as function,
    optional lastKey as any
) as function
```


## Details

Параметрлер келесідей:<ul><li><code>binaryFormat</code> параметрі кілт мәнінің екілік пішімін көрсетеді.</li><li><code>group</code> параметрі белгілі элементтер тобы туралы ақпаратты қамтамасыз етеді.</li><li><code>extra</code> қосымша параметрін кез келген кілттен кейінгі күтпеген мән үшін екілік пішімдегі мәнді қайтаратын функцияны көрсету үшін пайдалануға болады.  Егер <code>extra</code> параметрі көрсетілмесе, онда күтпеген кілт мәндері бар болса, қате орын алады.</li></ul><code>group</code> параметр элемен анықтамаларының тізімін көрсетеді.  Әр элемент анықтамасы 3-5 мәнді қамтитын тізім болып табылады, келесідей:<ul><li>Кілт мәні.  Элементке сәйкес кілттің мәні.  Бұл элементтер жиынында бірегей болуы керек.</li><li>Элементтің пішімі.  Элементтің мәніне сәйкес екілік пішім.  Бұл әр элементте әр түрлі пішім болуына мүмкіндік береді. </li><li>Элементтің қайталануы.   Элемент топта неше рет көрсетілетіні күтілетінін білдіретін <code>BinaryOccurrence.Type</code> мәні.   Міндетті элементтер жоқ болса, қате орын алады.  Міндетті немесе қосымша қайталанатын элементтер күтпеген кілт мәндері сияқты өңделеді.</li><li>Әдепкі элементтің мәні (қосымша).  Егер әдепкі элемент мәні элемент анықтамалары тізімінде болса және мәні нөл болмаса, онда ол әдепкі мәннің орнына пайдаланылады.  Қайталанатын немесе қосымша элементтердің әдепкі мәні — null, әл қайталанатын мәндердің әдепкі мәні — бос тізім \{ }.</li><li>Элемент мәнін түрлендіру (қосымша).   Егер элемен анықтамалары тізімінде элемент мәнін түрлендіру функциясы бар болса және мәні нөл емес болса, онда ол қайтарылмай тұрып элемент мәнін түрлендіру үшін шақырылады.  Түрлендіру функциясы элемент енгізбеде болса ғана шақырылады (ол ешқашан әдепкі мәнмен шақырылмайды). </li></ul>


## Examples

### Example #1 
Төменде бір байттық кілт мәні, топта 4 күтілетін элемент бар, олардың барлығында кілттен кейін деректер байты бар деп есептеледі.  Элементтер енгізбеде келесідей көрсетіледі:&lt;ul&gt;&lt;li&gt;1-кілт міндетті болып табылады және 11 мәнімен көрсетіледі.&lt;/li&gt;&lt;li&gt;2-кілт қайталанады және 22 мәнімен екі рет көрсетіледі, сөйтіп \{ 22, 22 } мәніне әкеледі.&lt;/li&gt;&lt;li&gt;3-кілт міндетті емес және көрсетілмейді, сөйтіп нөл мәніне әкеледі.&lt;/li&gt;&lt;li&gt;4-кілт қайталанады, бірақ көрсетілмейді, сөйтіп \{ } мәніне әкеледі.&lt;/li&gt;&lt;li&gt;5-кілт топтың бөлігі болып табылмайды, бірақ 55 мәнімен бір рет көрсетіледі.  Қосымша функция 5 кілт мәнімен шақырылады және сол мәнге сәйкес пішімді (BinaryFormat.Byte) қайтарады.  55 мәні оқылады және жойылады.&lt;/li&gt;&lt;/ul&gt;
```powerquery
let
    b = #binary({
        1, 11,
        2, 22,
        2, 22,
        5, 55,
        1, 11
    }),
    f = BinaryFormat.Group(
        BinaryFormat.Byte,
        {
            {1, BinaryFormat.Byte, BinaryOccurrence.Required},
            {2, BinaryFormat.Byte, BinaryOccurrence.Repeating},
            {3, BinaryFormat.Byte, BinaryOccurrence.Optional},
            {4, BinaryFormat.Byte, BinaryOccurrence.Repeating}
        },
        (extra) => BinaryFormat.Byte
    )
in
    f(b)
```

Result: 
```powerquery
{11, {22, 22}, null, {}}
```


### Example #2 
Келесі мысал элемент мәнін түрлендіруді және әдепкі элемент мәнін көрсетеді.   1-кілт бар қайталанатын элемент List.Sum пайдаланып мәндер тізімінің қосындысын есептейді.  2-кілт бар қосымша элементтің әдепкі мәні нөлдің орнына 123 болады.
```powerquery
let
    b = #binary({
        1, 101,
        1, 102
    }),
    f = BinaryFormat.Group(
        BinaryFormat.Byte,
        {
            {1, BinaryFormat.Byte, BinaryOccurrence.Repeating,
              0, (list) => List.Sum(list)},
            {2, BinaryFormat.Byte, BinaryOccurrence.Optional, 123}
        }
    )
in
    f(b)
```

Result: 
```powerquery
{203, 123}
```
