# Binary.View


## Description

Сұрау және әрекет операциялары үшін пайдаланушы анықтаған өңдегіштер арқылы екілік мәнді жасайды немесе кеңейтеді.


## Syntax

```powerquery
Binary.View(
    binary as binary,
    handlers as record
) as binary
```


## Details

Көрініске операция қолданылғанда <code>handlers</code> ішінде көрсетілген функциялар операцияның әдепкі әрекетінің орнына пайдаланылатын <code>binary</code> көрінісін қайтарады.<br /><code>binary</code> көрсетілген болса, барлық өңдегіш функциялар міндетті емес болып табылады. <code>binary</code> көрсетілмеген болса, <code>GetStream</code> өңдегіш функция қажет. Операция үшін өңдегіш функция көрсетілмеген болса, оның орнына <code>binary</code> дегенге операцияның әдепкі әрекеті қолданылады (<code>GetExpression</code> жағдайын қоспағанда).<br />Өңдегіш функциялар <code>binary</code> қатысты операцияны қолдану нәтижесіне (немесе <code>GetExpression</code> жағдайында нәтижесіндегі көрініске) семантикалық түрде баламалы.<br />Өңдегіш функция қате тудырса, көрініске операцияның әдепкі әрекеті қолданылады.<br /><code>Binary.View</code> дегенді деректер көзін қысқартуды (M сұрауларының көз операцияларына өтуі, мысалы, файл бөлімін жүктеп алу үшін) жүзеге асыру үшін пайдалануға болады.<br /><code>Binary.View</code> толығырақ сипаттамасын жарияланған Power Query теңшелетін қосқыш құжаттамасынан қараңыз.<br />


## Examples

### Example #1 
Ұзындықты анықтау үшін деректерге қатынасуды қажет етпейтін негізгі көріністі жасайды.
```powerquery
Binary.View(
    null,
    [
        GetLength = () => 12,
        GetStream = () => Text.ToBinary("hello world!")
    ]
)
```

Result: 
```powerquery
Text.ToBinary("hello world!")
```
