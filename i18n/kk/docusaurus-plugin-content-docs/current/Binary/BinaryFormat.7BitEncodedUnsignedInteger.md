# BinaryFormat.7BitEncodedUnsignedInteger


## Description

7 биттік ұзындығы айнымалы кодтауды пайдаланып кодталған 64 биттік таңбасыз бүтін санды оқитын екілік пішім.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
