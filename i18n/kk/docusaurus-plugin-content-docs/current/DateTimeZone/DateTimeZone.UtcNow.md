# DateTimeZone.UtcNow


## Description

Ағымдағы күнді және уақытты UTC (GMT уақыт белдеуі) пішімінде қайтарады.


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Ағымдағы күнді және уақытты UTC (GMT уақыт белдеуі) пішімінде қайтарады.


## Examples

### Example #1 
Ағымдағы күнді және уақытты UTC түрінде алу.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
