# Number.BitwiseShiftRight


## Description

Биттер жиынын оңға жылжытады.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

<code>number1</code> оң жаққа қарай көрсетілген <code>number2</code> биттер санына биттік жылжытуды орындаудың нәтижесін қайтарады.



## Category
Number.Bytes
