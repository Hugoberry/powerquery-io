# Number.Factorial


## Description

Санның факториалын қайтарады.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

<code>number</code> санының факториалын қайтарады.


## Examples

### Example #1 
10 факториалын анықтау.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
