# Number.BitwiseAnd


## Description

Екі енгізбе арасында биттік &#34;And&#34; әрекетін орындаудың нәтижесін қайтарады.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

<code>number1</code> және <code>number2</code> арасында биттік "And" әрекетін орындаудың нәтижесін қайтарады.



## Category
Number.Bytes
