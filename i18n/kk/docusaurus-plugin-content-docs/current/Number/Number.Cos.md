# Number.Cos


## Description

Санның косинусын қайтарады.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

<code>number</code> косинусын қайтарады.


## Examples

### Example #1 
0 бұрышының косинусын анықтау.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
