# Number.Atan2


## Description

Екі сан бөліндісінің арктангенсін қайтарады.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

<code>y</code> және <code>x</code> екі сан бөліндісінің арктангенсін қайтарады. Бөлу <code>y</code>/<code>x</code> түрінде көрсетіледі.
