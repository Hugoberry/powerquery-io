# Number.Random


## Description

Кездейсоқ санды қайтарады.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

0 мен 1 аралығындағы кездейсоқ санды қайтарады.


## Examples

### Example #1 
Кездейсоқ санды алу.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
