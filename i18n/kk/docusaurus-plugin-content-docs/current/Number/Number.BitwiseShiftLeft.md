# Number.BitwiseShiftLeft


## Description

Биттер жиынын солға жылжытады.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

<code>number1</code> солға қарай көрсетілген <code>number2</code> биттер санына биттік жылжытуды орындаудың нәтижесін қайтарады.



## Category
Number.Bytes
