# Number.BitwiseNot


## Description

Әр бит енгізбенің қарама-қарсы мәні болып табылатын байтты қайтарады.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

<code>number</code> мәнінде биттік "Not" әрекетін орындаудың нәтижесін қайтарады.



## Category
Number.Bytes
