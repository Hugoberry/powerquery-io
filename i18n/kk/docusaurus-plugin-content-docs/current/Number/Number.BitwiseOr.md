# Number.BitwiseOr


## Description

Екі енгізбе арасында биттік &#34;Or&#34; әрекетін орындаудың нәтижесін қайтарады.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

<code>number1</code> және <code>number2</code> арасында биттік "Or" әрекетін орындаудың нәтижесін қайтарады.



## Category
Number.Bytes
