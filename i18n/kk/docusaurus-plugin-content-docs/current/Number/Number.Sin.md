# Number.Sin


## Description

Санның синусын қайтарады.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

<code>number</code> синусын қайтарады.


## Examples

### Example #1 
0 бұрышының синусын анықтау.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
