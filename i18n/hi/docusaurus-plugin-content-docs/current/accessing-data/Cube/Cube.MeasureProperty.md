# Cube.MeasureProperty


किसी माप का गुण (कक्ष गुण) लौटाता है.


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Remarks

<code>measure</code> माप का <code>propertyName</code> गुण लौटाता है.

Cube
