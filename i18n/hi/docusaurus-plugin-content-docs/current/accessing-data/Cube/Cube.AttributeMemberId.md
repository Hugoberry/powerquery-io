# Cube.AttributeMemberId


सदस्य गुण मान से अद्वितीय सदस्य पहचानकर्ता लौटाता है.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Remarks

सदस्य गुण मान से अद्वितीय सदस्य पहचानकर्ता लौटाता है. <code>attribute</code>. किसी अन्य मान के लिए नल लौटाता है.
