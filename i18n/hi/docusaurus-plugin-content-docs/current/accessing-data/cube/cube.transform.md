# Cube.Transform


क्यूब फ़ंक्शन्स की एक सूची लागू करता है.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Remarks

क्यूब फ़ंक्शन्स, <code>transforms</code>, की सूची को <code>cube</code> पर लागू करता है.
