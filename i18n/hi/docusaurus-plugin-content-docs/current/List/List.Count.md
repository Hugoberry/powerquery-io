# List.Count


सूची के आइटमों की संख्या लौटाता है.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

सूची <code>list</code> के आइटमों की संख्या लौटाता है.


## Examples

### Example #1 
सूची \{1, 2, 3} के मानों की संख्या प्राप्त करें.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
