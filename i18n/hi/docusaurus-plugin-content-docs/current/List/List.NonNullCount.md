# List.NonNullCount


## Description

सूची के गैर-नल आइटमों की संख्या लौटाता है.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

सूची <code>list</code> के गैर-नल आइटमों की संख्या लौटाता है.
