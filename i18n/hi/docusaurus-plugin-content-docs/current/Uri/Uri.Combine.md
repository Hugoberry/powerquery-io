# Uri.Combine


## Description

एक पूर्ण URI लौटाता है जो इनपुट आधारित URI और रिलेटिव URI का संयोजन होता है.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

एक पूर्ण URI लौटाता है जो इनपुट <code>baseUri</code> और <code>relativeUri</code> का संयोजन होता है.
