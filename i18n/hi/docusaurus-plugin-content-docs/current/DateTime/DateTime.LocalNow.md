# DateTime.LocalNow


## Description

वर्तमान दिनांक और समय को स्थानीय समय क्षेत्र में लौटाता है.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

आपके सिस्टम के वर्तमान दिनांक और समय पर सेट <code>datetime</code> मान लौटाता है.

DateTime
