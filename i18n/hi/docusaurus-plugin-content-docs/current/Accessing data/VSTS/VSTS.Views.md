# VSTS.Views


## Description

खाता और प्रोजेक्ट के नाम दर्ज करें.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Azure DevOps द्वारा प्रस्तावित OData फ़ीड्स की तालिका लौटाता है.
