# VSTS.AnalyticsViews


## Description

खाता और प्रोजेक्ट के नाम दर्ज करें.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Azure DevOps द्वारा प्रस्तावित विश्लेषणात्मक दृश्यों की तालिका लौटाता है.
