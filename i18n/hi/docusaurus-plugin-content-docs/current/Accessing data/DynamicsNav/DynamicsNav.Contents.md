# DynamicsNav.Contents


अपने Dynamics NAV Odata सेवा समाप्ति बिंदु का URL दर्ज करें.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

प्रासंगिक Dynamics NAV डेटा के साथ तालिका लौटाता है.
