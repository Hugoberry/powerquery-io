# Cube.ApplyParameter


क्यूब पर पैरामीटर लागू करने के बाद उसे लौटाता है.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

<code>cube</code> पर <code>arguments</code> के साथ <code>parameter</code> लागू करने के बाद क्यूब लौटाता है.

Cube
