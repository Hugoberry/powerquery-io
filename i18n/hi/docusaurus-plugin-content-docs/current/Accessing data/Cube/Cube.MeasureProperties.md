# Cube.MeasureProperties


## Description

घन में विस्तृत उपलब्ध माप गुणों के सेट वाली तालिका लौटाता है.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

क्यूब में विस्तृत मापों के लिए उपलब्ध गुणों के सेट वाली एक तालिका लौटाता है.

Cube
