# Web.Page


## Description

अपनी घटक संरचनाओं में भंग HTML दस्तावेज़ की सामग्री, साथ ही टैग्स निकालने के बाद पूर्ण दस्तावेज़ और उसके पाठ का निरूपण लौटाता है.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
