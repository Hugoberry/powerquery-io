# Webtrends.Tables


## Description

अपने Webtrends खाते से संबद्ध प्रोफ़ाइल ID दर्ज करें.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

प्रासंगिक Webtrends डेटा वाली एक तालिका लौटाता है.
