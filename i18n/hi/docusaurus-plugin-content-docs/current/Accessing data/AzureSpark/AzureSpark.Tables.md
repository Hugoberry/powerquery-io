# AzureSpark.Tables


Azure स्पार्क इंस्टैंस में तालिकाओं को सूचीबद्ध करें.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Remarks

Azure स्पार्क इंस्टैंस में तालिकाओं को सूचीबद्ध करें.


## Examples

### Example #1 
Azure स्पार्क इंस्टैंस में तालिकाओं को सूचीबद्ध करें.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
