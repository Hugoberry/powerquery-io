# AtScale.Cubes


किसी AtScale से क्यूब डेटा आयात/DirectQuery करें.


## Syntax

```powerquery
AtScale.Cubes(
    server as Uri.Type,
    optional options as record
) as table
```


## Remarks

AtScale से सर्वर <code>सर्वर</code> पर क्यूब डेटा लौटाता है. निम्न विकल्पों को नियंत्रित करने के लिए एक वैकल्पिक रिकॉर्ड पैरामीटर, <code>विकल्प</code>, निर्दिष्ट किया जा सकता है:<ul>        <li><code>TypedMeasureColumns</code>: एक तार्किक मान इंगित करता है कि जोड़े गए माप स्तंभों के प्रकारों के लिए बहुआयामी या तालिकाबद्ध मॉडल में निर्दिष्ट प्रकार उपयोग किए जाएँगे या नहीं. false पर सेट होने से प्रकार "संख्या" सभी माप स्तंभों के लिए उपयोग किया जाएगा. इस विकल्प के लिए डिफ़ॉल्ट मान false है.</li>        <li><code>CommandTimeout</code>: वह अवधि (सेकंड में), जो यह नियंत्रित करती है कि रद्द किए जाने से पहले सर्वर-साइड क्वेरी को कितनी देर तक चलाए जाने की अनुमति है. डिफ़ॉल्ट मान ड्राइवर-आश्रित है. </li>        <li><code>ConnectionTimeout</code>: वह अवधि (सेकंड में), जो यह नियंत्रित करती है कि सर्वर से कनेक्शन बनाने का प्रयास छोड़ने से पहले कितनी देर तक प्रतीक्षा करनी है. </li></ul>रिकॉर्ड पैरामीटर [option1 = value1, option2 = value2...] के रूप में निर्दिष्ट है.
