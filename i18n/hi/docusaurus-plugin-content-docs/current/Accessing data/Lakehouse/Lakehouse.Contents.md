# Lakehouse.Contents


किसी झील के पास बने घर से डेटा आयात करें


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Remarks

झील के पास बने उन सभी घरों से कनेक्ट करें जिन तक आपकी पहुँच है, और उन निकायों को चुनें जिनका आप उपयोग करना चाहते हैं
