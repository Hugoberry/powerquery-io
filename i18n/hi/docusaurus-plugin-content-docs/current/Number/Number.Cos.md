# Number.Cos


संख्या का कोसिन लौटाता है.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

<code>number</code> का कोसिन लौटाता है.


## Examples

### Example #1 
0 कोण का कोसिन प्राप्त करें.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
