# Number.BitwiseNot


## Description

जहाँ प्रत्येक बिट, इनपुट के विपरीत होता है, वहाँ एक बाइट लौटाता है.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

<code>number</code> पर बिटवाइज़ "Not" कार्रवाई के निष्पादन का परिणाम लौटाता है.

Number.Bytes
