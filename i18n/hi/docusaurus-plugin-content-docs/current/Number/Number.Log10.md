# Number.Log10


## Description

संख्या का आधार-10 लघुगणक लौटाता है.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

संख्या, <code>number</code> का आधार-10 लघुगणक लौटाता है. अगर <code>number</code> नल है, तो <code>Number.Log10</code> नल लौटाता है.


## Examples

### Example #1 
2 का आधार-10 लघुगणक प्राप्त करें.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
