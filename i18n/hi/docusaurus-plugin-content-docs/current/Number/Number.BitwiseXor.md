# Number.BitwiseXor


दो इनपुट के बीच बिटवाइज़ &#34;XOR&#34; (अनन्य-OR) के निष्पादन का परिणाम लौटाता है.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> और <code>number2</code> के बीच बिटवाइज़ "XOR" (अनन्य-OR) के निष्पादन का परिणाम लौटाता है.
