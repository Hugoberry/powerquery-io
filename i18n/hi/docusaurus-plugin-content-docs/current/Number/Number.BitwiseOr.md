# Number.BitwiseOr


दो इनपुट के बीच बिटवाइज़ &#34;Or&#34; के निष्पादन का परिणाम लौटाता है.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> और <code>number2</code> के बीच बिटवाइज़ "Or" के निष्पादन का परिणाम लौटाता है.
