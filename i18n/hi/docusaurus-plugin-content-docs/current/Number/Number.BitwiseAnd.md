# Number.BitwiseAnd


दो इनपुट के बीच बिटवाइज़ &#34;And&#34; कार्रवाई के निष्पादन का परिणाम लौटाता है.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> और <code>number2</code> के बीच बिटवाइज़ "And" कार्रवाई के निष्पादन का परिणाम लौटाता है.
