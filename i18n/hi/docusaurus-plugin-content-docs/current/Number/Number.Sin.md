# Number.Sin


संख्या का सिन लौटाता है.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

<code>number</code> का सिन लौटाता है.


## Examples

### Example #1 
0 कोण का सिन प्राप्त करें.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
