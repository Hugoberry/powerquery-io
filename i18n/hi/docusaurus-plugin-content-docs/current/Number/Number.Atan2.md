# Number.Atan2


## Description

दो संख्याओं के विभाजन का आर्कटेंजेंट लौटाता है.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

दो संख्याओं, <code>y</code> और <code>x</code>, के विभाजन का आर्कटेंजेंट लौटाता है. विभाजन को <code>y</code>/<code>x</code> के रूप में बनाया जाएगा.
