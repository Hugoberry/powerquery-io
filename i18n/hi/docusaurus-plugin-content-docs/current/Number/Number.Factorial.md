# Number.Factorial


## Description

संख्या का क्रमगुणन लौटाता है.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

संख्या <code>number</code> का क्रमगुणन लौटाता है.


## Examples

### Example #1 
10 का क्रमगुणन प्राप्त करें.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
