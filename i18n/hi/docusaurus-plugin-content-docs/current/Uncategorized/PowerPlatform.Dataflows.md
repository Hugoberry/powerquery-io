# PowerPlatform.Dataflows


## Description

किसी डेटा प्रवाह से डेटा आयात करें


## Syntax

```powerquery
PowerPlatform.Dataflows(
    optional options as record
) as table
```


## Details

अपनी पहुँच वाले सभी डेटा प्रवाहों से कनेक्ट करें, और उन निकायों को चुनें, जिनका आप उपयोग करना चाहते हैं.
