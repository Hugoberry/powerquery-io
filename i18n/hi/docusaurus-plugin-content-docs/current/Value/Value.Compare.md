# Value.Compare


## Description

पहला मान दूसरे मान से कम, बराबर या अधिक है या नहीं, इसके आधार पर -1, 0, या 1 लौटाता है.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```
