# Value.Equals


## Description

लौटाता है कि क्या दो मान बराबर हैं या नहीं.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

अगर मान <code>value1</code> मान <code>value2</code> के बराबर है, तो true, अन्यथा false लौटाता है.



## Category
Values
