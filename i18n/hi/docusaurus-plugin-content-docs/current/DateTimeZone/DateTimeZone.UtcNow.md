# DateTimeZone.UtcNow


वर्तमान दिनांक और समय को UTC (GMT समय क्षेत्र) में लौटाता है.


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

वर्तमान दिनांक और समय को UTC (GMT समय क्षेत्र) में लौटाता है.


## Examples

### Example #1 
वर्तमान दिनांक और समय को UTC में प्राप्त करें.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
