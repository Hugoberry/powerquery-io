# DateTimeZone.FixedUtcNow


## Description

वर्तमान दिनांक और समय को UTC (GMT समय क्षेत्र) में लौटाता है. यह मान निश्चित होता है और अनुक्रमिक कॉल के साथ नहीं बदलेगा.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
