# Text.Length


## Description

वर्णों की संख्‍या लौटाता है.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

पाठ <code>text</code> के वर्णों की संख्या लौटाता है.


## Examples

### Example #1 
खोजें कि पाठ &#34;Hello World&#34; में कितने वर्ण हैं.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
