# Text.FromBinary


डेटा को बाइनरी फ़ॉर्म से पाठ में डीकोड करता है.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

<code>encoding</code> प्रकार का उपयोग करके डेटा, <code>binary</code>, को बाइनरी मान से पाठ मान में डीकोड करता है.
