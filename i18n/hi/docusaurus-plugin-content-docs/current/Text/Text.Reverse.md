# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

उपलब्ध कराए गए <code>text</code> को रिवर्स कर देता है.


## Examples

### Example #1 
&#34;123&#34; पाठ को रिवर्स करें.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
