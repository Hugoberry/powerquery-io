# Function.InvokeAfter


निर्दिष्ट अवधि के गुज़र जाने के बाद दिए गए फ़ंक्शन को इनवोक करता है.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Remarks

अवधि <code>delay</code> के गुज़र जाने के बाद <code>function</code> इनवोक करने का परिणाम लौटाता है.
