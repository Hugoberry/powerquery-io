# Type.TableColumn


## Description

तालिका में स्तंभ का प्रकार लौटाता है.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

तालिका प्रकार <code>tableType</code> में स्तंभ <code>column</code> का प्रकार लौटाता है.

Type
