# Type.ReplaceFacets


## Description

किसी प्रकार के फ़ैसिट बदल देता है.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

<code>type</code> प्रकार के फ़ैसिट को रिकॉर्ड <code>facets</code> में शामिल फ़ैसिट के साथ बदल देता है.
