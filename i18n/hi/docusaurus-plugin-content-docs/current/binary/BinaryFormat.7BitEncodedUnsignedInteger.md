# BinaryFormat.7BitEncodedUnsignedInteger


एक बाइनरी स्वरूप जो 64-बिट अहस्ताक्षरित पूर्णांक को पढ़ता है, जिसको 7-बिट चर-लंबाई एन्कोडिंग का उपयोग करके एन्कोड किया गया था.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks

एक बाइनरी स्वरूप जो 64-बिट अहस्ताक्षरित पूर्णांक को पढ़ता है, जिसको 7-बिट चर-लंबाई एन्कोडिंग का उपयोग करके एन्कोड किया गया था.
