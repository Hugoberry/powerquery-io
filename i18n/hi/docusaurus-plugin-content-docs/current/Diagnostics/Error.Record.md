# Error.Record


कारण, संदेश और विवरण के लिए प्रदान किए गए पाठ मान से एक त्रुटि रिकॉर्ड लौटाता है.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
