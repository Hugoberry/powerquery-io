# Value.Alternates


वैकल्पिक क्वेरी योजनाएँ व्यक्त करता है.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Remarks

Value. व्यंजक (Value. अनुकूलन (...)) के माध्यम से प्राप्त क्वेरी योजना व्यंजक में वैकल्पिक क्वेरी योजनाएँ व्यक्त करता है. अन्य उपयोगों के लिए लक्षित नहीं है.
