# BinaryFormat.UnsignedInteger64


## Description

बाइनरी स्वरूप जो 64-बिट अहस्ताक्षरित पूर्णांक को पढ़ता है.


## Syntax

```powerquery
BinaryFormat.UnsignedInteger64(
    binary as binary
) as any
```
