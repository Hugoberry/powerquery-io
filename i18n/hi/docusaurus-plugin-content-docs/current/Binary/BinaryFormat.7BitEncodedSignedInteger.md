# BinaryFormat.7BitEncodedSignedInteger


## Description

एक बाइनरी स्वरूप जो 64-बिट हस्ताक्षरित पूर्णांक को पढ़ता है, जिसको 7-बिट चर-लंबाई एन्कोडिंग का उपयोग करके एन्कोड किया गया था.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details
