# BinaryFormat.SignedInteger32


## Description

बाइनरी स्वरूप जो 32-बिट हस्ताक्षरित पूर्णांक को पढ़ता है.


## Syntax

```powerquery
BinaryFormat.SignedInteger32(
    binary as binary
) as any
```
