# BinaryFormat.Double


## Description

बाइनरी स्वरूप जो 8-बाइट IEEE दोहरी-परिशुद्धता फ़्लोटिंग पॉइंट मान को पढ़ता है.


## Syntax

```powerquery
BinaryFormat.Double(
    binary as binary
) as any
```
