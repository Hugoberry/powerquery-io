# BinaryFormat.UnsignedInteger16


## Description

बाइनरी स्वरूप जो 16-बिट अहस्ताक्षरित पूर्णांक को पढ़ता है.


## Syntax

BinaryFormat.UnsignedInteger16(
    binary as binary
) as any
