# Number.Random


यादृच्छिक संख्या लौटाता है.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

0 और 1 के बीच एक यादृच्छिक संख्या लौटाता है.


## Examples

### Example #1 
यादृच्छिक संख्या प्राप्त करें.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
