# Number.BitwiseShiftRight


बिट सेट को दाएँ शिफ़्ट करता है.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

बिट <code>number2</code> की संख्या निर्दिष्ट करके, <code>number1</code> पर बिटवाइज़ दाएँ शिफ़्ट करें के निष्पादन का परिणाम लौटाता है.
