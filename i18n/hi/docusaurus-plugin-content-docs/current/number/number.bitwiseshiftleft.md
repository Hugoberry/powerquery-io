# Number.BitwiseShiftLeft


बिट सेट को बाएँ शिफ़्ट करता है.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Remarks

बिट <code>number2</code> की संख्या निर्दिष्ट करके, <code>number1</code> पर बिटवाइज़ बाएँ शिफ़्ट करें के निष्पादन का परिणाम लौटाता है.
