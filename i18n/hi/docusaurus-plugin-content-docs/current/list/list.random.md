# List.Random


यादृच्छिक संख्याओं की सूची लौटाता है.


## Syntax

```powerquery
List.Random(
    count as number,
    optional seed as number
) as list
```


## Remarks

0 और 1 के बीच यादृच्छिक संख्याओं की एक ऐसी सूची लौटाता है, जिसमें जनरेट किए जाने वाले मानों की संख्या और एक वैकल्पिक सीड मान दिया हुआ होता है.<ul>   <li><code>count</code>: जनरेट किए जाने वाले यादृच्छिक मानों की संख्या.</li>   <li><code>seed</code>:  <i>[वैकल्पिक]</i> यादृच्छिक संख्या जनरेटर को सीड करने के लिए एक सांख्यिक मान का उपयोग किया जाता है. यदि छोड़ दिया जाए, तो आप जितनी बार फ़ंक्शन को कॉल करते हैं, उतनी बार यादृच्छिक संख्याओं की एक अनन्य सूची जनरेट की जाती है. यदि आप किसी संख्या वाला सीड मान निर्दिष्ट करते हैं, तो फ़ंक्शन को की जाने वाली प्रत्येक कॉल यादृच्छिक संख्याओं की समान सूची जनरेट करती है.</li></ul>


## Examples

### Example #1 
3 यादृच्छिक संख्याओं की सूची बनाएँ.
```powerquery
List.Random(3)
```

Result: 
```powerquery
{0.992332, 0.132334, 0.023592}
```


### Example #2 
मूल मान निर्दिष्ट करते हुए, 3 यादृच्छिक संख्याओं की सूची बनाएँ.
```powerquery
List.Random(3, 2)
```

Result: 
```powerquery
{0.883002, 0.245344, 0.723212}
```
