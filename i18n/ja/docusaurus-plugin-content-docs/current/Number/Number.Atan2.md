# Number.Atan2


## Description

2 つの数値の間で除算を行った結果のアークタンジェントを返します。


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

2 つの数 <code>y</code> と <code>x</code> の間で除算を行った結果のアークタンジェントを返します。除算の式は <code>y</code>/<code>x</code> となります。



## Category
Number.Trigonometry
