# Number.Abs


## Description

数値の絶対値を返します。


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Details

<code>number</code> の絶対値を返します。<code>number</code> が null の場合、<code>Number.Abs</code> は null を返します。    <ul>        <li><code>number</code>: 絶対値を計算する対象の <code>number</code>。</li>      </ul>


## Examples

### Example #1 
-3 の絶対値。
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
