# Number.BitwiseShiftRight


ビットを右へシフトします。


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

指定されたビット数 <code>number2</code> のビットごとの右シフトを <code>number1</code> に対して実行した結果を返します。



## Category
Number.Bytes
