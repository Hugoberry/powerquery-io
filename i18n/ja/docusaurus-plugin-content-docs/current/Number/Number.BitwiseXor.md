# Number.BitwiseXor


## Description

2 つの入力間でビットごとの XOR (排他的 OR) 演算を行った結果を返します。


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

<code>number1</code> と <code>number2</code> の間でビットごとの XOR (排他的 OR) 演算を行った結果を返します。



## Category
Number.Bytes
