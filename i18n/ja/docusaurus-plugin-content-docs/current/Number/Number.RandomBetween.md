# Number.RandomBetween


## Description

2 つの数値の間の乱数を返します。


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

<code>bottom</code> と <code>top</code> の間の乱数を返します。


## Examples

### Example #1 
1 と 5 の間の乱数を取得します。
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
