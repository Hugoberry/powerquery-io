# Number.Random


## Description

乱数を返します。


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

0 と 1 の間の乱数を返します。


## Examples

### Example #1 
乱数を取得します。
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
