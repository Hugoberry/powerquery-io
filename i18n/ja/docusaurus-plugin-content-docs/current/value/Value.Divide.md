# Value.Divide


最初の値を 2 番目の値で除算した結果を返します。


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Remarks

<code>value1</code> を <code>value2</code> で除算した結果を返します。省略可能な <code>precision</code> パラメーターを指定できます。既定では <code>Precision.Double</code> が使用されます。
