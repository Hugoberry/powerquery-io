# Text.FromBinary


## Description

バイナリ形式のデータをテキストにデコードします。


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

<code>encoding</code> 型を使用して、バイナリ値のデータ (<code>binary</code>) をテキスト値にデコードします。



## Category
Text.Conversions from and to text
