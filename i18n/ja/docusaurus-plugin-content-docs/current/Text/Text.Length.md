# Text.Length


## Description

文字の数を返します。


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

テキスト <code>text</code> 内の文字の数を返します。


## Examples

### Example #1 
テキスト &#34;Hello World&#34; 内の文字の数を調べます。
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
