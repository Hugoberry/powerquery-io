# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

指定された <code>text</code> を反転します。


## Examples

### Example #1 
テキスト &#34;123&#34; を反転します。
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
