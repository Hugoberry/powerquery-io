# Text.ToBinary


テキストをバイナリ形式にエンコードします。


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Remarks

指定された <code>encoding</code> を使用して、指定されたテキスト値 (<code>text</code>) をバイナリ値にエンコードします。
