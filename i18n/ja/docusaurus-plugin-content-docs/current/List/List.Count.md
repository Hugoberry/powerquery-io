# List.Count


リスト内の項目の数を返します。


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

リスト <code>list</code> 内の項目の数を返します。


## Examples

### Example #1 
リスト \{1, 2, 3} 内の値の数を求めます。
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
