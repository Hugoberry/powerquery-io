# VSTS.Contents


Azure DevOps Analytics サービスの URL を入力します。


## Syntax

```powerquery
VSTS.Contents(
    url as text,
    optional options as record
) as binary
```


## Remarks

Azure DevOps Analytics サービス URL からダウンロードされたコンテンツをバイナリ値として返します。この関数は、アクセス可能なすべての組織でグローバルな資格情報を使用します。
