# BinaryFormat.7BitEncodedUnsignedInteger


7 ビットの可変長エンコーディングを使用してエンコードされた 64 ビット符号なし整数を読み取るバイナリ形式です。


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks

7 ビットの可変長エンコーディングを使用してエンコードされた 64 ビット符号なし整数を読み取るバイナリ形式です。
