# BinaryFormat.7BitEncodedSignedInteger


7 ビットの可変長エンコーディングを使用してエンコードされた 64 ビット符号付き整数を読み取るバイナリ形式です。


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

7 ビットの可変長エンコーディングを使用してエンコードされた 64 ビット符号付き整数を読み取るバイナリ形式です。
