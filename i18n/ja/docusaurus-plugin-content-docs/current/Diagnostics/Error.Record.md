# Error.Record


理由、メッセージ、および詳細の指定されたテキスト値からエラー レコードを返します。


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Remarks

理由、メッセージ、および詳細の指定されたテキスト値からエラー レコードを返します。
