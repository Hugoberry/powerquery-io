# DateTimeZone.FixedUtcNow


## Description

UTC (GMT タイム ゾーン) での現在の日付と時刻を返します。この値は固定されているため、連続呼び出しでは変更されません。


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
