# DateTimeZone.LocalNow


## Description

ローカル タイム ゾーンでの現在の日付と時刻を返します。


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

システムの現在の日付と時刻に設定された <code>datetimezone</code> 値を返します。    返された値には、ローカル タイム ゾーンを表すタイム ゾーン情報が含まれています。

DateTimeZone
