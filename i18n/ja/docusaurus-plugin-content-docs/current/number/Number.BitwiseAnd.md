# Number.BitwiseAnd


2 つの入力間でビットごとの AND 演算を行った結果を返します。


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> と <code>number2</code> の間でビットごとの AND 演算を行った結果を返します。

Number.Bytes
