# Number.Factorial


数値の階乗を返します。


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

数値 <code>number</code> の階乗を返します。


## Examples

### Example #1 
10 の階乗を求めます。
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
