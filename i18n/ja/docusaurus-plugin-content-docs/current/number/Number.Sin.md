# Number.Sin


数値のサインを返します。


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

<code>number</code> のサインを返します。


## Examples

### Example #1 
角度 0 のサインを求めます。
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
