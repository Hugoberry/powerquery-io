# Number.Tan


数値のタンジェントを返します。


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

<code>number</code> のタンジェントを返します。


## Examples

### Example #1 
角度 1 のタンジェントを求めます。
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
