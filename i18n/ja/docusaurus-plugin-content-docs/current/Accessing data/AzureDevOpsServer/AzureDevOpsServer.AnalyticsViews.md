# AzureDevOpsServer.AnalyticsViews


## Description

組織とプロジェクトの名前を入力します。


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Azure DevOps が提供する Analytics ビューのテーブルを返します。
