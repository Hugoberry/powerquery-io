# Cube.ApplyParameter


## Description

パラメーターを適用後、キューブを返します。


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

<code>arguments</code> を含む <code>parameter</code> を <code>cube</code> に適用後、キューブを返します。



## Category
Cube
