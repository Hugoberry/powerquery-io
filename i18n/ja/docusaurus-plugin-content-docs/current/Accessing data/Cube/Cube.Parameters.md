# Cube.Parameters


## Description

キューブに適用できるパラメーターのセットを含むテーブルを返します。


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Details

<code>cube</code> に適用できるパラメーターのセットを含むテーブルを返します。各パラメーターは、そのパラメーターとパラメーターの引数が適用されている <code>cube</code> を取得するために呼び出すことができる関数です。

Cube
