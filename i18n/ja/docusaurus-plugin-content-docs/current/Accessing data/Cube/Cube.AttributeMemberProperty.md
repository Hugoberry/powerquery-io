# Cube.AttributeMemberProperty


## Description

ディメンション属性のプロパティを返します。


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

ディメンション属性 <code>attribute</code> のプロパティ <code>propertyName</code> を返します。



## Category
Cube
