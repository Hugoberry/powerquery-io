# Cube.Measures


## Description

使用できるメジャーのセットを含むテーブルを返します。


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

<code>cube</code> 内の使用できるメジャーのセットを含むテーブルを返します。    それぞれのメジャーが関数として表されます。メジャーは、Cube.AddMeasureColumn を使用してキューブに適用できます。
