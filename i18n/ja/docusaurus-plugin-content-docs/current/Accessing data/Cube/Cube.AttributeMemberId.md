# Cube.AttributeMemberId


## Description

メンバー プロパティ値から一意のメンバー識別子を返します。


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

メンバー プロパティ値から一意のメンバー識別子を返します。<code>attribute</code>。その他の値に対しては null を返します。



## Category
Cube
