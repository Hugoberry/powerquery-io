# Cube.MeasureProperty


## Description

メジャー (セル プロパティ) のプロパティを返します。


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

メジャー <code>measure</code> のプロパティ <code>propertyName</code> を返します。



## Category
Cube
