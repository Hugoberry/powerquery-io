# Cube.DisplayFolders


オブジェクト (ディメンションやメジャーなど) の表示フォルダー階層を表す入れ子になったテーブルのツリーを返します。


## Syntax

```powerquery
Cube.DisplayFolders(
    cube as table
) as table
```


## Remarks

<code>cube</code> で使用できるオブジェクトの表示フォルダー階層 (ディメンションやメジャーなど) を表すテーブルの入れ子になったツリーを返します。
