# ActiveDirectory.Domains


## Description

指定されたドメイン (指定されなかった場合は現在のコンピューターのドメイン) と同じフォレスト内の Active Directory ドメインの一覧を返します。


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Details

指定されたドメイン (指定されなかった場合は現在のコンピューターのドメイン) と同じフォレスト内の Active Directory ドメインの一覧を返します。
