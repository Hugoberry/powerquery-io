# GoogleAnalytics.Accounts


## Description

Google Analytics アカウントを返します。


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

現在の資格情報からアクセス可能な Google Analytics アカウントを返します。
