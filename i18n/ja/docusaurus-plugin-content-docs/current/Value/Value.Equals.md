# Value.Equals


## Description

2 つの値が等しいかどうかを示す値を返します。


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

値 <code>value1</code> が値 <code>value2</code> と等しい場合は true、それ以外の場合は false を返します。



## Category
Values
