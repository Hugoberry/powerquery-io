# Value.Subtract


2 つの値の差を返します。


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Remarks

<code>value1</code> と <code>value2</code> の差を返します。省略可能な <code>precision</code> パラメーターを指定できます。既定では <code>Precision.Double</code> が使用されます。
