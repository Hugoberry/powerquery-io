# Value.NullableEquals


## Description

2 つの値が等しいかどうかを示す値を返します。


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

引数 '<code>value1</code>', '<code>value2</code>' が NULL であるか、Value.Equals に等しい場合は、NULL を返します。



## Category
Values
