# Value.Add


## Description

2 つの値の合計を返します。


## Syntax

```powerquery
Value.Add(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

<code>value1</code> と <code>value2</code> の合計を返します。省略可能な <code>precision</code> パラメーターを指定できます。既定では <code>Precision.Double</code> が使用されます。



## Category
Values.Arithmetic operations
