# Type.TableSchema


指定したテーブルの種類の列の説明 (つまりスキーマ) を含むテーブルを返します。


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Remarks

<code>tableType</code> の列を説明するテーブルを返します。<br />結果として返されるテーブルの説明については、<code>Table.Schema</code> のドキュメントを参照してださい。<br />
