# BinaryFormat.ByteOrder


## Description

関数で指定されたバイト順でバイナリ形式を返します。


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

<code>binaryFormat</code> で指定されたバイト順でバイナリ形式を返します。既定のバイト順は <code>ByteOrder.BigEndian</code> です。
