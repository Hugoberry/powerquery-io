# Table.FirstValue


## Description

傳回資料表第一個資料列的第一個資料行，或傳回指定的預設值。


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

傳回資料表 <code>table</code> 第一個資料列的第一個資料行，或傳回指定的預設值。



## Category
Table.Row operations
