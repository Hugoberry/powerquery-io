# Tables.GetRelationships


取得一組資料表間的關聯性。


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Remarks

取得一組資料表間的關聯性。集合 <code>tables</code> 假設為具有類似於導覽資料表的結構。由 <code>dataColumn</code> 所定義的資料行包含實際的資料表。
