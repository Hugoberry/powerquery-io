# Table.ViewError


在檢視定義的處理常式透過 [表格檢視]建立已修改的錯誤記錄擲回時，不會觸發後援。


## Syntax

```powerquery
Table.ViewError(
    errorRecord as record
) as record
```


## Remarks

從 <code>errorRecord</code> 在檢視定義的處理常式透過 [表格檢視]建立已修改的錯誤記錄擲回時，不會觸發後援。
