# Text.FromBinary


## Description

將二進位格式的資料解碼成文字。


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

使用 <code>encoding</code> 類型，將二進位值的資料 <code>binary</code> 解碼成文字值。
