# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

反轉提供的 <code>text</code>。


## Examples

### Example #1 
反轉文字 &#34;123&#34;。
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
