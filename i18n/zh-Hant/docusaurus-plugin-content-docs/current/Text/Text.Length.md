# Text.Length


傳回字元數。


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

傳回文字 <code>text</code> 中的字元數。


## Examples

### Example #1 
找出文字 &#34;Hello World&#34; 有多少個字元。
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
