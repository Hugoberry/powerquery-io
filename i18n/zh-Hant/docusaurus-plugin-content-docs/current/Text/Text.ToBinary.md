# Text.ToBinary


## Description

將文字編碼成二進位格式。


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

使用指定的 <code>encoding</code>，將指定的文字值 <code>text</code> 編碼成二進位值。
