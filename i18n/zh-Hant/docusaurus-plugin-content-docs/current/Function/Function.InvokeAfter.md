# Function.InvokeAfter


## Description

在達到指定的持續時間之後，叫用指定的函數。


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

經過持續時間 <code>delay</code> 之後，傳回叫用 <code>function</code> 的結果。



## Category
Function
