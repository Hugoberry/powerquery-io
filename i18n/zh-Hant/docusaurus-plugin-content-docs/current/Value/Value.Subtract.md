# Value.Subtract


傳回兩個值的差額。


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Remarks

傳回 <code>value1</code> 與 <code>value2</code> 的差額。可指定選擇性 <code>precision</code> 參數，預設使用 <code>Precision.Double</code>。
