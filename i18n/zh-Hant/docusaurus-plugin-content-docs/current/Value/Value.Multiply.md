# Value.Multiply


## Description

傳回兩個值的乘積。


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

傳回 <code>value1</code> 乘以 <code>value2</code> 的乘積。可指定選擇性 <code>precision</code> 參數，預設使用 <code>Precision.Double</code>。



## Category
Values.Arithmetic operations
