# Number.RandomBetween


傳回兩個數字之間的亂數。


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Remarks

傳回 <code>bottom</code> 到 <code>top</code> 之間的亂數。


## Examples

### Example #1 
取得 1 到 5 之間的亂數。
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
