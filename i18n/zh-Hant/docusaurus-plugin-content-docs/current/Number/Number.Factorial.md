# Number.Factorial


傳回數字的階乘。


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

傳回數字 <code>number</code> 的階乘。


## Examples

### Example #1 
找出 10 的階乘。
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
