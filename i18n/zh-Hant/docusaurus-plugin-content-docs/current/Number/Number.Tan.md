# Number.Tan


傳回數字的正切值。


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

傳回 <code>number</code> 的正切值。


## Examples

### Example #1 
找出角度 1 的正切值。
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
