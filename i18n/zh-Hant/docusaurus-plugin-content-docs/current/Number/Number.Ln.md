# Number.Ln


傳回數字的自然對數。


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Remarks

傳回數字 <code>number</code> 的自然對數。如果 <code>number</code> 為 Null，<code>Number.Ln</code> 會傳回 Null。


## Examples

### Example #1 
取得 15 的自然對數。
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
