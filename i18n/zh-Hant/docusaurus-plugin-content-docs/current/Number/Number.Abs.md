# Number.Abs


傳回數字的絕對值。


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Remarks

傳回 <code>number</code> 的絕對值。如果 <code>number</code> 是 Null，<code>Number.Abs</code> 會傳回 Null。    <ul>        <li><code>number</code>: 要計算其絕對值的 <code>number</code>。</li>      </ul>


## Examples

### Example #1 
-3 的絕對值。
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
