# Number.Atan2


## Description

傳回兩個數字相除的反正切值。


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

傳回 <code>y</code> 和 <code>x</code> 這兩個數字相除的反正切值。除式列為 <code>y</code>/<code>x</code>。



## Category
Number.Trigonometry
