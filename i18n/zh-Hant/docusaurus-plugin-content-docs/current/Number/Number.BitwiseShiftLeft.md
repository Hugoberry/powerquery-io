# Number.BitwiseShiftLeft


## Description

將一組位元向左移。


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

傳回對 <code>number1</code> 執行位元左移指定位元數 <code>number2</code> 的結果。



## Category
Number.Bytes
