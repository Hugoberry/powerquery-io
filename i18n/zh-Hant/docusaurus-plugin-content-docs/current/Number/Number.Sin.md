# Number.Sin


傳回數字的正弦值。


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

傳回 <code>number</code> 的正弦值。


## Examples

### Example #1 
找出角度 0 的正弦值。
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
