# Number.IsEven


## Description

指出值是否為偶數。


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

指出值 <code>number</code> 是否為偶數，如果是偶數，傳回 <code>true</code>，否則傳回 <code>false</code>。


## Examples

### Example #1 
檢查 625 是否為偶數。
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
檢查 82 是否為偶數。
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
