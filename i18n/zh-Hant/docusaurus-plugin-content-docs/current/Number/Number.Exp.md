# Number.Exp


## Description

求 e 的指定次方。


## Syntax

```powerquery
Number.Exp(
    number as number
) as number
```


## Details

傳回 e 的 <code>number</code> 次方 (指數函數) 結果。      <ul>        <li><code>number</code>: 要計算其指數函數的 <code>number</code>。如果 <code>number</code> 是 Null，<code>Number.Exp</code> 會傳回 Null。 </li>      </ul>


## Examples

### Example #1 
取 e 的 3 次方。
```powerquery
Number.Exp(3)
```

Result: 
```powerquery
20.085536923187668
```
