# Number.Cos


傳回數字的餘弦值。


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

傳回 <code>number</code> 的餘弦值。


## Examples

### Example #1 
找出角度 0 的餘弦值。
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
