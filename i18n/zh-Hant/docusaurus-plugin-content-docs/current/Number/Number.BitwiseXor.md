# Number.BitwiseXor


傳回在兩個輸入之間執行位元 &#34;XOR&#34; (互斥 OR) 的結果。


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Remarks

傳回在 <code>number1</code> 和 <code>number2</code> 之間執行位元 "XOR" (互斥 OR) 的結果。



## Category
Number.Bytes
