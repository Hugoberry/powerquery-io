# Error.Record


## Description

根據提供的文字值傳回錯誤記錄，以取得原因、訊息和詳細資料。


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
