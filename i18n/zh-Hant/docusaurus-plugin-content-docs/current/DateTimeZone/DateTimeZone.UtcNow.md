# DateTimeZone.UtcNow


傳回 UTC (GMT 時區) 的目前日期和時間。


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

傳回 UTC (GMT 時區) 的目前日期和時間。


## Examples

### Example #1 
取得 UTC 的目前日期和時間。
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
