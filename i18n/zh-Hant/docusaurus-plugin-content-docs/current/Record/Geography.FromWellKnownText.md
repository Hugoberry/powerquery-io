# Geography.FromWellKnownText


## Description

將採用 Well-Known Text (WKT) 格式且代表地理值的文字轉譯成結構化記錄。


## Syntax

```powerquery
Geography.FromWellKnownText(
    input as text
) as record
```


## Details

將採用 Well-Known Text (WKT) 格式且代表地理值的文字轉譯成結構化記錄。WKT 是開放地理空間協會 (OGC) 所定義的標準格式，是眾多資料庫 (包括 SQL Server 在內) 使用的一般序列化格式。
