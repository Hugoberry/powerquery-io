# Exchange.Contents


傳回來自 Microsoft Exchange 帳戶的目錄。


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Remarks

從 Microsoft Exchange 帳戶 <code>mailboxAddress</code> 傳回內容的資料表。如果未指定 <code>mailboxAddress</code>，將會使用認證的預設帳戶。
