# Hdfs.Contents


## Description

傳回表格，內含在 Hadoop 檔案系統的指定資料夾中找到的檔案和資料夾的屬性和內容。


## Syntax

```powerquery
Hdfs.Contents(
    url as text
) as table
```


## Details

傳回表格，內含在 Hadoop 檔案系統的資料夾 URL <code>url</code> 找到的每個資料夾和檔案各一列。每一列包含資料夾或檔案的屬性，以及其內容的連結。
