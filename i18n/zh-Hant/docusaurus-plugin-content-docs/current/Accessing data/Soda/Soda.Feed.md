# Soda.Feed


## Description

傳回內容中的資料表，此內容位於根據 SODA 2.0 API 格式化的指定 URL。


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Details

傳回內容中的資料表，此內容位於根據 SODA 2.0 API 格式化的指定 URL <code>url</code>。URL 必須指向以 .csv 副檔名結尾、與 SODA 相容的有效來源。
