# ActiveDirectory.Domains


## Description

傳回與指定網域 (如果未指定網域，則使用目前電腦的網域) 相同樹系的 Active Directory 網域清單。


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Details

傳回與指定網域 (如果未指定網域，則使用目前電腦的網域) 相同樹系的 Active Directory 網域清單。
