# Cube.ReplaceDimensions


## Description

取代 Cube.Dimensions 傳回的維度集合。


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Details

取代 <code>Cube.Dimensions</code> 傳回的維度集合。    例如，此函式可用來將識別碼資料行新增至維度屬性，讓資料來源可以依識別碼分組，而非依顯示的值。
