# Cube.AttributeMemberProperty


傳回維度屬性的屬性。


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

傳回維度屬性 <code>attribute</code> 的屬性 <code>propertyName</code>。



## Category
Cube
