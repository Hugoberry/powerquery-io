# AzureDevOpsServer.AnalyticsViews


## Description

輸入組織和專案名稱。


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

傳回 Azure DevOps 提供的 Analytics 檢視資料表。
