# AzureDevOpsServer.Views


## Description

輸入組織和專案名稱。


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

傳回 Azure DevOps 提供的 OData 摘要資料表。
