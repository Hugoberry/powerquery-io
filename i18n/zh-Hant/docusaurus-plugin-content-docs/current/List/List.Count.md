# List.Count


## Description

傳回清單中的項目數。


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

傳回清單 <code>list</code> 中的項目數。


## Examples

### Example #1 
找出清單 \{1, 2, 3} 中值的數目。
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
