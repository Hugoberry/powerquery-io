# Type.TableSchema


## Description

傳回包含指定資料表類型之資料行 (例如結構描述) 描述的資料表。


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

傳回描述 <code>tableType</code> 之資料行的資料表。<br />如需結果資料表的描述，請參閱 <code>Table.Schema</code> 的文件。<br />
