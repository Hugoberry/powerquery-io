# Binary.ViewError


建立已修改的錯誤記錄，在由檢視表定義的處理常式 (透過 Binary.View) 擲回時，不會觸發後援。


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Remarks

從 <code>errorRecord</code> 建立已修改的錯誤記錄，在由檢視表定義的處理常式 (透過 Binary.View) 擲回時，不會觸發後援。
