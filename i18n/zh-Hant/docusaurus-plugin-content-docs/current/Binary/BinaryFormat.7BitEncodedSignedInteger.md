# BinaryFormat.7BitEncodedSignedInteger


一種二進位格式，可讀取使用 7 位元可變長度編碼方式編碼的 64 位元帶正負號的整數。


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

一種二進位格式，可讀取使用 7 位元可變長度編碼方式編碼的 64 位元帶正負號的整數。
