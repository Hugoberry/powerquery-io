# BinaryFormat.ByteOrder


## Description

傳回具有函數所指定位元組順序的二進位格式。


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

傳回具有 <code>binaryFormat</code> 所指定位元組順序的二進位格式。預設位元組順序是 <code>ByteOrder.BigEndian</code>。
