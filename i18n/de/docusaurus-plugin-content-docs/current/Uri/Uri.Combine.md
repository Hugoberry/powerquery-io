# Uri.Combine


Gibt den absoluten URI (Kombination des Eingabebasis-URI und des relativen URI) zurück.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Remarks

Gibt den absoluten URI (Kombination aus der Eingabe "<code>baseUri</code>" und "<code>relativeUri</code>") zurück.
