# List.NonNullCount


Gibt die Anzahl von Elementen ungleich NULL in der Liste zurück.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Gibt die Anzahl von Elementen ungleich NULL in der Liste "<code>list</code>" zurück.

List.Information
