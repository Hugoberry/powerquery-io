# Value.Compare


Gibt -1, 0 oder 1 zurück, je nachdem, ob der erste Wert im Vergleich zum zweiten kleiner, gleich oder größer ist.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```


## Remarks

Gibt -1, 0 oder 1 zurück, je nachdem, ob der erste Wert im Vergleich zum zweiten kleiner, gleich oder größer ist.
