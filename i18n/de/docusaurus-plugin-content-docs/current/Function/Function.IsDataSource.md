# Function.IsDataSource


## Description

Gibt an, ob eine bestimmte Funktion als Datenquelle betrachtet wird oder nicht.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Gibt an, ob "<code>function</code>" als Datenquelle betrachtet wird oder nicht.

Type
