# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Kehrt die Angabe von "<code>text</code>" um.


## Examples

### Example #1 
Kehrt den Text &#34;123&#34; um.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
