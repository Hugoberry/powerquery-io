# DateTime.LocalNow


## Description

Gibt das aktuelle Datum und die aktuelle Uhrzeit in der lokalen Zeitzone zurück.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Gibt einen <code>datetime</code>-Wert zurück, der auf das aktuelle Datum und die aktuelle Uhrzeit im System festgelegt ist.



## Category
DateTime
