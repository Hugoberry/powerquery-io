# Text.Length


Gibt die Zeichenanzahl zurück.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

Gibt die Anzahl von Zeichen im Text "<code>text</code>" zurück.


## Examples

### Example #1 
Ermittelt die Anzahl von Zeichen im Text &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
