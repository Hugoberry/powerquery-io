# Value.Equals


## Description

Gibt zurück, ob zwei Werte gleich sind.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Gibt "true" zurück, wenn der Wert <code>value1</code> gleich dem Wert <code>value2</code> ist. Andernfalls wird "false" zurückgegeben.



## Category
Values
