# Cube.MeasureProperty


Gibt eine Eigenschaft eines Measures (Zelleigenschaft) zurück.


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Remarks

Gibt die Eigenschaft "<code>propertyName</code>" des Measures "<code>measure</code>" zurück.



## Category
Cube
