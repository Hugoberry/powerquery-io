# Cube.Transform


## Description

Wendet eine Liste von Cubefunktionen an.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Wendet die Liste der Cubefunktionen '<code>transforms</code>' auf '<code>cube</code>' an.



## Category
Cube
