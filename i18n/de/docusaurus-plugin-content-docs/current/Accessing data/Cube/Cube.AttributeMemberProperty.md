# Cube.AttributeMemberProperty


Gibt eine Eigenschaft eines Dimensionsattributs zurück.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

Gibt die Eigenschaft "<code>propertyName</code>" des Dimensionsattributs "<code>attribute</code>" zurück.



## Category
Cube
