# AzureDevOpsServer.Views


## Description

Hiermit werden die Organisations- und Projektnamen eingegeben.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Hiermit wird eine Tabelle der über Azure DevOps bereitgestellten OData-Feeds zurückgegeben.
