# AzureDevOpsServer.Feed


## Description

Azure DevOps Services-Feed


## Syntax

```powerquery
AzureDevOpsServer.Feed(
    url as text,
    optional options as record
) as table
```


## Details

Hiermit wird eine Tabelle der über Azure DevOps bereitgestellten OData-Feeds zurückgegeben.
