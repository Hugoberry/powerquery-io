# GoogleAnalytics.Accounts


## Description

Gibt Google Analytics-Konten zurück.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

Gibt Google Analytics-Konten zurück, auf die mit den aktuellen Anmeldeinformationen zugegriffen werden kann.

Accessing data
