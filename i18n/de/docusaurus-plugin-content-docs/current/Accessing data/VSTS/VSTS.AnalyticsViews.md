# VSTS.AnalyticsViews


Hiermit werden die Organisations- und Projektnamen eingegeben.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Gibt eine Tabelle der Analytics-Ansichten zurück, die über Azure DevOps bereitgestellt werden.
