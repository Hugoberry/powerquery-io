# Twilio.Tables


## Description

Geben Sie die Anzahl von Monaten ein, für die Twilio-Verlaufsdaten abgerufen werden sollen.


## Syntax

```powerquery
Twilio.Tables(
    historyMonths as number
) as table
```


## Details

Gibt eine Tabelle mit relevanten Twilio-Daten zurück.
