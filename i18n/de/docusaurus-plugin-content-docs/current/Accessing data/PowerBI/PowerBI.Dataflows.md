# PowerBI.Dataflows



## Syntax

```powerquery
PowerBI.Dataflows(
    optional options as record
) as table
```


## Details

Stellen Sie eine Verbindung mit allen Power BI-Dataflows her, auf die Sie zugreifen können, und wählen Sie die Entitäten, die Sie verwenden möchten.
