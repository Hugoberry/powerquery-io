# Entdecken Sie die Sprache
Die Power Query M Formelsprache ist optimiert für den Aufbau von hochflexiblen Daten-Mashup-Abfragen. Es handelt sich um eine funktionale, groß- und kleinschreibungssensitive Sprache, ähnlich wie F#. Die Power Query Formelsprache wird in einer Reihe von Microsoft-Produkten wie Power BI Desktop, Excel und Analysis Services verwendet.

Diese Website dient als Referenz für Power Query-Funktionen und bietet schnellen und direkten Zugang zu detaillierten Informationen und praktischen Beispielen für jede Funktion.
:::note
Bitte beachten Sie, dass der Autor keine Urheberrechte an den auf dieser Website präsentierten Materialien beansprucht. Die Informationen werden gesammelt und bereitgestellt in der Hoffnung, dass sie für diejenigen nützlich sein werden, die ihr Wissen und ihre Anwendung von Power Query erweitern möchten.
:::

Die Suchfunktionalität auf dieser Website sollte den Entdeckungsprozess der breiten Palette von Power Query-Funktionen vereinfachen. Stellen Sie sicher, dass Sie sie ausprobieren (Strg+K).

![Suchfunktionalität bereitgestellt von Algolia](/img/search.png)

Für diejenigen, die an einer tieferen Erkundung oder Implementierungsdetails interessiert sind, können weitere Informationen im folgenden GitHub-Repository gefunden werden: [https://github.com/Hugoberry/powerquery-io](https://github.com/Hugoberry/powerquery-io). Dieses Repository kann zusätzliche Beispiele, Updates und Ressourcen enthalten, die für die Entwicklung und Nutzung von Power Query relevant sind.
