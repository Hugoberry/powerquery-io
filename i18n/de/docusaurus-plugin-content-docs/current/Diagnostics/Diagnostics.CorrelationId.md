# Diagnostics.CorrelationId


## Description

Gibt einen nicht transparenten Bezeichner zurück, um eingehende Anforderungen mit ausgehenden Anforderungen zu korrelieren.


## Syntax

```powerquery
Diagnostics.CorrelationId(

) as text
```
