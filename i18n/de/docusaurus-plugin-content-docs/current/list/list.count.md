# List.Count


Gibt die Anzahl von Elementen in der Liste zurück.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Gibt die Anzahl von Elementen in der Liste "<code>list</code>" zurück.


## Examples

### Example #1 
Ermittelt die Anzahl von Werten in der Liste &#34;\{1, 2, 3}&#34;.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
