# Number.Random


Gibt eine Zufallszahl zurück.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Gibt eine Zufallszahl zwischen 0 und 1 zurück.


## Examples

### Example #1 
Ruft eine Zufallszahl ab.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
