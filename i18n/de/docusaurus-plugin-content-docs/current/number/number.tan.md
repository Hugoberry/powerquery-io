# Number.Tan


Gibt den Tangens der Zahl zurück.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

Gibt den Tangens von <code>number</code> zurück.


## Examples

### Example #1 
Ermittelt den Tangens des Winkels 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
