# Number.Sin


## Description

Gibt den Sinus der Zahl zurück.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Gibt den Sinus von <code>number</code> zurück.


## Examples

### Example #1 
Ermittelt den Sinus des Winkels 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
