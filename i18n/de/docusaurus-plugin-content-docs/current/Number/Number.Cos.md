# Number.Cos


Gibt den Kosinus der Zahl zurück.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Gibt den Kosinus von <code>number</code> zurück.


## Examples

### Example #1 
Ermittelt den Kosinus des Winkels 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
