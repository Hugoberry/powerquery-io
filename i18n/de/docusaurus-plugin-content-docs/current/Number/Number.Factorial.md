# Number.Factorial


## Description

Gibt die Fakultät der Zahl zurück.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Gibt die Fakultät der Zahl <code>number</code> zurück.


## Examples

### Example #1 
Ermittelt die Fakultät von 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
