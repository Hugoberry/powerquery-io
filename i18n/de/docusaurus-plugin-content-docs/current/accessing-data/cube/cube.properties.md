# Cube.Properties


Gibt eine Tabelle mit dem Satz der verfügbaren Eigenschaften für Dimensionen zurück, die im Cube erweitert sind.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```


## Remarks
