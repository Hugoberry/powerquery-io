# DynamicsNav.Contents


Geben Sie die URL für Ihren Dynamics NAV-OData-Dienstendpunkt ein.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

Gibt eine Tabelle mit relevanten Dynamics NAV-Daten zurück.
