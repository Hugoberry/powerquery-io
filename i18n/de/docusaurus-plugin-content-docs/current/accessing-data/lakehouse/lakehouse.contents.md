# Lakehouse.Contents


Daten aus einem Lakehouse importieren


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Remarks

Stellen Sie eine Verbindung mit allen Lakehouses her, auf die Sie zugreifen können, und wählen Sie die Entitäten, die Sie verwenden möchten
