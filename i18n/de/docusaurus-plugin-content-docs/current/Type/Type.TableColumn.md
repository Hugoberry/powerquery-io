# Type.TableColumn


## Description

Gibt den Typ einer Spalte in einer Tabelle zurück.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Gibt den Typ der Spalte '<code>column</code>' in der Tabelle vom Typ '<code>tableType</code>' zurück.



## Category
Type
