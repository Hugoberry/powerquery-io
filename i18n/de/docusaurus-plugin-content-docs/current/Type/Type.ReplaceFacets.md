# Type.ReplaceFacets


## Description

Ersetzt die Facets eines Typs.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Ersetzt die Facets von <code>type</code> durch die Facets, die im Datensatz <code>facets</code> enthalten sind.



## Category
Type
