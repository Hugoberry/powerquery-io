# DateTimeZone.UtcNow


Gibt das aktuelle Datum und die aktuelle Uhrzeit in koordinierter Weltzeit (GMT-Zeitzone) zurück.


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

Gibt das aktuelle Datum und die aktuelle Uhrzeit in koordinierter Weltzeit (GMT-Zeitzone) zurück.


## Examples

### Example #1 
Ruft das aktuelle Datum und die aktuelle Uhrzeit in UTC ab.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
