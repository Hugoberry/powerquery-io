# DateTimeZone.UtcNow


## Description

Az aktuális dátumot és időt adja vissza az UTC (egyezményes világidő) időzónában.


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Az aktuális dátumot és időt adja vissza az UTC (egyezményes világidő) időzónában.


## Examples

### Example #1 
Az aktuális dátum és idő lekérése az UTC időzónában
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
