# Type.TableColumn


## Description

Egy táblában szereplő oszlop típusát adja vissza.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

A(z) <code>tableType</code> táblatípusban szereplő <code>column</code> oszlop típusát adja vissza.

Type
