# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Megfordítja a megadott <code>text</code> elemet.


## Examples

### Example #1 
Az „123” szöveg megfordítása.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
