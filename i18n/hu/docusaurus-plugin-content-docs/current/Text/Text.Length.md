# Text.Length


## Description

Visszaadja a karakterek számát.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

A(z) <code>text</code> szövegben található karakterek számát adja vissza.


## Examples

### Example #1 
Megállapítja, hogy hány karakter szerepel a „Hello World” szövegben.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
