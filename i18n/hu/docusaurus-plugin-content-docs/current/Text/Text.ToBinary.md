# Text.ToBinary


## Description

Bináris formában kódolja a szöveget.


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

Bináris értékké kódolja a megadott <code>text</code> szöveges értéket a megadott <code>encoding</code> használatával.
