# Number.BitwiseOr


## Description

A két bemenet közötti bitenkénti „Vagy” művelet eredményét adja vissza.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

A(z) <code>number1</code> és <code>number2</code> közötti bitenkénti „Vagy” művelet eredményét adja vissza.

Number.Bytes
