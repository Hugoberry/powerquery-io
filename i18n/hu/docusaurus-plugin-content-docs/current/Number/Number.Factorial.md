# Number.Factorial


## Description

Visszaadja a szám faktoriálisát.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

A(z) <code>number</code> szám faktoriálisát adja vissza.


## Examples

### Example #1 
A 10 faktoriálisának meghatározása
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
