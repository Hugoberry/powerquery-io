# Number.BitwiseXor


## Description

A két bemenet közötti bitenkénti „XOR” (kizáró vagy) művelet eredményét adja vissza.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

A(z) <code>number1</code> és <code>number2</code> közötti bitenkénti „XOR” (kizáró vagy) művelet eredményét adja vissza.



## Category
Number.Bytes
