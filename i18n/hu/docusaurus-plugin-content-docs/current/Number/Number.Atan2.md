# Number.Atan2


## Description

A két szám hányadosának arkusz tangensét adja vissza.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

A két szám – <code>y</code> és <code>x</code> – hányadosának arkusz tangensét adja vissza. A hányados <code>y</code>/<code>x</code> alakú lesz.
