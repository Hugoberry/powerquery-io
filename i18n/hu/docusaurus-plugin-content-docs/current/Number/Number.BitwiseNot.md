# Number.BitwiseNot


## Description

Egy olyan bájtot ad vissza, amelyben mindegyik bit ellentétes a bemenetben szereplővel.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

A(z) <code>number</code> értéken elvégzett bitenkénti „Negálás” művelet eredményét adja vissza.



## Category
Number.Bytes
