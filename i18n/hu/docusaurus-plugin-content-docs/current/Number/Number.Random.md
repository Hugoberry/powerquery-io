# Number.Random


## Description

Egy véletlenszerű számot ad vissza.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Egy 0 és 1 közötti véletlenszerű számot ad vissza.


## Examples

### Example #1 
Egy véletlenszerű szám lekérése
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
