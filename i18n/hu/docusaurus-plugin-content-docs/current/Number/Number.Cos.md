# Number.Cos


## Description

Visszaadja a szám koszinuszát.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

A(z) <code>number</code> koszinuszát adja vissza.


## Examples

### Example #1 
A 0 radiános szög koszinuszának meghatározása
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
