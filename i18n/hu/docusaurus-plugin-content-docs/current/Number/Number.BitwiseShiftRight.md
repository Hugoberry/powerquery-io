# Number.BitwiseShiftRight


## Description

Jobbra eltolja a bitkészletet.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

A(z) <code>number1</code> értéken elvégzett, a megadott <code>number2</code> bittel való bitenkénti jobbra eltolási művelet eredményét adja vissza.

Number.Bytes
