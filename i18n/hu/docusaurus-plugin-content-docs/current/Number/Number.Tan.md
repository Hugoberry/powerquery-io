# Number.Tan


## Description

Visszaadja a szám tangensét.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

A(z) <code>number</code> tangensét adja vissza.


## Examples

### Example #1 
Az 1 radiános szög tangensének meghatározása
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
