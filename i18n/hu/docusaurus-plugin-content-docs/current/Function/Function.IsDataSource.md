# Function.IsDataSource


## Description

Visszaadja, hogy az adott függvény adatforrásnak számít-e.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Visszaadja, hogy a(z) <code>function</code> adatforrásnak számít-e.

Type
