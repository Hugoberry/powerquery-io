# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Bináris formátum, amely 7 bites változó hosszúságú kódolás használatával kódolt 64 bites, előjel nélküli egész számokat olvas.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
