# BinaryFormat.7BitEncodedSignedInteger


## Description

Bináris formátum, amely 7 bites változó hosszúságú kódolás használatával kódolt 64 bites, előjeles egész számokat olvas.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
