# Error.Record


## Description

Egy hibarekordot ad vissza az ok, üzenet és részletek megadott szöveges értékeiből.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
