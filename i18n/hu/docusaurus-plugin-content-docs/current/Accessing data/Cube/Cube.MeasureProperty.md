# Cube.MeasureProperty


## Description

Egy mérték egyik tulajdonságát adja vissza (cellatulajdonság).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

A(z) <code>measure</code> mérték <code>propertyName</code> tulajdonságát adja vissza.



## Category
Cube
