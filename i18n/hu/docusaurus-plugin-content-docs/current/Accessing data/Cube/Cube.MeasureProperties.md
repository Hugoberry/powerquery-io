# Cube.MeasureProperties


## Description

Egy táblát ad vissza, amely a kockában kibontott mértékek elérhető tulajdonságainak készletét tartalmazza.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```
