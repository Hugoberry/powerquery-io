# Cube.Transform


## Description

Kockafüggvények listáját alkalmazza.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

A kockafüggvények listáját (<code>transforms</code>) alkalmazza a következőre: <code>cube</code>.

Cube
