# Web.Page


## Description

A HTML-dokumentum tartalmát adja vissza a dokumentumot alkotó struktúrákra lebontva, valamint a teljes dokumentumot és annak a címkék eltávolításával előállított szövegét jelölő objektumot is visszaad.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
