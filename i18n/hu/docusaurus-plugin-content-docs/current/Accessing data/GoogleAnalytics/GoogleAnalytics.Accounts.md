# GoogleAnalytics.Accounts


## Description

Google Analytics-fiókokat ad vissza.


## Syntax

```powerquery

    optional options as record
) as table
```


## Details

Olyan Google Analytics-fiókokat ad vissza, amelyek elérhetők a jelenlegi hitelesítő adatokkal.
