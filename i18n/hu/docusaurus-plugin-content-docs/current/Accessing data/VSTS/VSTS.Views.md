# VSTS.Views


## Description

Adja meg a vállalat- és projektneveket.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Az Azure DevOps által kínált OData-csatornák tábláját adja vissza.
