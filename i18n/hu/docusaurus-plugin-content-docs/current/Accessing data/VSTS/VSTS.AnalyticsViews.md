# VSTS.AnalyticsViews


## Description

Adja meg a vállalat- és projektneveket.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Az Azure DevOps által kínált Analytics-nézetek tábláját adja vissza.
