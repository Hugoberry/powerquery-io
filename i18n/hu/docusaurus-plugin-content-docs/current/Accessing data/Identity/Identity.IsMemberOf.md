# Identity.IsMemberOf


## Description

Meghatározza, hogy egy identitás egy identitásgyűjtemény tagja-e.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details

Meghatározza, hogy egy identitás egy identitásgyűjtemény tagja-e.
