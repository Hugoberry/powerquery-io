# DynamicsNav.Contents


## Description

Adja meg a Dynamics NAV OData-szolgáltatásvégpontjának URL-címét.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

A vonatkozó Dynamics NAV-adatokat tartalmazó táblát adja vissza.
