# Marketo.Tables


## Description

Adja meg a fiókjával társított Marketo REST API-végpont URL-címét.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

A vonatkozó Marketo-adatokat tartalmazó táblát adja vissza.
