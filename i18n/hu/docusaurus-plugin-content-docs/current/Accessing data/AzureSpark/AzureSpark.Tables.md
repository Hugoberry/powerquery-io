# AzureSpark.Tables


## Description

Azure Spark-példányban lévő táblák listázása.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Details

Azure Spark-példányban lévő táblák listázása.


## Examples

### Example #1 
Azure Spark-példányban lévő táblák listázása.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
