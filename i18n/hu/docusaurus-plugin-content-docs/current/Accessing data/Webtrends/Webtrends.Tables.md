# Webtrends.Tables


## Description

Adja meg a Webtrends-fiókjával társított profilazonosítót.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

A vonatkozó Webtrends-adatokat tartalmazó táblát adja vissza.
