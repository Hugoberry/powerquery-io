# List.Count


## Description

A listán szereplő elemek számát adja vissza.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

A(z) <code>list</code> listán szereplő elemek számát adja vissza.


## Examples

### Example #1 
Az \{1, 2, 3} listán levő értékek számának meghatározása
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
