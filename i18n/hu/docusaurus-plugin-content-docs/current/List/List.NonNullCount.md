# List.NonNullCount


## Description

A listán szereplő nem null értékű elemek számát adja vissza.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

A(z) <code>list</code> listán szereplő nem null értékű elemek számát adja vissza.

List.Information
