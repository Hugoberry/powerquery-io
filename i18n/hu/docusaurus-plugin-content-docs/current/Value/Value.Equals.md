# Value.Equals


## Description

Azt adja vissza, hogy két érték egyenlő-e.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Igaz értéket ad vissza, ha a(z) <code>value1</code> érték egyenlő a(z) <code>value2</code> értékkel, egyéb esetben pedig hamis értéket ad vissza.



## Category
Values
