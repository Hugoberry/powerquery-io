# Value.Divide


## Description

Az első érték második értékkel való osztásának eredményét adja vissza.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

A(z) <code>value1</code> érték <code>value2</code> értékkel való osztásának eredményét adja vissza. Megadható egy nem kötelező <code>precision</code> paraméter, a függvény alapértelmezés szerint a <code>Precision.Double</code> beállítást használja.
