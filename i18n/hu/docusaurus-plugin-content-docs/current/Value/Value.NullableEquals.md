# Value.NullableEquals


## Description

Azt adja vissza, hogy két érték egyenlő-e.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Null értéket ad vissza, ha a(z) „<code>value1</code>” és „<code>value2</code>” argumentumok bármelyike null értékű, egyéb esetben pedig a Value.Equals értékét adja vissza.
