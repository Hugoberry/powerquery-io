# Value.Alternates


## Description

Alternatív lekérdezésterveket fejez ki.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Alternatív lekérdezésterveket fejez ki a Value.Expression(Value.Optimize(...)) függvényből szerzett lekérdezésterv-kifejezésben. Más felhasználásra nem ajánlott.



## Category
Expression
