# Value.Subtract


## Description

A két érték különbségét adja vissza.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

A(z) <code>value1</code> és <code>value2</code> értékek különbségét adja vissza. Megadható egy nem kötelező <code>precision</code> paraméter, a függvény alapértelmezés szerint a <code>Precision.Double</code> beállítást használja.
