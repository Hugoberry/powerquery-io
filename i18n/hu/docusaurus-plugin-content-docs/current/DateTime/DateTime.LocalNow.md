# DateTime.LocalNow


## Description

Az aktuális dátumot és időt adja vissza a helyi időzónában.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Egy olyan <code>datetime</code> típusú értéket ad vissza, amely a rendszer aktuális dátumát és idejét tartalmazza.

DateTime
