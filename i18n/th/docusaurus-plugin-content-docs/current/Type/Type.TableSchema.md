# Type.TableSchema


## Description

ส่งคืนค่าตารางที่ประกอบด้วยคำอธิบายเกี่ยวกับคอลัมน์็ (เช่น Schema) สำหรับชนิดตารางที่ระบุไว้เฉพาะ


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

ส่งคืนค่าตารางที่อธิบายเกี่ยวกับคอลัมน์ <code>tableType</code><br />ดูเอกสารประกอบเกี่ยวกับ <code>Table.Schema</code> สำหรับคำอธิบายตารางที่แสดงผลลัพธ์<br />
