# Type.TableColumn


## Description

ส่งกลับชนิดของคอลัมน์ในตาราง


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

ส่งกลับชนิดของคอลัมน์ <code>column</code> ในชนิดตาราง <code>tableType</code>



## Category
Type
