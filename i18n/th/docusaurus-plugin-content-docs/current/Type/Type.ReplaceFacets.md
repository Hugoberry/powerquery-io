# Type.ReplaceFacets


## Description

เปลี่ยนข้อเท็จจริงของชนิด


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

เปลี่ยนแปลงข้อเท็จจริงของ <code>type</code> โดยมีข้อเท็จจริงอยู่ในระเบียน <code>facets</code>
