# List.Count


## Description

ส่งกลับจำนวนข้อมูลในรายการ


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

ส่งกลับจำนวนข้อมูลในรายการ <code>list</code>


## Examples

### Example #1 
หาจำนวนของค่าในรายการ \{1, 2, 3}
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
