# Geometry.FromWellKnownText


แปลข้อความแสดงแทนค่าเรขาคณิตที่อยู่ในรูปแบบ Well-Known Text (WKT) เป็นระเบียนที่มีโครงสร้าง


## Syntax

```powerquery
Geometry.FromWellKnownText(
    input as text
) as record
```


## Remarks

แปลข้อความแสดงแทนค่าเรขาคณิตที่อยู่ในรูปแบบ Well-Known Text (WKT) เป็นระเบียนที่มีโครงสร้าง WKT เป็นรูปแบบมาตรฐานที่กำหนดโดย Open Geospatial Consortium (OGC) และเป็นรูปแบบการจัดเรียงแบบอนุกรมทั่วไปที่ใช้โดยฐานข้อมูล รวมทั้ง SQL Server
