# Cube.ReplaceDimensions


แทนที่ชุดของมิติที่ส่งกลับโดย Cube.Dimensions


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Remarks

แทนที่ชุดมิติที่ส่งกลับโดย <code>Cube.Dimensions</code>    ตัวอย่างเช่น ฟังก์ชันนี้สามารถใช้เพื่อเพิ่มคอลัมน์ ID ไปยังแอตทริบิวต์มิติ เพื่อให้แหล่งข้อมูลสามารถจัดกลุ่มบน ID แทนที่จะเป็นค่าที่แสดง
