# Web.Page


ส่งกลับเนื้อหาของเอกสาร HTML ที่แยกย่อยเป็นโครงสร้างหน่วยประกอบ พร้อมกับการแสดงเอกสารแบบเต็มและข้อความเอกสารหลังจากเอาแท็กออก


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Remarks
