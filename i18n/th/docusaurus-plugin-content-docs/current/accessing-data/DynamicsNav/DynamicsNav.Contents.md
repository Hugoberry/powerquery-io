# DynamicsNav.Contents


ป้อน URL ของตำแหน่งข้อมูลบริการ OData ของ Dynamics NAV ของคุณ


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

ส่งกลับตารางพร้อมข้อมูลของ Dynamics NAV ที่เกี่ยวข้อง
