# Excel.CurrentWorkbook


ส่งกลับเนื้อหาของเวิร์กบุ๊ก Excel ปัจจุบัน


## Syntax

```powerquery
Excel.CurrentWorkbook(

) as table
```


## Remarks

ส่งกลับเนื้อหาของเวิร์กบุ๊ก Excel ปัจจุบันที่ส่งกลับตาราง ช่วงที่มีชื่อ และอาร์เรย์แบบไดนามิก ไม่เหมือนกับ <code>Excel.Workbook</code> ที่จะไม่ส่งกลับแผ่นงาน
