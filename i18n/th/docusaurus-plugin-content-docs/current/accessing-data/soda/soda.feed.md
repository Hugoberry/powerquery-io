# Soda.Feed


ส่งกลับตารางจากเนื้อหาที่ URL ที่ระบุซึ่งมีการจัดรูปแบบตาม SODA 2.0 API


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Remarks

ส่งกลับตารางจากเนื้อหาที่ URL <code>url</code> ที่ระบุซึ่งมีการจัดรูปแบบตาม SODA 2.0 API URL ต้องชี้ไปยังแหล่งข้อมูลที่สอดคล้องกับ SODA ที่ถูกต้องซึ่งลงท้ายด้วยนามสกุล .csv
