# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

ย้อนกลับ <code>text</code> ที่ระบุ


## Examples

### Example #1 
ย้อนกลับข้อความ &#34;123&#34;
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
