# Text.Length


ส่งกลับจำนวนของอักขระ


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

ส่งกลับจำนวนอักขระในข้อความ <code>text</code>


## Examples

### Example #1 
หาจำนวนอักขระที่อยู่ในข้อความ &#34;Hello World&#34;
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
