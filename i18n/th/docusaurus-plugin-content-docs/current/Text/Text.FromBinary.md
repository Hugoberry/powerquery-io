# Text.FromBinary


ถอดรหัสข้อมูลจากรูปแบบไบนารีให้เป็นข้อความ


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

ถอดรหัสข้อมูล <code>binary</code> จากค่าไบนารีให้เป็นค่าข้อความโดยใช้ชนิด <code>encoding</code>
