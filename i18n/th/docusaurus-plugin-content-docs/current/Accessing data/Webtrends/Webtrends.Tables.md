# Webtrends.Tables


กรอกรหัสโปรไฟล์ที่เกียวข้องกับบัญชี Webtrends ของคุณ


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Remarks

นำกลับตารางพร้อมด้วยข้อมูล Webtrends ที่เกี่ยวข้อง
