# Smartsheet.Tables


## Description

ส่งกลับตารางของแผ่นงาน, รายงาน, โฟลเดอร์ และพื้นที่ทำงานจาก API ของ Smartsheet


## Syntax

```powerquery
Smartsheet.Tables(

) as table
```


## Details

นำกลับการซ้อนตารางของชีท, รายงาน แฟ้ม และพื้นที่ทำงานที่ยังว่างอยู่จาก API ของ Smartsheet


## Examples

### Example #1 
นำกลับชีท, แฟ้ม, รายงานและพื้นที่ทำงานที่ยังว่างอยู่ที่ระดับสูงสุดของลำดับขึ้น Smartsheet
```powerquery
Smartsheet.Tables()
```

Result: 
```powerquery
ตารางของแฟ้ม, รายงาน และพื้นที่ทำงานจากระดับสูงสุดของลำดับขั้น Smartsheet
```
