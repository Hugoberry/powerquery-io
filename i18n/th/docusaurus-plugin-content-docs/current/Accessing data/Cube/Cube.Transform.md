# Cube.Transform


## Description

นำรายการฟังก์ชันคิวบ์ไปใช้


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

นำรายการฟังก์ชันคิวบ์, <code>transforms</code>, ไปใช้บน <code>cube</code>



## Category
Cube
