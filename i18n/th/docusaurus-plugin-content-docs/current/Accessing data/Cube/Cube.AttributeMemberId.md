# Cube.AttributeMemberId


## Description

ส่งผลตัวระบุหมายเลขเฉพาะจากค่าคุณสมบัติสมาชิก


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

ส่งผลตัวระบุหมายเลขเฉพาะจากค่าคุณสมบัติสมาชิก <code>attribute</code> ส่งกลับค่า null สำหรับค่าอื่นๆ



## Category
Cube
