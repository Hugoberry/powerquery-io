# Cube.MeasureProperty


## Description

ส่งกลับคุณสมบัติของหน่วยวัด (คุณสมบัติเซลล์)


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

ส่งกลับคุณสมบัติ <code>propertyName</code> ของหน่วยวัด <code>measure</code>



## Category
Cube
