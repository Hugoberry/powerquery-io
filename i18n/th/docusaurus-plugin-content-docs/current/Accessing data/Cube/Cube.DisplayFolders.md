# Cube.DisplayFolders


ส่งกลับทรีของตารางที่ซ้อนกันที่แสดงลําดับชั้นโฟลเดอร์การแสดงผลของวัตถุ (ตัวอย่างเช่น มิติและหน่วยวัด)


## Syntax

```powerquery
Cube.DisplayFolders(
    cube as table
) as table
```


## Remarks

ส่งกลับทรีของตารางที่ซ้อนกันที่แสดงลําดับชั้นโฟลเดอร์การแสดงผลของวัตถุ (ตัวอย่างเช่น มิติและหน่วยวัด) ที่พร้อมใช้งานใน <code>cube</code>
