# Cube.Parameters


ส่งคืนตารางที่ประกอบด้วยชุดพารามิเตอร์ที่สามารถกำหนดใช้สำหรับคิวบ์ได้


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Remarks

ส่งคืนตารางที่ประกอบด้วยชุดพารามิเตอร์ที่สามารถกำหนดไใช้สำหรับ <code>cube</code> ได้ พารามิเตอร์แต่ละชุดเป็นฟังก์ชันที่สามารถเรียกใช้เพื่อรับ <code>cube</code> ด้วยพารามิเตอร์และอาร์กิวเมนต์ที่กำหนดใช้
