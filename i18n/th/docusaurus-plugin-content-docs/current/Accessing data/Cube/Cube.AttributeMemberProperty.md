# Cube.AttributeMemberProperty


## Description

ส่งกลับคุณสมบัติของแอตทริบิวต์มิติ


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

ส่งกลับคุณสมบัติ <code>propertyName</code> ของแอตทริบิวต์มิติ <code>attribute</code>



## Category
Cube
