# Cube.MeasureProperties


## Description

ส่งกลับตารางที่มีชุดคุณสมบัติการวัดที่พร้อมใช้งานที่ขยายในคิวบ์


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

ส่งกลับตารางที่มีชุดคุณสมบัติที่พร้อมใช้งานสำหรับการวัดที่ขยายในคิวบ์
