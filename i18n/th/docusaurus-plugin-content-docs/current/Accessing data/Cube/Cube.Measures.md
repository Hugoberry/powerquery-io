# Cube.Measures


## Description

ส่งกลับตารางที่มีชุดการวัดที่มีให้ใช้งาน


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

ส่งกลับตารางที่มีชุดการวัดที่พร้อมใช้งานภายใน <code>cube</code>    แต่ละการวัดจะถูกแสดงเป็นฟังก์ชัน สามารถนำการวัดไปใช้กับคิวบ์ได้โดยใช้ Cube.AddMeasureColumn
