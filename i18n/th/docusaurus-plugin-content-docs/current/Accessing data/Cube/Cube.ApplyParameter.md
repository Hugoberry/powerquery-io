# Cube.ApplyParameter


## Description

ส่งคืนคิวบ์หลังจากกำหนดใช้พารามิเตอร์


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

ส่งคืนคิวบ์หลังจากกำหนดใช้ <code>parameter</code> ด้วย <code>arguments</code> เป็น <code>cube</code>



## Category
Cube
