# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Details

ตัวเชื่อมต่อ Teams Analytics ช่วยให้คุณสามารถรับข้อมูลเชิงลึกเกี่ยวกับการใช้งาน Teams ของคุณได้ ข้อมูลจะพร้อมใช้งานสำหรับทั้งกิจกรรมส่วนตัวของคุณใน Teams ตลอดจนกิจกรรมใน Teams ที่คุณเป็นสมาชิกอยู่<br />ข้อมูลจะมีให้ก่อน 90 วัน <br /> สำหรับรายละเอียดของชนิดข้อมูลที่พร้อมใช้งาน โปรดดูที่เอกสารวิธีใช้ต่อไปนี้: <br /><ul><li><b>การใช้งาน Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>กิจกรรมของผู้ใช้ Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
