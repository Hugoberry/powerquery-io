# Hdfs.Contents


## Description

ส่งกลับตารางที่มีคุณสมบัติและเนื้อหาของแฟ้มและโฟลเดอร์ที่พบในโฟลเดอร์ที่ระบุจากระบบแฟ้ม Hadoop


## Syntax

```powerquery
Hdfs.Contents(
    url as text
) as table
```


## Details

ส่งกลับตารางที่มีแถวสำหรับแต่ละโฟลเดอร์และแฟ้มที่พบใน URL โฟลเดอร์ <code>url</code> จากระบบแฟ้ม Hadoop แต่ละแถวประกอบด้วยคุณสมบัติของโฟลเดอร์หรือแฟ้มและลิงก์ไปยังเนื้อหา
