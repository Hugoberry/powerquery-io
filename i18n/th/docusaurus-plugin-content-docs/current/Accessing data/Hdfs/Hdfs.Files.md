# Hdfs.Files


ส่งกลับตารางที่มีคุณสมบัติและเนื้อหาของแฟ้มที่พบในโฟลเดอร์ที่ระบุและโฟลเดอร์ย่อยจากระบบแฟ้ม Hadoop


## Syntax

```powerquery
Hdfs.Files(
    url as text
) as table
```


## Remarks

ส่งกลับตารางที่มีแถวสำหรับแต่ละแฟ้มที่พบใน URL โฟลเดอร์ <code>url</code> และโฟลเดอร์ย่อยจากระบบแฟ้ม Hadoop แต่ละแถวประกอบด้วยคุณสมบัติของแฟ้มและลิงก์ไปยังเนื้อหา
