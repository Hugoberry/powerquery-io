# HdInsight.Containers


## Description

ส่งกลับตารางการนำทางที่มีคอนเทนเนอร์ที่พบในบัญชีที่ระบุจากชุดเก็บข้อมูลประจำตัวที่จัดเก็บ Azure


## Syntax

```powerquery
HdInsight.Containers(
    account as text
) as table
```


## Details

ส่งกลับตารางการนำทางที่มีแถวของแต่ละคอนเทนเนอร์ที่พบใน URL บัญชี <code>account</code> จากชุดเก็บข้อมูลประจำตัวที่จัดเก็บ Azure แต่ละแถวประกอบด้วยการเชื่อมโยงไปยัง BLOB ของคอนเทนเนอร์
