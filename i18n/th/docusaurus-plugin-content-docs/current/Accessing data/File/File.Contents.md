# File.Contents


ส่งกลับเนื้อหาของแฟ้มที่ระบุเป็นไบนารี


## Syntax

```powerquery
File.Contents(
    path as text,
    optional options as record
) as binary
```


## Remarks

ส่งกลับเนื้อหาของไฟล์ <code>path</code> เป็นไบนารี พารามิเตอร์ <code>options</code> มีไว้สําหรับใช้ภายในเท่านั้นในขณะนี้
