# VSTS.Contents


## Description

ใส่ URL ของ Azure DevOps Analytics Service ของคุณ


## Syntax

```powerquery
VSTS.Contents(
    url as text,
    optional options as record
) as binary
```


## Details

ส่งกลับเนื้อหาที่ดาวน์โหลดจาก URL ของ Azure DevOps Analytics Service เป็นค่าไบนารี ฟังก์ชันนี้จะใช้ข้อมูลประจำตัวส่วนกลางสำหรับองค์กรที่เข้าถึงได้ทั้งหมด
