# VSTS.AnalyticsViews


## Description

ใส่ชื่อองค์กรและโครงการ


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

ส่งกลับตารางมุมมองการวิเคราะห์ที่เสนอโดย Azure DevOps
