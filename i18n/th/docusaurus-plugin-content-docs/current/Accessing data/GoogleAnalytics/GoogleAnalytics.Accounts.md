# GoogleAnalytics.Accounts


## Description

ส่งคืนบัญชี Google Analytics


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

ส่งคืนบัญชี Google Analytics ซึ่งสามารถเข้าถึงจากข้อมูลประจำตัวปัจจุบัน
