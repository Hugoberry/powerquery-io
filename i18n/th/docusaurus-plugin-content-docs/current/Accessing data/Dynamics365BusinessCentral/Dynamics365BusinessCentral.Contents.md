# Dynamics365BusinessCentral.Contents


ใส่สภาพแวดล้อมและบริษัท Dynamics 365 Business Central ของคุณ


## Syntax

```powerquery
Dynamics365BusinessCentral.Contents(
    optional company as text
) as table
```


## Remarks

ส่งกลับตารางพร้อมข้อมูลของ Dynamics 365 Business Central ที่เกี่ยวข้อง
