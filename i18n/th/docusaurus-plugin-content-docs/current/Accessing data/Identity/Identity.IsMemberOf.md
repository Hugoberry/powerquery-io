# Identity.IsMemberOf


## Description

กำหนดว่าข้อมูลประจำตัวเป็นสมาชิกของคอลเลกชันข้อมูลประจำตัวหรือไม่


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details
