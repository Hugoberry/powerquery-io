# ActiveDirectory.Domains


## Description

ส่งกลับรายการโดเมน Active Directory ในฟอเรสต์เดียวกับโดเมนที่ระบุหรือรายการโดเมนของเครื่องปัจจุบันหากไม่มีการระบุโดเมน


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```
