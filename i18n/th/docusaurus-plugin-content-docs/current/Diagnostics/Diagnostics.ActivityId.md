# Diagnostics.ActivityId


## Description

ส่งคืนตัวระบุแบบย่อสำหรับการประเมินที่ใช้งานอยู่ในปัจจุบัน


## Syntax

```powerquery
Diagnostics.ActivityId(

) as text
```


## Details
