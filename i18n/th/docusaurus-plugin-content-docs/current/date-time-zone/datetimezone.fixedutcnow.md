# DateTimeZone.FixedUtcNow


ส่งกลับวันที่และเวลาในโซนเวลาใน UTC (โซนเวลา GMT) ค่านี้จะคงที่และไม่เปลี่ยนแปลงในการเรียกใช้ที่ต่อเนื่องกัน


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```


## Remarks

ส่งกลับวันที่และเวลาในโซนเวลาใน UTC (โซนเวลา GMT) ค่านี้จะคงที่และไม่เปลี่ยนแปลงในการเรียกใช้ที่ต่อเนื่องกัน
