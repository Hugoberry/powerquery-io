# Table.PartitionValues


ส่งกลับข้อมูลเกี่ยวกับวิธีการแบ่งพาร์ติชันตาราง


## Syntax

```powerquery
Table.PartitionValues(
    table as table
) as table
```


## Remarks

ส่งกลับข้อมูลเกี่ยวกับวิธีการแบ่งพาร์ติชันตาราง  ตารางจะถูกส่งกลับในกรณีที่แต่ละคอลัมน์เป็นคอลัมน์พาร์ติชันในตารางเดิม และแต่ละแถวจะสอดคล้องกับพาร์ติชันในตารางเดิม
