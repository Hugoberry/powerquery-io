# Table.ViewError


สร้างระเบียนข้อผิดพลาดที่ปรับเปลี่ยนซึ่งจะไม่ทริกเกอร์การย้อนกลับเมื่อส่งออกโดยตัวจัดการที่กำหนดไว้ในมุมมอง (ผ่าน Table.View)


## Syntax

```powerquery
Table.ViewError(
    errorRecord as record
) as record
```


## Remarks

สร้างระเบียนข้อผิดพลาดที่ปรับเปลี่ยนจาก <code>errorRecord</code> ซึ่งจะไม่ทริกเกอร์การย้อนกลับเมื่อส่งออกโดยตัวจัดการที่กำหนดไว้ในมุมมอง (ผ่าน Table.View)
