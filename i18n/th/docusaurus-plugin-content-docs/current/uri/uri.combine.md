# Uri.Combine


ส่งกลับ URI แบบสัมบูรณ์ที่ได้จากการรวม URI พื้นฐานของค่าที่ป้อนและ URI ที่เกี่ยวข้อง


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Remarks

ส่งกลับ URI แบบสัมบูรณ์ที่ได้จากการรวมค่าที่ป้อน <code>baseUri</code> และ <code>relativeUri</code>
