# BinaryFormat.SignedInteger16


รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 16 บิต


## Syntax

```powerquery
BinaryFormat.SignedInteger16(
    binary as binary
) as any
```


## Remarks

รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 16 บิต
