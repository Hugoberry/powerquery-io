# BinaryFormat.UnsignedInteger16


## Description

รูปแบบไบนารีที่อ่านจำนวนเต็มที่ไม่มีเครื่องหมาย 16 บิต


## Syntax

```powerquery
BinaryFormat.UnsignedInteger16(
    binary as binary
) as any
```
