# Binary.ViewError


## Description

สร้างระเบียนข้อผิดพลาดที่ปรับเปลี่ยนซึ่งจะไม่ทริกเกอร์การย้อนกลับเมื่อส่งออกโดยตัวจัดการที่กําหนดในมุมมอง (ผ่าน Binary.View)


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Details

สร้างระเบียนข้อผิดพลาดที่ปรับเปลี่ยนจาก <code>errorRecord</code> ซึ่งจะไม่ทริกเกอร์การย้อนกลับเมื่อส่งออกโดยตัวจัดการที่กําหนดในมุมมอง (ผ่าน Binary.View)
