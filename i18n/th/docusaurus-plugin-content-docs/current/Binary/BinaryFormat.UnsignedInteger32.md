# BinaryFormat.UnsignedInteger32


## Description

รูปแบบไบนารีที่อ่านจำนวนเต็มที่ไม่มีเครื่องหมาย 32 บิต


## Syntax

```powerquery
BinaryFormat.UnsignedInteger32(
    binary as binary
) as any
```


## Details
