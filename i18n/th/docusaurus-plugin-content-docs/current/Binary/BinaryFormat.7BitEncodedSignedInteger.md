# BinaryFormat.7BitEncodedSignedInteger


รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 64 บิตที่ถูกเข้ารหัสลับโดยใช้การเข้ารหัสลับความยาวตัวแปร 7 บิต


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 64 บิตที่ถูกเข้ารหัสลับโดยใช้การเข้ารหัสลับความยาวตัวแปร 7 บิต
