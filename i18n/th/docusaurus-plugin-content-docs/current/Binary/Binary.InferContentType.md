# Binary.InferContentType


## Description

อ่านสตรีมแบบไบนารีและลองกำหนดชนิดเนื้อหาและข้อมูลรูปแบบของสตรีม


## Syntax

```powerquery
Binary.InferContentType(
    source as binary
) as record
```


## Details

ส่งกลับระเบียนที่มีเขตข้อมูล Content.Type ที่ประกอบด้วยชนิด MIME ที่อนุมาน    ถ้าชนิดเนื้อหาที่อนุมานเป็นข้อความ/\* และมีการตรวจพบโค้ดเพจที่เข้ารหัส และนอกจากนี้ส่งกลับเขตข้อมูล Content.Encoding ที่ประกอบด้วยการเข้ารหัสของสตรีม    ถ้าชนิดเนื้อหาที่อนุมานเป็นข้อความ/csv และรูปแบบถูกคั่น นอกจากนี้ยังส่งกลับเขตข้อมูล Csv.PotentialDelimiter ซึ่งประกอบด้วยตารางสำหรับการวิเคราะห์ของตัวคั่นที่เป็นไปได้    ถ้าชนิดเนื้อหาที่อนุมานเป็นข้อความ/csv และรูปแบบมีความกว้างคงที่ นอกจากนี้ยังส่งกลับเขตข้อมูล Csv.PotentialPositions ที่ประกอบด้วยรายการสำหรับการวิเคราะห์ของตำแหน่งที่ตั้งคอลัมน์ที่มีความกว้างคงที่ที่เป็นไปได้
