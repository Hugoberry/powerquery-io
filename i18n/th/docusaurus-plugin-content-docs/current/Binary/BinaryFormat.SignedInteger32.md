# BinaryFormat.SignedInteger32


รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 32 บิต


## Syntax

```powerquery
BinaryFormat.SignedInteger32(
    binary as binary
) as any
```


## Remarks

รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 32 บิต
