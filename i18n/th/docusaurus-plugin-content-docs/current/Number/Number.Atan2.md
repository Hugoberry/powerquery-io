# Number.Atan2


ส่งกลับค่าอาร์กแทนเจนต์ของการหารสองจำนวน


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Remarks

ส่งกลับค่าอาร์กแทนเจนต์ของการหารสองจำนวน ได้แก่ <code>y</code> และ <code>x</code> การหารจะเขียนโครงสร้างได้เป็น <code>y</code>/<code>x</code>
