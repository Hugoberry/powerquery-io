# Number.Sin


## Description

ส่งกลับค่าไซน์ของตัวเลข


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

ส่งกลับค่าไซน์ของ <code>number</code>


## Examples

### Example #1 
หาค่าไซน์ของมุม 0
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
