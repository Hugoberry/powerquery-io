# Number.Random


ส่งกลับตัวเลขสุ่ม


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

คืนค่าตัวเลขสุ่มระหว่าง 0 และ 1


## Examples

### Example #1 
รับตัวเลขสุ่ม
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
