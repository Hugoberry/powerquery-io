# Number.Tan


## Description

ส่งกลับค่าแทนเจนต์ของตัวเลข


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

ส่งกลับค่าแทนเจนต์ของ <code>number</code>


## Examples

### Example #1 
หาค่าแทนเจนต์ของมุม 1
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
