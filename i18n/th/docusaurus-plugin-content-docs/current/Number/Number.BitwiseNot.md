# Number.BitwiseNot


## Description

ส่งกลับไบต์ซึ่งแต่ละบิตตรงกันข้ามกับค่าที่ป้อน


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

ส่งกลับผลลัพธ์ของการดำเนินการในระดับบิต "Not" ใน <code>number</code>



## Category
Number.Bytes
