# Number.BitwiseXor


## Description

ส่งกลับผลลัพธ์ของการดำเนินการในระดับบิต &#34;XOR&#34; (Exclusive-OR) ระหว่างค่าที่ป้อนสองรายการ


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

ส่งกลับผลลัพธ์ของการดำเนินการในระดับบิต "XOR" (Exclusive-OR) ระหว่าง <code>number1</code> กับ <code>number2</code>



## Category
Number.Bytes
