# Number.RandomBetween


## Description

ส่งกลับตัวเลขสุ่มระหว่างสองตัวเลข


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

ส่งกลับตัวเลขสุ่มระหว่าง <code>bottom</code> ถึง <code>top</code>


## Examples

### Example #1 
รับตัวเลขสุ่มระหว่าง 1 ถึง 5
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
