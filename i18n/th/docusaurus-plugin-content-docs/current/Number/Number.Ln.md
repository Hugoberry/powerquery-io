# Number.Ln


ส่งกลับลอการิทึมธรรมชาติของตัวเลข


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Remarks

ส่งคืนอัลกอริทึ่มสำหรับตัวเลข <code>number</code> หาก <code>number</code> เป็น Null <code>Number.Ln</code> จะส่งคืนค่าเป็น Null


## Examples

### Example #1 
รับลอการิทึมธรรมชาติของ 15
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
