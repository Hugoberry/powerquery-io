# Number.Cos


ส่งกลับค่าโคไซน์ของตัวเลข


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

ส่งกลับค่าโคไซน์ของ <code>number</code>


## Examples

### Example #1 
หาค่าโคไซน์ของมุม 0
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
