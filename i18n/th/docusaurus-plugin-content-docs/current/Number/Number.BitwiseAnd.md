# Number.BitwiseAnd


## Description

ส่งกลับผลลัพธ์ของการดำเนินการในระดับบิต &#34;And&#34; ระหว่างค่าที่ป้อนสองรายการ


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

ส่งกลับผลลัพธ์ของการดำเนินการในระดับบิต "And" ระหว่าง <code>number1</code> กับ <code>number2</code>



## Category
Number.Bytes
