# Number.BitwiseShiftLeft


เลื่อนชุดบิตที่กำหนดไปทางซ้าย


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Remarks

ส่งกลับผลลัพธ์ของการดำเนินการเลื่อนบิตไปทางซ้ายใน <code>number1</code> ตามจำนวนบิต <code>number2</code> ที่ระบุ
