# Number.Log10


ส่งกลับลอการิทึมฐาน 10 ของตัวเลข


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Remarks

ส่งคืนลอการิทึมฐาน 10 ของตัวเลข <code>number</code> ถ้า <code>number</code> เป็น null <code>Number.Log10</code> จะส่งกลับ null


## Examples

### Example #1 
รับลอการิทึมฐาน 10 ของ 2
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
