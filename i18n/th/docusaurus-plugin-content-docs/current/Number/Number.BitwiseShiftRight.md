# Number.BitwiseShiftRight


## Description

เลื่อนชุดบิตที่กำหนดไปทางขวา


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

ส่งกลับผลลัพธ์ของการดำเนินการเลื่อนบิตไปทางขวาใน <code>number1</code> ตามจำนวนบิต <code>number2</code> ที่ระบุ

Number.Bytes
