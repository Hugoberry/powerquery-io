# Error.Record


ส่งกลับระเบียนข้อผิดพลาดจากค่าข้อความที่ระบุสำหรับเหตุผล ข้อความ และรายละเอียด


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Remarks
