# DateTimeZone.UtcNow


## Description

ส่งกลับวันที่และเวลาในโซนเวลาใน UTC (โซนเวลา GMT)


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

ส่งกลับวันที่และเวลาในโซนเวลาใน UTC (โซนเวลา GMT)


## Examples

### Example #1 
รับวันที่และเวลาปัจจุบันใน UTC
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
