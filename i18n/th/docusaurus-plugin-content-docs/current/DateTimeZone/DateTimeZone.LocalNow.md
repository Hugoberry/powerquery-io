# DateTimeZone.LocalNow


## Description

ส่งกลับวันที่และเวลาในโซนเวลาท้องถิ่น


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

ส่งกลับค่า <code>datetimezone</code> เพื่อตั้งค่าวันที่และเวลาปัจจุบันในระบบ     ค่าที่ส่งกลับมีข้อมูลโซนเวลาที่แสดงโซนเวลาท้องถิ่น

DateTimeZone
