# Function.IsDataSource


## Description

ส่งคืนค่าว่าจะมีการใช้ฟังก์ชันที่กำหนดไว้เป็นแหล่งข้อมูลหรือไม่


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

ส่งคืนค่าว่าจะมีการใช้ <code>function</code> เป็นแหล่งข้อมูลหรือไม่
