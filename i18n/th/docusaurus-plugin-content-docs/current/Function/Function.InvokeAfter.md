# Function.InvokeAfter


## Description

เรียกฟังก์ชันที่กำหนดไว้หลังผ่านระยะเวลาที่ระบุไว้


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

ส่งกลับผลลัพธ์ของการเรียก <code>function</code> หลังผ่านระยะเวลา <code>delay</code>



## Category
Function
