# Number.Factorial


ส่งกลับค่าแฟกทอเรียลของตัวเลข


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

ส่งกลับค่าแฟกทอเรียลของตัวเลข <code>number</code>


## Examples

### Example #1 
หาค่าแฟกทอเรียลของ 10
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
