# Value.Equals


## Description

ส่งกลับว่าค่าสองค่าเท่ากันหรือไม่


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

ส่งกลับ True ถ้าค่า <code>value1</code> เท่ากับค่า <code>value2</code> และ False ถ้าไม่ใช่



## Category
Values
