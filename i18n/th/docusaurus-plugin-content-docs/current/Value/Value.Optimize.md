# Value.Optimize


ส่งสัญญาณ Value.Expression เพื่อส่งกลับนิพจน์ที่ปรับให้เหมาะสมสำหรับค่า


## Syntax

```powerquery
Value.Optimize(
    value as any
) as any
```


## Remarks

เมื่อใช้ใน Value.Expression ถ้า <code>value</code> เป็นคิวรีที่สามารถปรับให้เหมาะสมได้ ฟังก์ชันนี้จะระบุว่าควรส่งกลับนิพจน์ที่ได้รับการปรับให้เหมาะสม ไม่เช่นนั้น <code>value</code> จะถูกส่งไปโดยไม่มีผล
