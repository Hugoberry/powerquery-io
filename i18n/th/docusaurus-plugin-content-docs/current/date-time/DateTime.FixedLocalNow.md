# DateTime.FixedLocalNow


ส่งกลับวันที่และเวลาในโซนเวลาท้องถิ่น ค่านี้จะคงที่และไม่เปลี่ยนแปลงในการเรียกใช้ที่ต่อเนื่องกัน


## Syntax

```powerquery
DateTime.FixedLocalNow(

) as datetime
```


## Remarks

ส่งกลับค่า <code>datetime</code> เพื่อตั้งค่าวันที่และเวลาปัจจุบันในระบบ ค่านี้จะคงที่และไม่เปลี่ยนแปลงในการเรียกใช้ที่ต่อเนื่องกัน ซึ่งแตกต่างจาก DateTime.LocalNow ที่อาจส่งกลับค่าอื่นในกรณีที่มีการใช้นิพจน์
