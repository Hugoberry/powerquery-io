# Table.FirstValue


## Description

ส่งคืนคอลัมน์แรกของแถวแรกจากตาราง หรือค่าเริ่มต้นตามที่ระบุไว้


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

ส่งคืนคอลัมน์แรกของแถวแรกจากตาราง <code>table</code> หรือค่าเริ่มต้นตามที่ระบุไว้
