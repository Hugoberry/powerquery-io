# BinaryFormat.7BitEncodedUnsignedInteger


รูปแบบไบนารีที่อ่านจำนวนเต็มที่ไม่มีเครื่องหมาย 64 บิตที่ถูกเข้ารหัสลับโดยใช้การเข้ารหัสลับความยาวตัวแปร 7 บิต


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks

รูปแบบไบนารีที่อ่านจำนวนเต็มที่ไม่มีเครื่องหมาย 64 บิตที่ถูกเข้ารหัสลับโดยใช้การเข้ารหัสลับความยาวตัวแปร 7 บิต
