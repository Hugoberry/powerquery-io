# BinaryFormat.UnsignedInteger64


รูปแบบไบนารีที่อ่านจำนวนเต็มที่ไม่มีเครื่องหมาย 64 บิต


## Syntax

```powerquery
BinaryFormat.UnsignedInteger64(
    binary as binary
) as any
```


## Remarks
