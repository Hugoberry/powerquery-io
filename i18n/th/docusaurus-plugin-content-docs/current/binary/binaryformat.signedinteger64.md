# BinaryFormat.SignedInteger64


รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 64 บิต


## Syntax

```powerquery
BinaryFormat.SignedInteger64(
    binary as binary
) as any
```


## Remarks

รูปแบบไบนารีที่อ่านจำนวนเต็มที่มีเครื่องหมาย 64 บิต
