# Value.Alternates


แสดงแผนคิวรีสำรอง


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Remarks

แสดงแผนคิวรีสำรองภายในนิพจน์แผนคิวรีที่ได้รับผ่านค่านิพจน์ Value.Expression(Value.Optimize(...)) ไม่ได้มีไว้สำหรับการใช้งานอื่นๆ
