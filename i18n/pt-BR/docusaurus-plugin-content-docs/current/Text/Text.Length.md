# Text.Length


## Description

Retorna o número de caracteres.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Retorna o número de caracteres no texto <code>text</code>.


## Examples

### Example #1 
Localize quantos caracteres existem no texto &#34;Olá, Mundo&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
