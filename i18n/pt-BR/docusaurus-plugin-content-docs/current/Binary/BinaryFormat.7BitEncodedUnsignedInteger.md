# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Um formato binário que lê um número inteiro de 64 bits sem sinal que foi codificado usando a codificação de comprimento variável de 7 bits.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details

Um formato binário que lê um número inteiro de 64 bits sem sinal que foi codificado usando a codificação de comprimento variável de 7 bits.
