# Uri.Combine


## Description

Retorna um URI absoluto que é a combinação do URI base de entrada e URI relativo.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Retorna um URI absoluto que é a combinação do <code>baseUri</code> de entrada e do <code>relativeUri</code>.

Uri
