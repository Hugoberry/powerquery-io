# Error.Record


## Description

Retorna um registro de erro dos valores de texto fornecidos para motivo, mensagem e detalhes.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details

Retorna um registro de erro dos valores de texto fornecidos para motivo, mensagem e detalhes.
