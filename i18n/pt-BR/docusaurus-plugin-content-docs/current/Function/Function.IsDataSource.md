# Function.IsDataSource


## Description

Retorna se uma função em particular ou não, for considerada uma fonte de dados.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Retorna se <code>function</code> ou não, for considerada uma fonte de dados.
