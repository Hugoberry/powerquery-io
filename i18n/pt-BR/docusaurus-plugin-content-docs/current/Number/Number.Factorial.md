# Number.Factorial


## Description

Retorna o fatorial do número.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Retorna o fatorial do número <code>number</code>.


## Examples

### Example #1 
Localize o fatorial 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
