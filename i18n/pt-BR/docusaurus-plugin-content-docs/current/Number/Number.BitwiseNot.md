# Number.BitwiseNot


## Description

Retorna um byte em que cada bit é o oposto da entrada.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Retorna o resultado da execução de uma operação bit a bit "Not" em <code>number</code>.



## Category
Number.Bytes
