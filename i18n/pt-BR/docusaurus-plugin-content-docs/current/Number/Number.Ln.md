# Number.Ln


Retorna o logaritmo natural do número.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Remarks

Retorna o logaritmo natural de um número, <code>number</code>. Se <code>number</code> for nulo, <code>Number.Ln</code> retornará um valor nulo.


## Examples

### Example #1 
Obtenha o logaritmo natural 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
