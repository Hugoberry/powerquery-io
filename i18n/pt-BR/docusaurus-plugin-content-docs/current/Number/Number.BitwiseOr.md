# Number.BitwiseOr


Retorna o resultado da execução de uma operação bit a bit &#34;Or&#34; entre as duas entradas.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Retorna o resultado da execução de uma operação bit a bit "Or" entre <code>number1</code> e <code>number2</code>.



## Category
Number.Bytes
