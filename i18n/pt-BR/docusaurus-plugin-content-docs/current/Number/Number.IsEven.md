# Number.IsEven


Indica se o valor é par.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Remarks

Indica se o valor, <code>number</code>, é par retornando <code>true</code>; do contrário, o valor <code>false</code> será retornado.


## Examples

### Example #1 
Verifique se 625 é um número par.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Verifique se 82 é um número par.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
