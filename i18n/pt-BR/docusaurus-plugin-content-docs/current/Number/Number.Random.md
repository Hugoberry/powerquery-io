# Number.Random


## Description

Retorna um número aleatório.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Retorna um número aleatório entre 0 e 1.


## Examples

### Example #1 
Obtenha um número aleatório.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
