# Number.Sin


## Description

Retorna o seno do número.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Retorna o seno de <code>number</code>.


## Examples

### Example #1 
Localize o seno do ângulo 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
