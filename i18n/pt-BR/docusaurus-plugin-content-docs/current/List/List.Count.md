# List.Count


Retorna o número de itens na lista.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Retorna o número de itens na lista <code>list</code>.


## Examples

### Example #1 
Localize o número de valores na lista \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
