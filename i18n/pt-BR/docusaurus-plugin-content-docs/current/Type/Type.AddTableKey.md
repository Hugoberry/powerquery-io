# Type.AddTableKey


Adiciona uma chave ao tipo de tabela especificado.


## Syntax

```powerquery
Type.AddTableKey(
    table as type,
    columns as list,
    isPrimary as logical
) as type
```


## Remarks

Adiciona uma chave ao tipo de tabela especificado.
