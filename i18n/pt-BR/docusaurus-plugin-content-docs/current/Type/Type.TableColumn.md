# Type.TableColumn


Retorna o tipo de uma coluna em uma tabela.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Remarks

Retorna o tipo da coluna <code>column</code> no tipo de tabela <code>tableType</code>.
