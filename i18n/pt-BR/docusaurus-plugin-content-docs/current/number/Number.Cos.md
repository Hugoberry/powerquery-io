# Number.Cos


Retorna o cosseno do número.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Retorna o cosseno de <code>number</code>.


## Examples

### Example #1 
Localize o cosseno do ângulo 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
