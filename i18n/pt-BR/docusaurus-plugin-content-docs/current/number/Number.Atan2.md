# Number.Atan2


Retorna o arco tangente da divisão dos dois números.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Remarks

Retorna o arco tangente da divisão dos dois números, <code>y</code> e <code>x</code>. A divisão será criada como <code>y</code>/<code>x</code>.
