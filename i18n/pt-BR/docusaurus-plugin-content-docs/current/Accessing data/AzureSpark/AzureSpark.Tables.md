# AzureSpark.Tables


Lista as tabelas em uma instância do Azure Spark.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Remarks

Lista as tabelas em uma instância do Azure Spark.


## Examples

### Example #1 
Lista as tabelas em uma instância do Azure Spark.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
