# Web.Page


Retorna o conteúdo do documento HTML decomposto em suas estruturas constitutivas, bem como uma representação do documento completo e respectivo texto após a remoção das marcas.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Remarks

Retorna o conteúdo do documento HTML decomposto em suas estruturas constitutivas, bem como uma representação do documento completo e respectivo texto após a remoção das marcas.
