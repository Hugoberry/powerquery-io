# VSTS.Views


## Description

Insira os nomes da organização e do projeto.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Retorna uma tabela de feeds OData oferecidos pelo Azure DevOps.
