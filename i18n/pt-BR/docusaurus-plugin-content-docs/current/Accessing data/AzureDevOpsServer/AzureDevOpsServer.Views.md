# AzureDevOpsServer.Views


Insira os nomes da organização e do projeto.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Remarks

Retorna uma tabela de feeds OData oferecidos pelo Azure DevOps.
