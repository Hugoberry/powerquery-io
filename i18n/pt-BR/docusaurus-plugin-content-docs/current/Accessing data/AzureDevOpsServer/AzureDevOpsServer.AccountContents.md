# AzureDevOpsServer.AccountContents


Insira a URL do Serviço Analítico do Azure DevOps.


## Syntax

```powerquery
AzureDevOpsServer.AccountContents(
    url as text,
    optional options as record
) as table
```


## Remarks

Retorna os conteúdos baixados da URL do Serviço Analítico do Azure DevOps como um valor binário. Essa função usa credenciais diferentes para organizações diferentes.
