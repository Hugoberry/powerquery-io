# Cube.AttributeMemberProperty


## Description

Retorna uma propriedade de um atributo de dimensão.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Retorna a propriedade <code>propertyName</code> do atributo de dimensão <code>attribute</code>.



## Category
Cube
