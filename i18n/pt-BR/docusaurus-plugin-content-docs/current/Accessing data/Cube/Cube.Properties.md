# Cube.Properties


## Description

Retorna uma tabela que contém o conjunto de propriedades disponíveis para dimensões expandidas no cubo.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```


## Details

Retorna uma tabela que contém o conjunto de propriedades disponíveis para dimensões expandidas no cubo.
