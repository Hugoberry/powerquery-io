# MailChimp.TablesV2


Retorna uma tabela com os dados-chave do MailChimp.


## Syntax

```powerquery
MailChimp.TablesV2(

) as table
```


## Remarks

Retorna uma tabela com os dados-chave do MailChimp.


## Examples

### Example #1 
Retorna uma tabela com os dados-chave do MailChimp.
```powerquery
MailChimp.Tables()
```

Result:
