# AzureDevOpsServer.AnalyticsViews


Insira os nomes da organização e do projeto.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Retorna uma tabela das exibições de Análise oferecidas pelo Azure DevOps.
