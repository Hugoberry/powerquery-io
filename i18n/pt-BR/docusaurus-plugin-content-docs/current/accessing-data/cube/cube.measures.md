# Cube.Measures


Retorna a tabela que contém o conjunto de medidas disponíveis.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Remarks

Retorna a tabela que contém o conjunto de medidas disponíveis no <code>cube</code>.    Cada medida é representada como uma função. As medidas podem ser aplicadas ao cubo usando Cube.AddMeasureColumn.
