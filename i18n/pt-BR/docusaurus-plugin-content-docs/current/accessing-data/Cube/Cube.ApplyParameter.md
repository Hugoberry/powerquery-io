# Cube.ApplyParameter


Retorna um cubo após a aplicação de um parâmetro.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

Retorna um cubo após a aplicação de <code>parameter</code> com <code>arguments</code> a <code>cube</code>.

Cube
