# Identity.IsMemberOf


Determina se uma identidade é um membro de uma coleção de identidades.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Remarks

Determina se uma identidade é um membro de uma coleção de identidades.
