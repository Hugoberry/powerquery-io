# Type.TableSchema


Retorna uma tabela que contém uma descrição das colunas (p.ex., o esquema) do tipo de tabela especificado.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Remarks

Retorna uma tabela que descreve as colunas de <code>tableType</code>.<br />Confira a documentação de <code>Table.Schema</code> para obter uma descrição da tabela resultante.<br />
