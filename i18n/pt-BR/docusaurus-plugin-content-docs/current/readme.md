# Descubra a linguagem
A linguagem de fórmula Power Query M é otimizada para construir consultas de mashup de dados altamente flexíveis. É uma linguagem funcional, sensível a maiúsculas e minúsculas, semelhante ao F#. A Linguagem de Fórmula Power Query é usada em vários produtos da Microsoft, como Power BI Desktop, Excel e Analysis Services.

Este site serve como uma referência para funções do Power Query, oferecendo acesso rápido e direto a informações detalhadas e exemplos práticos para cada função.
:::note
Por favor, note que o autor não reivindica quaisquer direitos autorais sobre os materiais apresentados neste site. As informações são coletadas e fornecidas na esperança de que sejam úteis para aqueles que procuram expandir seu conhecimento e aplicação do Power Query.
:::

A funcionalidade de pesquisa neste site deve simplificar o processo de descoberta da ampla gama de funções do Power Query. Certifique-se de conferir (Ctrl+K).
