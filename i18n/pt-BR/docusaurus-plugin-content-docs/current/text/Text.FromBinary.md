# Text.FromBinary


Decodifica dados de um formato binário em texto.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

Decodifica os dados, <code>binary</code>, de um valor binário em valor de texto, usando o tipo <code>encoding</code>.
