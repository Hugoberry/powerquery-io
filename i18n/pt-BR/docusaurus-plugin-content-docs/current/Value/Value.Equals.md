# Value.Equals


Especifica se os dois valores são iguais.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Remarks

Retornará true se o valor <code>value1</code> for igual ao valor <code>value2</code>; do contrário, false será retornado.

Values
