# Value.Alternates


## Description

Expressa planos de consulta alternativos.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Expressa planos de consulta alternativos em uma expressão do plano de consulta obtida por meio de Value.Expression(Value.Optimize(...)). Não se destina a outros usos.



## Category
Expression
