# DateTimeZone.UtcNow


Retorna a data e a hora atuais em UTC (fuso horário GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

Retorna a data e a hora atuais em UTC (fuso horário GMT).


## Examples

### Example #1 
Obtenha a data e a hora atuais em UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
