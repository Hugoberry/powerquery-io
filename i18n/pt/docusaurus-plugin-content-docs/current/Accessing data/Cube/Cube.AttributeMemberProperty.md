# Cube.AttributeMemberProperty


## Description

Returns a property of a dimension attribute.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Returns the property <code>propertyName</code> of dimension attribute <code>attribute</code>.



## Category
Cube
