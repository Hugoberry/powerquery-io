# Cube.MeasureProperty


## Description

Returns a property of a measure (cell property).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Returns the property <code>propertyName</code> of measure <code>measure</code>.



## Category
Cube
