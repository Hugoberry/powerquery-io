# Web.Page


## Description

Returns the contents of the HTML document broken into its constituent structures, as well as a representation of the full document and its text after removing tags.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Details

Returns the contents of the HTML document broken into its constituent structures, as well as a representation of the full document and its text after removing tags.
