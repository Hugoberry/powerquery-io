# Value.Equals


## Description

Returns whether two values are equal.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Returns true if value <code>value1</code> is equal to value <code>value2</code>, false otherwise.



## Category
Values
