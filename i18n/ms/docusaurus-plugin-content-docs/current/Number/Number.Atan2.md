# Number.Atan2


## Description

Mengembalikan lengkok tangen bagi pembahagian dua nombor.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Mengembalikan lengkok tangen bagi pembahagian dua nombor, <code>y</code> dan <code>x</code>. Pembahagian akan dibina sebagai <code>y</code>/<code>x</code>.
