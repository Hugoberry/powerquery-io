# Number.Random


## Description

Mengembalikan nombor rawak.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Mengembalikan nombor rawak antara 0 dan 1.


## Examples

### Example #1 
Dapatkan nombor rawak.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
