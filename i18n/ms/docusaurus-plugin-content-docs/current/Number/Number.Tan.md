# Number.Tan


## Description

Mengembalikan tangen bagi nombor.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Mengembalikan tangen bagi <code>number</code>.


## Examples

### Example #1 
Cari tangen bagi sudut 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
