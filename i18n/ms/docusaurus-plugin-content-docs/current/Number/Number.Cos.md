# Number.Cos


## Description

Mengembalikan kosinus bagi nombor.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Mengembalikan kosinus bagi <code>number</code>.


## Examples

### Example #1 
Cari kosinus bagi sudut 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
