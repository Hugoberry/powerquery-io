# Number.BitwiseAnd


## Description

Mengembalikan hasil daripada melaksanakan operasi bit &#34;Dan&#34; antara dua input.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

Mengembalikan hasil daripada melaksanakan operasi bit "Dan" antara <code>number1</code> dan <code>number2</code>.



## Category
Number.Bytes
