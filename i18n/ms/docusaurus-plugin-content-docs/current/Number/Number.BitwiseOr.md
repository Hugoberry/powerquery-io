# Number.BitwiseOr


## Description

Mengembalikan hasil daripada melaksanakan bit &#34;Atau&#34; antara dua input.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Mengembalikan hasil daripada melaksanakan bit "Atau" antara <code>number1</code> dan <code>number2</code>.



## Category
Number.Bytes
