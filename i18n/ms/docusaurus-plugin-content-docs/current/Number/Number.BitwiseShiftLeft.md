# Number.BitwiseShiftLeft


## Description

Menganjak set bit ke kiri.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Mengembalikan hasil daripada melaksanakan anjakan bit ke kiri pada <code>number1</code>, sebanyak bilangan bit yang ditentukan <code>number2</code>.

Number.Bytes
