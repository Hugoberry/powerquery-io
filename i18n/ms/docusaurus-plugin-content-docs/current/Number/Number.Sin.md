# Number.Sin


## Description

Mengembalikan sinus bagi nombor.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Mengembalikan sinus bagi <code>number</code>.


## Examples

### Example #1 
Cari sinus bagi sudut 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
