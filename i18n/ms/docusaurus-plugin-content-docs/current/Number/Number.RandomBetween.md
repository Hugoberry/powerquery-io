# Number.RandomBetween


## Description

Mengembalikan nombor rawak antara dua nombor.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Mengembalikan nombor rawak antara <code>bottom</code> dan <code>top</code>.


## Examples

### Example #1 
Dapatkan nombor rawak antara 1 dan 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
