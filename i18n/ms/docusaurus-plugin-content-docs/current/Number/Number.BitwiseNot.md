# Number.BitwiseNot


## Description

Mengembalikan bait yang mana setiap bit ialah bertentangan input.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Mengembalikan hasil daripada melaksanakan operasi bit "Not" pada <code>number</code>.



## Category
Number.Bytes
