# Value.Multiply


## Description

Mengembalikan hasil daripada dua nilai.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Mengembalikan hasil daripada mendarab <code>value1</code> dengan <code>value2</code>. Parameter <code>precision</code> pilihan boleh ditentukan, <code>Precision.Double</code> digunakan secara lalai.
