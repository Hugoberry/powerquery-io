# Value.NullableEquals


## Description

Mengembalikan sama ada dua nilai adalah sama.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Mengembalikan nol jika mana-mana argumen '<code>value1</code>', '<code>value2</code>' adalah nol, sebaliknya bersamaan dengan Value.Equals.

Values
