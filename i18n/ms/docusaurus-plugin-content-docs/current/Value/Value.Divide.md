# Value.Divide


## Description

Mengembalikan hasil daripada membahagi nilai pertama dengan nilai kedua.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Mengembalikan hasil daripada membahagi <code>value1</code> dengan <code>value2</code>. Parameter <code>precision</code> pilihan boleh ditentukan, <code>Precision.Double</code> digunakan secara lalai.
