# Value.Equals


## Description

Mengembalikan sama ada dua nilai adalah sama.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Mengembalikan benar jika nilai <code>value1</code> adalah sama dengan nilai <code>value2</code>, palsu jika sebaliknya.

Values
