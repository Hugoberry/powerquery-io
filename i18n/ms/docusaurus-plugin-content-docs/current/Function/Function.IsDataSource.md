# Function.IsDataSource


## Description

Mengembalikan sama ada fungsi tertentu dianggap sebagai sumber data.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Mengemablikan sama ada <code>function</code> dianggap sebagai sumber data.
