# GeometryPoint.From


## Description

Mencipta rekod yang mewakili titik geometri daripada bahagian.


## Syntax

```powerquery
GeometryPoint.From(
    x as number,
    y as number,
    optional z as number,
    optional m as number,
    optional srid as number
) as record
```


## Details

Mencipta rekod yang mewakili titik geometri daripada bahagian unsurnya, seperti koordinat X, koordinat Y dan jika ada, koordinat Z dan ukuran (M). Pengecam rujukan ruang pilihan (SRID) boleh diberikan jika berbeza daripada nilai lalai (0).



## Category
Record.Serialization
