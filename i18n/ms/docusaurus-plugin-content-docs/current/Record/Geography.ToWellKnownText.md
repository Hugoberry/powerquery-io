# Geography.ToWellKnownText


## Description

Menterjemahkan nilai titik geografi berstruktur ke dalam representasi Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Menterjemahkan nilai titik geografi berstruktur ke dalam representasi Well-Known Text (WKT) seperti yang ditakrifkan oleh Open Geospatial Consortium (OGC), juga format penyirian yang digunakan oleh kebanyakan pangkalan data termasuk SQL Server.
