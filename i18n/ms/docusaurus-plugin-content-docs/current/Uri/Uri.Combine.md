# Uri.Combine


## Description

Mengembalikan URI mutlak yang merupakan gabungan URI asas input dan URI relatif.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Mengembalikan URI mutlak yang merupakan gabungan input <code>baseUri</code> dan <code>relativeUri</code>.
