# Web.Page


## Description

Mengembalikan kandungan dokumen HTML yang dipecahkan menjadi struktur juzuknya, serta sebagai perwakilan dokumen penuh dan teksnya selepas mengalih keluar tag.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
