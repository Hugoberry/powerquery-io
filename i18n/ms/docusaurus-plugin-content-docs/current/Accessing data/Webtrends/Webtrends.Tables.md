# Webtrends.Tables


## Description

Masukkan ID Profil yang berkaitan dengan akaun Webtrends anda.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Mengembalikan jadual dengan data Webtrends yang berkaitan.
