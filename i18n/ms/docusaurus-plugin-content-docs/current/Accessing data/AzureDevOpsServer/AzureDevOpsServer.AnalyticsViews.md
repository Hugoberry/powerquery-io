# AzureDevOpsServer.AnalyticsViews


## Description

Masukkan nama organisasi dan projek.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Mengembalikan jadual bagi pandangan Analitis yang ditawarkan oleh Azure DevOps.
