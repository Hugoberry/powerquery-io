# AzureDevOpsServer.Views


## Description

Masukkan nama organisasi dan projek.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Mengembalikan jadual suapan OData yang ditawarkan oleh Azure DevOps.
