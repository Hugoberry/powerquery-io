# Cube.Transform


## Description

Menggunakan senarai fungsi kiub.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Menggunakan senarai fungsi kiub, <code>transforms</code> pada <code>cube</code>.
