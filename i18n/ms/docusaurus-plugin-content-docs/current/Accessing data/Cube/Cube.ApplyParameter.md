# Cube.ApplyParameter


## Description

Mengembalikan kiub selepas menggunakan parameter padanya.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Mengembalikan kiub selepas menggunakan <code>parameter</code> dengan <code>arguments</code> ke <code>cube</code>.



## Category
Cube
