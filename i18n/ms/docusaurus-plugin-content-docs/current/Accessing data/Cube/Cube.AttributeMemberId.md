# Cube.AttributeMemberId


## Description

Kembalikan pengecam ahli unik daripada nilai sifat ahli.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Kembalikan pengecam ahli unik daripada nilai sifat ahli. <code>attribute</code>. Kembalikan nol bagi sebarang nilai yang lain.
