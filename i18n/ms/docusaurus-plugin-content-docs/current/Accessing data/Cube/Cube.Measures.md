# Cube.Measures


## Description

Mengembalikan jadual yang mengandungi set ukuran yang tersedia.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

Mengembalikan jadual yang mengandungi set ukuran yang tersedia dalam <code>cube</code>.     Setiap ukuran diwakili sebagai fungsi. Ukuran boleh digunakan pada kiub menggunakan Cube.AddMeasureColumn.



## Category
Cube
