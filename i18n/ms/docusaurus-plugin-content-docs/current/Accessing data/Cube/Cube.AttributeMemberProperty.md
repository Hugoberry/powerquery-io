# Cube.AttributeMemberProperty


## Description

Mengembalikan sifat bagi atribut dimensi.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Mengembalikan sifat <code>propertyName</code> atribut dimensi <code>attribute</code>.



## Category
Cube
