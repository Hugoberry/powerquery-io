# Cube.MeasureProperty


## Description

Mengembalikan sifat ukuran (sifat sel).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Mengembalikan sifat <code>propertyName</code> atribut dimensi <code>measure</code>.



## Category
Cube
