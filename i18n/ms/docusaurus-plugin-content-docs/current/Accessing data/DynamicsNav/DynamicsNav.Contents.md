# DynamicsNav.Contents


## Description

Masukkan URL titik tamat perkhidmatan Odata NAV Dinamik anda.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Mengembalikan jadual dengan data NAV Dinamik yang berkaitan.
