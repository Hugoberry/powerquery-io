# Text.ToBinary


## Description

Mengekodkan teks kepada bentuk perduaan.


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

Mengekodkan nilai teks yang ditentukan, <code>text</code>, kepada nilai perduaan menggunakan <code>encoding</code> yang ditentukan.
