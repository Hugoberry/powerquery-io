# Text.Length


## Description

Mengembalikan bilangan aksara.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Mengembalikan bilangan aksara dalam teks <code>text</code>.


## Examples

### Example #1 
Cari bilangan aksara yang ada dalam teks &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
