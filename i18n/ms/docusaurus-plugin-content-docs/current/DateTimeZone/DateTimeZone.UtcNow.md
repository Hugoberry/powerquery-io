# DateTimeZone.UtcNow


## Description

Mengembalikan tarikh dan masa semasa dalam UTC (zon waktu GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Mengembalikan tarikh dan masa semasa dalam UTC (zon waktu GMT).


## Examples

### Example #1 
Dapatkan tarikh &amp; masa semasa dalam UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
