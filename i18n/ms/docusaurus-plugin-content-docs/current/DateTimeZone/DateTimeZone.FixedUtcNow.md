# DateTimeZone.FixedUtcNow


## Description

Mengembalikan tarikh dan masa semasa dalam UTC (zon waktu GMT). Nilai ini adalah tetap dan tidak akan berubah dengan panggilan berikutnya.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
