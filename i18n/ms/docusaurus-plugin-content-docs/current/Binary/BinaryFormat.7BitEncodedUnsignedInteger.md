# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Format perduaan yang membaca integer tanpa tanda 64 bit yang dikodkan menggunakan pengekodan panjang boleh ubah 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
