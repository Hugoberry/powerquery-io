# BinaryFormat.7BitEncodedSignedInteger


## Description

Format perduaan yang membaca integer bertandatangan 64 bit yang dikodkan menggunakan pengekodan panjang boleh ubah 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
