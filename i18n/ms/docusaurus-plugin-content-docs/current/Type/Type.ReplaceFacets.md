# Type.ReplaceFacets


## Description

Menggantikan faset jenis.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Menggantikan faset <code>type</code> dengan faset yang terkandung dalam rekod <code>facets</code>.
