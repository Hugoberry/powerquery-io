# Type.TableColumn


## Description

Mengembalikan jenis lajur dalam jadual.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Mengembalikan jenis lajur <code>column</code> dalam jenis jadual <code>tableType</code>.



## Category
Type
