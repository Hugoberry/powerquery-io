# Type.ListItem


## Description

Kembalikan jenis item daripada jenis senarai.


## Syntax

```powerquery
Type.ListItem(
    type as type
) as type
```


## Details

Kembalikan jenis item daripada senarai <code>type</code>.


## Examples

### Example #1 
Cari jenis item daripada senarai &lt;code&gt;\{number}&lt;/code&gt;.
```powerquery
Type.ListItem(type {number})
```

Result: 
```powerquery
type number
```
