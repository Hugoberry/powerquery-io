# DateTime.LocalNow


## Description

Mengembalikan tarikh dan masa semasa dalam zon waktu tempatan.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Mengembalikan nilai <code>datetime</code> yang disetkan kepada tarikh dan masa semasa pada sistem.



## Category
DateTime
