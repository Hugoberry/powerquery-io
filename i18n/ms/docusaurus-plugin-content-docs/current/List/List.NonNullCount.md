# List.NonNullCount


## Description

Mengembalikan bilangan item bukan nol dalam senarai.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Mengembalikan bilangan item bukan nol dalam senarai <code>list</code>.

List.Information
