# List.Count


## Description

Mengembalikan bilangan item dalam senarai.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Mengembalikan bilangan item dalam senarai <code>list</code>.


## Examples

### Example #1 
Cari bilangan nilai dalam senarai \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
