# Marketo.Tables


## Description

Καταχωρήστε τη διεύθυνση URL του τελικού σημείου API Marketo REST που σχετίζεται με το λογαριασμό σας.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Επιστρέφει έναν πίνακα με σχετικά δεδομένα Marketo.
