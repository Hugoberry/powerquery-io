# Web.Page


## Description

Επιστρέφει τα περιεχόμενα του εγγράφου HTML κατακερματισμένα στα δομικά στοιχεία του, καθώς και μια αναπαράσταση του πλήρους εγγράφου και του κειμένου του μετά την κατάργηση των ετικετών.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
