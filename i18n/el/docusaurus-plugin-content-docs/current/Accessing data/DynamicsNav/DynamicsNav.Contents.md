# DynamicsNav.Contents


## Description

Εισαγάγετε τη διεύθυνση URL του τελικού σημείου της υπηρεσίας OData Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Επιστρέφει έναν πίνακα με σχετικά δεδομένα Dynamics NAV.
