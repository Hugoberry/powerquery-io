# Identity.IsMemberOf


## Description

Καθορίζει αν μια ταυτότητα είναι μέλος μιας συλλογής ταυτοτήτων.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details

Καθορίζει αν μια ταυτότητα είναι μέλος μιας συλλογής ταυτοτήτων.
