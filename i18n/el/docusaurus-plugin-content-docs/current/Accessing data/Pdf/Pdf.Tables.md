# Pdf.Tables


## Description

Επιστρέφει τους πίνακες που βρέθηκαν σε ένα αρχείο PDF.


## Syntax

```powerquery
Pdf.Tables(
    pdf as binary,
    optional options as record
) as table
```


## Details

Επιστρέφει τυχόν πίνακες που βρίσκονται στο <code>pdf</code>. Μπορεί να παρασχεθεί μια προαιρετική παράμετρος εγγραφής <code>options</code> για τον καθορισμό πρόσθετων ιδιοτήτων. Η εγγραφή μπορεί να περιέχει τα ακόλουθα πεδία:    <ul><li><code>Implementation</code> : Η έκδοση του αλγορίθμου που θα χρησιμοποιηθεί κατά τον προσδιορισμό πινάκων. Οι παλιές εκδόσεις είναι διαθέσιμες μόνο για λόγους συμβατότητας με προηγούμενες εκδόσεις, προκειμένου τα παλιά ερωτήματα να μην καταστρέφονται από ενημερώσεις αλγορίθμων. Η νεότερη έκδοση θα πρέπει να δίνει πάντα τα καλύτερα αποτελέσματα. Οι έγκυρες τιμές είναι &quot;1.3&quot;, &quot;1.2&quot;, &quot;1.1&quot; ή null.</li><li><code>StartPage</code> : Καθορίζει την πρώτη σελίδα στην περιοχή σελίδων προς εξέταση. Προεπιλογή: 1.</li><li><code>EndPage</code> : Καθορίζει την τελευταία σελίδα στην περιοχή σελίδων προς εξέταση. Προεπιλογή: η τελευταία σελίδα του εγγράφου.</li><li><code>MultiPageTables</code> : Ελέγχει αν παρόμοιοι πίνακες σε διαδοχικές σελίδες θα συνδυάζονται αυτόματα σε έναν πίνακα. Προεπιλογή: true.</li><li><code>EnforceBorderLines</code> : Ελέγχει αν οι γραμμές περιγράμματος επιβάλλονται πάντα ως όρια κελιού (όταν είναι true) ή χρησιμοποιούνται απλώς ως μία ένδειξη ανάμεσα σε πολλές για τον καθορισμό των ορίων κελιού (όταν είναι false). Προεπιλογή: false.</li></ul>    


## Examples

### Example #1 
Επιστρέφει τους πίνακες που περιέχονται στο sample.pdf.
```powerquery
Pdf.Tables(File.Contents("c:\sample.pdf"))
```

Result: 
```powerquery
#table({"Name", "Kind", "Data"}, ...)
```
