# GoogleAnalytics.Accounts


## Description

Επιστρέφει τους λογαριασμούς του Google Analytics.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

Επιστρέφει τους λογαριασμούς του Google Analytics που είναι προσβάσιμοι από το τρέχον διαπιστευτήριο.

Accessing data
