# Cdm.MapToEntity


## Description

Επιστρέφει έναν πίνακα με στήλες που έχουν αντιστοιχιστεί στα χαρακτηριστικά μιας οντότητας στο κοινό μοντέλο δεδομένων, συμπεριλαμβανομένων των τύπων δεδομένων.


## Syntax

```powerquery
Cdm.MapToEntity(
    table as table,
    columnDefinitions as any,
    optional defaultType as type,
    optional defaultCdmClass as text
) as table
```


## Details

Επιστρέφει έναν πίνακα με στήλες που έχουν αντιστοιχιστεί στα χαρακτηριστικά μιας οντότητας στο κοινό μοντέλο δεδομένων, συμπεριλαμβανομένων των τύπων δεδομένων. Οι στήλες από τον ορισμό οντότητας θα προστεθούν, αν δεν έχουν αντιστοιχιστεί, και οι στήλες που δεν έχουν αντιστοιχιστεί θα καταργηθούν.
