# AzureDevOpsServer.Views


## Description

Εισαγάγετε ονόματα οργανισμού και έργου.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Επιστρέφει έναν πίνακα τροφοδοσιών OData που παρέχονται από το Azure DevOps.
