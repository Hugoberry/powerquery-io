# Cube.Properties


## Description

Επιστρέφει έναν πίνακα που περιέχει το σύνολο των διαθέσιμων ιδιοτήτων για διαστάσεις που αναπτύσσονται στον κύβο.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```
