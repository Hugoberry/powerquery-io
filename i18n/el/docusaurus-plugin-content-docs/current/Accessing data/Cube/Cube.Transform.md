# Cube.Transform


## Description

Εφαρμόζει μια λίστα συναρτήσεων κύβου.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Εφαρμόζει τις συναρτήσεις κύβου λίστας, <code>transforms</code>, στο <code>cube</code>.

Cube
