# Cube.ApplyParameter


## Description

Επιστρέφει έναν κύβο, αφού εφαρμοστεί παράμετρος σε αυτόν.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Επιστρέφει έναν κύβο, αφού εφαρμοστεί <code>parameter</code> με <code>arguments</code> σε <code>cube</code>.



## Category
Cube
