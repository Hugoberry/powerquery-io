# Cube.MeasureProperty


## Description

Επιστρέφει μια ιδιότητα μιας μέτρησης (ιδιότητα κελιού).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Επιστρέφει την ιδιότητα <code>propertyName</code> της μέτρησης <code>measure</code>.



## Category
Cube
