# AzureStorage.Tables


## Description

Επιστρέφει έναν πίνακα περιήγησης που περιέχει τους πίνακες που βρέθηκαν στον καθορισμένο λογαριασμό από έναν θάλαμο χώρου αποθήκευσης του Azure.


## Syntax

```powerquery
AzureStorage.Tables(
    account as text,
    optional options as record
) as table
```


## Details

Επιστρέφει έναν πίνακα περιήγησης που περιέχει μια γραμμή για κάθε πίνακα που βρέθηκε στη διεύθυνση URL του λογαριασμού, <code>account</code>, από έναν θάλαμο αποθήκευσης Azure. Κάθε γραμμή περιέχει μια σύνδεση προς τον πίνακα Azure. Μπορεί να παρασχεθεί μια προαιρετική παράμετρος εγγραφής, <code>options</code>, για τον καθορισμό πρόσθετων ιδιοτήτων. Η εγγραφή μπορεί να περιέχει τα ακόλουθα πεδία:    <ul><li><code>Timeout</code> : Μια διάρκεια που ελέγχει για πόσο χρονικό διάστημα θα γίνει αναμονή πριν από την εγκατάλειψη της αίτησης στον διακομιστή. Η προεπιλεγμένη τιμή εξαρτάται από την προέλευση.</li></ul>
