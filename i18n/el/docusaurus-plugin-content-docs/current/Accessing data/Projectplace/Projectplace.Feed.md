# Projectplace.Feed


## Description

Εισαγάγετε τη διεύθυνση URL του λογαριασμού σας Planview Projectplace.


## Syntax

```powerquery
Projectplace.Feed(
    ODataURL as text
) as table
```


## Details

Επιστρέφει έναν πίνακα με σχετικά δεδομένα Planview Projectplace.
