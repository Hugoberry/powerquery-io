# List.RemoveFirstN


## Description

Επιστρέφει μια λίστα που παραλείπει τον καθορισμένο αριθμό στοιχείων από την αρχή της λίστας.


## Syntax

```powerquery
List.RemoveFirstN(
    list as list,
    optional countOrCondition as any
) as list
```


## Details

Επιστρέφει μια λίστα που καταργεί το πρώτο στοιχείο της λίστας <code>list</code>. Εάν η λίστα <code>list</code> είναι κενή, επιστρέφεται μια κενή λίστα.Αυτή η συνάρτηση μπορεί να λάβει μια προαιρετική παράμετρο, <code>countOrCondition</code>, προκειμένου να υποστηρίξει την κατάργηση πολλαπλών τιμών όπως περιγράφεται παρακάτω.  <ul> <li>Εάν καθοριστεί αριθμός, καταργούνται μέχρι τόσα στοιχεία. </li> <li>Εάν καθοριστεί συνθήκη, η λίστα που επιστρέφεται ξεκινά με το πρώτο στοιχείο της λίστας <code>list</code> που ικανοποιεί τα κριτήρια. Από τη στιγμή που ένα στοιχείο δεν ικανοποιεί τη συνθήκη, δεν εξετάζονται περαιτέρω στοιχεία. </li> <li>Εάν αυτή η παράμετρος είναι null, παρατηρείται η προεπιλεγμένη συμπεριφορά. </li> </ul>


## Examples

### Example #1 
Δημιουργήστε μια λίστα από τη λίστα \{1, 2, 3, 4, 5} χωρίς τους πρώτους 3 αριθμούς.
```powerquery
List.RemoveFirstN({1, 2, 3, 4, 5}, 3)
```

Result: 
```powerquery
{4, 5}
```


### Example #2 
Δημιουργήστε μια λίστα από τη λίστα \{5, 4, 2, 6, 1} που ξεκινά με αριθμό μικρότερο του 3.
```powerquery
List.RemoveFirstN({5, 4, 2, 6, 1}, each _ > 3)
```

Result: 
```powerquery
{2, 6, 1}
```
