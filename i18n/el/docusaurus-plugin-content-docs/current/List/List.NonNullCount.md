# List.NonNullCount


## Description

Επιστρέφει τον αριθμό των στοιχείων της λίστας που δεν είναι null.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Επιστρέφει τον αριθμό των στοιχείων που δεν είναι null στη λίστα <code>list</code>.



## Category
List.Information
