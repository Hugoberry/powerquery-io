# List.Union


## Description

Επιστρέφει την ένωση των τιμών λίστας που βρέθηκαν στην είσοδο.


## Syntax

```powerquery
List.Union(
    lists as list,
    optional equationCriteria as any
) as list
```


## Details

Λαμβάνει μια λίστα με τις λίστες <code>lists</code>, ενώνει τα στοιχεία των επιμέρους λιστών και τα επιστρέφει στη λίστα εξόδου. Ως αποτέλεσμα αυτού, η επιστρεφόμενη λίστα περιέχει όλα τα στοιχεία όλων των λιστών εισόδου.    Αυτή η λειτουργία διατηρεί παραδοσιακή σημασιολογία θύλακα, επομένως οι διπλότυπες τιμές εκλαμβάνονται ως συμφωνία ως μέρος της ένωσης.    Μια προαιρετική τιμή κριτηρίων ισότητας, <code>equationCriteria</code>, μπορεί να καθοριστεί για τον έλεγχο ισότητας. 


## Examples

### Example #1 
Δημιουργήστε μια ένωση των λιστών \{1..5}, \{2..6}, \{3..7}.
```powerquery
List.Union({{1..5}, {2..6}, {3..7}})
```

Result: 
```powerquery
{1, 2, 3, 4, 5, 6, 7}
```
