# Number.BitwiseNot


## Description

Επιστρέφει ένα byte όπου κάθε bit είναι το αντίθετο της εισόδου.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Επιστρέφει το αποτέλεσμα μιας λειτουργίας bit "Not" στο <code>number</code>.



## Category
Number.Bytes
