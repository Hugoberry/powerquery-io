# Number.Tan


## Description

Επιστρέφει την εφαπτομένη του αριθμού.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Επιστρέφει την εφαπτομένη του <code>number</code>.


## Examples

### Example #1 
Βρείτε την εφαπτομένη της γωνίας 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
