# Number.Random


## Description

Επιστρέφει έναν τυχαίο αριθμό.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Επιστρέφει έναν τυχαίο αριθμό ανάμεσα στο 0 και το 1.


## Examples

### Example #1 
Λάβετε έναν τυχαίο αριθμό.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
