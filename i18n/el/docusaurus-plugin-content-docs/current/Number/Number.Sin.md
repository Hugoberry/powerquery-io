# Number.Sin


## Description

Επιστρέφει το ημίτονο του αριθμού.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Επιστρέφει το ημίτονο του <code>number</code>.


## Examples

### Example #1 
Βρείτε το ημίτονο της γωνίας 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
