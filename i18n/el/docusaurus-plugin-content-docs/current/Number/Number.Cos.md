# Number.Cos


## Description

Επιστρέφει το συνημίτονο του αριθμού.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Επιστρέφει το συνημίτονο του <code>number</code>.


## Examples

### Example #1 
Βρείτε το συνημίτονο της γωνίας 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
