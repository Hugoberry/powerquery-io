# Number.Atan2


## Description

Επιστρέφει το τόξο εφαπτομένης του πηλίκου των δύο αριθμών.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Επιστρέφει το τόξο εφαπτομένης του πηλίκου των δύο αριθμών <code>y</code> και <code>x</code>. Η διαίρεση συντάσσεται ως <code>y</code>/<code>x</code>.
