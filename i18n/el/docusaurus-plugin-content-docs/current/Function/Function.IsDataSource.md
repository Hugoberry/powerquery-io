# Function.IsDataSource


## Description

Προσδιορίζει αν μια συγκεκριμένη συνάρτηση θεωρείται προέλευση δεδομένων ή όχι και επιστρέφει τα αποτελέσματα.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Προσδιορίζει αν το <code>function</code> θεωρείται προέλευση δεδομένων ή όχι και επιστρέφει τα αποτελέσματα.
