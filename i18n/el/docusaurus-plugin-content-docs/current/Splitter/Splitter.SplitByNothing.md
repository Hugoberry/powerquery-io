# Splitter.SplitByNothing


## Description

Επιστρέφει μια συνάρτηση που δεν διαιρεί, επιστρέφοντας το όρισμά της ως λίστα ενός στοιχείου.


## Syntax

```powerquery
Splitter.SplitByNothing(

) as function
```
