# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Μια δυαδική μορφή που διαβάζει έναν ακέραιο 64 bit χωρίς πρόσημο που κωδικοποιήθηκε με κωδικοποίηση μεταβλητού μήκους 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details
