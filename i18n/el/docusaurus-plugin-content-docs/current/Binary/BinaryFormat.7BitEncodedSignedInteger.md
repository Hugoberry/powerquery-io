# BinaryFormat.7BitEncodedSignedInteger


## Description

Μια δυαδική μορφή που διαβάζει έναν ακέραιο 64 bit με πρόσημο που κωδικοποιήθηκε με κωδικοποίηση μεταβλητού μήκους 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details
