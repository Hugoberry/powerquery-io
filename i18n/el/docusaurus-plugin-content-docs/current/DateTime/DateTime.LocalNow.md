# DateTime.LocalNow


## Description

Επιστρέφει την τρέχουσα ημερομηνία και ώρα στην τοπική ζώνη ώρας.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Επιστρέφει μια τιμή <code>datetime</code> που έχει οριστεί στην τρέχουσα ημερομηνία και ώρα του συστήματος.



## Category
DateTime
