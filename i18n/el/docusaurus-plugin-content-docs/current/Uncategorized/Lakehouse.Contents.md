# Lakehouse.Contents


## Description

Εισαγωγή δεδομένων από ένα Lakehouse


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Details

Συνδεθείτε σε όλα τα Lakehouses στα οποία έχετε πρόσβαση και επιλέξτε τις οντότητες που θέλετε να χρησιμοποιήσετε
