# Error.Record


## Description

Επιστρέφει μια εγγραφή σφάλματος από τις παρεχόμενες τιμές κειμένου για την αιτία, το μήνυμα και τις λεπτομέρειες.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
