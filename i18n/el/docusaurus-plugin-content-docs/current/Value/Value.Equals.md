# Value.Equals


## Description

Επιστρέφει αν δύο τιμές είναι ίσες.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Επιστρέφει true εάν η τιμή <code>value1</code> ισούται με την τιμή <code>value2</code>, διαφορετικά false.



## Category
Values
