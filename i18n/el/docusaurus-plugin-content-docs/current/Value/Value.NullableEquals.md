# Value.NullableEquals


## Description

Επιστρέφει αν δύο τιμές είναι ίσες.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Επιστρέφει την τιμή null εάν κάποιο από τα δύο ορίσματα '<code>value1</code>', '<code>value2</code>' είναι null, διαφορετικά ισοδυναμεί με Value.Equals.



## Category
Values
