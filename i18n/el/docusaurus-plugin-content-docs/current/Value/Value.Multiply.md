# Value.Multiply


## Description

Επιστρέφει το γινόμενο των δύο τιμών.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Επιστρέφει το διανυσματικό γινόμενο του πολλαπλασιασμού <code>value1</code> επί <code>value2</code>. Μπορεί να καθοριστεί μια προαιρετική παράμετρος <code>precision</code>, από προεπιλογή χρησιμοποιείται η <code>Precision.Double</code>.
