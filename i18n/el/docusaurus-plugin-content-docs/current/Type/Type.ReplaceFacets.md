# Type.ReplaceFacets


## Description

Αντικαθιστά τις κλάσεις facet ενός τύπου.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Αντικαθιστά τις κλάσεις facet του <code>type</code> με τις κλάσεις facet που περιέχονται στην εγγραφή <code>facets</code>.

Type
