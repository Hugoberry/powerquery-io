# Type.TableColumn


## Description

Επιστρέφει τον τύπο μιας στήλης σε έναν πίνακα.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Επιστρέφει τον τύπο μιας στήλης <code>column</code> σε έναν τύπο πίνακα <code>tableType</code>.

Type
