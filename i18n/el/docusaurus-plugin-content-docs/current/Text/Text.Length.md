# Text.Length


## Description

Επιστρέφει τον αριθμό των χαρακτήρων.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Επιστρέφει τον αριθμό χαρακτήρων του κειμένου <code>text</code>.


## Examples

### Example #1 
Βρείτε πόσοι χαρακτήρες υπάρχουν στο κείμενο &#34;Hello, World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
