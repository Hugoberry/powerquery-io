# DateTimeZone.UtcNow


## Description

Επιστρέφει την τρέχουσα ημερομηνία και ώρα σε μορφή UTC (η ζώνη ώρας GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Επιστρέφει την τρέχουσα ημερομηνία και ώρα σε μορφή UTC (η ζώνη ώρας GMT).


## Examples

### Example #1 
Λάβετε την τρέχουσα ημερομηνία και ώρα σε μορφή UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
