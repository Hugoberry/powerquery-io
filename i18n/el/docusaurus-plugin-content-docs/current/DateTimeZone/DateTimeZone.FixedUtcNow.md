# DateTimeZone.FixedUtcNow


## Description

Επιστρέφει την τρέχουσα ημερομηνία και ώρα σε μορφή UTC (η ζώνη ώρας GMT). Αυτή η τιμή είναι σταθερή και δεν θα αλλάξει με τις διαδοχικές κλήσεις.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
