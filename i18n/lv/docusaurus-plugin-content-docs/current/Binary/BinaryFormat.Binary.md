# BinaryFormat.Binary


## Description

Atgriež bināru formātu, kas satur bināru vērtību.


## Syntax

```powerquery
BinaryFormat.Binary(
    optional length as any
) as function
```


## Details

Atgriež bināru formātu, kas satur bināru vērtību. Ja ir norādīts <code>length</code>, tas nozīmē, ka binārajā vērtībā ir tik daudz baitu. Ja nav norādīts <code>length</code>, tas nozīmē, ka binārajā vērtībā ir atlikušie baiti. Vērtību <code>length</code> var norādīt kā skaitli vai kā bināru formātu garumam, kas atrodas bināro datu priekšā.
