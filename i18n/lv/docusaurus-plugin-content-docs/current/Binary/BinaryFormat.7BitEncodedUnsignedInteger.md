# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Binārs formāts, kas satur 64 bitu veselo skaitli bez zīmes, kurš tika iekodēts, izmantojot 7 bitu mainīga garuma kodējumu.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details
