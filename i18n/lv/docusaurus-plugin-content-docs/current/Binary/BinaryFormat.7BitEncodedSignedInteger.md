# BinaryFormat.7BitEncodedSignedInteger


## Description

Binārs formāts, kas satur 64 bitu veselo skaitli ar zīmi, kurš tika iekodēts, izmantojot 7 bitu mainīga garuma kodējumu.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details

Binārs formāts, kas satur 64 bitu veselo skaitli ar zīmi, kurš tika iekodēts, izmantojot 7 bitu mainīga garuma kodējumu.
