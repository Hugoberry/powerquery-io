# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Apgriež norādīto <code>text</code>.


## Examples

### Example #1 
Apgrieziet tekstu 123.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
