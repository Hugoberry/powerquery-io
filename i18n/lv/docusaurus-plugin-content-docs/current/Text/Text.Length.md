# Text.Length


## Description

Tiek atgriezts rakstzīmju skaits.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Tiek atgriezts rakstzīmju skaits tekstā <code>text</code>.


## Examples

### Example #1 
Uzziniet, cik rakstzīmju ir tekstā Hello World.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
