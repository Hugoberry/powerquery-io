# List.Count


## Description

Tiek atgriezts saraksta vienumu skaits.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Tiek atgriezts saraksta <code>list</code> vienumu skaits.


## Examples

### Example #1 
Iegūstiet saraksta \{1, 2, 3} vērtību skaitu.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
