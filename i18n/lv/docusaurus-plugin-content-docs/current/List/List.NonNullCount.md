# List.NonNullCount


## Description

Tiek atgriezts to saraksta vienumu skaits, kuru vērtība nav Null.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Tiek atgriezts to saraksta <code>list</code> vienumu skaits, kuru vērtība nav Null.
