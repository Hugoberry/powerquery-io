# Number.Cos


## Description

Tiek atgriezts skaitļa kosinuss.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Tiek atgriezts skaitļa <code>number</code> kosinuss.


## Examples

### Example #1 
Iegūstiet 0 grādu leņķa kosinusu.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
