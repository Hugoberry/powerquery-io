# Number.BitwiseShiftRight


## Description

Bitu kopa tiek pārbīdīta pa labi.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Tiek atgriezts rezultāts, kas tiek iegūts, veicot vērtības <code>number1</code> bitu pārbīdi pa labi par norādīto bitu skaitu <code>number2</code>.



## Category
Number.Bytes
