# Number.Sin


## Description

Tiek atgriezts skaitļa sinuss.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Tiek atgriezts skaitļa <code>number</code> sinuss.


## Examples

### Example #1 
Iegūstiet 0 grādu leņķa sinusu.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
