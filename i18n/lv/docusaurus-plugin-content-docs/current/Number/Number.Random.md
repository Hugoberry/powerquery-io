# Number.Random


## Description

Tiek atgriezts gadījumskaitlis.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Tiek atgriezts gadījumskaitlis diapazonā no 0 līdz 1.


## Examples

### Example #1 
Iegūstiet gadījumskaitli.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
