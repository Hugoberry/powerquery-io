# Number.Atan2


## Description

Tiek atgriezts divu skaitļu dalījuma arktangenss.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Tiek atgriezts divu skaitļu <code>y</code> un <code>x</code> dalījuma arktangenss. Dalījums tiek veidots šādi: <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
