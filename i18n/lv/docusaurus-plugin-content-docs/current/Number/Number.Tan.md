# Number.Tan


## Description

Tiek atgriezts skaitļa tangenss.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Tiek atgriezts skaitļa <code>number</code> tangenss.


## Examples

### Example #1 
Iegūstiet 1 grāda leņķa tangensu.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
