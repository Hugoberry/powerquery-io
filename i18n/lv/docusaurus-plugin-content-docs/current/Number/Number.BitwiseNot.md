# Number.BitwiseNot


## Description

Tiek atgriezts baits, kurā katra bita vērtība ir pretēja ievades datiem.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Tiek atgriezts rezultāts, kas tiek iegūts, ar vērtību <code>number</code> veicot bitu operāciju Not.



## Category
Number.Bytes
