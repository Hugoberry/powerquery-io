# Number.Factorial


## Description

Tiek atgriezts skaitļa faktoriāls.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Tiek atgriezts skaitļa <code>number</code> faktoriāls.


## Examples

### Example #1 
Iegūstiet skaitļa 10 faktoriālu.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
