# Type.TableColumn


## Description

Tiek atgriezts kolonnas tips tabulā.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Tiek atgriezts kolonnas tips <code>column</code> tabulas tipā <code>tableType</code>.

Type
