# Type.ListItem


## Description

Atgriež vienuma tipu no saraksta tipa.


## Syntax

```powerquery
Type.ListItem(
    type as type
) as type
```


## Details

Atgriež vienuma tipu no saraksta <code>type</code>.


## Examples

### Example #1 
Atrodiet vienuma tipu sarakstā &lt;code&gt;\{number}&lt;/code&gt;.
```powerquery
Type.ListItem(type {number})
```

Result: 
```powerquery
type number
```
