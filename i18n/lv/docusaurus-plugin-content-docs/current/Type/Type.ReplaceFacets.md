# Type.ReplaceFacets


## Description

Aizstāj tipa fasetes.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Aizstāj <code>type</code> fasetes ar fasetēm, kuras atrodas ierakstā <code>facets</code>.

Type
