# DateTimeZone.UtcNow


## Description

Atgriež pašreizējo datumu un laiku UTC (GMT laika joslā).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Atgriež pašreizējo datumu un laiku UTC (GMT laika joslā).


## Examples

### Example #1 
Iegūstiet pašreizējo datumu un laiku UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
