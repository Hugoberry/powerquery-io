# Web.Page


## Description

Tiek atgriezts tā HTML dokumenta saturs, kas ir sadalīts tā saistītajās struktūrās, kā arī kā pilna dokumenta atveidojums un tā teksts pēc tagu noņemšanas.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
