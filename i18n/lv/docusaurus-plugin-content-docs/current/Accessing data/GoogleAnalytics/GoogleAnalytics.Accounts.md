# GoogleAnalytics.Accounts


## Description

Atgriež Google Analytics kontus.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

Atgriež Google Analytics kontus, kam var piekļūt, izmantojot pašreizējos akreditācijas datus.
