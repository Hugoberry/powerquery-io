# Marketo.Tables


## Description

Ievadiet tā Marketo REST API galapunkta vietrādi URL, kas ir saistīts ar jūsu kontu.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Šī funkcija atgriež tabulu, kurā norādīti attiecīgi dati no pakalpojuma Marketo.
