# VSTS.Views


## Description

Ievadiet organizācijas un projekta nosaukumu.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Tiek atgriezta Azure DevOps nodrošinātā OData plūsmu tabula.
