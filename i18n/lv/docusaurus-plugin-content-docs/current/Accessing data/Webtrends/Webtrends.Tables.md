# Webtrends.Tables


## Description

Ievadiet ar Webtrends kontu saistīto profila ID.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Atgriež tabulu ar atbilstošiem datiem no risinājuma Webtrends.
