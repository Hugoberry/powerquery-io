# AzureDevOpsServer.AnalyticsViews


## Description

Ievadiet organizācijas un projekta nosaukumu.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Tiek atgriezta Azure DevOps nodrošinātā Analytics skatu tabula.
