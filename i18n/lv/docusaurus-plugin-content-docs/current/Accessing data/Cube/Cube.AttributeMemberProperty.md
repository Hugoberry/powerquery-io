# Cube.AttributeMemberProperty


## Description

Atgriež dimensijas atribūta rekvizītu.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Atgriež rekvizītu <code>propertyName</code> no dimensijas atribūta <code>attribute</code>.



## Category
Cube
