# Cube.MeasureProperties


## Description

Atgriež tabulu ar pieejamo mēru rekvizītu, kas ir izvērsti kubā, kopu.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Atgriež tabulu ar pieejamo rekvizītu kopu mēriem, kas ir izvērsti kubā.
