# Cube.MeasureProperty


## Description

Atgriež mēra rekvizītu (šūnas rekvizīts).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Atgriež mēra <code>measure</code> rekvizītu <code>propertyName</code>.
