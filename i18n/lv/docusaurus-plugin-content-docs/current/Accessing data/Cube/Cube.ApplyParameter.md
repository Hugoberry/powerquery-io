# Cube.ApplyParameter


## Description

Atgriež kubu pēc parametra lietošanas.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Atgriež kubu pēc <code>parameter</code> ar <code>arguments</code> lietošanas vienumam <code>cube</code>.

Cube
