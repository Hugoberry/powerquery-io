# Value.Compare


## Description

Atgriež vērtību -1, 0 vai 1, pamatojoties uz to, vai pirmā vērtība ir mazāka par, vienāda ar vai lielāka par otro.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```
