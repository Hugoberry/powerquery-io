# Value.NullableEquals


## Description

Tiek atgriezts, vai abas vērtības ir vienādas.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Ja arguments <code>value1</code>, <code>value2</code> ir Null, tiek atgriezta vērtība Null, pretējā gadījumā tiek atgriezts Value.Equals ekvivalents.
