# Value.Equals


## Description

Tiek atgriezts, vai abas vērtības ir vienādas.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Ja vērtība <code>value1</code> ir vienāda ar vērtību <code>value2</code>, tiek atgriezta vērtība True, pretējā gadījumā tiek atgriezta vērtība False.
