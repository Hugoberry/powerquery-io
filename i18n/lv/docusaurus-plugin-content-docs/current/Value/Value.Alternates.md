# Value.Alternates


## Description

Izsaka alternatīvos vaicājumu plānus.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Izsaka alternatīvos vaicājumu plānus vaicājumu plāna izteiksmē, kas iegūta, izmantojot Value.Expression(Value.Optimize(...)). Nav paredzēts citām vajadzībām.



## Category
Expression
