# Error.Record


## Description

Atgriež kļūdas ierakstu no sniegtajām teksta vērtībām attiecībā uz iemeslu, ziņojumu un detalizētu informāciju.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
