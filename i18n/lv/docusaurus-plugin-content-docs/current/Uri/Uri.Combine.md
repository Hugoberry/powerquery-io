# Uri.Combine


## Description

Tiek atgriezts absolūts URI, kas tiek iegūts, apvienojot ievades bāzes URI un relatīvo URI.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Tiek atgriezts absolūts URI, kas tiek iegūts, apvienojot ievades <code>baseUri</code> un <code>relativeUri</code>.
