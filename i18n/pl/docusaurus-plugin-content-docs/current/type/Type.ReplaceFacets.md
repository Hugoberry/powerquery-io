# Type.ReplaceFacets


Zastępuje aspekty typu.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Remarks

Zastępuje aspekty elementu <code>type</code> aspektami zawartymi w rekordzie <code>facets</code>.
