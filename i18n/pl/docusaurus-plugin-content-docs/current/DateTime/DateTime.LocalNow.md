# DateTime.LocalNow


## Description

Zwraca bieżącą datę i godzinę w lokalnej strefie czasowej.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Zwraca wartość typu <code>datetime</code> reprezentującą aktualną datę i godzinę ustawioną w systemie.



## Category
DateTime
