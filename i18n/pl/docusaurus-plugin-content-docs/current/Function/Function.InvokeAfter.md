# Function.InvokeAfter


## Description

Wywołuje daną funkcję po upływie podanego okresu.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

Zwraca wynik wywołania funkcji <code>function</code> po upływie okresu <code>delay</code>.



## Category
Function
