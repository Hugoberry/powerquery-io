# Odkryj język
Język formuł Power Query M jest zoptymalizowany do tworzenia wysoko elastycznych zapytań do zestawów danych. Jest to język funkcyjny, uwzględniający wielkość liter, podobny do F#. Język formuł Power Query jest używany w wielu produktach Microsoftu, takich jak Power BI Desktop, Excel i Analysis Services.

Ta strona służy jako źródło odniesienia do funkcji Power Query, oferując szybki i bezpośredni dostęp do szczegółowych informacji oraz praktycznych przykładów dla każdej funkcji.
:::note
Proszę zauważyć, że autor nie rości sobie praw autorskich do materiałów prezentowanych na tej stronie. Informacje są zbierane i dostarczane w nadziei, że będą przydatne dla osób chcących poszerzyć swoją wiedzę i zastosowanie Power Query.
:::

Funkcjonalność wyszukiwania na tej stronie internetowej powinna ułatwić proces odkrywania szerokiej gamy funkcji Power Query. Upewnij się, że sprawdziłeś ją (Ctrl+K).

![Funkcjonalność wyszukiwania zapewniona przez Algolia](/img/search.png)

Dla osób zainteresowanych głębszym eksplorowaniem lub szczegółami implementacji, więcej informacji można znaleźć w następującym repozytorium GitHub: [https://github.com/Hugoberry/powerquery-io](https://github.com/Hugoberry/powerquery-io). To repozytorium może zawierać dodatkowe przykłady, aktualizacje i zasoby istotne dla rozwoju i użytkowania Power Query.
