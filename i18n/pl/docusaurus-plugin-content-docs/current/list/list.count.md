# List.Count


Zwraca liczbę elementów na liście.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Zwraca liczbę elementów na liście <code>list</code>.


## Examples

### Example #1 
Znajdź liczbę wartości na liście \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
