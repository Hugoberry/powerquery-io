# Value.Compare


Zwraca wartość -1, 0 lub 1 w zależności od tego, czy pierwsza wartość jest mniejsza, równa lub większa od drugiej.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```


## Remarks

Zwraca wartość -1, 0 lub 1 w zależności od tego, czy pierwsza wartość jest mniejsza, równa lub większa od drugiej.
