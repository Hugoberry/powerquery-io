# Table.FirstValue


Zwraca pierwszą kolumnę pierwszego wiersza tabeli lub określoną wartość domyślną.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Remarks

Zwraca pierwszą kolumnę pierwszego wiersza tabeli <code>table</code> lub określoną wartość domyślną.
