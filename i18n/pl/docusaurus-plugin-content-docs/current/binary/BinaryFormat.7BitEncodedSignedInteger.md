# BinaryFormat.7BitEncodedSignedInteger


Format binarny odczytujący 64-bitowe liczby całkowite ze znakiem, które zostały zakodowane przy użyciu 7-bitowego kodowania o zmiennej długości.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks
