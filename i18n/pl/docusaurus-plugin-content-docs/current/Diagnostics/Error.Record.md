# Error.Record


## Description

Zwraca rekord błędu z dostarczonych wartości tekstowych określających przyczynę, komunikat i szczegóły.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
