# Geometry.ToWellKnownText


Tłumaczy wartość strukturalnego punktu geometrycznego na jego reprezentację Well-Known Text (WKT).


## Syntax

```powerquery
Geometry.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Remarks

Tłumaczy wartość strukturalnego punktu geometrycznego na jego reprezentację Well-Known Text (WKT) zdefiniowaną przez konsorcjum Open Geospatial Consortium (OGC), a także format serializacji używany przez wiele baz danych, w tym SQL Server.
