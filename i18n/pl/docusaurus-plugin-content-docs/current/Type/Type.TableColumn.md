# Type.TableColumn


## Description

Zwraca typ kolumny w tabeli.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Zwraca typ kolumny <code>column</code> w tabeli typu <code>tableType</code>.



## Category
Type
