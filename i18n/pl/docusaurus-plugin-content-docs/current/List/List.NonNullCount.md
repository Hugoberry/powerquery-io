# List.NonNullCount


## Description

Zwraca liczbę elementów na liście, które mają wartość inną niż null.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Zwraca liczbę elementów na liście <code>list</code>, które mają wartość inną niż null.
