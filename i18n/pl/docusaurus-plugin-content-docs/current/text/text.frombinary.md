# Text.FromBinary


Dekoduje dane z formy binarnej na wartość tekstową.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

Dekoduje dane <code>binary</code> z wartości binarnej na wartość tekstową, używając typu <code>encoding</code>.
