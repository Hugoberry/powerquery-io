# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

Odwraca dostarczony tekst <code>text</code>.


## Examples

### Example #1 
Odwróć tekst „123”.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
