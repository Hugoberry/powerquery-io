# Identity.IsMemberOf


Określa, czy tożsamość należy do kolekcji tożsamości.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Remarks

Określa, czy tożsamość należy do kolekcji tożsamości.
