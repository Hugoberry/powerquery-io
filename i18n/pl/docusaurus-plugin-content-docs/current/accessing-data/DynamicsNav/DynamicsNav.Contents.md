# DynamicsNav.Contents


Wprowadź punkt końcowy usługi OData dla systemu Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

Zwraca tabelę zawierającą odpowiednie dane systemu Dynamics NAV.
