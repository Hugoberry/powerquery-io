# Cube.ApplyParameter


Zwraca moduł po zastosowaniu do niego parametru.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

Zwraca moduł po zastosowaniu parametru <code>parameter</code> z wartością <code>arguments</code> do <code>cube</code>.

Cube
