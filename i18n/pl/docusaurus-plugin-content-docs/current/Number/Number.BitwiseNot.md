# Number.BitwiseNot


## Description

Zwraca bajt, w którym każdy bit ma wartość przeciwną do bitu w wartości wejściowej.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Zwraca wynik wykonania bitowej opcji NOT na wartości <code>number</code>.



## Category
Number.Bytes
