# Number.BitwiseOr


Zwraca wynik wykonania bitowej opcji OR na dwóch wartościach wejściowych.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Zwraca wynik wykonania bitowej opcji OR na wartościach <code>number1</code> i <code>number2</code>.



## Category
Number.Bytes
