# Number.Cos


Zwraca cosinus liczby.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Zwraca cosinus liczby <code>number</code>.


## Examples

### Example #1 
Znajdź cosinus kąta 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
