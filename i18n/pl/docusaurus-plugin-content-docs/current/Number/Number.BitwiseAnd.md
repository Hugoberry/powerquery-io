# Number.BitwiseAnd


Zwraca wynik wykonania bitowej opcji AND na dwóch wartościach wejściowych.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Zwraca wynik wykonania bitowej opcji AND na wartościach <code>number1</code> i <code>number2</code>.

Number.Bytes
