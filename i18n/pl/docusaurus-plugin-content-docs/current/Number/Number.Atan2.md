# Number.Atan2


Zwraca arcus tangens ilorazu dwóch liczb.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Remarks

Zwraca arcus tangens ilorazu dwóch liczb (<code>y</code> i <code>x</code>). Dzielenie będzie wykonywane w formie <code>y</code>/<code>x</code>.
