# Number.BitwiseXor


## Description

Zwraca wynik wykonania bitowej opcji XOR (OR wyłączne) na dwóch wartościach wejściowych.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Zwraca wynik wykonania bitowej opcji XOR (OR wyłączne) na wartościach <code>number1</code> i <code>number2</code>.



## Category
Number.Bytes
