# Number.BitwiseShiftRight


Wykonuje przesunięcie bitów w prawo.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Zwraca wynik przesunięcia bitów w prawo w wartości <code>number1</code> o określoną liczbę bitów <code>number2</code>.



## Category
Number.Bytes
