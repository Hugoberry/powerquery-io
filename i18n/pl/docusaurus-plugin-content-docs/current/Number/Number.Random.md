# Number.Random


## Description

Zwraca liczbę losową.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Zwraca losową liczbę z zakresu od 0 do 1.


## Examples

### Example #1 
Utwórz liczbę losową.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
