# Number.Sin


## Description

Zwraca sinus liczby.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Zwraca sinus liczby <code>number</code>.


## Examples

### Example #1 
Znajdź sinus kąta 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
