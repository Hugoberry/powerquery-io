# Number.BitwiseShiftLeft


Wykonuje przesunięcie bitów w lewo.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Zwraca wynik przesunięcia bitów w lewo w wartości <code>number1</code> o określoną liczbę bitów <code>number2</code>.



## Category
Number.Bytes
