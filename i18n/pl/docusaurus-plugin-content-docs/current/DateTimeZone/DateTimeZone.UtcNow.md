# DateTimeZone.UtcNow


## Description

Zwraca bieżącą datę i godzinę w formacie UTC (strefa czasowa GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Zwraca bieżącą datę i godzinę w formacie UTC (strefa czasowa GMT).


## Examples

### Example #1 
Pobierz bieżącą datę i godzinę w formacie UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
