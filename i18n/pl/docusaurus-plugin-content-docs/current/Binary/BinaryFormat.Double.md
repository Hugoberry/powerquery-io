# BinaryFormat.Double


Format binarny odczytujący 8-bajtową wartość zmiennoprzecinkową o podwójnej precyzji, która jest zgodna ze standardem IEEE.


## Syntax

```powerquery
BinaryFormat.Double(
    binary as binary
) as any
```


## Remarks

Format binarny odczytujący 8-bajtową wartość zmiennoprzecinkową o podwójnej precyzji, która jest zgodna ze standardem IEEE.
