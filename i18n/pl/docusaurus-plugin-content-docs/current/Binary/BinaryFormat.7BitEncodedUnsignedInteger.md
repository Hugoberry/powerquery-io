# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Format binarny odczytujący 64-bitowe liczby całkowite bez znaku, które zostały zakodowane przy użyciu 7-bitowego kodowania o zmiennej długości.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
