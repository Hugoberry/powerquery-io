# BinaryFormat.Single


Format binarny odczytujący 4-bajtową wartość zmiennoprzecinkową o pojedynczej precyzji, która jest zgodna ze standardem IEEE.


## Syntax

```powerquery
BinaryFormat.Single(
    binary as binary
) as any
```


## Remarks
