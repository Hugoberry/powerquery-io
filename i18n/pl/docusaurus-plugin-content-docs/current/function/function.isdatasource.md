# Function.IsDataSource


Zwraca informację o tym, czy dana funkcja jest uznawana za źródło danych.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Remarks

Zwraca informację o tym, czy funkcja <code>function</code> jest uznawana za źródło danych.
