# AzureSpark.Tables


Wyświetl tabele w wystąpieniu usługi Azure Spark.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Remarks

Wyświetl tabele w wystąpieniu usługi Azure Spark.


## Examples

### Example #1 
Wyświetl tabele w wystąpieniu usługi Azure Spark.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
