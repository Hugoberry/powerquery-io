# GoogleAnalytics.Accounts


## Description

Zwraca konta usługi Google Analytics.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

Zwraca konta usługi Google Analytics, do których można uzyskać dostęp za pomocą bieżących poświadczeń.

Accessing data
