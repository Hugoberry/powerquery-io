# Lakehouse.Contents


Importuj dane z magazynu Lakehouse


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Remarks

Nawiąż połączenie ze wszystkimi magazynami Lakehouse, do których masz dostęp, i wybierz jednostki, których chcesz użyć
