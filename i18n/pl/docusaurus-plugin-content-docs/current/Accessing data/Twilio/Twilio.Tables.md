# Twilio.Tables


## Description

Wprowadź liczbę miesięcy, z których mają zostać pobrane dane historyczne usługi Twilio.


## Syntax

```powerquery
Twilio.Tables(
    historyMonths as number
) as table
```


## Details

Zwraca tabelę z właściwymi danymi usługi Twilio.
