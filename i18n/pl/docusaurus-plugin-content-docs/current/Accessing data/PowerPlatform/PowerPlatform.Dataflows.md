# PowerPlatform.Dataflows


## Description

Importuj dane z przepływu danych


## Syntax

```powerquery
PowerPlatform.Dataflows(
    optional options as record
) as table
```


## Details

Nawiąż połączenie z wszystkimi przepływami danych, do których masz dostęp, i wybierz jednostki, których chcesz użyć
