# Cube.MeasureProperty


Zwraca właściwości miary (właściwość komórki).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Remarks

Zwraca właściwość <code>propertyName</code> miary <code>measure</code>.



## Category
Cube
