# Cube.AttributeMemberProperty


## Description

Zwraca właściwość atrybutu wymiaru.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Zwraca właściwość <code>propertyName</code> atrybutu wymiaru <code>attribute</code>.



## Category
Cube
