# Webtrends.Tables


## Description

Podaj identyfikator profilu skojarzony z kontem rozwiązania Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Zwraca tabelę z odpowiednimi danymi rozwiązania Webtrends.
