# Xml.Document


Zwraca zawartość dokumentu w formacie XML jako tabelę hierarchiczną.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Remarks

Zwraca zawartość dokumentu w formacie XML jako tabelę hierarchiczną.
