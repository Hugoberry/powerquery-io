# Web.Page


## Description

Zwraca zawartość dokumentu HTML podzieloną na występujące w nim struktury, a także reprezentację pełnego dokumentu oraz jego tekstu po usunięciu tagów.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
