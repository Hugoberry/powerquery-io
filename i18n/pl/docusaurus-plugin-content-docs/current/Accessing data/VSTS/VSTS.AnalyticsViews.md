# VSTS.AnalyticsViews


Wprowadź nazwę organizacji i projektu.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Zwraca tabelę widoków analizy oferowanych przez usługę Azure DevOps.
