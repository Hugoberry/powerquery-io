# VSTS.Views


Wprowadź nazwę organizacji i projektu.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Remarks

Zwraca tabelę źródeł danych OData udostępnianych przez usługę Azure DevOps.
