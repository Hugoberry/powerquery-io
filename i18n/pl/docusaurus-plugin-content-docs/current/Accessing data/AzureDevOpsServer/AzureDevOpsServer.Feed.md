# AzureDevOpsServer.Feed


Źródło danych usługi Azure DevOps Services


## Syntax

```powerquery
AzureDevOpsServer.Feed(
    url as text,
    optional options as record
) as table
```


## Remarks

Zwraca tabelę źródeł danych OData udostępnianych przez usługę Azure DevOps.
