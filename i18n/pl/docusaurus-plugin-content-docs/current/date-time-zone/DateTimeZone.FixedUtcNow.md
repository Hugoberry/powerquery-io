# DateTimeZone.FixedUtcNow


Zwraca bieżącą datę i godzinę w formacie UTC (strefa czasowa GMT). Ta wartość jest stała i nie zmienia się w kolejnych wywołaniach.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```


## Remarks
