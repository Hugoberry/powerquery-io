# Text.Length


## Description

Zwraca liczbę znaków.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Zwraca liczbę znaków w tekście <code>text</code>.


## Examples

### Example #1 
Określ, ile znaków znajduje się w tekście „Hello World”.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
