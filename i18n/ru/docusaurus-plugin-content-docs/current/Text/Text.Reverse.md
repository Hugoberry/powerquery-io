# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

Обращает порядок указанного <code>text</code>.


## Examples

### Example #1 
Обратный порядок текста &#34;123&#34;.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
