# Text.Length


## Description

Возвращает число символов.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Возвращает число символов в тексте <code>text</code>.


## Examples

### Example #1 
Определить, сколько символов содержится в тексте &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
