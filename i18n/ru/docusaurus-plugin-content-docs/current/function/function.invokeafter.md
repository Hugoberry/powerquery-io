# Function.InvokeAfter


Вызывает заданную функцию спустя указанный период времени.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Remarks

Возвращает результат вызова <code>function</code> спустя период времени <code>delay</code>.



## Category
Function
