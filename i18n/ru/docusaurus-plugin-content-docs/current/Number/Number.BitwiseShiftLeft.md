# Number.BitwiseShiftLeft


## Description

Сдвигает набор битов влево.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Возвращает результат выполнения побитового сдвига влево над <code>number1</code> на указанное число битов <code>number2</code>.



## Category
Number.Bytes
