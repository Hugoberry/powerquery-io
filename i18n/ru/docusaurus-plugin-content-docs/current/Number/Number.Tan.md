# Number.Tan


## Description

Возвращает тангенс данного числа.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Возвращает тангенс <code>number</code>.


## Examples

### Example #1 
Найти тангенс угла 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
