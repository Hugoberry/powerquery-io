# Number.IsEven


## Description

Указывает, четно ли число.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

Указывает, четно ли значение <code>number</code>, возвращая <code>true</code>, если оно четно, и <code>false</code> - в противном случае.


## Examples

### Example #1 
Проверить, четно ли число 625.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Проверить, четно ли число 82.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
