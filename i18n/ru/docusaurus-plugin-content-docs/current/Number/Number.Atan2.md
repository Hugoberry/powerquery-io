# Number.Atan2


## Description

Возвращает арктангенс частного от деления двух чисел.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Возвращает арктангенс частного от деления двух чисел, <code>y</code> и <code>x</code>. Деление будет построено как <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
