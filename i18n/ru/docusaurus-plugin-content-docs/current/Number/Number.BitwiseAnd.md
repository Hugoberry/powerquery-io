# Number.BitwiseAnd


Возвращает результат выполнения побитовой операции AND с двумя входными значениями.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Возвращает результат выполнения побитовой операции AND со значениями <code>number1</code> и <code>number2</code>.



## Category
Number.Bytes
