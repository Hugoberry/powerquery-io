# Number.BitwiseShiftRight


## Description

Сдвигает заданные биты вправо.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Возвращает результат выполнения побитового сдвига вправо над <code>number1</code> на указанное число битов <code>number2</code>.



## Category
Number.Bytes
