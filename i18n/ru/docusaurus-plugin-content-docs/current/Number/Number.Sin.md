# Number.Sin


## Description

Возвращает синус числа.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Возвращает синус <code>number</code>.


## Examples

### Example #1 
Найти синус угла 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
