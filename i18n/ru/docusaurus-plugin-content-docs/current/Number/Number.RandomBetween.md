# Number.RandomBetween


## Description

Возвращает случайное число между 2 чисел.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Возвращает случайное число, расположенное между <code>bottom</code> и <code>top</code>.


## Examples

### Example #1 
Получить случайное число между 1 и 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
