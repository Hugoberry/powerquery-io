# Text.FromBinary


Декодирует данные из двоичного формата в текстовый.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

Декодирует данные <code>binary</code> из двоичного значения в текстовое, используя тип <code>encoding</code>.
