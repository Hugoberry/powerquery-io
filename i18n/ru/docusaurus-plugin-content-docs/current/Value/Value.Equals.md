# Value.Equals


## Description

Проверяет, равны ли два значения.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Возвращает значение true, если значение <code>value1</code> равно значению <code>value2</code>, в противном случае - false.



## Category
Values
