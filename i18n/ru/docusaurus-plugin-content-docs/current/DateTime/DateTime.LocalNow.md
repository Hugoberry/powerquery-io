# DateTime.LocalNow


Возвращает текущую дату и время в местном часовом поясе.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Remarks

Возвращает значение <code>datetime</code>, равное текущей дате и времени в системе.

DateTime
