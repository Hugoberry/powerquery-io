# Binary.ViewError


## Description

Создает измененную запись об ошибке, которая не запускает откат при вызове обработчиком, определенным в представлении (через Binary.View).


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Details

Создает измененную запись об ошибке из <code>errorRecord</code>, которая не запускает откат при вызове обработчиком, определенным в представлении (через Binary.View).

Binary
