# BinaryFormat.Double


## Description

Двоичный формат, который считывает 8-байтовое значение двойной точности с плавающей запятой в формате IEEE.


## Syntax

```powerquery
BinaryFormat.Double(
    binary as binary
) as any
```


## Details

Двоичный формат, который считывает 8-байтовое значение двойной точности с плавающей запятой в формате IEEE.
