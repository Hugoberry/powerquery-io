# Error.Record


Возвращает ошибочную запись из указанных текстовых значений для причины, сообщения и сведений.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Remarks

Возвращает ошибочную запись из указанных текстовых значений для причины, сообщения и сведений.
