# Diagnostics.CorrelationId


## Description

Возвращает непрозрачный идентификатор для сопоставления входящих запросов с исходящими.


## Syntax

```powerquery
Diagnostics.CorrelationId(

) as text
```


## Details

Возвращает непрозрачный идентификатор для сопоставления входящих запросов с исходящими.
