# Язык Power Query
Язык формул Power Query M оптимизирован для создания гибких запросов смешивания данных. Это функциональный язык, чувствительный к регистру, похожий на F#. Язык формул Power Query используется в ряде продуктов Microsoft, таких как Power BI Desktop, Excel и Analysis Services.

Этот сайт служит справочником по функциям Power Query, предлагая быстрый и прямой доступ к подробной информации и практическим примерам каждой функции.
:::note
Обратите внимание, что автор не претендует на авторские права на материалы, представленные на этом сайте. Информация собирается и предоставляется в надежде, что она будет полезна тем, кто хочет расширить свои знания и применение Power Query.
:::

Функциональность поиска на этом веб-сайте должна упростить процесс открытия широкого спектра функций Power Query. Убедитесь, что вы проверили её (Ctrl+K).

![Функциональность поиска, предоставленная Algolia](/img/search.png)

Для тех, кто заинтересован в более глубоком изучении или деталях реализации, дополнительная информация может быть найдена в следующем репозитории GitHub: [https://github.com/Hugoberry/powerquery-io](https://github.com/Hugoberry/powerquery-io). Этот репозиторий может содержать дополнительные примеры, обновления и ресурсы, актуальные для разработки и использования Power Query.
