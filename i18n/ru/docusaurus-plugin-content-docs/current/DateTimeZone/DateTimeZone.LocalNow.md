# DateTimeZone.LocalNow


## Description

Возвращает текущую дату и время в местном часовом поясе.


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

Возвращает значение <code>datetimezone</code>, равное текущей дате и времени в системе.    Возвращенное значение содержит сведения о часовом поясе, представляющем местный часовой пояс.
