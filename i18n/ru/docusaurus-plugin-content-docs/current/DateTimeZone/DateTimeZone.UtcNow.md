# DateTimeZone.UtcNow


Возвращает текущую дату и время в формате UTC (часовой пояс GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

Возвращает текущую дату и время в формате UTC (часовой пояс GMT).


## Examples

### Example #1 
Возвращает текущую дату и время в формате UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
