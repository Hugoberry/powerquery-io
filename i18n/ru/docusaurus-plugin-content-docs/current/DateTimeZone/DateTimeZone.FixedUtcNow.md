# DateTimeZone.FixedUtcNow


## Description

Возвращает текущую дату и время в формате UTC (часовой пояс GMT). Это значение зафиксировано и не меняется при последующих вызовах.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
