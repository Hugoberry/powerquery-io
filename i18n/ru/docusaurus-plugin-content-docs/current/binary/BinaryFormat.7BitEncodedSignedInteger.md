# BinaryFormat.7BitEncodedSignedInteger


Двоичный формат, который считывает 64-разрядное целое число со знаком, зашифрованное с использованием 7-разрядной кодировки с переменной длиной.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

Двоичный формат, который считывает 64-разрядное целое число со знаком, зашифрованное с использованием 7-разрядной кодировки с переменной длиной.
