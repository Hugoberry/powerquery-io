# BinaryFormat.7BitEncodedUnsignedInteger


Двоичный формат, который считывает 64-разрядное целое число без знака, зашифрованное с использованием 7-разрядной кодировки с переменной длиной.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks

Двоичный формат, который считывает 64-разрядное целое число без знака, зашифрованное с использованием 7-разрядной кодировки с переменной длиной.
