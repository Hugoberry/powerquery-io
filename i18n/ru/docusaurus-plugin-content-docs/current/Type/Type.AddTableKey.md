# Type.AddTableKey


## Description

Добавляет ключ к данному типу таблицы.


## Syntax

```powerquery
Type.AddTableKey(
    table as type,
    columns as list,
    isPrimary as logical
) as type
```


## Details

Добавляет ключ к данному типу таблицы.
