# Type.ReplaceFacets


## Description

Заменяет аспекты типа.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Заменяет аспекты <code>type</code> на аспекты из записи <code>facets</code>.
