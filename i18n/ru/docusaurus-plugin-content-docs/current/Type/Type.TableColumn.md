# Type.TableColumn


## Description

Возвращает тип столбца в таблице.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Возвращает тип столбца <code>column</code> в типе таблицы <code>tableType</code>.

Type
