# List.Count


Возвращает число элементов в списке.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Возвращает число элементов в списке <code>list</code>.


## Examples

### Example #1 
Определить количество значений в списке \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
