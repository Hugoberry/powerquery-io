# List.Random


Возвращает список случайных чисел.


## Syntax

```powerquery
List.Random(
    count as number,
    optional seed as number
) as list
```


## Remarks

Возвращает список случайных чисел от 0 до 1 для заданного количества создаваемых значений и необязательного начального значения.<ul>   <li><code>count</code>: число создаваемых случайных значений.</li>   <li><code>seed</code>:  <i>[необязательно]</i> числовое значение, используемое для инициализации генератора случайных чисел.  Если не указано, при каждом вызове функции создается уникальный список случайных чисел.  Если указано численное начальное значение, каждый вызов функции будет создавать один и тот же список случайных чисел для этого значения.</li></ul>


## Examples

### Example #1 
Создать список из 3 случайных чисел.
```powerquery
List.Random(3)
```

Result: 
```powerquery
{0.992332, 0.132334, 0.023592}
```


### Example #2 
Создать список из 3 случайных чисел с указанным начальным значением.
```powerquery
List.Random(3, 2)
```

Result: 
```powerquery
{0.883002, 0.245344, 0.723212}
```
