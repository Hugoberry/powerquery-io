# List.NonNullCount


Возвращает число элементов, отличных от NULL, в списке.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Возвращает число элементов, отличных от NULL, в списке <code>list</code>.

List.Information
