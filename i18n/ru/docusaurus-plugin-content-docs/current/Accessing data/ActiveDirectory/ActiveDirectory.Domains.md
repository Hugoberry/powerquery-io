# ActiveDirectory.Domains


Возвращает список доменов Active Directory в том же лесу, что и указанный домен, или из домена текущего компьютера, если значение не указано.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Remarks
