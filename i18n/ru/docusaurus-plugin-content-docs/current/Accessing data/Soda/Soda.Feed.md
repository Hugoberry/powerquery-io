# Soda.Feed


## Description

Возвращает таблицу с содержимым по указанному URL-адресу, отформатированным в соответствии с SODA 2.0 API.


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Details

Возвращает таблицу с содержимым по указанному URL-адресу <code>url</code>, отформатированному согласно SODA 2.0 API. URL-адрес должен указывать на действительный SODA-совместимый источник, который заканчивается на расширение CSV.
