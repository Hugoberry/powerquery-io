# Pdf.Tables


## Description

Возвращает все таблицы, найденные в PDF-файле.


## Syntax

```powerquery
Pdf.Tables(
    pdf as binary,
    optional options as record
) as table
```


## Details

Возвращает таблицы, найденные в <code>pdf</code>. Для указания дополнительных свойств может предоставляться необязательный параметр записи, <code>options</code>. Запись может содержать следующие поля:    <ul><li><code>Implementation</code> : Версия алгоритма, используемого для идентификации таблиц. Старые версии доступны только для обеспечения обратной совместимости, чтобы предотвратить нарушение работы старых запросов при обновлении алгоритмов. Самая последняя версия всегда обеспечивает наилучшие результаты. Допустимые значения: &quot;1.3&quot;, &quot;1.2&quot;, &quot;1.1&quot; или &quot;null&quot;.</li><li><code>StartPage</code> : Задает первую страницу диапазона страниц для проверки. Значение по умолчанию: 1.</li><li><code>EndPage</code> : Задает последнюю страницу диапазона страниц для проверки. Значение по умолчанию: номер последней страницы документа.</li><li><code>MultiPageTables</code> : Определяет, будут ли похожие таблицы на идущих одна за другой страницах автоматически объединяться в одну таблицу. Значение по умолчанию: true.</li><li><code>EnforceBorderLines</code> : Определяет, будут ли для определения границ ячеек использоваться только линии границ (при значении true) или они будут использоваться просто как одно из множества указаний (при значении false). Значение по умолчанию: false.</li></ul>    


## Examples

### Example #1 
Возвращает таблицы, содержащиеся в файле sample.pdf.
```powerquery
Pdf.Tables(File.Contents("c:\sample.pdf"))
```

Result: 
```powerquery
#table({"Name", "Kind", "Data"}, ...)
```
