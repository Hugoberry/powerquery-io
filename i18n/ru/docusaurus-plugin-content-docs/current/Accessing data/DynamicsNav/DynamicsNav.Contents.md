# DynamicsNav.Contents


Укажите URL-адрес конечной точки службы Dynamics NAV OData.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

Возвращает таблицу с подходящими данными Dynamics NAV OData.
