# GoogleAnalytics.Accounts


Возвращает учетные данные Google Analytics.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Remarks

Возвращает учетные данные Google Analytics, доступные с текущими учетными данными.
