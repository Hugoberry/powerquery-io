# Dynamics365BusinessCentral.Contents


## Description

Вход в вашу среду Dynamics 365 Business Central и в вашу организацию.


## Syntax

```powerquery
Dynamics365BusinessCentral.Contents(
    optional company as text
) as table
```


## Details

Возвращает таблицу с нужными данными Dynamics 365 Business Central.
