# AzureDevOpsServer.Views


## Description

Введите название организации и проекта.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Возвращает таблицу веб-каналов OData, предлагаемых Azure DevOps.
