# Identity.IsMemberOf


## Description

Определяет, входит ли удостоверение в коллекцию удостоверений.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details

Определяет, входит ли удостоверение в коллекцию удостоверений.
