# Identity.IsMemberOf


Определяет, входит ли удостоверение в коллекцию удостоверений.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Remarks

Определяет, входит ли удостоверение в коллекцию удостоверений.
