# SharePoint.Contents


Возвращает таблицу, содержащую содержимое сайта SharePoint.


## Syntax

```powerquery
SharePoint.Contents(
    url as text,
    optional options as record
) as table
```


## Remarks

Возвращает таблицу, содержащую по одной строке для каждой папки и документа, найденных на указанном сайте SharePoint (<code>url</code>). Каждая строка представляет свойства файла или папки и ссылку на содержимое. Можно указать <code>options</code> для управления следующими параметрами:    <ul><li><code>ApiVersion</code> : Число (14 или 15) или значение &quot;Auto&quot; (Автоматически), указывающее версию API SharePoint для использования на этом сайте. Если значение не задано, используется версия API&#160;14. Если задано значение Auto, версия сервера будет по возможности определяться автоматически. В случае невозможности будет использоваться значение по умолчанию&#160;— 14. Для неанглоязычных сайтов SharePoint требуется по меньшей мере версия&#160;15.</li></ul>
