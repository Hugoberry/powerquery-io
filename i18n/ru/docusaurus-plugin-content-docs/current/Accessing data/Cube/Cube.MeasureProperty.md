# Cube.MeasureProperty


## Description

Возвращает свойство меры (свойство ячейки).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Возвращает свойство <code>propertyName</code> меры <code>measure</code>.



## Category
Cube
