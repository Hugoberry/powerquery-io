# Cube.AttributeMemberProperty


## Description

Возвращает свойство атрибута измерения.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Возвращает свойство <code>propertyName</code> атрибута измерения <code>attribute</code>.



## Category
Cube
