# Cube.Parameters


Возвращает таблицу, содержащую набор параметров, которые можно применить к кубу.


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Remarks

Возвращает таблицу, содержащую набор параметров, которые можно применить к "<code>cube</code>". Каждый параметр представляет из себя функцию, которую можно вызвать для того, чтобы получить "<code>cube</code>" с примененными параметром и агрументами.
