# Hdfs.Files


## Description

Возвращает таблицу, содержащую свойства и содержимое файлов, найденных в заданной папке и вложенных папках из файловой системы Hadoop.


## Syntax

```powerquery
Hdfs.Files(
    url as text
) as table
```


## Details

Возвращает таблицу, содержащую одну строку для каждого файла, найденного по URL-адресу папки <code>url</code> и во вложенных папках из файловой системы Hadoop. Каждая строка содержит свойства файла и ссылки на его содержимое.
