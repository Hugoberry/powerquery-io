# VSTS.AnalyticsViews


Введите название организации и проекта.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Возвращает таблицу представлений службы аналитики, предлагаемых Azure DevOps.
