# Number.BitwiseOr


Возвращает результат выполнения побитовой операции OR с двумя входными значениями.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Возвращает результат выполнения побитовой операции OR со значениями <code>number1</code> и <code>number2</code>.



## Category
Number.Bytes
