# Number.BitwiseNot


Возвращает байт, где каждый бит является обратным по отношению к входному.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Remarks

Возвращает результат выполнения побитовой операции NOT над <code>number</code>.



## Category
Number.Bytes
