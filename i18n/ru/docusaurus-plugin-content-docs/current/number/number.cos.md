# Number.Cos


Возвращает косинус числа.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Возвращает косинус <code>number</code>.


## Examples

### Example #1 
Найти косинус угла 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
