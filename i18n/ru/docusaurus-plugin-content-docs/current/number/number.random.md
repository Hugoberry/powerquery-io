# Number.Random


Возвращает случайное число.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Возвращает случайное число от 0 до 1.


## Examples

### Example #1 
Возвращает случайное число.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
