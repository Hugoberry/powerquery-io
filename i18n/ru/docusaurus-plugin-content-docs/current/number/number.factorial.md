# Number.Factorial


Возвращает факториал числа.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

Возвращает факториал числа <code>number</code>.


## Examples

### Example #1 
Найти факториал 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
