# Webtrends.Tables


Введите идентификатор профиля, связанный с вашей учетной записью Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Remarks

Возвращает таблицу с актуальными данными Webtrends.
