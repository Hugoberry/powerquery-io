# Xml.Document


Возвращает содержимое XML-документа в виде иерархической таблицы.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Remarks

Возвращает содержимое XML-документа в виде иерархической таблицы.
