# File.Contents


Возвращает содержимое заданного файла в двоичном виде.


## Syntax

```powerquery
File.Contents(
    path as text,
    optional options as record
) as binary
```


## Remarks

Возвращает содержимое файла, <code>path</code> в виде двоичного кода. Параметр <code>options</code> в настоящее время предназначен только для внутреннего использования.
