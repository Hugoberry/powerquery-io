# Cube.Properties


Возвращает таблицу с набором доступных свойств измерений, которые расширяются в кубе.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```


## Remarks

Возвращает таблицу с набором доступных свойств измерений, которые расширяются в кубе.
