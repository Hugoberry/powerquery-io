# Cube.ApplyParameter


Возвращает куб после применения к нему параметра.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

Возвращает куб после применения "<code>parameter</code>" с "<code>arguments</code>" к "<code>cube</code>".

Cube
