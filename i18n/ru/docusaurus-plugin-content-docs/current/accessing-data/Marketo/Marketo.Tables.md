# Marketo.Tables


Введите URL-адрес конечной точки REST API Marketo, связанной с вашей учетной записью.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Remarks

Возвращает таблицу с актуальными данными Marketo.
