# List.Count


Retourne le nombre d&#39;éléments dans la liste.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Retourne le nombre d'éléments dans la liste <code>list</code>.


## Examples

### Example #1 
Recherche le nombre de valeurs dans la liste \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
