# List.NonNullCount


## Description

Retourne le nombre d&#39;éléments non null dans la liste.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Retourne le nombre d'éléments non null dans la liste <code>list</code>.
