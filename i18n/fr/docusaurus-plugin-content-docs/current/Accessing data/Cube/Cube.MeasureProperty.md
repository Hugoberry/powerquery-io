# Cube.MeasureProperty


Retourne une propriété d&#39;une mesure (propriété de cellule).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Remarks

Retourne la propriété <code>propertyName</code> de la mesure <code>measure</code>.



## Category
Cube
