# Cube.AttributeMemberProperty


## Description

Retourne une propriété d&#39;un attribut de dimension.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Retourne la propriété <code>propertyName</code> de l'attribut de dimension <code>attribute</code>.



## Category
Cube
