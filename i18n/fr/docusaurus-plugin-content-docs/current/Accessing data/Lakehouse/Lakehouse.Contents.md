# Lakehouse.Contents


## Description

Importer des données à partir d’un Lakehouse


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Details

Connectez-vous à tous les Lakehouses auxquels vous avez accès et choisissez les entités que vous souhaitez utiliser.
