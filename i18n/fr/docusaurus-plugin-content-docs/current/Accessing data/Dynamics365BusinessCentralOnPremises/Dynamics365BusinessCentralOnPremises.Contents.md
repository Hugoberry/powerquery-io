# Dynamics365BusinessCentralOnPremises.Contents


Entrez l&#39;URL de votre point de terminaison de service OData Dynamics 365 Business Central (local).


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text

```


## Remarks
