# MailChimp.TablesV2


Retourne un tableau avec des données MailChimp clés.


## Syntax

```powerquery
MailChimp.TablesV2(

) as table
```


## Remarks

Retourne un tableau avec des données MailChimp clés.


## Examples

### Example #1 
Retourne un tableau avec des données MailChimp clés.
```powerquery
MailChimp.Tables()
```
