# VSTS.Views


## Description

Entrez les noms de l&#39;organisation et du projet.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Retourne une table de flux OData offerts par Azure DevOps.
