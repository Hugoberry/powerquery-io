# VSTS.AnalyticsViews


Entrez les noms de l&#39;organisation et du projet.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Retourne une table de vues Analytics offertes par Azure DevOps.
