# Webtrends.Tables


## Description

Entrez l&#39;ID de profil associé à votre compte Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Retourne une table avec les données Webtrends appropriées.
