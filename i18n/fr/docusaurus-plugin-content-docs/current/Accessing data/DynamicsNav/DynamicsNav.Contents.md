# DynamicsNav.Contents


Entrez l&#39;URL du point de terminaison de service OData Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

Retourne une table contenant des données Dynamics NAV pertinentes.
