# AzureDevOpsServer.AnalyticsViews


## Description

Entrez les noms de l&#39;organisation et du projet.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Retourne une table de vues Analytics offertes par Azure DevOps.
