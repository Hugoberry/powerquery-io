# Xml.Document


## Description

Retourne le contenu du document XML en tant que table hiérarchique.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Details

Retourne le contenu du document XML en tant que table hiérarchique.
