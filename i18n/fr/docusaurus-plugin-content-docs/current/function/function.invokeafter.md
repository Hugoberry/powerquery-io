# Function.InvokeAfter


Appelle la fonction donnée une fois le délai spécifié écoulé.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Remarks

Retourne le résultat de l'appel de <code>function</code> une fois le délai <code>delay</code> écoulé.

Function
