# Number.Atan2


Renvoie l&#39;arc tangente de la division des deux nombres.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Remarks

Renvoie l'arc tangente de la division des deux nombres, <code>y</code> et <code>x</code>. La division sera construite comme <code>y</code>/<code>x</code>.
