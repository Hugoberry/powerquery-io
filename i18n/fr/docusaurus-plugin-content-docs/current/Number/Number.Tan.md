# Number.Tan


## Description

Retourne la tangente du nombre.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Retourne la tangente de <code>number</code>.


## Examples

### Example #1 
Recherche la tangente de l&#39;angle 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
