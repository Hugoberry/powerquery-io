# Number.BitwiseOr


## Description

Retourne le résultat d&#39;une opération au niveau du bit &#34; Or &#34; entre les deux entrées.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Retourne le résultat d'une opération au niveau du bit " Or " entre <code>number1</code> et <code>number2</code>.



## Category
Number.Bytes
