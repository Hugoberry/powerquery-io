# Number.RandomBetween


Retourne un nombre aléatoire entre deux nombres.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Remarks

Retourne un nombre aléatoire entre <code>bottom</code> et <code>top</code>.


## Examples

### Example #1 
Obtient un nombre aléatoire entre 1 et 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
