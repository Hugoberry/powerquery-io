# Number.Random


## Description

Retourne un nombre aléatoire.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Retourne un nombre aléatoire entre 0 et 1.


## Examples

### Example #1 
Obtient un nombre aléatoire.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
