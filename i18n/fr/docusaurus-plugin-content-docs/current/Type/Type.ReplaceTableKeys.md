# Type.ReplaceTableKeys


## Description

Retourne un nouveau type de table avec toutes les clés remplacées par la liste spécifiée de clés.


## Syntax

```powerquery
Type.ReplaceTableKeys(
    tableType as type,
    keys as list
) as type
```


## Details

Retourne un nouveau type de table avec toutes les clés remplacées par la liste spécifiée de clés.
