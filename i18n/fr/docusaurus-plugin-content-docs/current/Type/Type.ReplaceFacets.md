# Type.ReplaceFacets


## Description

Remplace les facettes d&#39;un type.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Remplace les facettes de <code>type</code> par les facettes contenues dans l'enregistrement <code>facets</code>.

Type
