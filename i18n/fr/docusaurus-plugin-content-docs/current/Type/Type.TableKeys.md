# Type.TableKeys


## Description

Retourne la liste éventuellement vide de clés pour le type de table spécifié.


## Syntax

```powerquery
Type.TableKeys(
    tableType as type
) as list
```


## Details

Retourne la liste éventuellement vide de clés pour le type de table spécifié.
