# Type.TableColumn


Retourne le type d&#39;une colonne dans une table.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Remarks

Retourne le type de la colonne <code>column</code> dans le type de table <code>tableType</code>.
