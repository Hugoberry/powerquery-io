# DateTimeZone.FixedUtcNow


## Description

Retourne la date et l&#39;heure actuelles en UTC (fuseau horaire GMT). Cette valeur est fixe et ne change pas lors des appels successifs.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
