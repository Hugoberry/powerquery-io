# DateTimeZone.UtcNow


## Description

Retourne la date et l&#39;heure actuelles en UTC (fuseau horaire GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Retourne la date et l'heure actuelles en UTC (fuseau horaire GMT).


## Examples

### Example #1 
Obtenez la date et l&#39;heure actuelles en UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
