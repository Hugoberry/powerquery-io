# Découvrez le langage
Le langage de formule Power Query M est optimisé pour construire des requêtes de fusion de données hautement flexibles. C'est un langage fonctionnel, sensible à la casse, similaire à F#. Le langage de formules Power Query est utilisé dans plusieurs produits Microsoft tels que Power BI Desktop, Excel et Analysis Services.

Ce site sert de référence pour les fonctions de Power Query, offrant un accès rapide et direct à des informations détaillées et à des exemples pratiques pour chaque fonction.
:::note
Veuillez noter que l'auteur ne revendique aucun droit d'auteur sur les matériaux présentés sur ce site. Les informations sont collectées et fournies dans l'espoir qu'elles seront utiles à ceux qui cherchent à élargir leurs connaissances et leur application de Power Query.
:::

La fonctionnalité de recherche sur ce site devrait simplifier le processus de découverte de la vaste gamme de fonctions Power Query. Assurez-vous de l'essayer (Ctrl+K).

![Fonctionnalité de recherche fournie par Algolia](/img/search.png)

Pour ceux qui sont intéressés par une exploration plus approfondie ou des détails d'implémentation, des informations supplémentaires peuvent être trouvées dans le répertoire GitHub suivant : [https://github.com/Hugoberry/powerquery-io](https://github.com/Hugoberry/powerquery-io). Ce dépôt peut contenir des exemples supplémentaires, des mises à jour et des ressources pertinentes pour le développement et l'utilisation de Power Query.
