# DateTime.LocalNow


## Description

Retourne la date et l&#39;heure actuelles dans le fuseau horaire local.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Retourne une valeur <code>datetime</code> définie selon la date et l'heure actuelles du système.
