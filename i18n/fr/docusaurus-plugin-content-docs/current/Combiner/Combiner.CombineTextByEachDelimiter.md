# Combiner.CombineTextByEachDelimiter


## Description

Retourne une fonction qui associe une liste de textes dans un seul fichier texte à l&#39;aide de chaque délimiteur spécifié en séquence.


## Syntax

```powerquery
Combiner.CombineTextByEachDelimiter(
    delimiters as list,
    optional quoteStyle as QuoteStyle.Type
) as function
```
