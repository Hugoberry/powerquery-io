# Value.Equals


## Description

Détermine si deux valeurs sont égales.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Retourne true si la valeur <code>value1</code> est égale à la valeur <code>value2</code>, sinon retourne false.



## Category
Values
