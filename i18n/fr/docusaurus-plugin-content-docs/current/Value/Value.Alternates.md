# Value.Alternates


Exprime les autres plans de requête.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Remarks

Exprime les autres plans de requête dans une expression de plan de requête obtenue via Value.Expression(Value.Optimize(...)). Non destiné à d’autres utilisations.



## Category
Expression
