# Number.BitwiseNot


Retourne un octet où chaque bit est l&#39;inverse de l&#39;entrée.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Remarks

Retourne le résultat d'une opération au niveau du bit " Not " sur <code>number</code>.



## Category
Number.Bytes
