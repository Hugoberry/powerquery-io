# Number.BitwiseAnd


Retourne le résultat d&#39;une opération au niveau du bit &#34; And &#34; entre les deux entrées.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Retourne le résultat d'une opération au niveau du bit " And " entre <code>number1</code> et <code>number2</code>.



## Category
Number.Bytes
