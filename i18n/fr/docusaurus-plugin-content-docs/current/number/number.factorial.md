# Number.Factorial


Retourne la factorielle du nombre.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

Retourne la factorielle du nombre <code>number</code>.


## Examples

### Example #1 
Recherche la factorielle de 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
