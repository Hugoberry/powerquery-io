# Cube.ApplyParameter


Retourne un cube après application d&#39;un paramètre à celui-ci.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

Retourne un cube après application de <code>parameter</code> avec <code>arguments</code> à <code>cube</code>.



## Category
Cube
