# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

Inverse le <code>text</code> fourni.


## Examples

### Example #1 
Inversez le texte &#34;123&#34;.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
