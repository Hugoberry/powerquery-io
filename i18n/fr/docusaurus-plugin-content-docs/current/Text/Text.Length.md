# Text.Length


## Description

Retourne le nombre de caractères.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Retourne le nombre de caractères dans le texte <code>text</code>.


## Examples

### Example #1 
Indiquez combien de caractères sont dans le texte &#34; Hello World &#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
