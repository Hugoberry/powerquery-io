# BinaryFormat.7BitEncodedSignedInteger


## Description

Un format binar care citeşte un număr întreg cu semn pe 64 de biţi care a fost codificat utilizând codificarea cu o lungime variabilă de 7 biţi.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
