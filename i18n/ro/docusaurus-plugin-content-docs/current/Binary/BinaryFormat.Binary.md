# BinaryFormat.Binary


## Description

Returnează un format binar care citeşte o valoare binară.


## Syntax

```powerquery
BinaryFormat.Binary(
    optional length as any
) as function
```


## Details

Returnează un format binar care citește o valoare binară.  Dacă se specifică <code>length</code>, valoarea binară va conține numărul respectiv de byți.  Dacă nu se specifică <code>length</code>, valoarea binară va conține numărul de byți rămași.  <code>length</code> poate fi specificat fie ca număr, fie ca format binar cu o lungime care apare înaintea datelor binare.
