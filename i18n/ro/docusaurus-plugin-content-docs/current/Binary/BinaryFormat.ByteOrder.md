# BinaryFormat.ByteOrder


## Description

Returnează un format binar cu ordinea de octeţi specificată de o funcţie.


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

Returnează un format binar cu ordinea de octeţi specificată de <code>binaryFormat</code>.  Ordinea de octeţi implicită este <code>ByteOrder.BigEndian</code>.
