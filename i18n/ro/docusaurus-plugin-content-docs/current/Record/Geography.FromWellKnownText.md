# Geography.FromWellKnownText


## Description

Translatează un text care reprezintă o valoare geografică în format WKT (Well-Known Text) într-o înregistrare structurată.


## Syntax

```powerquery
Geography.FromWellKnownText(
    input as text
) as record
```


## Details

Translatează un text care reprezintă o valoare geografică în format WKT (Well-Known Text) într-o înregistrare structurată. WKT este un format standard definit de Open Geospatial Consortium (OGC) și este formatul tipic de serializare utilizat de bazele de date, inclusiv SQL Server.

Record.Serialization
