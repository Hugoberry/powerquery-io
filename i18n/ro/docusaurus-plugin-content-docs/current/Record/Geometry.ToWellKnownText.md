# Geometry.ToWellKnownText


## Description

Translatează o valoare de punct geometric structurat în reprezentarea sa de WKT (Well-Known Text).


## Syntax

```powerquery
Geometry.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Translatează o valoare de punct geometric structurat în reprezentarea sa WKT (Well-Known Text), după cum este definită de Open Geospatial Consortium (OGC), precum și formatul de serializare utilizat de mai multe baze de date, inclusiv SQL Server.
