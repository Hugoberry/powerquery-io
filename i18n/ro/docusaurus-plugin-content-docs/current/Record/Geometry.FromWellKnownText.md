# Geometry.FromWellKnownText


## Description

Translatează textul care reprezintă o valoare geometrică în format WKT (Well-Known Text) într-o înregistrare structurată.


## Syntax

```powerquery
Geometry.FromWellKnownText(
    input as text
) as record
```


## Details

Translatează textul care reprezintă o valoare geometrică în format WKT (Well-Known Text) într-o înregistrare structurată. WKT este un format standard definit de Open Geospatial Consortium (OGC) și este formatul tipic de serializare utilizat de bazele de date, inclusiv SQL Server.

Record.Serialization
