# Geography.ToWellKnownText


## Description

Translatează o valoare de punct geografic structurat în reprezentarea sa WKT (Well-Known Text).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Translatează o valoare de punct geografic structurat într-o reprezentare WKT (Well-Known Text) după cum este definită de Open Geospatial Consortium (OGC), precum și formatul de serializare utilizat de mai multe baze de date, inclusiv SQL Server.
