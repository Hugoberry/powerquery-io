# Lines.ToText


## Description

Converteşte o listă de texte într-un singur text.  Valoarea lineSeparator specificată este adăugată la fiecare linie.  Dacă nu se specifică, atunci se utilizează caracterele de tip sfârşit de linie şi linie nouă.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```
