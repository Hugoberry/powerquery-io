# Smartsheet.Tables


## Description

Returnează un tabel de foi, rapoarte, foldere și spații de lucru de la API-ul Smartsheet


## Syntax

```powerquery
Smartsheet.Tables(

) as table
```


## Details

Returnează un tabel imbricat de foi, rapoarte, foldere și spații de lucru disponibile de la API-ul Smartsheet.


## Examples

### Example #1 
Returnează foi, foldere, rapoarte și spații de lucru disponibile la nivelul superior al ierarhiei Smartsheet
```powerquery
Smartsheet.Tables()
```

Result: 
```powerquery
Un tabel de foldere, rapoarte și spații de lucru de la nivelul superior al ierarhiei Smartsheet
```
