# PowerPlatform.Dataflows


## Description

Importați date dintr-un flux de date


## Syntax

```powerquery
PowerPlatform.Dataflows(
    optional options as record
) as table
```


## Details

Conectați-vă la toate fluxurile de date la care aveți acces și selectați entitățile pe care doriți să le utilizați.
