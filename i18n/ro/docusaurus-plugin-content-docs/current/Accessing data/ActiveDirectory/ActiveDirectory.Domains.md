# ActiveDirectory.Domains


## Description

Returnează o listă de domenii Active Directory în aceeaşi pădure ca şi domeniul specificat sau lista domeniului computerului curent dacă niciunul nu este specificat.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```
