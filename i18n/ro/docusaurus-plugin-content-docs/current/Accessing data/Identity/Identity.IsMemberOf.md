# Identity.IsMemberOf


## Description

Determină dacă o identitate este membra unei colecții de identități.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details

Determină dacă o identitate este membra unei colecții de identități.
