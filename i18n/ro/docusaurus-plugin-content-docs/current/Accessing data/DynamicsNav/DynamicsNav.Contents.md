# DynamicsNav.Contents


## Description

Introduceți adresa URL a punctului final al serviciului OData Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Returnează un tabel cu datele Dynamics NAV relevante.
