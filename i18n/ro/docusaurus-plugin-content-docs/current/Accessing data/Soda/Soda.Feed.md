# Soda.Feed


## Description

Returnează un tabel din conţinut la adresa URL specificată formatată conform interfeţei API SODA 2.0.


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Details

Returnează un tabel din conţinut la adresa URL specificată <code>url</code> formatată conform interfeţei API SODA 2.0. Adresa URL trebuie să indice la o sursă compatibilă cu SODA validă care se termină cu o extensie .csv.
