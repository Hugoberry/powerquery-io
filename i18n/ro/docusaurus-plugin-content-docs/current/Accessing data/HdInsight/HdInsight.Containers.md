# HdInsight.Containers


## Description

Returnează un tabel de navigare ce conține containerele găsite în contul specificat dintr-un seif de stocare Azure.


## Syntax

```powerquery
HdInsight.Containers(
    account as text
) as table
```


## Details

Returnează un tabel de navigare ce conține un rând pentru fiecare container găsit la adresa URL a contului, <code>account</code>, dintr-un seif de stocare Azure. Fiecare rând conține un link la bloburile containerului.
