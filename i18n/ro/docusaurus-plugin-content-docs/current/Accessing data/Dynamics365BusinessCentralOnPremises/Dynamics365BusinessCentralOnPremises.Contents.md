# Dynamics365BusinessCentralOnPremises.Contents


## Description

Introduceți URL-ul punctului dvs. final pentru serviciul OData Dynamics 365 Business Central (local).


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Returnează un tabel cu datele Dynamics 365 Business Central (local) relevante.
