# AzureDevOpsServer.AccountContents


## Description

Introduceți adresa URL a serviciului Azure DevOps Analytics.


## Syntax

```powerquery
AzureDevOpsServer.AccountContents(
    url as text,
    optional options as record
) as table
```


## Details

Returnează conținutul descărcat de la adresa URL Azure DevOps Analytics ca valoare binară. Această funcție utilizează acreditări diferite pentru organizații diferite.
