# Webtrends.Tables


## Description

Introduceți ID-ul de profil asociat contului dvs. Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Returnează un tabel cu datele Webtrends relevante.
