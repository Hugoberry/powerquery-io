# Dynamics365BusinessCentral.EnvironmentContents


## Description

Introduceți mediul și firma dvs. Dynamics 365 Business Central.


## Syntax

```powerquery
Dynamics365BusinessCentral.EnvironmentContents(
    optional environment as text,
    optional company as text
) as table
```


## Details

Returnează un tabel cu datele Dynamics 365 Business Central relevante.
