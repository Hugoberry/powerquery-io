# VSTS.Views


## Description

Introduceți numele organizației și al proiectului.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Returnează un tabel de fluxuri OData oferite de Azure DevOps.
