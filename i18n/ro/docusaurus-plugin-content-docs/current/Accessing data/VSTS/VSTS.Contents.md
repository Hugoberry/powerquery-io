# VSTS.Contents


## Description

Introduceți adresa URL a serviciului Azure DevOps Analytics.


## Syntax

```powerquery
VSTS.Contents(
    url as text,
    optional options as record
) as binary
```


## Details

Returnează conținutul descărcat de la adresa URL a Azure DevOps Analytics Service ca valoare binară. Această funcție utilizează acreditările globale pentru întreaga organizație care poate fi accesată.
