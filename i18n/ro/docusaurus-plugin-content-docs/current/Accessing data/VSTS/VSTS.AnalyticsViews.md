# VSTS.AnalyticsViews


## Description

Introduceți numele organizației și al proiectului.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Returnează un tabel de vizualizări analitice oferite de Azure DevOps.
