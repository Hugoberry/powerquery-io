# Excel.CurrentWorkbook


## Description

Returnează conținutul registrului de lucru Excel curent.


## Syntax

```powerquery
Excel.CurrentWorkbook(

) as table
```


## Details

Returnează conținutul registrului de lucru Excel curent. Returnează tabele, zone denumite și matrice dinamice. Spre deosebire de <code>Excel.Workbook</code>, nu returnează foi.
