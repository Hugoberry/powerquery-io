# Cube.AttributeMemberId


## Description

Returnează identificatorul de membru unic din valoarea de proprietate a membrilor.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Returnează identificatorul de membru unic dintr-o valoare de proprietate a membrilor. <code>attribute</code>. Returnează nul pentru orice alte valori.
