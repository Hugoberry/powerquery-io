# Cube.Measures


## Description

Returnează un tabel ce conţine setul de măsuri disponibile.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

Returnează un tabel care conține setul de măsuri disponibile din cadrul <code>cube</code>.    Fiecare măsură este reprezentată ca funcție. Măsurile pot fi aplicate la cub utilizând Cube.AddMeasureColumn.



## Category
Cube
