# Cube.MeasureProperties


## Description

Returnează un tabel care conține setul de proprietăți de măsură disponibile care sunt extinse în cub.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Returnează un tabel care conține setul de proprietăți disponibile pentru măsurile care sunt extinse în cub.



## Category
Cube
