# Cube.MeasureProperty


## Description

Returnează o proprietate a unei măsuri (proprietate de celulă).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Returnează proprietatea <code>propertyName</code> a măsurii <code>measure</code>.



## Category
Cube
