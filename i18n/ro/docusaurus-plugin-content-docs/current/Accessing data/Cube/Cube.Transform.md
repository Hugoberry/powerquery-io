# Cube.Transform


## Description

Aplică o listă de funcţii de cub.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Aplică lista de funcţii de cub, <code>transforms</code>, pe <code>cube</code>.



## Category
Cube
