# Cube.ApplyParameter


## Description

Returnează un cub după aplicarea unui parametru la acesta.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Returnează un cub după aplicarea <code>parameter</code> cu <code>arguments</code> la <code>cube</code>.



## Category
Cube
