# Cube.Parameters


## Description

Returnează un tabel care conține setul de parametri ce se pot aplica la cub.


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Details

Returnează un tabel care conține setul de parametri ce se pot aplica la <code>cube</code>. Fiecare parametru este o funcție care poate fi invocată pentru a obține <code>cube</code> cu parametrul și argumentele sale aplicate.

Cube
