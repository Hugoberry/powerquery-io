# Cube.Dimensions


## Description

Returnează un tabel ce conţine setul de dimensiuni disponibile.


## Syntax

```powerquery
Cube.Dimensions(
    cube as table
) as table
```


## Details

Returnează un tabel ce conţine setul de dimensiuni disponibile în <code>cube</code>. Fiecare dimensiune este un tabel ce conţine un set de atribute de dimensiune şi fiecare atribut de dimensiune este reprezentat ca şi coloană în tabelul de dimensiuni. Dimensiunile pot fi extinse în cub utilizând Cube.AddAndExpandDimensionColumn.
