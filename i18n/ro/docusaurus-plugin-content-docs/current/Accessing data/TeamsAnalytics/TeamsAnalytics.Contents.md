# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Details

Conectorul Teams Analytics vă permite să obțineți detalii despre utilizarea Teams. Datele sunt disponibile atât pentru activitatea personală în Teams, cât și pentru activitatea din Teams din care faceți parte.<br />Datele sunt disponibile pentru ultimele 90 de zile. <br /> Pentru detalii despre tipurile de date disponibile, consultați documentația de ajutor următoare: <br /><ul><li><b>Utilizarea Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>Activitatea utilizatorilor Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
