# Web.Page


## Description

Returnează conţinutul documentului HTML divizat în structurile sale constituente, precum şi o reprezentare a documentului complet şi textul acestuia după eliminarea etichetelor.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
