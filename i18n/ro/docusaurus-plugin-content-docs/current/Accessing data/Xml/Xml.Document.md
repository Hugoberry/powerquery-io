# Xml.Document


## Description

Returnează conţinutul documentului XML ca tabel ierarhic.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Details

Returnează conţinutul documentului XML ca tabel ierarhic.
