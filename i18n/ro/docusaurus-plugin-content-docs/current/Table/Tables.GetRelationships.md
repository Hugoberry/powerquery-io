# Tables.GetRelationships


## Description

Obține relațiile dintr-un set de tabele.


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Details

Obține relațiile dintr-un set de tabele. Se presupune că setul <code>tables</code> are o structură similară cu cea a unui tabel de navigare. Coloana definită de <code>dataColumn</code> conține tabelele de date propriu-zise.
