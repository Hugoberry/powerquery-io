# Table.FirstValue


## Description

Returnează prima coloană a primului rând al tabelului sau o valoare implicită specificată.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

Returnează prima coloană a primului rând al tabelului <code>table</code> sau o valoare implicită specificată.



## Category
Table.Row operations
