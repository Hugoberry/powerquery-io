# Text.FromBinary


## Description

Decodifică datele dintr-o formă binară în text.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

Decodifică datele, <code>binary</code>, dintr-o valoare binară într-o valoare text, utilizând tipul <code>encoding</code>.



## Category
Text.Conversions from and to text
