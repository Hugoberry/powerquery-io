# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Inversează <code>text</code> furnizat.


## Examples

### Example #1 
Inversați textul „123”.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
