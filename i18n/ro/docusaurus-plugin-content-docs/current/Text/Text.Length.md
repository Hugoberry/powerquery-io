# Text.Length


## Description

Returnează numărul de caractere.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Returnează numărul de caractere din textul <code>text</code>.


## Examples

### Example #1 
Aflaţi câte caractere sunt în textul „Hello World”.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
