# DateTimeZone.UtcNow


## Description

Returnează data şi ora curente în UTC (fusul orar GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Returnează data şi ora curente în UTC (fusul orar GMT).


## Examples

### Example #1 
Obţineţi data şi ora curente în UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
