# DateTimeZone.FixedUtcNow


## Description

Returnează data şi ora curente în UTC (fusul orar GMT). Această valoare este fixă şi nu se va modifica cu apelurile succesive.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
