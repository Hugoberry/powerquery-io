# DateTimeZone.LocalNow


## Description

Returnează data şi ora curente în fusul orar local.


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

Returnează o valoare <code>datetimezone</code> setată la data și ora curente ale sistemului.    Valoarea returnată conține informații despre fusul orar care reprezintă fusul orar local.
