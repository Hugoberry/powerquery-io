# Type.TableSchema


## Description

Returnează un tabel ce conține o descriere a coloanelor (de ex., schema) tipului de tabel specificat.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

Returnează un tabel care descrie coloanele din <code>tableType</code>.<br />Consultați documentația <code>Table.Schema</code> pentru o descriere a tabelului rezultat.<br />
