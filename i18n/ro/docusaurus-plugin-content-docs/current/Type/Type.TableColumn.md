# Type.TableColumn


## Description

Returnează tipul unei coloane dintr-un tabel.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Returnează tipul coloanei <code>column</code> din tipul de tabel <code>tableType</code>.



## Category
Type
