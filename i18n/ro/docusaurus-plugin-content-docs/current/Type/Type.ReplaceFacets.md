# Type.ReplaceFacets


## Description

Înlocuiește fațetele unui tip.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Înlocuiește fațetele pentru <code>type</code> cu fațetele cuprinse în înregistrarea <code>facets</code>.

Type
