# Lakehouse.Contents


## Description

Importați date dintr-un Lakehouse


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Details

Conectați-vă la toate Lakehouse-urile la care aveți acces și selectați entitățile pe care doriți să le utilizați
