# Number.BitwiseOr


## Description

Returnează rezultatul efectuării operaţiunii „Or” la nivel de bit între cele două date introduse.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnează rezultatul efectuării operaţiunii „Or” la nivel de bit între <code>number1</code> şi <code>number2</code>.

Number.Bytes
