# Number.Log10


## Description

Returnează logaritmul în baza 10 al numărului.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Returnează logaritmul în baza 10 al unui număr, <code>number</code>. Dacă <code>number</code> este nul, <code>Number.Log10</code> returnează nul.


## Examples

### Example #1 
Aflați logaritmul în baza 10 pentru 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
