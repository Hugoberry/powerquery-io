# Number.Sin


## Description

Returnează sinusul numărului.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Returnează sinusul <code>number</code>.


## Examples

### Example #1 
Aflaţi sinusul unghiului 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```




## Category
Number.Trigonometry
