# Number.Atan2


## Description

Returnează arctangenta împărţirii a două numere.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Returnează arctangenta împărţirii a două numere, <code>y</code> şi <code>x</code>. Împărţirea va fi construită ca <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
