# Number.BitwiseNot


## Description

Returnează un octet în care fiecare bit este opusul datelor introduse.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Returnează rezultatul efectuării operaţiunii „Not” la nivel de bit pe <code>number</code>.



## Category
Number.Bytes
