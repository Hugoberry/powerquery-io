# Number.RandomBetween


## Description

Returnează un număr aleatoriu cuprins între două numere.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Returnează un număr aleatoriu cuprins între <code>bottom</code> şi <code>top</code>.


## Examples

### Example #1 
Obţineţi un număr aleatoriu cuprins între 1 şi 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
