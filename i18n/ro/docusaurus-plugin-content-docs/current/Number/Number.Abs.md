# Number.Abs


## Description

Returnează valoarea absolută a numărului.


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Details

Returnează valoarea absolută a <code>number</code>. Dacă <code>number</code> este nul, <code>Number.Abs</code> returnează nul.    <ul>        <li><code>number</code>: o valoare <code>number</code> pentru care urmează să fie calculată valoarea absolută.</li>      </ul>


## Examples

### Example #1 
Valoarea absolută -3.
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
