# Number.Tan


## Description

Returnează tangenta numărului.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Returnează tangenta <code>number</code>.


## Examples

### Example #1 
Aflaţi tangenta unghiului 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```




## Category
Number.Trigonometry
