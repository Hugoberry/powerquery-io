# Number.BitwiseAnd


## Description

Returnează rezultatul efectuării unei operaţiuni „And” la nivel de bit între cele două date introduse.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnează rezultatul efectuării operaţiunii „And” la nivel de bit între <code>number1</code> şi <code>number2</code>.

Number.Bytes
