# Number.BitwiseShiftRight


## Description

Deplasează setul de biţi spre dreapta.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnează rezultatul efectuării unei deplasări spre dreapta la nivel de bit pe <code>number1</code>, cu numărul specificat de biţi <code>number2</code>.



## Category
Number.Bytes
