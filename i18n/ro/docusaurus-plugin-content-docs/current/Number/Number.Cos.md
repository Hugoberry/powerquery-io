# Number.Cos


## Description

Returnează cosinusul numărului.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Returnează cosinusul <code>number</code>.


## Examples

### Example #1 
Aflaţi cosinusul unghiului 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
