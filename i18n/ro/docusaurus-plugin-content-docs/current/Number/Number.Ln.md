# Number.Ln


## Description

Returnează logaritmul natural al numărului.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Returnează logaritmul natural al unui număr, <code>number</code>. Dacă <code>number</code> este nul, <code>Number.Ln</code> returnează nul.


## Examples

### Example #1 
Obţineţi logaritmul natural pentru 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
