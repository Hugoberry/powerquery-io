# Number.BitwiseShiftLeft


## Description

Deplasează setul de biţi spre stânga.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Returnează rezultatul efectuării unei deplasări spre stânga la nivel de bit pe <code>number1</code>, cu numărul specificat de biţi <code>number2</code>.



## Category
Number.Bytes
