# Number.Factorial


## Description

Returnează produsul factorial al numărului.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Returnează produsul factorial al numărului <code>number</code>.


## Examples

### Example #1 
Găsiţi produsul factorial pentru 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
