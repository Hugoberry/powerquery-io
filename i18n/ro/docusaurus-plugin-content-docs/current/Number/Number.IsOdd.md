# Number.IsOdd


## Description

Indică dacă valoarea este impară.


## Syntax

```powerquery
Number.IsOdd(
    number as number
) as logical
```


## Details

Indică dacă valoarea este impară. Returnează <code>true</code> dacă <code>number</code> este un număr impar; în caz contrar, returnează <code>false</code>.


## Examples

### Example #1 
Verificaţi dacă 625 este un număr impar.
```powerquery
Number.IsOdd(625)
```

Result: 
```powerquery
true
```


### Example #2 
Verificaţi dacă 82 este un număr impar.
```powerquery
Number.IsOdd(82)
```

Result: 
```powerquery
false
```
