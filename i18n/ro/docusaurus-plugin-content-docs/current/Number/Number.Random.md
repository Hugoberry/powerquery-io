# Number.Random


## Description

Returnează un număr aleatoriu.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Returnează un număr aleatoriu cuprins între 0 și 1.


## Examples

### Example #1 
Obţineţi un număr aleatoriu.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
