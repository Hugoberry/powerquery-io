# Value.Alternates


## Description

Exprimă planurile alternative de interogare.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Exprimă planurile alternative de interogare dintr-o expresie plan de interogare obținută prin intermediul Value.Expression(Value.Optimize(...)). Nu este destinat pentru alte utilizări.



## Category
Expression
