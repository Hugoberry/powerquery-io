# Value.Add


## Description

Returnează suma celor două valori.


## Syntax

```powerquery
Value.Add(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Returnează suma dintre <code>value1</code> şi <code>value2</code>. Un parametru <code>precision</code> opţional poate fi specificat; în mod implicit, se utilizează <code>Precision.Double</code>.

Values.Arithmetic operations
