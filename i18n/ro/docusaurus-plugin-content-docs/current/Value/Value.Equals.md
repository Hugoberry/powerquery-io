# Value.Equals


## Description

Returnează dacă două valori sunt egale.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Returnează true dacă valoarea <code>value1</code> este egală cu valoarea <code>value2</code>; în caz contrar, returnează false.

Values
