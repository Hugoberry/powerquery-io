# Value.NullableEquals


## Description

Returnează dacă două valori sunt egale.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Returnează null dacă oricare argument „<code>value1</code>”, „<code>value2</code>” este null; în caz contrar, este echivalent cu Value.Equals.



## Category
Values
