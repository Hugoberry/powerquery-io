# Value.Multiply


## Description

Returnează produsul celor două valori.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Returnează produsul înmulţirii <code>value1</code> cu <code>value2</code>. Un parametru <code>precision</code> opţional poate fi specificat; în mod implicit, se utilizează <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
