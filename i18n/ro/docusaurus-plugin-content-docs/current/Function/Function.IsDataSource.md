# Function.IsDataSource


## Description

Returnează dacă o funcție anume este sau nu considerată o sursă de date.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Returnează dacă <code>function</code> este sau nu considerată o sursă de date.
