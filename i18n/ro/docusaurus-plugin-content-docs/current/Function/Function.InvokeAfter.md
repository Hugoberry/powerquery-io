# Function.InvokeAfter


## Description

Invocă funcția dată după expirarea duratei specificate.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

Returnează rezultatul invocării <code>function</code> după ce trece durata <code>delay</code>.



## Category
Function
