# Uri.Combine


## Description

Returnează un identificator URI absolut care reprezintă combinaţia dintre identificatorul URI de intrare de bază şi identificatorul URI relativ.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Returnează un identificator URI absolut care este combinaţia dintre <code>baseUri</code> de intrare şi <code>relativeUri</code>.
