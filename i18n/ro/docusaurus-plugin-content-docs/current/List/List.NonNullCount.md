# List.NonNullCount


## Description

Returnează numărul de elemente care nu sunt null din listă.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Returnează numărul de elemente care nu sunt null din lista <code>list</code>.



## Category
List.Information
