# List.Count


## Description

Returnează numărul de elemente din listă.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Returnează numărul de elemente din lista <code>list</code>.


## Examples

### Example #1 
Aflaţi numărul de valori din lista \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
