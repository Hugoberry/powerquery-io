# Error.Record


## Description

Returnează o înregistrare de eroare din valorile text furnizate pentru motiv, mesaj şi detalii.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
