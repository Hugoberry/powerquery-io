# Table.PartitionValues


## Description

Vráti informácie o spôsobe rozdelenia tabuľky.


## Syntax

```powerquery
Table.PartitionValues(
    table as table
) as table
```


## Details

Vráti informácie o spôsobe rozdelenia tabuľky.  Vráti sa tabuľka, v ktorej každý stĺpec je stĺpcom oblasti v pôvodnej tabuľke a každý riadok zodpovedá oblasti v pôvodnej tabuľke.

Table.Information
