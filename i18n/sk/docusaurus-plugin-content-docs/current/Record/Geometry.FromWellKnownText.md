# Geometry.FromWellKnownText


## Description

Preloží text predstavujúci geometrickú hodnotu vo formáte Well-Known Text (WKT) do štruktúrovaného záznamu.


## Syntax

```powerquery
Geometry.FromWellKnownText(
    input as text
) as record
```


## Details

Preloží text predstavujúci geometrickú hodnotu vo formáte Well-Known Text (WKT) do štruktúrovaného záznamu. WKT je štandardný formát definovaný konzorciom Open Geospatial Consortium (OGC) a je to typický formát serializácie používaný databázami vrátane SQL Servera.
