# Geometry.ToWellKnownText


## Description

Preloží štruktúrovanú hodnotu geometrického bodu na jej reprezentáciu vo formáte Well-Known Text (WKT).


## Syntax

```powerquery
Geometry.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Preloží štruktúrovanú hodnotu geometrického bodu na jej reprezentáciu vo formáte Well-Known Text (WKT), ako je definovaná konzorciom Open Geospatial Consortium (OGC), ako aj na formát serializácie používaný databázami vrátane SQL Servera.
