# Number.Atan2


## Description

Vráti arkustangens delenia dvoch čísel.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Vráti arkustangens delenia dvoch čísel <code>y</code> a <code>x</code>. Delenie sa skonštruuje ako <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
