# Number.BitwiseXor


## Description

Vráti výsledok vykonania bitovej operácie &#34;XOR&#34; (Exclusive-OR) medzi dvomi vstupmi.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Vráti výsledok vykonania bitovej operácie "XOR" (Exclusive-OR) medzi vstupmi <code>number1</code> a <code>number2</code>.



## Category
Number.Bytes
