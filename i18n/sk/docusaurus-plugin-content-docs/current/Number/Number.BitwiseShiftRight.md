# Number.BitwiseShiftRight


## Description

Posunie množinu bitov doprava.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Vráti výsledok vykonania bitového posunu doprava vo vstupe <code>number1</code> o stanovený počet bitov <code>number2</code>.



## Category
Number.Bytes
