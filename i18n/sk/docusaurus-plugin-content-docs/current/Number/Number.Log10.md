# Number.Log10


## Description

Vráti dekadický logaritmus (so základom 10) daného čísla.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Vráti dekadický logaritmus (so základom 10) čísla <code>number</code>. Ak hodnota <code>number</code> je null, funkcia <code>Number.Log10</code> vráti hodnotu null.


## Examples

### Example #1 
Získajte dekadický logaritmus (so základom 10) čísla 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
