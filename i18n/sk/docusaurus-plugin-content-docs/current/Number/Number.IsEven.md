# Number.IsEven


## Description

Udáva, či je hodnota párna.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

Udáva, či je hodnota <code>number</code> párna. Ak je párna, vráti hodnotu <code>true</code>, inak vráti hodnotu <code>false</code>.


## Examples

### Example #1 
Skontrolujte, či je číslo 625 párne.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Skontrolujte, či je číslo 82 párne.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
