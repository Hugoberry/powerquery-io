# Cube.MeasureProperties


## Description

Vráti tabuľku, ktorá obsahuje množinu dostupných vlastností mierky, ktoré sú rozložené v kocke.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Vráti tabuľku, ktorá obsahuje množinu dostupných vlastností pre miery, ktoré sú rozbalené v kocke.

Cube
