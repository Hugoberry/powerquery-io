# Cube.Measures


## Description

Vráti tabuľku obsahujúcu množinu dostupných mierok.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

Vráti tabuľku obsahujúcu množinu dostupných mierok v rámci kocky <code>cube</code>.    Každá mierka je znázornená ako funkcia. Mierky možno v kocke použiť pomocou funkcie Cube.AddMeasureColumn.
