# Cube.Transform


## Description

Použije zoznam funkcií kocky.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Použije zoznam <code>transforms</code> funkcií kocky v kocke <code>cube</code>.
