# Cube.AttributeMemberId


## Description

Vráti jedinečný identifikátor člena z hodnoty vlastnosti členov.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Vráti jedinečný identifikátor člena z hodnoty vlastnosti členov. <code>attribute</code>. Vráti hodnotu null pre všetky ostatné hodnoty.

Cube
