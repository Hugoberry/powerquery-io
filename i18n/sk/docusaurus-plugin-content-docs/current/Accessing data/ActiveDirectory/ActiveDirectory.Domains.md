# ActiveDirectory.Domains


## Description

Vráti zoznam domén služby Active Directory v rovnakej doménovej štruktúre ako zadaná doména alebo vráti doménu aktuálneho počítača v prípade, že nie je zadaná.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```
