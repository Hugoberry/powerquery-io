# Exchange.Contents


## Description

Vráti tabuľku obsahu z konta Microsoft Exchange.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Details

Vráti obsah z konta Microsoft Exchange <code>mailboxAddress</code>. Ak nie je zadané konto <code>mailboxAddress</code>, použije sa predvolené konto poverenia.
