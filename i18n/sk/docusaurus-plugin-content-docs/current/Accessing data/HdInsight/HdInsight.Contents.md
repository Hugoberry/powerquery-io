# HdInsight.Contents


## Description

Vráti navigačnú tabuľku obsahujúcu kontajnery nachádzajúce sa v zadanom konte z trezora ukladacieho priestoru platformy Azure.


## Syntax

```powerquery
HdInsight.Contents(
    account as text
) as table
```


## Details

Vráti navigačnú tabuľku obsahujúcu riadok pre každý kontajner nachádzajúci sa na URL adrese konta, <code>account</code>, z trezora ukladacieho priestoru platformy Azure. Každý riadok obsahuje prepojenie na objekty BLOB kontajnera.
