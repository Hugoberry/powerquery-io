# BinaryFormat.ByteOrder


## Description

Vráti binárny formát s poradím bajtov zadaným pomocou funkcie.


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

Vráti binárny formát s poradím bajtov zadaným pomocou výrazu <code>binaryFormat</code>.  Predvolené poradie bajtov je <code>ByteOrder.BigEndian</code>.
