# Binary.ViewError


## Description

Vytvorí upravený záznam chýb, ktorý nespustí návrat pri vygenerovaní obslužným programom definovaným v zobrazení (prostredníctvom funkcie Binary.View).


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Details

Vytvorí upravený záznam chýb z <code>errorRecord</code>, ktorý nespustí návrat pri vygenerovaní obslužným programom definovaným v zobrazení (prostredníctvom funkcie Binary.View).
