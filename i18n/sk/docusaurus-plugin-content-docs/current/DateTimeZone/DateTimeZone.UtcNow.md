# DateTimeZone.UtcNow


## Description

Vráti aktuálny dátum a čas v čase UTC (časové pásmo GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Vráti aktuálny dátum a čas v čase UTC (časové pásmo GMT).


## Examples

### Example #1 
Získajte aktuálny dátum a čas v pásme UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
