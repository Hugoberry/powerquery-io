# List.Count


## Description

Vráti počet položiek v zozname.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Vráti počet položiek v zozname <code>list</code>.


## Examples

### Example #1 
Zistite počet hodnôt v zozname \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
