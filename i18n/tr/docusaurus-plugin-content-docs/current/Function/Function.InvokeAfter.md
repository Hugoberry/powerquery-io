# Function.InvokeAfter


## Description

Belirtilen süre dolduktan sonra verilen işlevi çağırır.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

<code>delay</code> süresi dolduktan sonra <code>function</code> çağırmanın sonucunu döndürür.



## Category
Function
