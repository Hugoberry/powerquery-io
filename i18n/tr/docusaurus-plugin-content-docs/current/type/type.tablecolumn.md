# Type.TableColumn


Bir tablodaki sütunun türünü döndürür.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Remarks

<code>tableType</code> tablo türündeki <code>column</code> sütununun türünü döndürür.
