# Value.Equals


## Description

İki değerin eşit olup olmadığını döndürür.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

<code>value1</code> değeri <code>value2</code> değerine eşitse doğru, aksi halde yanlış değerini döndürür.



## Category
Values
