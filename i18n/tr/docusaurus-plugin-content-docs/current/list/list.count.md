# List.Count


Listedeki öğe sayısını döndürür.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

<code>list</code> listesindeki öğe sayısını döndürür.


## Examples

### Example #1 
\{1, 2, 3} listesindeki değerlerin sayısını bulur.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
