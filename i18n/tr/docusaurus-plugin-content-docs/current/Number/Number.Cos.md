# Number.Cos


## Description

Sayının kosinüsünü döndürür.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

<code>number</code> değerinin kosinüsünü döndürür.


## Examples

### Example #1 
0 açısının kosinüsünü bulur.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
