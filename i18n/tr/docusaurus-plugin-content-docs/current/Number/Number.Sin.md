# Number.Sin


## Description

Sayının sinüsünü döndürür.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

<code>number</code> değerinin sinüsünü döndürür.


## Examples

### Example #1 
0 açısının sinüsünü bulur.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
