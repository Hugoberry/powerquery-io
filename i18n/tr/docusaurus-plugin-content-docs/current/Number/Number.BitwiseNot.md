# Number.BitwiseNot


Her bitin girdinin karşıtı olduğu bir bayt döndürür.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Remarks

<code>number</code> üzerinde bit tabanlı "Not" işlemi uygulamanın sonucunu döndürür.



## Category
Number.Bytes
