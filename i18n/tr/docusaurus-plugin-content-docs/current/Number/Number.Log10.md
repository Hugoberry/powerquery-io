# Number.Log10


## Description

Sayının 10 tabanındaki logaritmasını döndürür.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Bir sayının (<code>number</code>) 10 tabanındaki logaritmasını döndürür. <code>number</code> null ise <code>Number.Log10</code> null döndürür.


## Examples

### Example #1 
2&#39;nin 10 tabanındaki logaritmasını alır.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
