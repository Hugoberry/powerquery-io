# Number.Tan


Sayının tanjantını döndürür.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

<code>number</code> değerinin tanjantını döndürür.


## Examples

### Example #1 
1 açısının tanjantını bulur.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
