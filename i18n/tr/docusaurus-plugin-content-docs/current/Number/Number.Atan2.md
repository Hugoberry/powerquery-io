# Number.Atan2


## Description

İki sayının bölümünün arktanjantını döndürür.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

<code>y</code> ve <code>x</code> sayılarının bölümünün arktanjantını döndürür. Bölüm <code>y</code>/<code>x</code> olarak oluşturulur.



## Category
Number.Trigonometry
