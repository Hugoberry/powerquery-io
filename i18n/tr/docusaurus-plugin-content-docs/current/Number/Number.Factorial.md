# Number.Factorial


## Description

Sayının faktöriyelini döndürür.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

<code>number</code> sayısının faktöriyelini döndürür.


## Examples

### Example #1 
10&#39;un faktöriyelini bulur.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
