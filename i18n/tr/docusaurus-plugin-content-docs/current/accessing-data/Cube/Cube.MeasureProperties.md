# Cube.MeasureProperties


Küpte genişletilen ölçülerin özellikleri kümesini içeren bir tablo döndürür.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Remarks

Küpte genişletilen ölçüler için kullanılabilen özelliklerin kümesini içeren bir tablo döndürür.
