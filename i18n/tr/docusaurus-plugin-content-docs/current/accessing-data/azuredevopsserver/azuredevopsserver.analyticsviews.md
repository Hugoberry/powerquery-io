# AzureDevOpsServer.AnalyticsViews


Kuruluşu ve proje adlarını girin.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Azure DevOps tarafından sunulan Analytics görünümlerinin bir tablosunu döndürür.
