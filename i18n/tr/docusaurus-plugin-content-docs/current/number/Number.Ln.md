# Number.Ln


Sayının doğal logaritmasını döndürür.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Remarks

Sayıların doğal logaritmasını döndürür, <code>number</code>. <code>number</code> null ise <code>Number.Ln</code> null döndürür.


## Examples

### Example #1 
15&#39;in doğal logaritmasını alır.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
