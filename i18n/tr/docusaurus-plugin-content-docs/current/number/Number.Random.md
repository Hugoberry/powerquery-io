# Number.Random


Rastgele bir sayı döndürür.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

0 ve 1 arasında rastgele bir sayı döndürür.


## Examples

### Example #1 
Rastgele bir sayı alır.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
