# Number.BitwiseAnd


İki girdi arasında bit tabanlı &#34;And&#34; işlemi uygulamanın sonucunu döndürür.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> ve <code>number2</code> arasında bit tabanlı "And" işlemi uygulamanın sonucunu döndürür.

Number.Bytes
