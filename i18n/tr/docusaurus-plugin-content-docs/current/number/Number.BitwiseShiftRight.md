# Number.BitwiseShiftRight


Bit kümesini sağa kaydırır.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> üzerinde <code>number2</code> belirtilen bit sayısı kadar sağa bit tabanlı kaydırma gerçekleştirmenin sonucunun döndürür.
