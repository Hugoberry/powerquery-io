# Number.BitwiseShiftLeft


Bit kümesini sola kaydırır.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Remarks

<code>number1</code> üzerinde <code>number2</code> belirtilen bit sayısı kadar sola bit tabanlı kaydırma gerçekleştirmenin sonucunun döndürür.

Number.Bytes
