# Number.RandomBetween


İki sayı arasında rastgele bir sayı döndürür.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Remarks

<code>bottom</code> ve <code>top</code> arasında rastgele bir sayı döndürür.


## Examples

### Example #1 
1 ve 5 arasında rastgele bir sayı alır.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
