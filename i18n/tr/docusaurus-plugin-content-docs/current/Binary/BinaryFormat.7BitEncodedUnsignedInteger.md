# BinaryFormat.7BitEncodedUnsignedInteger


## Description

7 bit değişken uzunlukta kodlama kullanılarak kodlanan bir 64 bit işaretsiz tamsayıyı okuyan bir ikili biçimi.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details
