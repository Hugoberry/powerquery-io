# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Sağlanan <code>text</code> değerini tersine çevirir.


## Examples

### Example #1 
&#34;123&#34; metnini tersine çevirir.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
