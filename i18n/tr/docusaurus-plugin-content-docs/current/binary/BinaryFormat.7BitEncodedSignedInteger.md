# BinaryFormat.7BitEncodedSignedInteger


Bir 7 bit değişken uzunluğunda kodlama kullanılarak kodlanan bir 64 bit işaretli tamsayıyı okuyan bir ikili biçimi.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

Bir 7 bit değişken uzunluğunda kodlama kullanılarak kodlanan bir 64 bit işaretli tamsayıyı okuyan bir ikili biçimi.
