# DateTimeZone.FixedUtcNow


UTC (GMT saat dilimi) biçiminde geçerli tarih ve saati döndürür. Bu değer sabittir ve sonraki çağrılarla değişmez.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
