# Uri.Combine


Girdi taban URI&#39;si ile göreli URL birleşimi olan tam bir URI döndürür.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Remarks

<code>baseUri</code> ve <code>relativeUri</code> girdisinin birleşimi olan tam bir URI döndürür.
