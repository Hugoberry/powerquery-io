# Type.ReplaceFacets


## Description

Bir türe ilişkin modelleri değiştirir.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

<code>type</code> modellerini <code>facets</code> kaydında bulunan modellerle değiştirir.
