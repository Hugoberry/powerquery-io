# Webtrends.Tables


## Description

Webtrends hesabınızla ilişkilendirilen Profil Kimliğinizi girin.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

İlgili Webtrends verilerinin yer aldığı bir tablo döndürür.
