# Cube.AttributeMemberProperty


Boyut özniteliğinin bir özelliğini döndürür.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

<code>attribute</code> boyut özniteliğinin <code>propertyName</code> özelliğini döndürür.



## Category
Cube
