# Cube.Transform


## Description

Küp işlevleri listesini uygular.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

<code>cube</code> üzerinde, <code>transforms</code> küp işlevleri listesini uygular.

Cube
