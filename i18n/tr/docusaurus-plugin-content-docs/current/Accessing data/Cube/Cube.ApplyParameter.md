# Cube.ApplyParameter


## Description

Parametre uygulandıktan sonra bir küp döndürür.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

<code>cube</code> öğesine <code>arguments</code> ile <code>parameter</code> uygulandıktan sonra bir küp döndürür.



## Category
Cube
