# AzureDevOpsServer.Views


Kuruluşu ve proje adlarını girin.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record

```


## Remarks

Azure DevOps tarafından sunulan bir OData akışları tablosu döndürür.
