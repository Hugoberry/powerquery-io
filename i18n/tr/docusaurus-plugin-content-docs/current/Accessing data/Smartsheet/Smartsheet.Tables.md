# Smartsheet.Tables


Smartsheet API&#39;sinden sayfaların, raporların, klasörlerin ve çalışma alanlarının bulunduğu bir tablo döndürür


## Syntax

```powerquery
Smartsheet.Tables(

) as table
```


## Remarks

Smartsheet API'sinden sayfaların, raporların, klasörlerin ve çalışma alanlarının bulunduğu iç içe bir tablo döndürür


## Examples

### Example #1 
Smartsheet hiyerarşisinin en üst düzeyinde bulunan sayfaları, klasörleri, raporları ve çalışma alanlarını döndürür
```powerquery
Smartsheet.Tables()
```

Result: 
```powerquery
Smartsheet hiyerarşisinin en üst düzeyinde bulunan sayfaların, klasörlerin, raporların ve çalışma alanlarının yer aldığı bir tablo
```
