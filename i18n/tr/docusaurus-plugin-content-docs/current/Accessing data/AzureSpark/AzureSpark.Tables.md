# AzureSpark.Tables


## Description

Tabloları bir Azure Spark örneğinde listeleyin.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Details

Tabloları bir Azure Spark örneğinde listeleyin.


## Examples

### Example #1 
Tabloları bir Azure Spark örneğinde listeleyin.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
