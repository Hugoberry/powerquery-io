# VSTS.AnalyticsViews


## Description

Kuruluşu ve proje adlarını girin.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Azure DevOps tarafından sunulan Analytics görünümlerinin bir tablosunu döndürür.
