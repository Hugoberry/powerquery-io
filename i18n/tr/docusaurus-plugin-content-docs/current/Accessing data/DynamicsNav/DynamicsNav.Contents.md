# DynamicsNav.Contents


Dynamics NAV OData hizmet uç noktanızın URL&#39;sini girin.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

İlgili Dynamics NAV verilerinin bulunduğu bir tablo döndürür.
