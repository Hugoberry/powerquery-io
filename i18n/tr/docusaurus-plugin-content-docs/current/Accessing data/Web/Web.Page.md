# Web.Page


## Description

HTML belgesinin içeriğini destekçi yapılarına ayrılmış şekilde ve tam belgenin gösterimini ve etiketler kaldırıldıktan sonraki metnini döndürür.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
