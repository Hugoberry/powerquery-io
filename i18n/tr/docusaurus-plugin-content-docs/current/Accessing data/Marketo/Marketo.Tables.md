# Marketo.Tables


Hesabınızla ilişkili olan Marketo REST API uç noktasına ilişkin URL&#39;yi belirtin.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Remarks

İlgili Marketo verilerinin yer aldığı bir tablo döndürür.
