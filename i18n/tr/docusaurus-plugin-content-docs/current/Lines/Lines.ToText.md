# Lines.ToText


Metin listesini tek bir metne dönüştürür.  Belirtilen lineSeparator her satıra eklenir.  Belirtilmezse, satır başı ve satır akış karakterleri kullanılır.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```


## Remarks
