# Error.Record


## Description

Neden, ileti ve ayrıntı için sağlanan metin değerlerinden bir hata kaydı döndürür.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
