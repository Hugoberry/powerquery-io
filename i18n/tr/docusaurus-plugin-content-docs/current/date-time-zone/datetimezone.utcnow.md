# DateTimeZone.UtcNow


UTC (GMT saat dilimi) biçiminde geçerli tarih ve saati döndürür.


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

UTC (GMT saat dilimi) biçiminde geçerli tarih ve saati döndürür.


## Examples

### Example #1 
Geçerli tarih ve saati UTC biçiminde alır.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
