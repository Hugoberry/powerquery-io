# List.NonNullCount


## Description

Listedeki null olmayan öğe sayısını döndürür.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

<code>list</code> listesindeki null olmayan öğe sayısını döndürür.



## Category
List.Information
