# Text.Length


Karakterlerin sayısını döndürür.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

<code>text</code> metnindeki karakter sayısını verir.


## Examples

### Example #1 
&#34;Hello World&#34; metninde kaç karakterin olduğunu bulur.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
