# List.Count


Retourneert het aantal items in de lijst.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Retourneert het aantal items in de lijst <code>list</code>.


## Examples

### Example #1 
Het aantal waarden zoeken in de lijst \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
