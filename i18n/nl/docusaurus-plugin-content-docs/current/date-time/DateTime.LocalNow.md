# DateTime.LocalNow


Retourneert de huidige datum en tijd in de lokale tijdzone.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Remarks

Retourneert een <code>datetime</code>waarde die is ingesteld op de huidige datum en tijd op het systeem.



## Category
DateTime
