# Number.BitwiseShiftLeft


## Description

Verschuift de bitset naar links.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Retourneert het resultaat van een bitwise-verschuiving naar links op <code>number1</code> conform het aantal opgegeven bits <code>number2</code>.



## Category
Number.Bytes
