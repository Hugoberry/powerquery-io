# Number.Tan


## Description

Retourneert de tangens van het getal.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Retourneert de tangens van <code>number</code>.


## Examples

### Example #1 
De tangens van de hoek 1 zoeken.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
