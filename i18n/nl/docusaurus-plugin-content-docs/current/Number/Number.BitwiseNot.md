# Number.BitwiseNot


## Description

Retourneert een byte waar elke bit het tegenovergestelde van de invoer is.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Retourneert het resultaat van de uitvoering van een bitwise "Not"-bewerking op <code>number</code>.



## Category
Number.Bytes
