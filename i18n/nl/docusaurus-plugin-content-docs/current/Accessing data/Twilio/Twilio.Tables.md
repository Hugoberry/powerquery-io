# Twilio.Tables


## Description

Voer het aantal maanden aan historische Twilio-gegevens in dat u wilt ontvangen.


## Syntax

```powerquery
Twilio.Tables(
    historyMonths as number
) as table
```


## Details

Hiermee wordt een tabel met relevante Twilio-gegevens geretourneerd.
