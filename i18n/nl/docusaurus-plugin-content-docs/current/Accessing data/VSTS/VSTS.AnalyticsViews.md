# VSTS.AnalyticsViews


Voer een organisatie en projectnamen in.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Remarks

Hiermee wordt een tabel met Analytics-weergaven van Azure DevOps geretourneerd.
