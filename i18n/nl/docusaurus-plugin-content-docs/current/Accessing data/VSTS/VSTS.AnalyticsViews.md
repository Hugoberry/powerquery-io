# VSTS.AnalyticsViews


## Description

Voer een organisatie en projectnamen in.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Hiermee wordt een tabel met Analytics-weergaven van Azure DevOps geretourneerd.
