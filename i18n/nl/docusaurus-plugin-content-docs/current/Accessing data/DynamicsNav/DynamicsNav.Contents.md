# DynamicsNav.Contents


## Description

Voer de URL in van het OData-service-eindpunt voor Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Hiermee wordt een tabel met de relevante Dynamics NAV-gegevens geretourneerd.
