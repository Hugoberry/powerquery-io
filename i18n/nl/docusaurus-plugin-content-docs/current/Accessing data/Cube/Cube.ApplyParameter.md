# Cube.ApplyParameter


## Description

Retourneert een kubus nadat er een parameter op is toegepast.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Retourneert een kubus nadat <code>parameter</code> met <code>arguments</code> op <code>cube</code> is toegepast.



## Category
Cube
