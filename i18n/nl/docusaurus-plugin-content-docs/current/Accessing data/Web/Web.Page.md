# Web.Page


## Description

Retourneert de inhoud van het HTML-document, opgesplitst in alle structuren van dat document, alsmede een weergave van het volledige document en de tekst van dat document na verwijdering van de tags.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
