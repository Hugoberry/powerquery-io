# Type.ListItem


## Description

Retourneert een itemtype van een lijsttype.


## Syntax

```powerquery
Type.ListItem(
    type as type
) as type
```


## Details

Retourneert een itemtype van een lijst <code>type</code>.


## Examples

### Example #1 
Itemtype zoeken in de lijst &lt;code&gt;\{number}&lt;/code&gt;.
```powerquery
Type.ListItem(type {number})
```

Result: 
```powerquery
type number
```
