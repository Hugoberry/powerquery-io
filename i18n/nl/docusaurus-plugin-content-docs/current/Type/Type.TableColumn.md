# Type.TableColumn


## Description

Retourneert het type van een kolom in een tabel.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Retourneert het type van de kolom <code>column</code> in het tabeltype <code>tableType</code>.



## Category
Type
