# Type.ReplaceFacets


Hiermee worden de facetten van een type vervangen.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Remarks

Hiermee worden de facetten van <code>type</code> vervangen door de facetten in de record <code>facets</code>.
