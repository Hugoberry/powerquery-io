# Webtrends.Tables


Voer de profiel-id in die is gekoppeld aan uw Webtrends-account.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```
