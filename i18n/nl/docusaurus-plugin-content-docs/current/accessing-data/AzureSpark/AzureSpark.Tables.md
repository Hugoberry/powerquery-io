# AzureSpark.Tables


Geef de tabellen in een Azure Spark-exemplaar weer.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Remarks

Geef de tabellen in een Azure Spark-exemplaar weer.


## Examples

### Example #1 
Geef de tabellen in een Azure Spark-exemplaar weer.

AzureSpark.Tables("contoso.azurehdinsight.net")
