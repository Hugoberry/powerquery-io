# VSTS.Views


Voer een organisatie en projectnamen in.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Remarks

Hiermee wordt een tabel met OData-inhoud van Azure DevOps geretourneerd.
