# Number.Sin


Retourneert de sinus van het getal.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

Retourneert de sinus van <code>number</code>.


## Examples

### Example #1 
De sinus van de hoek 0 zoeken.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
