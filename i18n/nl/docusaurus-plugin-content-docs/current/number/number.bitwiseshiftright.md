# Number.BitwiseShiftRight


Verschuift de bitset naar rechts.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Retourneert het resultaat van een bitwise-verschuiving naar rechts op <code>number1</code> conform het aantal opgegeven bits <code>number2</code>.



## Category
Number.Bytes
