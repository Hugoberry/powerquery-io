# Number.Cos


Retourneert de cosinus van het getal.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Retourneert de cosinus van <code>number</code>.


## Examples

### Example #1 
De cosinus van de hoek 0 zoeken.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
