# List.NonNullCount


Retourneert het aantal items uit de lijst die niet gelijk zijn aan nul.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Retourneert het aantal items uit de lijst <code>list</code> die niet gelijk zijn aan nul.

List.Information
