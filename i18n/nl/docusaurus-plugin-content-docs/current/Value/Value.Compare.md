# Value.Compare


## Description

Hiermee wordt -1, 0 of 1 geretourneerd op basis van of de eerste waarde kleiner is dan, gelijk is aan of groter is dan de tweede.


## Syntax

```powerquery
Value.Compare(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as number
```
