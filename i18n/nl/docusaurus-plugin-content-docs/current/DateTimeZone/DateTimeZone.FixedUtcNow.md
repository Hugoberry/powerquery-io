# DateTimeZone.FixedUtcNow


## Description

Retourneert de huidige datum en tijd in UTC (de GMT-tijdzone). Deze waarde is vast en verandert niet met opeenvolgende aanroepen.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
