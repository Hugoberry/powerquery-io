# DateTimeZone.UtcNow


## Description

Retourneert de huidige datum en tijd in UTC (de GMT-tijdzone).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Retourneert de huidige datum en tijd in UTC (de GMT-tijdzone).


## Examples

### Example #1 
De huidige datum en tijd in UTC ophalen.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
