# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Hiermee wordt de opgegeven <code>text</code> omgekeerd.


## Examples

### Example #1 
Hiermee wordt de tekst &#39;123&#39; omgekeerd.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
