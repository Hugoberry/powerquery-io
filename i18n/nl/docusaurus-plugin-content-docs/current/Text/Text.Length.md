# Text.Length


## Description

Retourneert het aantal tekens.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Retourneert het aantal tekens uit de tekst <code>text</code>.


## Examples

### Example #1 
Uitzoeken uit hoeveel tekens de tekst &#34;Hallo wereld&#34; bestaat.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
