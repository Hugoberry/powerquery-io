# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Een binaire indeling die een 64-bits geheel getal zonder teken leest dat is gecodeerd met een 7-bits codering met variabele lengte.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
