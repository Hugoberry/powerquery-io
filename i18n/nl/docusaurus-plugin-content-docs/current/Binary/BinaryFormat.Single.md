# BinaryFormat.Single


## Description

Een binaire indeling die een IEEE zwevende-kommagetal met enkele precisie met een lengte van 4 bytes leest.


## Syntax

```powerquery
BinaryFormat.Single(
    binary as binary
) as any
```
