# BinaryFormat.7BitEncodedSignedInteger


## Description

Een binaire indeling die een 64-bits geheel getal met teken leest dat is gecodeerd met een 7-bits codering met variabele lengte.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
