# Text.ToBinary


## Description

Mengkodekan teks ke format biner.


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

Mengkodekan nilai teks tertentu, <code>text</code>, ke nilai biner menggunakan <code>encoding</code> yang ditetapkan.

Text.Conversions from and to text
