# Text.Length


## Description

Menghasilkan jumlah karakter.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Menghasilkan jumlah karakter pada teks <code>text</code>.


## Examples

### Example #1 
Mencari jumlah karakter pada teks &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
