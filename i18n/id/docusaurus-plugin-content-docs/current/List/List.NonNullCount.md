# List.NonNullCount


## Description

Menghasilkan jumlah item non-null dalam daftar.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Menghasilkan jumlah item non-null dalam daftar <code>list</code>.

List.Information
