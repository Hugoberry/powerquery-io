# List.Count


## Description

Menghasilkan jumlah item dalam daftar.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Menghasilkan jumlah item dalam daftar <code>list</code>.


## Examples

### Example #1 
Mencari jumlah nilai dalam daftar \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
