# DateTime.LocalNow


## Description

Menghasilkan tanggal dan waktu saat ini dalam zona waktu lokal.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Menghasilkan nilai <code>datetime</code> yang ditetapkan ke tanggal dan waktu saat ini pada sistem.



## Category
DateTime
