# Error.Record


## Description

Menghasilkan data kesalahan alasan, pesan, dan rincian dari nilai teks yang tersedia.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details

Menghasilkan data kesalahan alasan, pesan, dan rincian dari nilai teks yang tersedia.
