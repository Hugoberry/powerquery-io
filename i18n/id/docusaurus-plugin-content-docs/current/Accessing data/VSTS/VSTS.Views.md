# VSTS.Views


## Description

Masukkan nama organisasi dan proyek.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Menghasilkan tabel umpan OData yang diberikan oleh Azure DevOps.
