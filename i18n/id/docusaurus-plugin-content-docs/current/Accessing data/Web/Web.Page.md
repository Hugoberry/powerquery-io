# Web.Page


## Description

Menghasilkan konten dokumen HTML yang dipecah menjadi struktur konstituennya, dan representasi dokumen penuh beserta teksnya setelah menghapus tag.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
