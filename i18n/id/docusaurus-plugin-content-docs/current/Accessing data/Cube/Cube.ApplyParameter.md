# Cube.ApplyParameter


## Description

Menghasilkan kubus setelah menerapkan parameter ke kubus.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Menghasilkan kubus setelah menerapkan <code>parameter</code> dengan <code>arguments</code> ke <code>cube</code>.



## Category
Cube
