# Cube.AttributeMemberProperty


## Description

Mengembalikan properti atribut dimensi.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Mengembalikan properti <code>propertyName</code> dari atribut dimensi <code>attribute</code>.



## Category
Cube
