# Cube.Transform


## Description

Menerapkan daftar fungsi kubus.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Menerapkan daftar fungsi kubus, <code>transforms</code>, pada <code>cube</code>.



## Category
Cube
