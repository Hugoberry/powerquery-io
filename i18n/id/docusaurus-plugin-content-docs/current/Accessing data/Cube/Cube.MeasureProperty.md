# Cube.MeasureProperty


## Description

Mengembalikan properti pengukuran (properti sel).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Mengembalikan properti <code>propertyName</code> dari pengukuran <code>measure</code>.



## Category
Cube
