# Webtrends.Tables


## Description

Masukkan ID Profil yang terkait dengan akun Webtrends Anda.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Menghasilkan tabel dengan data Webtrends yang relevan.
