# DynamicsNav.Contents


## Description

Masukkan URL titik akhir layanan Dynamics NAV OData Anda.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Mengembalikan tabel dengan data Dynamics NAV yang relevan.
