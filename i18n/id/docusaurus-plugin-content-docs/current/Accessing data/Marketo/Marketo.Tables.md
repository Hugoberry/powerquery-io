# Marketo.Tables


## Description

Masukkan URL titik akhir REST API Marketo yang berhubungan dengan akun Anda.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Mengembalikan tabel dengan data Marketo yang relevan.
