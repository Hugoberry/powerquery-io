# Identity.IsMemberOf


## Description

Menentukan apakah identitas merupakan anggota dari kumpulan identitas.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Details

Menentukan apakah identitas merupakan anggota dari kumpulan identitas.
