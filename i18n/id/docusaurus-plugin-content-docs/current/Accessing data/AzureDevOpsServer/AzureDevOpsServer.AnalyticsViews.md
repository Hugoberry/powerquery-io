# AzureDevOpsServer.AnalyticsViews


## Description

Masukkan nama organisasi dan proyek.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Menghasilkan tabel tampilan Analytics yang diberikan oleh Azure DevOps.
