# Number.Random


## Description

Menghasilkan bilangan acak.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Menghasilkan bilangan acak antara 0 dan 1.


## Examples

### Example #1 
Mengambil bilangan acak.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
