# Number.BitwiseShiftRight


## Description

Menggeser kumpulan bit ke kanan.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Memberikan hasil dari menjalankan penggeseran bitwise ke kanan pada <code>number1</code>, berdasarkan jumlah bit yang ditetapkan <code>number2</code>.



## Category
Number.Bytes
