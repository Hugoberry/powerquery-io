# Number.BitwiseShiftLeft


## Description

Menggeser kumpulan bit ke kiri.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Memberikan hasil dari menjalankan penggeseran bitwise ke kiri pada <code>number1</code>, berdasarkan jumlah bit yang ditetapkan <code>number2</code>.



## Category
Number.Bytes
