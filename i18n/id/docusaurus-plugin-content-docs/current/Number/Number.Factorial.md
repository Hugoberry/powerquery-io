# Number.Factorial


## Description

Menghasilkan faktorial bilangan.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Menghasilkan faktorial bilangan <code>number</code>.


## Examples

### Example #1 
Mencari faktorial 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
