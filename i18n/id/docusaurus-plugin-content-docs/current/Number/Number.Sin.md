# Number.Sin


## Description

Menghasilkan sinus bilangan.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Menghasilkan sinus <code>number</code>.


## Examples

### Example #1 
Mencari sinus sudut 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
