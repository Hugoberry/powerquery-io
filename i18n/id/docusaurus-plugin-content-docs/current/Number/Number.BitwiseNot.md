# Number.BitwiseNot


## Description

Menghasilkan byte dengan setiap bit merupakan kebalikan dari input.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Memberikan hasil dari menjalankan operasi bitwise "Not" pada <code>number</code>.



## Category
Number.Bytes
