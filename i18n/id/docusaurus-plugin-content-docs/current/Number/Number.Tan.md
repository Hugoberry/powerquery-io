# Number.Tan


## Description

Menghasilkan tangen bilangan.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Menghasilkan tangen <code>number</code>.


## Examples

### Example #1 
Mencari tangen sudut 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
