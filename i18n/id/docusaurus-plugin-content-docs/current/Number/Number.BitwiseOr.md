# Number.BitwiseOr


## Description

Memberikan hasil dari menjalankan operasi bitwise &#34;Or&#34; di antara kedua input.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Memberikan hasil dari menjalankan operasi bitwise "Or" antara <code>number1</code> dan <code>number2</code>.



## Category
Number.Bytes
