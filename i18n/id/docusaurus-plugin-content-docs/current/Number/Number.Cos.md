# Number.Cos


## Description

Menghasilkan kosinus bilangan.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Menghasilkan kosinus <code>number</code>.


## Examples

### Example #1 
Mencari kosinus sudut 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
