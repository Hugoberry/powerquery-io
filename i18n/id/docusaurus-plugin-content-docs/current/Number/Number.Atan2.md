# Number.Atan2


## Description

Menghasilkan tangen inversi divisi dua bilangan.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Menghasilkan tangen inversi divisi dua bilangan, yakni <code>y</code> dan <code>x</code>. Divisi akan dibuat seperti <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
