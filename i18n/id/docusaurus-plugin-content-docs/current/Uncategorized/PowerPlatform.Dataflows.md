# PowerPlatform.Dataflows


## Description

Mengimpor data dari alur data


## Syntax

```powerquery

    optional options as record
) as table
```


## Details

Hubungkan ke semua alur data yang dapat Anda akses, dan pilih entitas yang ingin digunakan.
