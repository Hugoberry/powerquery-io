# Uri.Combine


## Description

Menghasilkan URI absolut yang merupakan kombinasi URI dasar input dan URI relatif.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Menghasilkan URI absolut yang merupakan kombinasi input <code>baseUri</code> dan <code>relativeUri</code>.

Uri
