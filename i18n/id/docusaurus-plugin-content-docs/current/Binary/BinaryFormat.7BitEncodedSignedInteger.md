# BinaryFormat.7BitEncodedSignedInteger


## Description

Format biner yang membaca bilangan bulat bertanda 64-bit yang dikodekan menggunakan pengodean panjang variabel 7-bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
