# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Format biner yang membaca bilangan bulat tidak bertanda 64-bit yang dikodekan menggunakan pengodean panjang variabel 7-bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
