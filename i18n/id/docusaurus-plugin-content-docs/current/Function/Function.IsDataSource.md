# Function.IsDataSource


## Description

Mengembalikan apakah fungsi tertentu dianggap sebagai sumber data atau tidak.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Mengembalikan apakah <code>function</code> dianggap sebagai sumber data atau tidak.
