# Type.ReplaceFacets


## Description

Mengganti faset jenis.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Mengganti faset <code>type</code> dengan faset yang terdapat dalam rekaman <code>facets</code>.

Type
