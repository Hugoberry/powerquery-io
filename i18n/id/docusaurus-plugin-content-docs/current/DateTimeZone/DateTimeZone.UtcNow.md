# DateTimeZone.UtcNow


## Description

Menghasilkan tanggal dan waktu saat ini di UTC (zona waktu GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Menghasilkan tanggal dan waktu saat ini di UTC (zona waktu GMT).


## Examples

### Example #1 
Mengambil tanggal &amp; waktu saat ini dalam UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
