# DateTimeZone.FixedUtcNow


## Description

Menghasilkan tanggal dan waktu saat ini di UTC (zona waktu GMT). Nilai ini tetap dan tidak akan berubah dengan pemanggilan berturut-turut.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
