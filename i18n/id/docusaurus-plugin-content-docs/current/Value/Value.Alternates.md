# Value.Alternates


## Description

Ekspresikan rencana kueri alternatif.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Ekspresikan rencana kueri alternatif dalam ekspresi rencana kueri yang diperoleh melalui Value.Expression (Value.Optimize(...)). Tidak dimaksudkan untuk penggunaan lainnya.



## Category
Expression
