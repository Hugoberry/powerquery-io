# Value.Equals


## Description

Menghasilkan apakah dua nilai sama.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Menghasilkan benar jika nilai <code>value1</code> sama dengan nilai <code>value2</code>, atau salah.



## Category
Values
