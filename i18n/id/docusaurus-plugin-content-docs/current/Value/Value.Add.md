# Value.Add


## Description

Menghasilkan jumlah dari dua nilai.


## Syntax

```powerquery
Value.Add(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Menghasilkan jumlah dari <code>value1</code> dan <code>value2</code>. Parameter <code>precision</code> opsional dapat ditetapkan, secara default <code>Precision.Double</code> akan digunakan.

Values.Arithmetic operations
