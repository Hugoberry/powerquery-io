# Value.Multiply


## Description

Menghasilkan produk dari dua nilai.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Menghasilkan produk perkalian <code>value1</code> dengan <code>value2</code>. Parameter <code>precision</code> opsional dapat ditetapkan, secara default <code>Precision.Double</code> akan digunakan.



## Category
Values.Arithmetic operations
