# Value.NullableEquals


## Description

Menghasilkan apakah dua nilai sama.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Menghasilkan null jika argumen '<code>value1</code>', '<code>value2</code>' adalah null, atau sama dengan Value.Equals.



## Category
Values
