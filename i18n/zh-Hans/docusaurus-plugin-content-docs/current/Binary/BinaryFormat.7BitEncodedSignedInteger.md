# BinaryFormat.7BitEncodedSignedInteger


读取使用 7 位可变长度编码进行编码的 64 位带符号整数的二进制格式。


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

读取使用 7 位可变长度编码进行编码的 64 位带符号整数的二进制格式。
