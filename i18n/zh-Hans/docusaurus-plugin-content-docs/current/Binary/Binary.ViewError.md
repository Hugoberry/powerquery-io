# Binary.ViewError


创建修改的错误记录，当在视图上(通过 Binary.View)定义的处理程序引发该记录时，它不会触发回退。


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Remarks

从 <code>errorRecord</code> 创建修改的错误记录，当在视图上(通过 Binary.View)定义的处理程序引发该记录时，它不会触发回退。
