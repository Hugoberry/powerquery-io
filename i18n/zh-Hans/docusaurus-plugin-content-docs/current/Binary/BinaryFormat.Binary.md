# BinaryFormat.Binary


## Description

返回读取二进制值的二进制格式。


## Syntax

```powerquery
BinaryFormat.Binary(
    optional length as any
) as function
```


## Details

返回读取二进制值的二进制格式。如果指定了 <code>length</code>，则该二进制值将包含这些字节。如果未指定 <code>length</code>，则该二进制值将包含剩余字节。可将 <code>length</code> 指定为数值，也可以指定为置二进制数据之前长度的二进制格式。
