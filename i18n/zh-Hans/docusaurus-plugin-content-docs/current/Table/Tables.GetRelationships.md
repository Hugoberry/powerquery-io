# Tables.GetRelationships


## Description

获取一组表之间的关系。


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Details

获取一组表之间的关系。假定集 <code>tables</code> 的结构与导航表的结构相似。<code>dataColumn</code> 定义的列包含实际数据表。
