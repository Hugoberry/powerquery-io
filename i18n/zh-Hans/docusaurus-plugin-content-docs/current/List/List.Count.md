# List.Count


返回列表中的项数。


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

返回列表 <code>list</code> 中的项数。


## Examples

### Example #1 
查看列表 \{1, 2, 3} 中的值数。
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
