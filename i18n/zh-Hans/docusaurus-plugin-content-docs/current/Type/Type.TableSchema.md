# Type.TableSchema


## Description

返回包含指定表类型的列(即架构)的说明的表。


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

返回说明 <code>tableType</code> 的列的表。<br />请参阅有关 <code>Table.Schema</code> 的文档，以了解生成的表的说明。<br />
