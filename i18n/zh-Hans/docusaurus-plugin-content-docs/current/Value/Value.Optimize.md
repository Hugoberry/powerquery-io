# Value.Optimize


## Description

表示 Value.Expression，它用于为值返回已优化的表达式。


## Syntax

```powerquery
Value.Optimize(
    value as any
) as any
```


## Details

在 Value.Expression 中使用时，如果 <code>value</code> 表示可优化的查询，则此函数指示应返回已优化的表达式。否则，<code>value</code> 将通过且不产生任何影响。



## Category
Expression
