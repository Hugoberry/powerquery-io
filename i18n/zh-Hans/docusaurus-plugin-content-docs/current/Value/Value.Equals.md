# Value.Equals


## Description

返回两个值是否相等。


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

如果值 <code>value1</code> 等于值 <code>value2</code>，则返回 true，否则返回 false。



## Category
Values
