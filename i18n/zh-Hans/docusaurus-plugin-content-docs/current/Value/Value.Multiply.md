# Value.Multiply


## Description

返回两个值的乘积。


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

返回将 <code>value1</code> 与 <code>value2</code> 相乘的乘积。可以指定一个可选 <code>precision</code> 参数，默认情况下使用 <code>Precision.Double</code>。



## Category
Values.Arithmetic operations
