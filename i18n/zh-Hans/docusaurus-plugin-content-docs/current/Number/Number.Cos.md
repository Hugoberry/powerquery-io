# Number.Cos


## Description

返回数的余弦。


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

返回 <code>number</code> 的余弦。


## Examples

### Example #1 
计算角 0 的余弦。
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```




## Category
Number.Trigonometry
