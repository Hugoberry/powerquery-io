# Number.BitwiseXor


返回对两个输入执行按位 &#34;XOR&#34; (异或)所得的结果。


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Remarks

返回对 <code>number1</code> 和 <code>number2</code> 执行按位 "XOR" (异或)所得的结果。



## Category
Number.Bytes
