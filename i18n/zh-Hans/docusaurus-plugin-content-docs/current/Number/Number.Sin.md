# Number.Sin


## Description

返回数的正弦。


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

返回 <code>number</code> 的正弦。


## Examples

### Example #1 
计算角 0 的正弦。
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```




## Category
Number.Trigonometry
