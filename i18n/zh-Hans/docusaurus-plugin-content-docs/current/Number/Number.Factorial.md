# Number.Factorial


## Description

返回数的阶乘。


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

返回数 <code>number</code> 的阶乘。


## Examples

### Example #1 
计算 10 的阶乘。
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```

Number.Operations
