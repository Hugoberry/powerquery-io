# Number.Ln


返回数的自然对数。


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Remarks

返回数 <code>number</code> 的自然对数。如果 <code>number</code> 为 null，则 <code>Number.Ln</code> 返回 null。


## Examples

### Example #1 
获取 15 的自然对数。
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
