# Number.BitwiseOr


## Description

返回对两个输入执行按位 &#34;Or&#34; 所得的结果。


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

返回对 <code>number1</code> 和 <code>number2</code> 执行按位 "Or" 所得的结果。



## Category
Number.Bytes
