# Number.Exp


计算 e 的指定次幂。


## Syntax

```powerquery
Number.Exp(
    number as number
) as number
```


## Remarks

返回计算 e 的 <code>number</code> 次幂(指数函数)所得的结果。      <ul>        <li><code>number</code>: 要计算其指数函数的 <code>number</code>。如果 <code>number</code> 为 null，则 <code>Number.Exp</code> 返回 null。</li>      </ul>


## Examples

### Example #1 
计算 e 的 3 次幂。
```powerquery
Number.Exp(3)
```

Result: 
```powerquery
20.085536923187668
```
