# Number.Log10


返回数值的以 10 为底的对数。


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Remarks

返回数值 <code>number</code> 的以 10 为底的对数。如果 <code>number</code> 为 null，则 <code>Number.Log10</code> 返回 null。


## Examples

### Example #1 
获取 2 的以 10 为底的对数。
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
