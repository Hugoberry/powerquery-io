# Number.RandomBetween


返回两个数之间的一个随机数。


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Remarks

返回 <code>bottom</code> 和 <code>top</code> 之间的一个随机数。


## Examples

### Example #1 
获取 1 和 5 之间的一个随机数。
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
