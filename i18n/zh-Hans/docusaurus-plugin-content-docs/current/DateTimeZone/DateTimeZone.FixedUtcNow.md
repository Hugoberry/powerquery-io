# DateTimeZone.FixedUtcNow


## Description

返回采用 UTC (GMT 时区)的当前日期和时间。该值是固定的，因此将不会随着连续调用而更改。


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```


## Details
