# DateTimeZone.UtcNow


## Description

返回采用 UTC (GMT 时区)的当前日期和时间。


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

返回采用 UTC (GMT 时区)的当前日期和时间。


## Examples

### Example #1 
获取采用 UTC 的当前日期和时间。
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
