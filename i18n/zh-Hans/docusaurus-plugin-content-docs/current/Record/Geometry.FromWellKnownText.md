# Geometry.FromWellKnownText


将 Well-Known Text (WKT)格式的表示几何值的文本转换为结构化记录。


## Syntax

```powerquery
Geometry.FromWellKnownText(
    input as text
) as record
```


## Remarks

将 Well-Known Text (WKT)格式的表示几何值的文本转换为结构化记录。WKT 是开放地理空间信息联盟(OGC)定义的一种标准格式，也是 SQL Server 等数据库使用的典型序列化格式。
