# Text.FromBinary


## Description

将数据从二进制解码为文本。


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

使用 <code>encoding</code> 类型将数据 <code>binary</code> 从二进制值解码为文本值。
