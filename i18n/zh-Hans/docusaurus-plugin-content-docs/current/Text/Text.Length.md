# Text.Length


返回字符数。


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

返回文本 <code>text</code> 中的字符数。


## Examples

### Example #1 
查找文本 &#34;Hello World&#34; 中有多少个字符。
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
