# Text.ToBinary


将文本编码为二进制格式。


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Remarks

使用指定的 <code>encoding</code> 将给定的文本值 <code>text</code> 编码为二进制值。
