# Dynamics365BusinessCentral.ApiContents


输入你的 Dynamics 365 Business Central 环境和公司。


## Syntax

```powerquery
Dynamics365BusinessCentral.ApiContents(
    optional environment as text,
    optional company as text,
    optional apiRoute as text
) as table
```


## Remarks

返回包含相关 Dynamics 365 Business Central 数据的表。
