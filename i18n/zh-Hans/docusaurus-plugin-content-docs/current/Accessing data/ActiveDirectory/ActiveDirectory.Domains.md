# ActiveDirectory.Domains


返回与指定域或当前计算机的域(如果未指定任何域)处于同一个林中的 Active Directory 域的列表。


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Remarks

返回与指定域或当前计算机的域(如果未指定任何域)处于同一个林中的 Active Directory 域的列表。
