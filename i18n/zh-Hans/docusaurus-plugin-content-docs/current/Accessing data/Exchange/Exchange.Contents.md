# Exchange.Contents


返回来自 Microsoft Exchange 帐户的目录。


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Remarks

返回 Microsoft Exchange 帐户 <code>mailboxAddress</code> 中的内容表。如果未指定 <code>mailboxAddress</code>，则将使用凭据的默认帐户。
