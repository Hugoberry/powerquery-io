# Dynamics365BusinessCentralOnPremises.Contents


输入 Dynamics 365 Business Central (本地) OData 服务终结点的 URL。


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Remarks

返回包含相关 Dynamics 365 Business Central (本地)数据的表。
