# AzureDevOpsServer.Views


## Description

输入组织和项目名称。


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

返回包含 Azure DevOps 提供的 OData 源的表。
