# AzureDevOpsServer.AnalyticsViews


## Description

输入组织和项目名称。


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

返回包含 Azure DevOps 提供的“分析”视图的表。
