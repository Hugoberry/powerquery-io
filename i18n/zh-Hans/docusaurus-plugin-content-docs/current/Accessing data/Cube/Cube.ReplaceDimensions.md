# Cube.ReplaceDimensions


## Description

替换 Cube.Dimensions 返回的维度集。


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Details

替换 <code>Cube.Dimensions</code> 返回的维度集。    例如，此函数可用于向维度属性添加 ID 列，以便数据源可以按 ID 而不是显示的值进行分组。
