# Cube.AttributeMemberProperty


返回维度属性的属性。


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

返回维度属性 <code>attribute</code> 的属性 <code>propertyName</code>。



## Category
Cube
