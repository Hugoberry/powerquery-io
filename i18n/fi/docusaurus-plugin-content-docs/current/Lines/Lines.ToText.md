# Lines.ToText


## Description

Muuntaa tekstiluettelon yksittäiseksi tekstiksi.  Määritetty lineSeparator-kohde lisätään kullekin riville.  Jos sitä ei määritetä, käytetään rivinvaihtomerkkejä (CR ja LF).


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```
