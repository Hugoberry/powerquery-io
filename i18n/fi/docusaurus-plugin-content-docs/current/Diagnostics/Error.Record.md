# Error.Record


## Description

Palauttaa virhetietueen annetuista tekstiarvoista syylle, sanomalle ja tiedoille.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
