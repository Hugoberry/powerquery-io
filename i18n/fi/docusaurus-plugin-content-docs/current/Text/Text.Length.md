# Text.Length


## Description

Palauttaa merkkien määrän.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Palauttaa tekstin <code>text</code> merkkien määrän.


## Examples

### Example #1 
Selvitä, montako merkkiä on tekstissä &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
