# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Tekee annetusta kohteesta <code>text</code> käänteisen.


## Examples

### Example #1 
Tee tekstistä 123 käänteinen.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
