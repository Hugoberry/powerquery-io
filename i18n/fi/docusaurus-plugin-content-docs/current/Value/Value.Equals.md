# Value.Equals


## Description

Palauttaa tiedon siitä, ovatko kaksi arvoa yhtä suuret.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Palauttaa arvon true, jos arvo <code>value1</code> on yhtä suuri kuin arvo <code>value2</code>, ja muutoin arvon false.



## Category
Values
