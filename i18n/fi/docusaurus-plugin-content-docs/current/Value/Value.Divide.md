# Value.Divide


## Description

Palauttaa ensimmäisen arvon toisella arvolla jakamisen tuloksen.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Palauttaa luvun <code>value1</code> luvulla <code>value2</code> jakamisen tuloksen. Valinnainen parametri <code>precision</code> voidaan määrittää. Oletusarvon mukaan käytetään kohdetta <code>Precision.Double</code>.
