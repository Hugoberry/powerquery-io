# Value.Subtract


## Description

Palauttaa kahden arvon eron.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Palauttaa kohteiden <code>value1</code> ja <code>value2</code> eron. Valinnainen parametri <code>precision</code> voidaan määrittää. Oletusarvon mukaan käytetään kohdetta <code>Precision.Double</code>.
