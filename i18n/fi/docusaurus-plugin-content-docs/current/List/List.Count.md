# List.Count


## Description

Palauttaa luettelon kohteiden määrän.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Palauttaa luettelon <code>list</code> kohteiden määrän.


## Examples

### Example #1 
Selvitä luettelon \{1, 2, 3} arvojen määrä
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
