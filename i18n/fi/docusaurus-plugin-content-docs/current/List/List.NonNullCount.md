# List.NonNullCount


## Description

Palauttaa luettelon muiden kuin tyhjäarvoisten kohteiden määrän.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Palauttaa luettelon <code>list</code> muiden kuin tyhjäarvoisten kohteiden määrän.
