# DateTime.LocalNow


## Description

Palauttaa nykyisen päivämäärän ja ajan paikallisella aikavyöhykkeellä.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Palauttaa <code>datetime</code>-arvon, joka on asetettu järjestelmän nykyiseksi päivämääräksi ja kellonajaksi.



## Category
DateTime
