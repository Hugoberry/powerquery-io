# DynamicsNav.Contents


## Description

Anna Dynamics NAV:n OData-palvelun päätepisteesi URL-osoite.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Palauttaa taulukon, joka sisältää olennaiset Dynamics NAV -tiedot.
