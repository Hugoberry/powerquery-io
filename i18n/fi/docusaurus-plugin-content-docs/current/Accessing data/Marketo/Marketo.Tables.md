# Marketo.Tables


## Description

Anna tiliisi liittyvän Marketo REST API -päätepisteen URL-osoite.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Palauttaa taulukon, jossa on asiaankuuluvia Marketo-tietoja.
