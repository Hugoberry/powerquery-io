# Web.Page


## Description

Palauttaa HTML-tiedoston sisällön jaettuna sen muodostaviin rakenteisiin ja koko tiedoston sekä sen tekstin esityksen, kun tunnisteet on poistettu.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
