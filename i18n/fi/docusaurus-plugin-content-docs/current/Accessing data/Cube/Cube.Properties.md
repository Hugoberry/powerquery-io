# Cube.Properties


## Description

Palauttaa taulukon, joka sisältää kuutiossa laajennettujen dimensioiden käytettävissä olevien ominaisuuksien joukon.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```
