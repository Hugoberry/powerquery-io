# Cube.Transform


## Description

Ottaa käyttöön kuutiofunktioiden luettelon.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Ottaa käyttöön kuutiofunktioiden luettelon <code>transforms</code> kohteessa <code>cube</code>.



## Category
Cube
