# Cube.AttributeMemberProperty


## Description

Palauttaa dimensiomääritteen ominaisuuden.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Palauttaa dimensioattribuutin <code>attribute</code> ominaisuuden <code>propertyName</code>.

Cube
