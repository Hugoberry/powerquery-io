# Cube.MeasureProperty


## Description

Palauttaa mittayksikön ominaisuuden (solun ominaisuuden).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Palauttaa mittayksikön <code>measure</code> ominaisuuden <code>propertyName</code>.



## Category
Cube
