# Lakehouse.Contents


## Description

Tietojen tuominen Lakehousesta


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Details

Muodosta yhteys kaikkiin Lakehouse-kohteisiin, joihin sinulla on käyttöoikeus, ja valitse entiteetit, joita haluat käyttää
