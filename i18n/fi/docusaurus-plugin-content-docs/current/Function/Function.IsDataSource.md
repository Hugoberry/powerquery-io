# Function.IsDataSource


## Description

Palauttaa tiedon siitä, pidetäänkö jotakin tiettyä funktiota tietolähteenä.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Palauttaa tiedon siitä, pidetäänkö funktiota <code>function</code> tietolähteenä.

Type
