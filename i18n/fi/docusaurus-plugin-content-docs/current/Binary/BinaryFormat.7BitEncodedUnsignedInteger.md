# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Binaarimuoto, joka lukee 64-bittisen etumerkittömän kokonaisluvun, joka koodattiin käyttäen 7-bittistä vaihtelevan pituista koodausta.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
