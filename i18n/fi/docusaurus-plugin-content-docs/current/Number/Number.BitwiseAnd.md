# Number.BitwiseAnd


## Description

Palauttaa bittitason And-toiminnon kahden syötteen välillä suorittamisen tuloksen.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

Palauttaa bittitason And-toiminnon kohteiden <code>number1</code> ja <code>number2</code> välillä suorittamisen tuloksen.



## Category
Number.Bytes
