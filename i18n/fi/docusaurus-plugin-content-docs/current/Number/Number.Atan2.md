# Number.Atan2


## Description

Palauttaa kahden luvun jakamisen arkustangentin.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Palauttaa kahden luvun (<code>y</code> ja <code>x</code>) jakamisen arkustangentin. Jakaminen muodostetaan seuraavasti: <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
