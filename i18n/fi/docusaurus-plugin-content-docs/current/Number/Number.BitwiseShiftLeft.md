# Number.BitwiseShiftLeft


## Description

Siirtää bittijoukkoa vasemmalle.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Palauttaa bittitason määritetyn bittien määrän <code>number2</code> verran vasemmalle kohteessa <code>number1</code> siirtämisen tuloksen.



## Category
Number.Bytes
