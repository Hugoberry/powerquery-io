# Number.BitwiseShiftRight


## Description

Siirtää bittijoukkoa oikealle.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Palauttaa bittitason määritetyn bittien määrän <code>number2</code> verran kohteessa <code>number1</code> oikealle siirtämisen tuloksen.



## Category
Number.Bytes
