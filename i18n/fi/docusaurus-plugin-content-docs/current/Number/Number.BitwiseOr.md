# Number.BitwiseOr


## Description

Palauttaa bittitason Or-toiminnon kahden syötteen välillä suorittamisen tuloksen.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Palauttaa bittitason Or-toiminnon kohteiden <code>number1</code> ja <code>number2</code> välillä suorittamisen tuloksen.



## Category
Number.Bytes
