# Number.Random


## Description

Palauttaa satunnaisluvun.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Palauttaa satunnaisluvun välillä 0 ja 1.


## Examples

### Example #1 
Hae satunnaisluku.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
