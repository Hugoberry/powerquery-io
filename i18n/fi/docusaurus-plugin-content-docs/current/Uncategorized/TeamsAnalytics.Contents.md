# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Details

Teamsin Analytics-liittimen avulla voit saada merkityksellisiä tietoja Teamsin käytöstäsi. Tiedot ovat käytettävissä sekä henkilökohtaiseen toimintaasi Teamsissa että toimintaan Teamsissa, jonka jäsen olet.<br />Tiedot ovat käytettävissä edelliseltä 90 päivältä. <br /> Tietoja käytettävissä olevista tietotyypeistä tietoja on seuraavassa ohjeessa: <br /><ul><li><b>Teamsin käyttö:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>Teamsin käyttäjätoiminta:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
