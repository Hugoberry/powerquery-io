# DateTimeZone.FixedUtcNow


## Description

Palauttaa nykyisen päivämäärän ja ajan UTC-ajassa (GMT-aikavyöhykkeellä). Tämä arvo on kiinteä, eikä se muutu peräkkäisten kutsujen myötä.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
