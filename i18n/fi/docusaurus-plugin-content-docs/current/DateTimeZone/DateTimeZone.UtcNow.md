# DateTimeZone.UtcNow


## Description

Palauttaa nykyisen päivämäärän ja ajan UTC-ajassa (GMT-aikavyöhykkeellä).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Palauttaa nykyisen päivämäärän ja ajan UTC-ajassa (GMT-aikavyöhykkeellä).


## Examples

### Example #1 
Hae nykyinen päivämäärä ja aika UTC-muodossa.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
