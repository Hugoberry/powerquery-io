# List.NonNullCount


## Description

Devolve o número de itens não nulos existentes na lista.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Devolve o número de itens não nulos existentes na lista <code>list</code>.

List.Information
