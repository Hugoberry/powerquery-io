# Number.Random


## Description

Devolve um número aleatório.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Devolve um número aleatório entre 0 e 1.


## Examples

### Example #1 
Obter um número aleatório.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
