# Number.BitwiseNot


## Description

Devolve um byte em que cada bit é o oposto da entrada.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Devolve o resultado da execução de uma operação "Not" bit a bit em <code>number</code>.



## Category
Number.Bytes
