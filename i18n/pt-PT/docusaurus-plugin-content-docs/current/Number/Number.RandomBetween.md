# Number.RandomBetween


## Description

Devolve um número aleatório entre dois números.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Devolve um número aleatório entre <code>bottom</code> e <code>top</code>.


## Examples

### Example #1 
Obter um número aleatório entre 1 e 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
