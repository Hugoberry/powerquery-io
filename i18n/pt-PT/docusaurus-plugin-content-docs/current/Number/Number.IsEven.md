# Number.IsEven


## Description

Indica se o valor é par.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

Indica se o valor, <code>number</code>, é par devolvendo <code>true</code> se for par; caso contrário, <code>false</code>.


## Examples

### Example #1 
Verificar se 625 é um número par.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Verificar se 82 é um número par.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
