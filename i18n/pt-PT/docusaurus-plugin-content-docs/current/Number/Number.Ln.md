# Number.Ln


## Description

Devolve o logaritmo natural do número.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Devolve o logaritmo natural de um número, <code>number</code>. Se <code>number</code> for nulo, <code>Number.Ln</code> devolverá um valor nulo.


## Examples

### Example #1 
Obter o logaritmo natural de 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
