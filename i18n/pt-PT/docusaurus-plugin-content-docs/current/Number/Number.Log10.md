# Number.Log10


## Description

Devolve o logaritmo base 10 do número.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Devolve o logaritmo base 10 de um número, <code>number</code>. Se <code>number</code> for nulo, <code>Number.Log10</code> devolverá um nulo.


## Examples

### Example #1 
Obter o logaritmo base 10 de 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
