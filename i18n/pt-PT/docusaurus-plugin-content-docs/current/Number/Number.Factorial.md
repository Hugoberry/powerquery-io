# Number.Factorial


## Description

Devolve o factorial do número.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Devolve o factorial do número <code>number</code>.


## Examples

### Example #1 
Determinar o factorial de 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
