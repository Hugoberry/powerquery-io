# Number.Cos


## Description

Devolve o cosseno do número.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Devolve o cosseno de <code>number</code>.


## Examples

### Example #1 
Determinar o cosseno do ângulo 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
