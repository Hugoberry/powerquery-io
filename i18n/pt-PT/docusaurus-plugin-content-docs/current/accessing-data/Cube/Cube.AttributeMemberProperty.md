# Cube.AttributeMemberProperty


Devolve uma propriedade de um atributo de dimensão.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

Devolve a propriedade <code>propertyName</code> do atributo de dimensão <code>attribute</code>.

Cube
