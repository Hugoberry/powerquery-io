# Smartsheet.Tables


Devolve uma tabela de folhas, relatórios, pastas e áreas de trabalho da API do Smartsheet


## Syntax

```powerquery
Smartsheet.Tables(

) as table
```


## Remarks

Devolve uma tabela aninhada de folhas, relatórios, pastas e áreas de trabalho disponíveis da API do Smartsheet.


## Examples

### Example #1 
Devolve uma tabela de folhas, relatórios, pastas e áreas de trabalho disponíveis no nível mais elevado da hierarquia do Smartsheet
```powerquery
Smartsheet.Tables()
```

Result: 
```powerquery
Uma tabela de folhas, relatórios, pastas e áreas de trabalho do nível mais elevado da hierarquia do Smartsheet
```
