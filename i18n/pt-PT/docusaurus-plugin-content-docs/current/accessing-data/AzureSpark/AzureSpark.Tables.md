# AzureSpark.Tables


Listar as tabelas numa instância do Azure Spark.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Remarks

Listar as tabelas numa instância do Azure Spark.


## Examples

### Example #1 
Listar as tabelas numa instância do Azure Spark.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
