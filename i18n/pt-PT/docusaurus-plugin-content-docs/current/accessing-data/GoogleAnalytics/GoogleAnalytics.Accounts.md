# GoogleAnalytics.Accounts


Devolve as contas do Google Analytics.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Remarks

Devolve as contas do Google Analytics que são acessíveis a partir da credencial atual.
