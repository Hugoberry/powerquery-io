# Cube.ReplaceDimensions


Substitui o conjunto de dimensões devolvidas por Cube.Dimensions.


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Remarks

Substitui o conjunto de dimensões devolvidas por <code>Cube.Dimensions</code>. Por exemplo, esta função pode ser utilizada para adicionar uma coluna de ID a um atributo de dimensão, para que a origem de dados possa agrupar com base no ID em vez de no valor apresentado.
