# Uri.Combine


## Description

Devolve um URI absoluto que é a combinação do URI base e do URI relativo de entrada.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Devolve um URI absoluto que é a combinação do <code>baseUri</code> e <code>relativeUri</code> de entrada.

Uri
