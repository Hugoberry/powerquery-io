# Number.Atan2


Devolve o arco de tangente da divisão dos dois números.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Remarks

Devolve o arco de tangente da divisão dos dois números, <code>y</code> e <code>x</code>. A divisão será construída como <code>y</code>/<code>x</code>.
