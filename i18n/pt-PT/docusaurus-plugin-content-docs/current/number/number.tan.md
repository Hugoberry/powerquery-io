# Number.Tan


Devolve a tangente do número.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Remarks

Devolve a tangente de <code>number</code>.


## Examples

### Example #1 
Determinar a tangente do ângulo 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```
