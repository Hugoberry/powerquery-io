# Number.BitwiseOr


Devolve o resultado da execução de uma operação &#34;Or&#34; bit a bit entre as duas entradas.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Devolve o resultado da execução de uma operação "Or" bit a bit entre <code>number1</code> e <code>number2</code>.

Number.Bytes
