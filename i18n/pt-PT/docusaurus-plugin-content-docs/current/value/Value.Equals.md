# Value.Equals


Devolve se dois valores são iguais.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Remarks

Devolve true se o valor <code>value1</code> for igual ao valor <code>value2</code>; caso contrário, devolve false.

Values
