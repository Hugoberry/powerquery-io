# BinaryFormat.7BitEncodedSignedInteger


Um formato binário que lê um número inteiro com sinal de 64 bits codificado através de uma codificação de comprimento variável de 7 bits.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Remarks

Um formato binário que lê um número inteiro com sinal de 64 bits codificado através de uma codificação de comprimento variável de 7 bits.
