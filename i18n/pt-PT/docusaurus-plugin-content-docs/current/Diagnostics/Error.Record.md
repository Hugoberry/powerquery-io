# Error.Record


## Description

Devolve um registo de erro dos valores de texto fornecidos por razão, mensagem e detalhe.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details

Devolve um registo de erro dos valores de texto fornecidos por razão, mensagem e detalhe.
