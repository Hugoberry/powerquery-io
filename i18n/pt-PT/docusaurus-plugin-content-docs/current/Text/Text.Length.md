# Text.Length


Devolve o número de carateres.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Remarks

Devolve o número de carateres existente no texto <code>text</code>.


## Examples

### Example #1 
Determinar quantos carateres existem no texto &#34;Olá Mundo&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
