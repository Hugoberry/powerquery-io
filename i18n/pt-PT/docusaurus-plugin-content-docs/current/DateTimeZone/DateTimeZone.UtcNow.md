# DateTimeZone.UtcNow


Devolve a data e hora atuais em UTC (o fuso horário GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Remarks

Devolve a data e hora atuais em UTC (o fuso horário GMT).


## Examples

### Example #1 
Obter a data e hora atuais em UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
