# DateTimeZone.FixedUtcNow


Devolve a data e hora atuais em UTC (o fuso horário GMT). Este valor é fixo e não mudará com as chamadas sucessivas.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```


## Remarks
