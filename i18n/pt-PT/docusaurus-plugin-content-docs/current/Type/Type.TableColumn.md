# Type.TableColumn


## Description

Devolve o tipo de uma coluna numa tabela.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Devolve o tipo da coluna <code>column</code> no tipo de tabela <code>tableType</code>.



## Category
Type
