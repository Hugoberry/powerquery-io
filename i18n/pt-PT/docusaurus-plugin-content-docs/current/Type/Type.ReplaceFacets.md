# Type.ReplaceFacets


## Description

Substitui as facetas de um tipo.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Substitui as facetas de <code>type</code> pelas facetas contidas no registo <code>facets</code>.

Type
