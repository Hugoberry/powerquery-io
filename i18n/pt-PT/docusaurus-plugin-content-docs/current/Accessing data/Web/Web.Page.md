# Web.Page


Devolve os conteúdos do documento HTML divididos nas estruturas que o compõem, bem como uma representação do documento completo e do respetivo texto após a remoção dos tags.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Remarks
