# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Remarks

O conector do Teams Analytics permite-lhe obter informações sobre a sua utilização do Teams. Os dados estão disponíveis tanto para a sua atividade pessoal no Teams como para a atividade em Equipas das quais é membro.<br />Estão disponíveis dados para os 90 dias anteriores. <br /> Para mais detalhes sobre os tipos de dados disponíveis, consulte a seguinte documentação de ajuda: <br /><ul><li><b>Utilização do Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>Atividade de Utilizador do Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
