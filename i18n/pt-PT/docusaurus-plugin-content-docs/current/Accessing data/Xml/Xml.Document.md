# Xml.Document


## Description

Devolve o conteúdo do documento XML como uma tabela hierárquica.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Details

Devolve o conteúdo do documento XML como uma tabela hierárquica.
