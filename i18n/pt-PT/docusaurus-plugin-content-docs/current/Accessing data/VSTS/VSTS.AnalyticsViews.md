# VSTS.AnalyticsViews


## Description

Introduzir os nomes da organização e do projeto.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Devolve uma tabela de vistas de Análise oferecida pelo Azure DevOps.
