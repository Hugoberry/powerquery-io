# Cube.Measures


Devolve uma tabela que contém o conjunto de medidas disponíveis.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Remarks

Devolve uma tabela que contém o conjunto de medidas disponíveis no <code>cube</code>.    Cada medida é representada como uma função. É possível aplicar medidas ao cubo utilizando Cube.AddMeasureColumn.
