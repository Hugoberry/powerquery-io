# Cube.Transform


Aplica uma lista de funções de cubo.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Remarks

Aplica a lista de funções de cubo, <code>transforms</code>, ao <code>cube</code>.
