# Cube.MeasureProperties


## Description

Devolve uma tabela que contém o conjunto de propriedades disponíveis para as medidas que são expandidas no cubo.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Devolve uma tabela que contém o conjunto de propriedades disponíveis para as dimensões que são expandidas no cubo.



## Category
Cube
