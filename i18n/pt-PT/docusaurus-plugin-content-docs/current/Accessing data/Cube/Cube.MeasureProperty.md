# Cube.MeasureProperty


Devolve uma propriedade de uma medida (propriedade de célula).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Remarks

Devolve a propriedade <code>propertyName</code> do atributo de dimensão <code>measure</code>.



## Category
Cube
