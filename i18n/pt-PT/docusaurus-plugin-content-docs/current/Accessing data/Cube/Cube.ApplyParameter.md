# Cube.ApplyParameter


## Description

Devolve um cubo após aplicar-lhe um parâmetro.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Devolve um cubo após aplicar <code>parameter</code> com <code>arguments</code> a <code>cube</code>.



## Category
Cube
