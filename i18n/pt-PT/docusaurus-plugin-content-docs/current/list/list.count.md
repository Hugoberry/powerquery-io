# List.Count


Devolve o número de itens existentes na lista.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Remarks

Devolve o número de itens existentes na lista <code>list</code>.


## Examples

### Example #1 
Determinar o número de valores existentes na lista \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
