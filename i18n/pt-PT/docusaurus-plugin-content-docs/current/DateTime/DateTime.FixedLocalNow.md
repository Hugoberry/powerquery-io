# DateTime.FixedLocalNow


Devolve a data e hora atuais no fuso horário local. Este valor é fixo e não mudará com as chamadas sucessivas.


## Syntax

```powerquery
DateTime.FixedLocalNow(

) as datetime
```


## Remarks

Devolve um valor <code>datetime</code> definido para a data e hora atuais no sistema. Este valor é fixo e não mudará com as chamadas sucessivas, ao contrário de DateTime.LocalNow, que poderá devolver valores diferentes no decorrer da execução de uma expressão.
