# Lines.ToBinary


## Description

Converte uma lista de texto num valor binário utilizando a codificação e o lineSeparator especificados. O lineSeparator especificado é acrescentado a cada linha.  Se não for especificado, serão utilizados os carateres de símbolo de retorno e avanço de linha.


## Syntax

```powerquery
Lines.ToBinary(
    lines as list,
    optional lineSeparator as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```
