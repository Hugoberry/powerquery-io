# Type.TableSchema


Devolve uma tabela que contém uma descrição das colunas (isto é, o esquema) do tipo de tabela especificado.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Remarks

Devolve uma tabela que descreve as colunas de <code>tableType</code>.<br />Consulte a documentação <code>Table.Schema</code> para obter uma descrição da tabela resultante.<br />
