# Error.Record


Devuelve un registro de error de los valores de texto proporcionados para el motivo, el mensaje y el detalle.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Remarks

Devuelve un registro de error de los valores de texto proporcionados para el motivo, el mensaje y el detalle.
