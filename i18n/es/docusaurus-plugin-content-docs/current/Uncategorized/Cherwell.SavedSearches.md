# Cherwell.SavedSearches



## Syntax

```powerquery
Cherwell.SavedSearches(
    API URL as text,
    Client ID as text,
    optional Locale as text,
    optional Saved Search URL as text
) as table
```


## Details

Returns the results of a Saved Search from a Cherwell Service Management REST API (requires CSM version 10.2 or later).
