# Value.Equals


## Description

Devuelve si dos valores son iguales.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Devuelve true si el valor <code>value1</code> es igual que el valor <code>value2</code>; false, en caso contrario.



## Category
Values
