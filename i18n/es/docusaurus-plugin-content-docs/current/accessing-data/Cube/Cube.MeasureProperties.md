# Cube.MeasureProperties


Devuelve una tabla que contiene el conjunto de propiedades disponibles para las medidas que se expanden en el cubo.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Remarks

Devuelve una tabla que contiene el conjunto de propiedades disponibles para las medidas que se expanden en el cubo.
