# Exchange.Contents


Devuelve una tabla de contenido de una cuenta de Microsoft Exchange.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Remarks

Devuelve una tabla de contenido de la cuenta de Microsoft Exchange <code>mailboxAddress</code>. Si no se especifica <code>mailboxAddress</code>, se usar√° la cuenta de credenciales predeterminada.
