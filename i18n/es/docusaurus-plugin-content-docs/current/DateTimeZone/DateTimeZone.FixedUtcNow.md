# DateTimeZone.FixedUtcNow


## Description

Devuelve la fecha y la hora actuales en UTC (la zona horaria GMT). Este valor es fijo y no cambiará con las sucesivas llamadas.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
