# Number.Sin


Devuelve el seno del número.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

Devuelve el seno de <code>number</code>.


## Examples

### Example #1 
Buscar el seno del ángulo 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
