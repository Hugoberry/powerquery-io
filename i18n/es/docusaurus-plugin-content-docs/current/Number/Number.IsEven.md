# Number.IsEven


## Description

Indica si el valor es par.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

Indica si el valor <code>number</code>, está volviendo <code>true</code> incluso si es uniforme, <code>false</code> si no.


## Examples

### Example #1 
Comprobar si 625 es un número par.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Comprobar si 82 es un número par.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
