# Number.Random


## Description

Devuelve un número aleatorio.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Devuelve un número aleatorio entre 0 y 1.


## Examples

### Example #1 
Obtener un número aleatorio.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```

Number.Random
