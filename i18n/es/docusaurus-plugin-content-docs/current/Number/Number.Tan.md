# Number.Tan


## Description

Devuelve la tangente del número.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Devuelve la tangente de <code>number</code>.


## Examples

### Example #1 
Hallar la tangente del ángulo 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
