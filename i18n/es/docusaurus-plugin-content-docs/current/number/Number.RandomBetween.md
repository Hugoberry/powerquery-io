# Number.RandomBetween


Devuelve un número aleatorio entre dos números.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Remarks

Devuelve un número aleatorio entre <code>bottom</code> y <code>top</code>.


## Examples

### Example #1 
Obtener un número aleatorio entre 1 y 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
