# Number.Cos


Devuelve el coseno del número.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Devuelve el coseno de <code>number</code>.


## Examples

### Example #1 
Buscar el coseno del ángulo 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
