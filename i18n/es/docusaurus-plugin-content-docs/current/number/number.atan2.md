# Number.Atan2


Devuelve el arco tangente de la división de los dos números.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Remarks

Devuelve el arco tangente de la división de los dos números, <code>y</code> y <code>x</code>. La división se construirá como <code>y</code>/<code>x</code>.
