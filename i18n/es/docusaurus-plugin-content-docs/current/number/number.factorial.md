# Number.Factorial


Devuelve el factorial del número.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

Devuelve el factorial del número <code>number</code>.


## Examples

### Example #1 
Buscar el factorial de 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
