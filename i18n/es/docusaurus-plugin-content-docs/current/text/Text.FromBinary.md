# Text.FromBinary


Descodifica datos de un formato binario en texto.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

Descodifica los datos <code>binary</code>, de un valor binario en un valor de texto, usando el tipo <code>encoding</code>.
