# List.NonNullCount


Devuelve el número de elementos que no son NULL de la lista.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Devuelve el número de elementos que no son NULL de la lista <code>list</code>.
