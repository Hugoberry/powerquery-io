# List.Count


## Description

Devuelve el número de elementos de la lista.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Devuelve el número de elementos de la lista <code>list</code>.


## Examples

### Example #1 
Buscar el número de valores de la lista \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
