# AzureSpark.Tables


## Description

Lista de tablas en una instancia de Azure Spark.


## Syntax

```powerquery

    server as text,
    optional options as record
) as table
```


## Details

Lista de tablas en una instancia de Azure Spark.


## Examples

Lista de tablas en una instancia de Azure Spark.
