# Identity.IsMemberOf


Determina si una identidad es miembro de una colección de identidades.


## Syntax

```powerquery
Identity.IsMemberOf(
    identity as record,
    collection as record
) as logical
```


## Remarks

Determina si una identidad es miembro de una colección de identidades.
