# Web.Page


## Description

Devuelve el contenido del documento HTML dividido en sus estructuras constitutivas, así como una representación de todo el documento y su texto después de quitar las etiquetas.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```


## Details
