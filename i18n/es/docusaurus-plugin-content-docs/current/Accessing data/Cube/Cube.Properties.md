# Cube.Properties


Devuelve una tabla que contiene el conjunto de propiedades disponibles para las dimensiones que se expanden en el cubo.


## Syntax

```powerquery
Cube.Properties(
    cube as table
) as table
```


## Remarks

Devuelve una tabla que contiene el conjunto de propiedades disponibles para las dimensiones que se expanden en el cubo.
