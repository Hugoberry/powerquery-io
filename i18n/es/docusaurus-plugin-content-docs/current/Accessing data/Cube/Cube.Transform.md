# Cube.Transform


## Description

Aplica una lista de funciones de cubo.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Aplica la lista de funciones de cubo, <code>transforms</code>, en el <code>cube</code>.

Cube
