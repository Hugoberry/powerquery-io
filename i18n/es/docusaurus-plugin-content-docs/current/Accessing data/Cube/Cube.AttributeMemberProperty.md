# Cube.AttributeMemberProperty


Devuelve una propiedad de un atributo de dimensión.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

Devuelve la propiedad <code>propertyName</code> del atributo de dimensión <code>attribute</code>.

Cube
