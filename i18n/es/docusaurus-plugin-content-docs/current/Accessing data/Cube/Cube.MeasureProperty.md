# Cube.MeasureProperty


## Description

Devuelve la propiedad de una medida (propiedad de celda).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Devuelve la propiedad <code>propertyName</code> de la medida <code>measure</code>.



## Category
Cube
