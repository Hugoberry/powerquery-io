# Text.Length


## Description

Devuelve el número de caracteres.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Devuelve el número de caracteres del texto <code>text</code>.


## Examples

### Example #1 
Buscar cuántos caracteres hay en el texto &#34;Hola mundo&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
