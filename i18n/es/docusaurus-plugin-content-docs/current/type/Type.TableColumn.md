# Type.TableColumn


Devuelve el tipo de una columna de una tabla.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Remarks

Devuelve el tipo de la columna <code>column</code> de la tabla <code>tableType</code>.
