# Number.BitwiseOr


## Description

Devolve o resultado de realizar unha operación &#34;Or&#34; bit a bit entre as dúas entradas.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Devolve o resultado de realizar un "Or" bit a bit entre <code>number1</code> e <code>number2</code>.



## Category
Number.Bytes
