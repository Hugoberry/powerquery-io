# Number.Tan


## Description

Devolve a tanxente do número.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Devolve a tanxente de <code>number</code>.


## Examples

### Example #1 
Buscar a tanxente do ángulo 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
