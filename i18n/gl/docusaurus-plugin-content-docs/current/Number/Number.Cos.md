# Number.Cos


## Description

Devolve o coseno do número.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Devolve o coseno de <code>number</code>.


## Examples

### Example #1 
Buscar o coseno do ángulo 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
