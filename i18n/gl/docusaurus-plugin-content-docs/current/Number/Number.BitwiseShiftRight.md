# Number.BitwiseShiftRight


## Description

Despraza os bits definidos á dereita.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Devolve o resultado de realizar un desprazamento bit a bit á dereita en <code>number1</code>, o número especificado de bits <code>number2</code>.



## Category
Number.Bytes
