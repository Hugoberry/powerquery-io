# Number.BitwiseNot


## Description

Devolve un byte no que cada bit é o oposto da entrada.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Devolve o resultado de realizar unha operación "Not" bit a bit en <code>number</code>.



## Category
Number.Bytes
