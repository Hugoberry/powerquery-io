# Number.Sin


## Description

Devolve o seno do número.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Devolve o seno de <code>number</code>.


## Examples

### Example #1 
Buscar o seno do ángulo 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
