# Number.IsOdd


## Description

Indica se o valor é impar.


## Syntax

```powerquery
Number.IsOdd(
    number as number
) as logical
```


## Details

Indica se o valor é impar. Devolve <code>true</code> se <code>number</code> é un número impar, se non, <code>false</code>.


## Examples

### Example #1 
Comprobar se 625 é un número impar.
```powerquery
Number.IsOdd(625)
```

Result: 
```powerquery
true
```


### Example #2 
Comprobar se 82 é un número impar.
```powerquery
Number.IsOdd(82)
```

Result: 
```powerquery
false
```
