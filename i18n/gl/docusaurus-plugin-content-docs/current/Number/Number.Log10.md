# Number.Log10


## Description

Devolve o logaritmo en base 10 do número.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Devolve o logaritmo en base 10 dun número, <code>number</code>. Se <code>number</code> é nulo, <code>Number.Log10</code> devolve nulo.


## Examples

### Example #1 
Obtén o logaritmo en base 10 de 2.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
