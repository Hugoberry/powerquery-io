# Number.RandomBetween


## Description

Devolve un número aleatorio entre dous números.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Devolve un número aleatorio entre <code>bottom</code> e <code>top</code>.


## Examples

### Example #1 
Obter un número aleatorio entre 1 e 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
