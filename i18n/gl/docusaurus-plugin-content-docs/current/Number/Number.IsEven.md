# Number.IsEven


## Description

Indica se o valor é par.


## Syntax

```powerquery
Number.IsEven(
    number as number
) as logical
```


## Details

Indica se o valor, <code>number</code>, é par devolvendo <code>true</code> se é par, se non, <code>false</code>.


## Examples

### Example #1 
Comprobar se 625 é un número par.
```powerquery
Number.IsEven(625)
```

Result: 
```powerquery
false
```


### Example #2 
Comprobar se 82 é un número par.
```powerquery
Number.IsEven(82)
```

Result: 
```powerquery
true
```
