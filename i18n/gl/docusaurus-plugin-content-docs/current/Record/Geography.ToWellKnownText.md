# Geography.ToWellKnownText


## Description

Traduce un valor de punto xeográfico estruturado na súa representación de Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Traduce un valor de punto xeográfico estruturado na súa representación de Well-Known Text (WKT) segundo o define Open Geospatial Consortium (OGC), así como o formato de serialización que usan a maioría das bases de datos, incluído SQL Server.
