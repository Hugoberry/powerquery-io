# Geometry.ToWellKnownText


## Description

Traduce un valor de punto de xeométrico estruturado na súa representación de Well-Known Text (WKT).


## Syntax

```powerquery
Geometry.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Traduce un valor de punto de xeométrico estruturado na súa representación de Well-Known Text (WKT) segundo o define Open Geospatial Consortium (OGC), así como o formato de serialización que usan a maioría das bases de datos, incluído SQL Server.
