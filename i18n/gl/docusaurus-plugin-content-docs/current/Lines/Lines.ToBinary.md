# Lines.ToBinary


## Description

Converte unha lista de texto nun valor binario utilizando a codificación e o lineSeparator especificados. O lineSeparator especificado anéxase a cada liña.  Se non se especifica, utilízanse os caracteres de retorno de carro e avance de liña.


## Syntax

```powerquery
Lines.ToBinary(
    lines as list,
    optional lineSeparator as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```
