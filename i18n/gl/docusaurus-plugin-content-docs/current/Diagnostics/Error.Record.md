# Error.Record


## Description

Devolve un rexistro de erros dos valores de texto fornecidos por motivo, mensaxe e detalles.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
