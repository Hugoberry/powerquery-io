# Uri.Combine


## Description

Devolve un URI absoluto que é a combinación do URI base de entrada e o URI relativo.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Devolve un URI absoluto que é a combinación do <code>baseUri</code> de entrada e o <code>relativeUri</code>.
