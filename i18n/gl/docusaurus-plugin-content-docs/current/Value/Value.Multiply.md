# Value.Multiply


## Description

Devolve o produto de dous valores.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Devolve o produto de multiplicar <code>value1</code> por <code>value2</code>. Pódese especificar un parámetro opcional <code>precision</code>, por defecto, utilízase <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
