# Value.Add


## Description

Devolve a suma dos dous valores.


## Syntax

```powerquery
Value.Add(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Devolve a suma de <code>value1</code> e <code>value2</code>. Pódese especificar un parámetro opcional <code>precision</code>, por defecto, utilízase <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
