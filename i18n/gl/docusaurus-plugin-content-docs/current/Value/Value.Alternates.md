# Value.Alternates


## Description

Expresa plans de consulta alternativos.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Expresa plans de consulta alternativos cunha expresión do plan de consulta obtida a través de Value.Expression(Value.Optimize(...)). Non está concibido para outros usos.



## Category
Expression
