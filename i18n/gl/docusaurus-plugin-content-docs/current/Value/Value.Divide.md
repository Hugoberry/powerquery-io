# Value.Divide


## Description

Devolve o resultado de dividir o primeiro valor polo segundo.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Devolve o resultado de dividir <code>value1</code> por <code>value2</code>. Pódese especificar un parámetro opcional <code>precision</code>, por defecto, utilízase <code>Precision.Double</code>.
