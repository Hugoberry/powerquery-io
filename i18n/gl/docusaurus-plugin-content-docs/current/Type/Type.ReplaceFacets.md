# Type.ReplaceFacets


## Description

Substitúe as facetas dun tipo.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Substitúe as facetas de <code>type</code> coas facetas que contén o rexistro <code>facets</code>.

Type
