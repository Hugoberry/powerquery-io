# Type.TableColumn


## Description

Devolve o tipo de unha columna nunha táboa.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Devolve o tipo da columna <code>column</code> no tipo de táboa <code>tableType</code>.



## Category
Type
