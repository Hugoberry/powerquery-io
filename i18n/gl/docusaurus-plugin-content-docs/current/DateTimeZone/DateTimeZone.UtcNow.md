# DateTimeZone.UtcNow


## Description

Devolve a data e a hora actuais en UTC (o fuso horario GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Devolve a data e a hora actuais en UTC (o fuso horario GMT).


## Examples

### Example #1 
Obter a data e a hora actuais en UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
