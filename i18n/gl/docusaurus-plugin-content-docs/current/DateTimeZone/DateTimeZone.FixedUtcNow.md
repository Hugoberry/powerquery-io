# DateTimeZone.FixedUtcNow


## Description

Devolve a data e a hora actuais en UTC (o fuso horario GMT). Este valor é fixo e non se vai modificar con sucesivas chamadas.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
