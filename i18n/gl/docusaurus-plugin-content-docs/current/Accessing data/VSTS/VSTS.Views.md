# VSTS.Views


## Description

Indique os nomes da organización e do proxecto.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Devolve unha táboa de fontes OData proporcionada por Azure DevOps.
