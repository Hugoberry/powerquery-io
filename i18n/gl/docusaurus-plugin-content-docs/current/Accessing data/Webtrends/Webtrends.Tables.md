# Webtrends.Tables


## Description

Escribe o ID do perfil asociado coa túa conta de Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Devolve unha táboa con datos importantes de Webtrends.
