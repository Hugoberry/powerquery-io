# AzureDevOpsServer.AnalyticsViews


## Description

Indique os nomes da organización e do proxecto.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Devolve unha táboa de visualizacións de Analytics proporcionada por Azure DevOps.
