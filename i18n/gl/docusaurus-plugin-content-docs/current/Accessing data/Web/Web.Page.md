# Web.Page


## Description

Devolve o contido do documento HTML dividido nas súas estruturas constituíntes, así como unha representación do documento completo e seu texto despois de eliminar as etiquetas.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
