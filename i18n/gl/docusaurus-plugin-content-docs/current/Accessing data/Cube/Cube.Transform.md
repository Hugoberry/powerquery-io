# Cube.Transform


## Description

Aplica unha lista de funcións de cubo.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Aplica a lista de funcións de cubo, <code>transforms</code>, en <code>cube</code>.
