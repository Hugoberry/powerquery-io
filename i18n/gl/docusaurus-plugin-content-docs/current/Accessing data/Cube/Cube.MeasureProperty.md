# Cube.MeasureProperty


## Description

Devolve unha propiedade dunha medida (propiedade da cela).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Devolve a propiedade <code>propertyName</code> da medida <code>measure</code>.



## Category
Cube
