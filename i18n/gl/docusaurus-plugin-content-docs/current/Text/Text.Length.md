# Text.Length


## Description

Devolve o número de caracteres.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Devolve o número de caracteres no texto <code>text</code>.


## Examples

### Example #1 
Buscar o número de caracteres que hai no texto &#34;Ola, Mundo&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
