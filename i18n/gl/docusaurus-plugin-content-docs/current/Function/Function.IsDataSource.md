# Function.IsDataSource


## Description

Devolve se unha determinada función considérase ou non unha orixe de datos.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Devolve se <code>function</code> considérase ou non unha orixe de datos.
