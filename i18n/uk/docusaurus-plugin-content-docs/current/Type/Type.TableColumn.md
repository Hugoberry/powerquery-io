# Type.TableColumn


## Description

Повертає тип стовпця в таблиці.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Повертає тип стовпця <code>column</code> в таблиці типу <code>tableType</code>.



## Category
Type
