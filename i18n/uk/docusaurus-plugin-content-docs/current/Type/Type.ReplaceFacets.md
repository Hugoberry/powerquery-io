# Type.ReplaceFacets


## Description

Замінює аспекти типу.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Замінює аспекти типу "<code>type</code>" на аспекти, що містяться в записі "<code>facets</code>".
