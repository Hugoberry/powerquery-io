# Type.TableSchema


## Description

Повертає таблицю, що містить опис стовпців (тобто схему) указаного типу таблиці.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

Повертає таблицю з описом стовпців у таблиці "<code>tableType</code>".<br />Ознайомтеся з документацією про функцію <code>Table.Schema</code>, щоб переглянути відомості про таблицю, що в результаті генерується.<br />
