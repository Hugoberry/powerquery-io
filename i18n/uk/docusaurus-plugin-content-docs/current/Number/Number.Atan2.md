# Number.Atan2


## Description

Повертає арктангенс частки двох чисел.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Повертає арктангенс частки двох чисел, <code>y</code> і <code>x</code>. Частка буде побудована як <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
