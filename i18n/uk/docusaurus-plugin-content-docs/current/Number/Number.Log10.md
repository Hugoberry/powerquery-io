# Number.Log10


## Description

Повертає логарифм числа за основою 10.


## Syntax

```powerquery
Number.Log10(
    number as number
) as number
```


## Details

Повертає логарифм числа <code>number</code> за основою 10. Якщо параметр <code>number</code> має Null-значення, функція <code>Number.Log10</code> повертає Null-значення.


## Examples

### Example #1 
Отримати логарифм числа 2 за основою 10.
```powerquery
Number.Log10(2)
```

Result: 
```powerquery
0.3010299956639812
```
