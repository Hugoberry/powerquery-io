# Number.Cos


## Description

Повертає косинус числа.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Повертає косинус <code>number</code>.


## Examples

### Example #1 
Знайти косинус кута 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
