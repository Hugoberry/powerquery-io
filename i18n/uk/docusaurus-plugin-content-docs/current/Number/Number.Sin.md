# Number.Sin


## Description

Повертає синус числа.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Повертає синус <code>number</code>.


## Examples

### Example #1 
Знайти синус кута 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
