# Number.RandomBetween


## Description

Повертає випадкове число між двома числами.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Повертає випадкове число між <code>bottom</code> та <code>top</code>.


## Examples

### Example #1 
Отримати випадкове число між 1 і 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
