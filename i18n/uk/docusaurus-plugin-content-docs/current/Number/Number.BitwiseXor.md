# Number.BitwiseXor


## Description

Повертає результат виконання побітового &#34;XOR&#34; (виключне АБО) між двома вхідними даними.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Повертає результат виконання побітового "XOR" (виключне АБО) між <code>number1</code> і <code>number2</code>.



## Category
Number.Bytes
