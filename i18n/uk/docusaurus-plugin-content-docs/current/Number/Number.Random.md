# Number.Random


## Description

Повертає випадкове число.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Повертає випадкове число від 0 до 1.


## Examples

### Example #1 
Отримати випадкове число.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
