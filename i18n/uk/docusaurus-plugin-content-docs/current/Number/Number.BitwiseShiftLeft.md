# Number.BitwiseShiftLeft


## Description

Зсовує набір бітів ліворуч.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Повертає результат виконання побітового зсуву <code>number1</code> ліворуч на вказану кількість бітів <code>number2</code>.



## Category
Number.Bytes
