# Number.BitwiseOr


## Description

Повертає результат виконання побітового &#34;АБО&#34; між двома вхідними даними.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Повертає результат виконання побітового "АБО" між <code>number1</code> і <code>number2</code>.



## Category
Number.Bytes
