# Number.BitwiseShiftRight


## Description

Зсовує набір бітів праворуч.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Повертає результат виконання побітового зсуву <code>number1</code> праворуч на вказану кількість бітів <code>number2</code>.



## Category
Number.Bytes
