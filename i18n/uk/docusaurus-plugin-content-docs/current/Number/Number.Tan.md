# Number.Tan


## Description

Повертає тангенс числа.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Повертає тангенс <code>number</code>.


## Examples

### Example #1 
Знайти тангенс кута 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
