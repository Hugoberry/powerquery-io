# Number.Exp


## Description

Підносить e до заданого степеню.


## Syntax

```powerquery
Number.Exp(
    number as number
) as number
```


## Details

Повертає результат піднесення e до <code>number</code> степеня (експоненційна функція).      <ul>        <li><code>number</code>: значення <code>number</code>, для якого обчислюється експоненційна функція. Якщо <code>number</code> має Null-значення, функція <code>Number.Exp</code> повертає Null-значення. </li>      </ul>


## Examples

### Example #1 
Піднести e до 3 степеню.
```powerquery
Number.Exp(3)
```

Result: 
```powerquery
20.085536923187668
```
