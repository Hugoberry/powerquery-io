# Number.Factorial


## Description

Повертає факторіал числа.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Повертає факторіал числа <code>number</code>.


## Examples

### Example #1 
Знайти факторіал числа 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
