# Number.BitwiseNot


## Description

Повертає байт, в якому кожен біт протилежний до вхідних даних.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Повертає результат виконання побітової операції "НЕ" з <code>number</code>.



## Category
Number.Bytes
