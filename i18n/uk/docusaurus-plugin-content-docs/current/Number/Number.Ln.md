# Number.Ln


## Description

Повертає натуральний логарифм числа.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Повертає натуральний логарифм числа <code>number</code>. Якщо <code>number</code> має Null-значення, функція <code>Number.Ln</code> повертає Null-значення.


## Examples

### Example #1 
Отримати натуральний логарифм числа 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
