# Lines.ToText


## Description

Перетворює список тексту на єдиний текст.  Вказаний lineSeparator додається до кожного рядка.  Якщо не вказано, використовуються символи повернення каретки та перенесення рядка.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```


## Details
