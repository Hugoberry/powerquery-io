# List.FirstN


## Description

Повертає перший набір елементів у списку, вказавши кількість елементів, які необхідно повернути, або описову умову.


## Syntax

```powerquery
List.FirstN(
    list as list,
    countOrCondition as any
) as any
```


## Details

 <ul>  <li>Якщо вказано кількість, повертається елементів не більше, ніж указано. </li>  <li>Якщо вказано умову, повертаються всі елементи, які спочатку задовольняють цій умові. Як тільки один елемент не задовольняє умову, наступні елементи не розглядаються. </li> </ul>


## Examples

### Example #1 
Знайти початкові значення у списку \{3, 4, 5, -1, 7, 8, 2}, які більші 0.
```powerquery
List.FirstN({3, 4, 5, -1, 7, 8, 2}, each _ > 0)
```

Result: 
```powerquery
{3, 4, 5}
```
