# List.Random


## Description

Повертає список випадкових чисел.


## Syntax

```powerquery
List.Random(
    count as number,
    optional seed as number
) as list
```


## Details

Повертає список випадкових чисел від 0 до 1. Можна вказати кількість значень, які потрібно повернути, і початкове значення (необов’язково).<ul>   <li><code>count</code>: кількість випадкових значень, які слід створити.</li>   <li><code>seed</code>: <i>[необов’язково]</i> число, яке вказує генератору випадкових чисел початкове значення. Якщо пропущено, кожен виклик функції повертає унікальний список випадкових чисел. Якщо вказати цей параметр, кожен виклик функції створює той самий список випадкових чисел.</li></ul>


## Examples

### Example #1 
Створити список із 3 випадкових чисел.
```powerquery
List.Random(3)
```

Result: 
```powerquery
{0.992332, 0.132334, 0.023592}
```


### Example #2 
Створити список із 3 випадкових чисел з указаним початковим значенням.
```powerquery
List.Random(3, 2)
```

Result: 
```powerquery
{0.883002, 0.245344, 0.723212}
```
