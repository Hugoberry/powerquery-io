# List.Count


## Description

Повертає кількість елементів у списку.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Повертає кількість елементів у списку <code>list</code>.


## Examples

### Example #1 
Знайти кількість значень у списку \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
