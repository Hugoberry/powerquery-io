# List.NonNullCount


## Description

Повертає кількість ненульових елементів у списку.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Повертає кількість ненульових елементів у списку <code>list</code>.



## Category
List.Information
