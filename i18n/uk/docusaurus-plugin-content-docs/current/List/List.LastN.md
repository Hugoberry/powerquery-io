# List.LastN


## Description

Повертає останнє значення у списку.  Можна додатково вказати кількість значень, які необхідно повернути, або описову умову.


## Syntax

```powerquery
List.LastN(
    list as list,
    optional countOrCondition as any
) as any
```


## Details

Повертає останній елемент списку <code>list</code>. Якщо список пустий, створюється виняток. Ця функція приймає додатковий параметр, <code>countOrCondition</code>, для підтримки збирання кількох елементів або фільтрування елементів. <code>countOrCondition</code> можна вказати трьома способами. <ul>  <li>Якщо вказано кількість, повертається елементів не більше, ніж указано. </li>  <li>Якщо вказано умову, повертаються всі елементи, які спочатку задовольняють цій умові, починаючи з кінця списку. Як тільки один елемент не задовольняє умові, наступні елементи не розглядаються. </li>  <li>Якщо цей параметр має Null-значення, повертається останній елемент у списку.</li> </ul>


## Examples

### Example #1 
Знайти останнє значення у списку \{3, 4, 5, -1, 7, 8, 2}.
```powerquery
List.LastN({3, 4, 5, -1, 7, 8, 2}, 1)
```

Result: 
```powerquery
{2}
```


### Example #2 
Знайти останні значення у списку \{3, 4, 5, -1, 7, 8, 2}, які більші 0.
```powerquery
List.LastN({3, 4, 5, -1, 7, 8, 2}, each _ > 0)
```

Result: 
```powerquery
{7, 8, 2}
```
