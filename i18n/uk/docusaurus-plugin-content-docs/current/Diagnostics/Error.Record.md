# Error.Record


## Description

Повертає запис помилки з наданих текстових значень із зазначеними причиною, повідомленням та подробицями.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
