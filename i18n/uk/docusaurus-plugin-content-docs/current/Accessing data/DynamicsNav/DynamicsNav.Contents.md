# DynamicsNav.Contents


## Description

Введіть URL-адресу кінцевої точки OData служби Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Повертає таблицю з відповідними даними служби Dynamics NAV.
