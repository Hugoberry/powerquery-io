# Smartsheet.Tables


## Description

Повертає таблицю аркушів, звітів, папок і робочих областей з інтерфейсу Smartsheet API.


## Syntax

```powerquery
Smartsheet.Tables(

) as table
```


## Details

Повертає вкладену таблицю доступних аркушів, звітів, папок і робочих областей з інтерфейсу Smartsheet API.


## Examples

### Example #1 
Повертає аркуші, папки, звіти та робочі області, доступні на верхньому рівні ієрархії Smartsheet.
```powerquery
Smartsheet.Tables()
```

Result: 
```powerquery
Таблиця папок, звітів і робочих областей із верхнього рівня ієрархії Smartsheet
```
