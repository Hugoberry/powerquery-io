# VSTS.AccountContents


## Description

Введіть URL-адресу служби аналітики Azure DevOps.


## Syntax

```powerquery
VSTS.AccountContents(
    url as text,
    optional options as record
) as binary
```


## Details

Повертає вміст, завантажений за URL-адресою служби аналітики Azure DevOps, як двійкове значення. Ця функція використовує різні облікові дані для різних організацій.
