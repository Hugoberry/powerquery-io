# VSTS.AnalyticsViews


## Description

Введіть назви організації та проекту.


## Syntax

```powerquery
VSTS.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Повертає таблицю подань аналітики, доступної в Azure DevOps.
