# VSTS.Views


## Description

Введіть назви організації та проекту.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Повертає таблицю каналів OData, доступних в Azure DevOps.
