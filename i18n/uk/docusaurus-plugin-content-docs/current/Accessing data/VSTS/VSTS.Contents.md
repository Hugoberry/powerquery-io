# VSTS.Contents


## Description

Введіть URL-адресу служби аналітики Azure DevOps.


## Syntax

```powerquery
VSTS.Contents(
    url as text,
    optional options as record
) as binary
```


## Details

Повертає вміст, завантажений за URL-адресою служби аналітики VSTS, як двійкове значення. Ця функція використовує глобальні облікові дані для всіх доступних організацій.
