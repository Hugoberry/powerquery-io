# Dynamics365BusinessCentral.EnvironmentContents


## Description

Укажіть середовище та компанію з Dynamics 365 Business Central.


## Syntax

```powerquery
Dynamics365BusinessCentral.EnvironmentContents(
    optional environment as text,
    optional company as text
) as table
```


## Details

Повертає таблицю з відповідними даними Dynamics 365 Business Central.
