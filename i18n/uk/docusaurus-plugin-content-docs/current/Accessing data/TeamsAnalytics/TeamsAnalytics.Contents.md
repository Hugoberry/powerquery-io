# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Details

Цей аналітичний з’єднувач для Teams дає змогу отримувати висновки щодо використання рішення. Переглядайте дані щодо своїх окремих дій і активності в групах.<br />Відомості доступні за попередні 90 днів.<br />Щоб дізнатися більше про типи доступних даних, див. довідкову документацію, зокрема:<br /><ul><li><b>про використання Teams</b> – https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>про активність користувача</b> – https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
