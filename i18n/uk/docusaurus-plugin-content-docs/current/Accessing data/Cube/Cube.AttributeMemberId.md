# Cube.AttributeMemberId


## Description

Повертає унікальний ідентифікатор елемента зі значення властивості елемента.


## Syntax

```powerquery
Cube.AttributeMemberId(
    attribute as any
) as any
```


## Details

Повертає унікальний ідентифікатор елемента зі значення властивості елемента. <code>attribute</code>. Повертає Null-значення для будь-яких інших значень.
