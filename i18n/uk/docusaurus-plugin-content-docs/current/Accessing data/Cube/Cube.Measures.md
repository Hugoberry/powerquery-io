# Cube.Measures


## Description

Повертає таблицю, що містить набір доступних мір.


## Syntax

```powerquery
Cube.Measures(
    cube as any
) as table
```


## Details

Повертає таблицю, що містить набір доступних мір у значенні <code>cube</code>.    Кожна міра представлена як функція. Міри можна застосувати до куба за допомогою Cube.AddMeasureColumn.
