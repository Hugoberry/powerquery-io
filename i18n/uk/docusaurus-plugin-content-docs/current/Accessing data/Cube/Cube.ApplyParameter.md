# Cube.ApplyParameter


## Description

Повертає куб після застосування до нього параметра.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Повертає куб після застосування параметрів "<code>parameter</code>" і "<code>arguments</code>" до елемента "<code>cube</code>".



## Category
Cube
