# Cube.DisplayFolders


## Description

Повертає вкладене дерево таблиць, яке представляє ієрархію папок відображення об’єктів (як-от виміри та міри).


## Syntax

```powerquery
Cube.DisplayFolders(
    cube as table
) as table
```


## Details

Повертає вкладене дерево таблиць, яке представляє ієрархію папок відображення об’єктів (як-от виміри та міри), що можна використовувати в <code>cube</code>.
