# Cube.MeasureProperties


## Description

Повертає таблицю з набором доступних властивостей для мір, розгорнутих у кубі.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Details

Повертає таблицю з набором доступних властивостей для мір, розгорнутих у кубі.
