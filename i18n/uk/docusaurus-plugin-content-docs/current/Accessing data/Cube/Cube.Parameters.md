# Cube.Parameters


## Description

Повертає таблицю з набором параметрів, які можна застосувати до куба.


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Details

Повертає таблицю з набором параметрів, які можна застосувати до елемента "<code>cube</code>". Кожен параметр – це функція, яка повертає елемент "<code>cube</code>" із застосованим параметром і його аргументами.
