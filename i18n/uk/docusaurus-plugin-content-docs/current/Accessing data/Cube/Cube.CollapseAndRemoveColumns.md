# Cube.CollapseAndRemoveColumns


## Description

Змінює деталізацію вимірів контексту фільтра для куба, згортаючи атрибути, зіставлені з указаними стовпцями.


## Syntax

```powerquery
Cube.CollapseAndRemoveColumns(
    cube as table,
    columnNames as list
) as table
```


## Details

Змінює деталізацію вимірів контексту фільтра для <code>cube</code>, згортаючи атрибути, зіставлені з указаними стовпцями <code>columnNames</code>. Стовпці також видалено з табличного подання куба.
