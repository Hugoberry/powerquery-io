# Cube.Transform


## Description

Застосовує список функцій куба.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Застосовує список функцій куба, <code>transforms</code>, до <code>cube</code>.

Cube
