# Cube.AttributeMemberProperty


## Description

Повертає властивість атрибута виміру.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Повертає властивість "<code>propertyName</code>" атрибута виміру "<code>attribute</code>".



## Category
Cube
