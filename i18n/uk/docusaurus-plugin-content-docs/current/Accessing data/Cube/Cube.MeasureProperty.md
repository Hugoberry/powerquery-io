# Cube.MeasureProperty


## Description

Повертає властивість міри (властивість клітинки).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Повертає властивість <code>propertyName</code> міри <code>measure</code>.



## Category
Cube
