# Webtrends.Tables


## Description

Введіть ідентифікатор профілю, пов’язаний із вашим обліковим записом Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Повертає таблицю з відповідними даними Webtrends.
