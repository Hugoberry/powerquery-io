# Marketo.Tables


## Description

Введіть URL-адресу кінцевої точки інтерфейсу REST API програми Marketo, пов’язаної з вашим обліковим записом.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Повертає таблицю з відповідними даними Marketo.
