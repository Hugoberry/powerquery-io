# Exchange.Contents


## Description

Повернення змісту, знайденого за обліковим записом Microsoft Exchange.


## Syntax

```powerquery
Exchange.Contents(
    optional mailboxAddress as text
) as table
```


## Details

Повертає таблицю вмісту з облікового запису Microsoft Exchange <code>mailboxAddress</code>. Якщо параметр <code>mailboxAddress</code> не вказано, використовуються облікові дані стандартного облікового запису.
