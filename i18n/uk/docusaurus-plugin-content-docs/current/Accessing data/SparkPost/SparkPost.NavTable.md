# SparkPost.NavTable



## Syntax

```powerquery
SparkPost.NavTable(
    DaysToAggregate as number
) as table
```


## Details

Отримує надані з’єднувачем SparkPost вбудовані таблиці з узагальненими даними за вказану користувачем кількість днів. Оновлюючи ці таблиці або викликаючи SparkPost API за допомогою цього з’єднувача, пам’ятайте, що SparkPost API має суворе обмеження на частоту викликів. Якщо сервер SparkPost повертає код стану 429, це означає, що ви досягли обмеження частоти й повинні трохи зачекати, перш ніж здійснювати подальші виклики. Вибираючи значення для параметра "Кількість днів", майте на увазі, що цей інтерфейс API зберігає дані лише за 6 місяців.
