# Xml.Document


## Description

Повертає вміст XML-документа у вигляді ієрархічної таблиці.


## Syntax

```powerquery
Xml.Document(
    contents as any,
    optional encoding as TextEncoding.Type
) as table
```


## Details

Повертає вміст XML-документа у вигляді ієрархічної таблиці.
