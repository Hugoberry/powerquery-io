# AzureSpark.Tables


## Description

Створити список таблиць в екземплярі Azure Spark.


## Syntax

```powerquery
AzureSpark.Tables(
    server as text,
    optional options as record
) as table
```


## Details

Створити список таблиць в екземплярі Azure Spark.


## Examples

### Example #1 
Створити список таблиць в екземплярі Azure Spark.
```powerquery
AzureSpark.Tables("contoso.azurehdinsight.net")
```
