# GoogleAnalytics.Accounts


## Description

Повертає облікові записи Google Analytics.


## Syntax

```powerquery
GoogleAnalytics.Accounts(
    optional options as record
) as table
```


## Details

Повертає облікові записи Google Analytics, доступ до яких можна отримати з поточними обліковими даними.
