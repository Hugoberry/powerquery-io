# Dynamics365BusinessCentralOnPremises.Contents


## Description

Введіть URL-адресу кінцевої точки OData служби Dynamics 365 Business Central (локальний екземпляр).


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Повертає таблицю з відповідними даними Dynamics 365 Business Central (локальний екземпляр).
