# Excel.CurrentWorkbook


## Description

Повертає вміст поточної книги Excel.


## Syntax

```powerquery
Excel.CurrentWorkbook(

) as table
```


## Details

Повертає вміст поточної книги Excel: таблиці, іменовані діапазони та динамічні масиви. На відміну від <code>excel.Workbook</code> ця функція не повертає аркуші.
