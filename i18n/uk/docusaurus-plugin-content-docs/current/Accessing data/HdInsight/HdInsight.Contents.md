# HdInsight.Contents


## Description

Повертає навігаційну таблицю, яка містить контейнери, знайдені в указаному обліковому записі зі сховища Azure.


## Syntax

```powerquery
HdInsight.Contents(
    account as text
) as table
```


## Details

Повертає навігаційну таблицю, яка містить рядок для кожного контейнера, знайденого за URL-адресою облікового запису, <code>account</code>, зі сховища Azure. Кожен рядок містить посилання на BLOB-об’єкти контейнера.
