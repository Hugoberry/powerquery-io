# ActiveDirectory.Domains


## Description

Повертає список доменів Active Directory в лісі, у якому розташований вказаний домен або домен поточного комп&#39;ютера, якщо жодний домен не вказано.


## Syntax

```powerquery
ActiveDirectory.Domains(
    optional forestRootDomainName as text
) as table
```


## Details
