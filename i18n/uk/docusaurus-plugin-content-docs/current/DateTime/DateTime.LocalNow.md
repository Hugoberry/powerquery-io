# DateTime.LocalNow


## Description

Повертає поточну дату та час у місцевому часовому поясі.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Повертає значення <code>datetime</code>, встановлене для поточної дати й часу в системі.



## Category
DateTime
