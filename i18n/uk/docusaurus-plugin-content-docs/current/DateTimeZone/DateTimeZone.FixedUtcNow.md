# DateTimeZone.FixedUtcNow


## Description

Повертає поточну дату й час за часом UTC (за Гринвічем). Це значення – фіксоване, і воно не зміниться після наступних викликів.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
