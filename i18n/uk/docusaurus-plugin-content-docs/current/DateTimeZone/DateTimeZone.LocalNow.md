# DateTimeZone.LocalNow


## Description

Повертає поточну дату та час у місцевому часовому поясі.


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

Повертає значення <code>datetimezone</code>, установлене для поточних дати й часу в системі.    Повернуте значення містить інформацію про місцевий часовий пояс.
