# DateTimeZone.UtcNow


## Description

Повертає поточну дату й час за часом UTC (за Гринвічем).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Повертає поточну дату й час за часом UTC (за Гринвічем).


## Examples

### Example #1 
Визначення поточної дати й часу за UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
