# Tables.GetRelationships


## Description

Отримує зв’язки між таблицями набору.


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Details

Отримує зв’язки між таблицями набору. Припускається, що структура набору "<code>tables</code>" подібна до таблиці переходів. Визначений параметром "<code>dataColumn</code>" стовпець містить фактичні таблиці даних.
