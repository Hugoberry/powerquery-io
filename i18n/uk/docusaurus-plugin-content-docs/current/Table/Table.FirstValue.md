# Table.FirstValue


## Description

Повертає перший стовпець першого рядка таблиці або вказане стандартне значення.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

Повертає перший стовпець першого рядка таблиці <code>table</code> або вказане стандартне значення.

Table.Row operations
