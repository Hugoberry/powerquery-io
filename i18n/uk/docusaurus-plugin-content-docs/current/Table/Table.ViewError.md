# Table.ViewError


## Description

Створює модифікований запис про помилку, який не ініціює перехід при поверненні обробником, зазначеним у поданні (через Table.View).


## Syntax

```powerquery
Table.ViewError(
    errorRecord as record
) as record
```


## Details

Створює модифікований запис про помилку від <code>errorRecord</code>, який не ініціює перехід при поверненні обробником, зазначеним у поданні (через Table.View).
