# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Двійковий формат, який читає 64-розрядне беззнакове ціле число, закодоване за допомогою 7-розрядного кодування змінної довжини.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details
