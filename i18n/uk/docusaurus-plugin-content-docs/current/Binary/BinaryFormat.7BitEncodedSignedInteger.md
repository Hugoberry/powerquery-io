# BinaryFormat.7BitEncodedSignedInteger


## Description

Двійковий формат, який читає 64-розрядне ціле число зі знаком, закодоване за допомогою 7-розрядного кодування змінної довжини.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details
