# BinaryFormat.ByteOrder


## Description

Повертає двійковий формат із порядком байтів, указаними за допомогою функції.


## Syntax

```powerquery
BinaryFormat.ByteOrder(
    binaryFormat as function,
    byteOrder as ByteOrder.Type
) as function
```


## Details

Повертає двійковий формат із порядком байтів, указаними за допомогою <code>binaryFormat</code>.  Порядок байтів за замовчуванням: <code>ByteOrder.BigEndian</code>.
