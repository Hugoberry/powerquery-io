# Text.Length


## Description

Повертає кількість символів.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Повертає кількість символів у тексті <code>text</code>.


## Examples

### Example #1 
Знайти кількість символів у тексті &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
