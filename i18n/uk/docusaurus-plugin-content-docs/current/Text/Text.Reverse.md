# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Обертає порядок тексту <code>text</code> на зворотний.


## Examples

### Example #1 
Обернути текст &#34;123&#34;.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
