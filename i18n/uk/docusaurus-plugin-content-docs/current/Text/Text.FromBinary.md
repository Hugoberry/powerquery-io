# Text.FromBinary


## Description

Розшифровує дані з двійкового формату на текстовий.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

Розшифровує дані, <code>binary</code>, з двійкового значення на текстове за допомогою типу <code>encoding</code>.



## Category
Text.Conversions from and to text
