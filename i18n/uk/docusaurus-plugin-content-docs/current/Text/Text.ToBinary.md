# Text.ToBinary


## Description

Кодує текст у двійкову форму.


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

Кодує надане текстове значення, <code>text</code>, у двійкове значення за допомогою вказаного <code>encoding</code>.
