# Uri.Combine


## Description

Повертає абсолютний URI, який є комбінацією URI вхідної бази та відносного URI.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Повертає абсолютний URI, який є комбінацією вхідного <code>baseUri</code> та <code>relativeUri</code>.

Uri
