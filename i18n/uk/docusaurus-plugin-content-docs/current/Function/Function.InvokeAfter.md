# Function.InvokeAfter


## Description

Виклик указаної функції після визначеної тривалості.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

Повернення результату виклику <code>function</code> після визначеної тривалості (<code>delay</code>).



## Category
Function
