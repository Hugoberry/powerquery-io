# Value.Subtract


## Description

Повертає різницю двох значень.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Повертає різницю <code>value1</code> і <code>value2</code>. Можна вказати додатковий параметр <code>precision</code>. За замовчуванням використовується <code>Precision.Double</code>.
