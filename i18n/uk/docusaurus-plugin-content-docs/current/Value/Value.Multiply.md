# Value.Multiply


## Description

Повертає добуток двох значень.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Повертає добуток множення <code>value1</code> на <code>value2</code>. Можна вказати додатковий параметр <code>precision</code>. За замовчуванням використовується <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
