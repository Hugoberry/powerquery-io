# Value.Alternates


## Description

Указує альтернативні плани запитів.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Ця функція в складі виразу планів запитів, що отримується шляхом поєднання Value.Expression(Value.Optimize(…)), указує альтернативні плани запитів. Її не призначено для застосування з іншою метою.



## Category
Expression
