# Value.Divide


## Description

Повертає результат ділення першого значення на друге.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Повертає результат ділення <code>value1</code> на <code>value2</code>. Можна вказати додатковий параметр <code>precision</code>. За замовчуванням використовується <code>Precision.Double</code>.
