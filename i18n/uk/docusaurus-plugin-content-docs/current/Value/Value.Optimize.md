# Value.Optimize


## Description

Указує, що функція Value.Expression має повернути оптимізований вираз для значення.


## Syntax

```powerquery
Value.Optimize(
    value as any
) as any
```


## Details

Ця функція вказує, що має бути повернуто оптимізований вираз, якщо її вказано разом із функцією Value.Expression, а <code>value</code> – це запит, який можна оптимізувати. Інакше <code>value</code> буде передано без змін.



## Category
Expression
