# Value.Equals


## Description

Повертає результат, чи два значення рівні.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Повертає значення true, якщо значення <code>value1</code> дорівнює значенню <code>value2</code>, у протилежному випадку повертає значення false.



## Category
Values
