# Value.Add


## Description

Повертає суму двох значень.


## Syntax

```powerquery
Value.Add(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Повертає суму <code>value1</code> і <code>value2</code>. Можна вказати додатковий параметр <code>precision</code>. За замовчуванням використовується <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
