# Value.NullableEquals


## Description

Повертає результат, чи два значення рівні.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Повертає нуль, якщо один з аргументів "<code>value1</code>" або "<code>value2</code>" є нулем, у протилежному випадку є еквівалентним Value.Equals.

Values
