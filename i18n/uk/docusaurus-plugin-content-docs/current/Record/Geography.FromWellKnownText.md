# Geography.FromWellKnownText


## Description

Перетворює текст, що представляє географічне значення у форматі Well-Known Text (WKT), у структурований запис.


## Syntax

```powerquery
Geography.FromWellKnownText(
    input as text
) as record
```


## Details

Перетворює текст, що представляє географічне значення у форматі Well-Known Text (WKT), у структурований запис. WKT – це стандартний формат, визначений Відкритим геопросторовим консорціумом (OGC). Це типовий формат серіалізації, який використовують бази даних, зокрема SQL Server.
