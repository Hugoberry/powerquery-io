# Geography.ToWellKnownText


## Description

Перетворює значення структурованої географічної точки у формат Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Перетворює значення структурованої географічної точки у формат Well-Known Text (WKT), визначений Відкритим геопросторовим консорціумом (OGC), а також у формат серіалізації, який використовує багато баз даних, зокрема SQL Server.
