# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Binárny formát, ktorý prečíta 64-bitové celé číslo bez znamienka kódované pomocou 7-bitového kódovania s premenlivou dĺžkou.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
