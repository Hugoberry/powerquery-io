# BinaryFormat.Double


## Description

Binárny formát, ktorý prečíta 8-bajtovú hodnotu s pohyblivou desatinnou čiarkou s dvojitou presnosťou IEEE.


## Syntax

```powerquery
BinaryFormat.Double(
    binary as binary
) as any
```
