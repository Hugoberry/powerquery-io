# Binary.Combine


## Description

Kombinuje zoznam binárnych údajov do jedného binárneho údaja.


## Syntax

```powerquery
Binary.Combine(
    binaries as list
) as binary
```


## Details

Kombinuje zoznam binárnych údajov do jedného binárneho údaja.
