# BinaryFormat.7BitEncodedSignedInteger


## Description

Binárny formát, ktorý prečíta 64-bitové celé číslo so znamienkom kódované pomocou 7-bitového kódovania s premenlivou dĺžkou.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details
