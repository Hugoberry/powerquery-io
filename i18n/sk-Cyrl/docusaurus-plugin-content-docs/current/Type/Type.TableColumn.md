# Type.TableColumn


## Description

Vráti typ stĺpca v tabuľke.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Vráti typ stĺpca <code>column</code> v tabuľke typu <code>tableType</code>.
