# Type.TableSchema


## Description

Vráti tabuľku s popisom stĺpcov (napr. schému) zadaného typu tabuľky.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Details

Vráti tabuľku s popisom stĺpcov tabuľky <code>tableType</code>.<br />Popis výslednej tabuľky nájdete v dokumentácii pre <code>Table.Schema</code>.<br />

Type
