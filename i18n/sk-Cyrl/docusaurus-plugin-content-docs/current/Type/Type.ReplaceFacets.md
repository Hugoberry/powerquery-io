# Type.ReplaceFacets


## Description

Nahradí obmedzenia typu.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

Nahradí obmedzenia typu <code>type</code> za obmedzenia obsiahnuté v zázname <code>facets</code>.
