# Uri.Combine


## Description

Vráti absolútny identifikátor URI, ktorý je kombináciou vstupného identifikátora URI základu a relatívneho identifikátora URI.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Vráti absolútny identifikátor URI, ktorý je kombináciou vstupu <code>baseUri</code> a identifikátora <code>relativeUri</code>.
