# Number.Tan


## Description

Vráti tangens čísla.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Vráti tangens čísla <code>number</code>.


## Examples

### Example #1 
Nájdite tangens uhla 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
