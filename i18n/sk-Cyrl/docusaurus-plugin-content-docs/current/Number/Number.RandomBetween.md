# Number.RandomBetween


## Description

Vráti náhodné číslo medzi dvomi číslami.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Vráti náhodné číslo medzi číslami <code>bottom</code> a <code>top</code>.


## Examples

### Example #1 
Získajte náhodné číslo medzi číslami 1 a 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
