# Number.BitwiseOr


## Description

Vráti výsledok vykonania bitovej operácie &#34;Or&#34; medzi dvomi vstupmi.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Vráti výsledok vykonania bitovej operácie "Or" medzi vstupmi <code>number1</code> a <code>number2</code>.



## Category
Number.Bytes
