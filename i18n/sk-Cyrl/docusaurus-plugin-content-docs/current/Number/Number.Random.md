# Number.Random


## Description

Vráti náhodné číslo.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Vráti náhodné číslo v rozsahu 0 až 1.


## Examples

### Example #1 
Získajte náhodné číslo.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
