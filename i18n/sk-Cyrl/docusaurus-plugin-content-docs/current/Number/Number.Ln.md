# Number.Ln


## Description

Vráti prirodzený logaritmus čísla.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Vráti prirodzený logaritmus čísla <code>number</code>. Ak hodnota <code>number</code> je null, funkcia <code>Number.Ln</code> vráti hodnotu null.


## Examples

### Example #1 
Získajte prirodzený logaritmus čísla 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
