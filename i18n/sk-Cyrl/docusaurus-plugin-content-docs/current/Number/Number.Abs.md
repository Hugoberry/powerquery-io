# Number.Abs


## Description

Vráti absolútnu hodnotu čísla.


## Syntax

```powerquery
Number.Abs(
    number as number
) as number
```


## Details

Vráti absolútnu hodnotu čísla <code>number</code>. Ak hodnota <code>number</code> je null, funkcia <code>Number.Abs</code> vráti hodnotu null.    <ul>        <li><code>number</code>: Číslo <code>number</code>, pre ktoré sa má vypočítať absolútna hodnota.</li>      </ul>


## Examples

### Example #1 
Absolútna hodnota čísla –3.
```powerquery
Number.Abs(-3)
```

Result: 
```powerquery
3
```
