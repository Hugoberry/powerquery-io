# Number.BitwiseAnd


## Description

Vráti výsledok vykonania bitovej operácie &#34;And&#34; medzi dvomi vstupmi.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

Vráti výsledok vykonania bitovej operácie "And" medzi vstupmi <code>number1</code> a <code>number2</code>.



## Category
Number.Bytes
