# Number.Sin


## Description

Vráti sínus čísla.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Vráti sínus čísla <code>number</code>.


## Examples

### Example #1 
Nájdite sínus uhla 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
