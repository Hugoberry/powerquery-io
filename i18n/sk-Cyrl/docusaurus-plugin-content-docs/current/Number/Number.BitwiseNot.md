# Number.BitwiseNot


## Description

Vráti bajt, pričom každý bit je opačný voči vstupu.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Vráti výsledok vykonania bitovej operácie "Not" vo vstupe <code>number</code>.



## Category
Number.Bytes
