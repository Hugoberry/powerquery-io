# Number.Cos


## Description

Vráti kosínus čísla.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Vráti kosínus čísla <code>number</code>.


## Examples

### Example #1 
Nájdite kosínus uhla 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
