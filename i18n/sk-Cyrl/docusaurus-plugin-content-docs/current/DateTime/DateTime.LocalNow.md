# DateTime.LocalNow


## Description

Vráti aktuálny dátum a čas v miestnom časovom pásme.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Vráti hodnotu <code>datetime</code> nastavenú na aktuálny dátum a čas v systéme.

DateTime
