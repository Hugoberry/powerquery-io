# DateTimeZone.FixedUtcNow


## Description

Vráti aktuálny dátum a čas v čase UTC (časové pásmo GMT). Táto hodnota je pevne stanovená a nezmení sa pri ďalších volaniach.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
