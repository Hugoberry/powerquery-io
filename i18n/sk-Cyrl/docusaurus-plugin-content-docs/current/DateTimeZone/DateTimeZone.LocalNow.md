# DateTimeZone.LocalNow


## Description

Vráti aktuálny dátum a čas v miestnom časovom pásme.


## Syntax

```powerquery
DateTimeZone.LocalNow(

) as datetimezone
```


## Details

Vráti hodnotu <code>datetimezone</code> nastavenú na aktuálny dátum a čas v systéme.    Vrátená hodnota obsahuje informácie o časovom pásme predstavujúce miestne časové pásmo.
