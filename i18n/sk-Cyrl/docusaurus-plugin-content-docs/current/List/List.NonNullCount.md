# List.NonNullCount


## Description

Vráti počet položiek s hodnotami rôznymi od null v zozname.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Vráti počet položiek s hodnotami rôznymi od null v zozname <code>list</code>.

List.Information
