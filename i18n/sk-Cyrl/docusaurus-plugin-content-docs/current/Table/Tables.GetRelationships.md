# Tables.GetRelationships


## Description

Získa vzťahy v rámci množiny tabuliek.


## Syntax

```powerquery
Tables.GetRelationships(
    tables as table,
    optional dataColumn as text
) as table
```


## Details

Získa vzťahy v rámci množiny tabuliek. Predpokladá sa, že množina <code>tables</code> má podobnú štruktúru ako navigačná tabuľka. Stĺpec definovaný položkou <code>dataColumn</code> obsahuje skutočné tabuľky údajov.
