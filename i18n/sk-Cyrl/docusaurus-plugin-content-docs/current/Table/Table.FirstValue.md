# Table.FirstValue


## Description

Vráti prvý stĺpec prvého riadka tabuľky alebo zadanú predvolenú hodnotu.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details
