# Value.Equals


## Description

Vráti informácie o tom, či sa dve hodnoty rovnajú.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Vráti hodnotu True, ak sa hodnota <code>value1</code> rovná hodnote <code>value2</code>, v opačnom prípade vráti hodnotu False.
