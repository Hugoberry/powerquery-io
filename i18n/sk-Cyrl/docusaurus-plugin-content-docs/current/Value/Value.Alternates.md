# Value.Alternates


## Description

Vyjadruje alternatívne plány dotazov.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Vyjadruje alternatívne plány dotazov v rámci výrazu plánu dotazu získaného cez Value.Expression(Value.Optimize(...)). Nie je určené na iné použitie.



## Category
Expression
