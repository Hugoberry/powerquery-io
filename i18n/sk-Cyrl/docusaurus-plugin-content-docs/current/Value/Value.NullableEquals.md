# Value.NullableEquals


## Description

Vráti informácie o tom, či sa dve hodnoty rovnajú.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Vráti hodnotu null, ak niektorý z argumentov <code>value1</code>, <code>value2</code> má hodnotu null, v opačnom prípade vráti ekvivalent k hodnote Value.Equals.

Values
