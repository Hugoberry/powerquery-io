# Cube.AttributeMemberProperty


## Description

Vráti vlastnosť atribútu dimenzie.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Vráti vlastnosť <code>propertyName</code> atribútu dimenzie <code>attribute</code>.
