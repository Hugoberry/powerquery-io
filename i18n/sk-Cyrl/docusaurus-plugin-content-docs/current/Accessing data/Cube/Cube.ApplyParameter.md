# Cube.ApplyParameter


## Description

Vráti kocku, keď sa na ňu použije parameter.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Vráti kocku, keď sa <code>parameter</code> s <code>arguments</code> použije na <code>cube</code>.



## Category
Cube
