# Cube.MeasureProperty


## Description

Vráti vlastnosť miery (vlastnosť bunky).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Vráti vlastnosť <code>propertyName</code> miery <code>measure</code>.
