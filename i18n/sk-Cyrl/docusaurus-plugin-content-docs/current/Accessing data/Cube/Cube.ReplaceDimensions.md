# Cube.ReplaceDimensions


## Description

Nahradí množinu dimenzií vrátených vlastnosťou Cube.Dimensions.


## Syntax

```powerquery
Cube.ReplaceDimensions(
    cube as table,
    dimensions as any
) as table
```


## Details

Nahradí množinu dimenzií vrátených vlastnosťou <code>Cube.Dimensions</code>    Túto funkciu možno napríklad použiť na pridanie stĺpca ID do atribútu dimenzie, aby sa zdroj údajov mohol zoskupiť podľa ID, a nie podľa zobrazenej hodnoty.
