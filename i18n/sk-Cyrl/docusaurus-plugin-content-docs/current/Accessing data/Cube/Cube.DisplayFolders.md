# Cube.DisplayFolders


## Description

Vráti vnorený strom tabuliek, znázorňujúc hierarchiu zobrazovaného priečinka objektov (napríklad dimenzie a mierky).


## Syntax

```powerquery
Cube.DisplayFolders(
    cube as table
) as table
```


## Details

Vráti vnorený strom tabuliek, znázorňujúc hierarchiu zobrazovaného priečinka objektov (napríklad dimenzie a mierky) dostupných na použitie v kocke <code>cube</code>.
