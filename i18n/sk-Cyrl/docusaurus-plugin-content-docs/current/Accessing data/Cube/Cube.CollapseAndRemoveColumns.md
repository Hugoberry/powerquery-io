# Cube.CollapseAndRemoveColumns


## Description

Zmení granularitu dimenzie kontextu filtra pre kocku prostredníctvom zbalenia atribútov mapovaných na zadané stĺpce.


## Syntax

```powerquery
Cube.CollapseAndRemoveColumns(
    cube as table,
    columnNames as list
) as table
```


## Details

Zmení granularitu dimenzie kontextu filtra pre kocku <code>cube</code> prostredníctvom zbalenia atribútov mapovaných na zadané stĺpce <code>columnNames</code>. Z tabuľkového zobrazenia kocky sa odstránia aj stĺpce.
