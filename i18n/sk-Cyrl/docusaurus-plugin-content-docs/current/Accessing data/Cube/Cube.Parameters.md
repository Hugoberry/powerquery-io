# Cube.Parameters


## Description

Vráti tabuľku obsahujúcu množinu parametrov, ktoré možno použiť na kocku.


## Syntax

```powerquery
Cube.Parameters(
    cube as table
) as table
```


## Details

Vráti tabuľku obsahujúcu množinu parametrov, ktoré možno použiť na <code>cube</code>. Každý parameter je funkcia, ktorú možno vyvolať na získanie <code>cube</code> s parametrom a argumentmi, ktoré sú naň použité.
