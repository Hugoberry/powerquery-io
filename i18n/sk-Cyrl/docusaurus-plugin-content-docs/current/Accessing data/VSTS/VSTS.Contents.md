# VSTS.Contents


## Description

Zadajte URL adresu svojej služby Azure DevOps Analytics Service.


## Syntax

```powerquery
VSTS.Contents(
    url as text,
    optional options as record
) as binary
```


## Details

Vráti obsah stiahnutý z URL adresy služby Azure DevOps Analytics Service ako binárnu hodnotu. Táto funkcia využíva globálne prihlasovacie údaje pre všetky prístupné organizácie.
