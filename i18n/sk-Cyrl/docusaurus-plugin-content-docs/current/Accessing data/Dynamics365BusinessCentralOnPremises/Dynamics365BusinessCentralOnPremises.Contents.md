# Dynamics365BusinessCentralOnPremises.Contents


## Description

Zadajte URL adresu koncového bodu služby OData pre Dynamics 365 Business Central (lokálna verzia).


## Syntax

```powerquery
Dynamics365BusinessCentralOnPremises.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Vráti tabuľku s relevantnými údajmi z Dynamicsu 365 Business Central (lokálna verzia).
