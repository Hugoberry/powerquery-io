# Soda.Feed


## Description

Vráti tabuľku z obsahu na konkrétnej adrese URL formátovanej podľa rozhrania SODA 2.0 API.


## Syntax

```powerquery
Soda.Feed(
    url as text
) as table
```


## Details

Vráti tabuľku z obsahu na konkrétnej adrese URL <code>url</code> formátovanej podľa rozhrania SODA 2.0 API. Adresa URL musí odkazovať na platný zdroj kompatibilný s rozhraním SODA, ktorý končí na príponu .csv.
