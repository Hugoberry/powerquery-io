# Web.Page


## Description

Vráti obsah dokumentu HTML rozdelený do základných štruktúr, ako aj vyjadrenie úplného dokumentu a jeho textu po odstránení značiek.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
