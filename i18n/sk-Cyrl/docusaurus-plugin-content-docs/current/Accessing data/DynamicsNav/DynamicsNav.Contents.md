# DynamicsNav.Contents


## Description

Zadajte URL adresu koncového bodu služby OData pre Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Vráti tabuľku s relevantnými údajmi zo služby Dynamics NAV.
