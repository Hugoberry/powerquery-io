# Text.Length


## Description

Vráti počet znakov.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Vráti počet znakov v texte <code>text</code>.


## Examples

### Example #1 
Zistite, koľko znakov sa nachádza v texte &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
