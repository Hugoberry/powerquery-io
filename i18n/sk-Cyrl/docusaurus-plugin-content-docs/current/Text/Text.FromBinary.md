# Text.FromBinary


## Description

Dekóduje údaje z binárneho formátu na text.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Details

Dekóduje údaje <code>binary</code> z binárnej hodnoty na textovú hodnotu použitím typu <code>encoding</code>.
