# Lines.ToText


## Description

Konvertuje zoznam textu na jeden text.  Zadaný oddeľovač lineSeparator je pripojený ku každému riadku.  Ak nie je zadaný, použije sa koniec riadka a znaky posunu riadka.


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```


## Details
