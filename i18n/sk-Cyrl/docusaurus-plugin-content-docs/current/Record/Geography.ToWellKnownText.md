# Geography.ToWellKnownText


## Description

Preloží štruktúrovanú hodnotu geografického bodu na jej reprezentáciu vo formáte Well-Known Text (WKT).


## Syntax

```powerquery
Geography.ToWellKnownText(
    input as record,
    optional omitSRID as logical
) as text
```


## Details

Preloží štruktúrovanú hodnotu geografického bodu na jej reprezentáciu vo formáte Well-Known Text (WKT), ako je definovaná konzorciom Open Geospatial Consortium (OGC), ako aj na formát serializácie používaný databázami vrátane SQL Servera.



## Category
Record.Serialization
