# Marketo.Tables


## Description

Įveskite „Marketo“ REST API galinio punkto, susieto su jūsų paskyra, URL.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Pateikiama lentelė su atitinkamais „Marketo“ duomenimis.
