# Cube.AttributeMemberProperty


## Description

Pateikiama dimensijos atributo ypatybė.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Pateikiama dimensijos atributo <code>attribute</code> ypatybė <code>propertyName</code>.

Cube
