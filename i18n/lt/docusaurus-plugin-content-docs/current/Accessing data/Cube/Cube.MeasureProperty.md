# Cube.MeasureProperty


## Description

Pateikiama mato ypatybė (langelio ypatybė).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Pateikiama mato <code>measure</code> ypatybė <code>propertyName</code>.



## Category
Cube
