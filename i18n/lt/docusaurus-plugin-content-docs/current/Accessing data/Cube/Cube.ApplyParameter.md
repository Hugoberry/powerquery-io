# Cube.ApplyParameter


## Description

Pritaikius parametrą pateikia kubą.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

Pritaikius <code>parameter</code> su <code>arguments</code> <code>cube</code> pateikia kubą.
