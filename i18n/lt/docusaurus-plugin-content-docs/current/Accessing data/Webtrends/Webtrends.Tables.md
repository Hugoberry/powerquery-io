# Webtrends.Tables


## Description

Įveskite profilio ID, susietą su „Webtrends“ paskyra.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Pateikiama lentelė su susijusiais „Webtrends“ duomenimis.
