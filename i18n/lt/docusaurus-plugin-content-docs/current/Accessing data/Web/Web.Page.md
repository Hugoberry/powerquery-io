# Web.Page


## Description

Pateikiamas HTML dokumento, išskaidyto į sudedamųjų dalių struktūrą, turinys ir viso dokumento bei jo teksto po žymių pašalinimo išraiška.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
