# DateTimeZone.UtcNow


## Description

Dabartinė data ir laikas pateikiami naudojant UTC (GMT laiko juosta).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Dabartinė data ir laikas pateikiami naudojant UTC (GMT laiko juosta).


## Examples

### Example #1 
Gauti dabartinę datą ir laiką UTC pavidalu.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
