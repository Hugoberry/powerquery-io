# Type.TableColumn


## Description

Pateikiamas lentelės stulpelio tipas.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Pateikiamas lentelės tipo <code>tableType</code> <code>column</code> stulpelio tipas.
