# Type.ListItem


## Description

Pateikiamas elemento tipas iš sąrašo tipo.


## Syntax

```powerquery
Type.ListItem(
    type as type
) as type
```


## Details

Pateikiamas elemento tipas iš sąrašo <code>type</code>.


## Examples

### Example #1 
Raskite elemento tipą sąraše &lt;code&gt;\{number}&lt;/code&gt;.
```powerquery
Type.ListItem(type {number})
```

Result: 
```powerquery
type number
```
