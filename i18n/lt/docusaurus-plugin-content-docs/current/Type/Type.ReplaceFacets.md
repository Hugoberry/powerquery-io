# Type.ReplaceFacets


## Description

Pakeičiami tipo aspektai.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Details

<code>type</code> aspektai pakeičiami įraše <code>facets</code> esančiais aspektais.
