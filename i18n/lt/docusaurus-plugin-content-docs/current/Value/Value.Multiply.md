# Value.Multiply


## Description

Pateikiamas dviejų reikšmių rezultatas.


## Syntax

```powerquery
Value.Multiply(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Pateikiamas <code>value1</code> dauginimo iš <code>value2</code> rezultatas. Galima nurodyti pasirenkamą parametrą <code>precision</code>; pagal numatytuosius parametrus naudojamas <code>Precision.Double</code>.
