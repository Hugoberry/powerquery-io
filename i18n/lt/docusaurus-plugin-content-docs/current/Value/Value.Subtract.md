# Value.Subtract


## Description

Pateikiamas dviejų reikšmių skirtumas.


## Syntax

```powerquery
Value.Subtract(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Pateikiamas <code>value1</code> ir <code>value2</code> skirtumas. Galima nurodyti pasirenkamą parametrą <code>precision</code>; pagal numatytuosius parametrus naudojamas <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
