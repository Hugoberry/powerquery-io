# Value.NullableEquals


## Description

Nurodoma, ar dvi reikšmės yra lygios.


## Syntax

```powerquery
Value.NullableEquals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Pateikiama neapibrėžta reikšmė, jei argumentas „<code>value1</code>“ arba „<code>value2</code>“ yra neapibrėžtas; priešingu atveju atitinka Value.Equals.



## Category
Values
