# Value.Divide


## Description

Pateikiamas pirmosios reikšmės padalijimo iš antrosios rezultatas.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Pateikiamas <code>value1</code> padalijimo iš <code>value2</code> rezultatas. Galima nurodyti pasirenkamą parametrą <code>precision</code>; pagal numatytuosius parametrus naudojamas <code>Precision.Double</code>.
