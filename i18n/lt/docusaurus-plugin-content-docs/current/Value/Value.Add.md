# Value.Add


## Description

Pateikiama dviejų reikšmių suma.


## Syntax

```powerquery
Value.Add(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Pateikiama <code>value1</code> ir <code>value2</code> suma. Galima nurodyti pasirenkamą parametrą <code>precision</code>; pagal numatytuosius parametrus naudojamas <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
