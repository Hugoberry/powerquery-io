# Value.Equals


## Description

Nurodoma, ar dvi reikšmės yra lygios.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Pateikiama „true“, jei reikšmė <code>value1</code> yra lygi reikšmei <code>value2</code>; priešingu atveju pateikiama „false“.



## Category
Values
