# Number.Sin


## Description

Pateikiamas skaičiaus sinusas.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Pateikiamas <code>number</code> sinusas.


## Examples

### Example #1 
Raskite kampo 0 sinusą.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```

Number.Trigonometry
