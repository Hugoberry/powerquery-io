# Number.Factorial


## Description

Pateikiamas skaičiaus faktorialas.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Pateikiamas skaičiaus <code>number</code> faktorialas.


## Examples

### Example #1 
Raskite skaičiaus 10 faktorialą.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
