# Number.BitwiseXor


## Description

Pateikiamas bitų „XOR“ („Exclusive-OR“) operacijos tarp dviejų įvesčių vykdymo rezultatas.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Pateikiamas bitų „XOR“ („Exclusive-OR“) operacijos tarp <code>number1</code> ir <code>number2</code> vykdymo rezultatas.



## Category
Number.Bytes
