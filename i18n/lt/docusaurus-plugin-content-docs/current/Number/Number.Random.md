# Number.Random


## Description

Pateikiamas atsitiktinis skaičius.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Pateikiamas atsitiktinis skaičius nuo 0 iki 1.


## Examples

### Example #1 
Gaukite atsitiktinį skaičių.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
