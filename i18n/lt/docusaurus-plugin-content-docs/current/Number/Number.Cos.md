# Number.Cos


## Description

Pateikiamas skaičiaus kosinusas.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Pateikiamas <code>number</code> kosinusas.


## Examples

### Example #1 
Raskite kampo 0 kosinusą.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
