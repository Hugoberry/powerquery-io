# Number.BitwiseNot


## Description

Pateikiamas baitas, kuriame kiekvienas bitas yra priešingas įvesčiai.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Pateikiamas bitų „Not“ operacijos vykdymo <code>number</code> rezultatas.



## Category
Number.Bytes
