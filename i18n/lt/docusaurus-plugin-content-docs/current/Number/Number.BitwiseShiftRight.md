# Number.BitwiseShiftRight


## Description

Nustatyti bitai perstumiami į dešinę.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Pateikiamas bitų perstūmimo į dešinę vykdymo <code>number1</code> pagal nurodytą bitų <code>number2</code> skaičių rezultatas.



## Category
Number.Bytes
