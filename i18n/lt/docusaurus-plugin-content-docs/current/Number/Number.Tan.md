# Number.Tan


## Description

Pateikiamas skaičiaus tangentas.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Pateikiamas <code>number</code> tangentas.


## Examples

### Example #1 
Raskite kampo 1 tangentą.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
