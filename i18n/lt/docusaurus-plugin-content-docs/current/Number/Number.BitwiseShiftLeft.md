# Number.BitwiseShiftLeft


## Description

Nustatyti bitai perstumiami į kairę.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Pateikiamas bitų perstūmimo į kairę vykdymo <code>number1</code> pagal nurodytą bitų <code>number2</code> skaičių rezultatas.



## Category
Number.Bytes
