# List.Count


## Description

Pateikiamas sąrašo elementų skaičius.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Pateikiamas sąrašo elementų skaičius <code>list</code>.


## Examples

### Example #1 
Rasti sąrašo reikšmių skaičių \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
