# List.NonNullCount


## Description

Pateikiamas ne neapibrėžtų sąrašo elementų skaičius.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Pateikiamas ne neapibrėžtų sąrašo <code>list</code> elementų skaičius.
