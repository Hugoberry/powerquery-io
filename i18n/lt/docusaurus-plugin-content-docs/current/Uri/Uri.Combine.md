# Uri.Combine


## Description

Pateikiamas absoliutusis URI, kuris yra įvesties pagrindinio URI ir santykinio URI derinys.


## Syntax

```powerquery
Uri.Combine(
    baseUri as text,
    relativeUri as text
) as text
```


## Details

Pateikiamas absoliutusis URI, kuris yra įvesties <code>baseUri</code> ir <code>relativeUri</code> derinys.
