# Function.IsDataSource


## Description

Pateikiama, neatsižvelgiant į tai, ar konkreti funkcija laikoma duomenų šaltiniu.


## Syntax

```powerquery
Function.IsDataSource(
    function as function
) as logical
```


## Details

Pateikiama, neatsižvelgiant į tai, ar <code>function</code> laikomas duomenų šaltiniu.
