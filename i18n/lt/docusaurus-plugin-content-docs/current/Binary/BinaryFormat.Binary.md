# BinaryFormat.Binary


## Description

Pateikiamas dvejetainis formatas, kuris perskaito dvejetainę reikšmę.


## Syntax

```powerquery
BinaryFormat.Binary(
    optional length as any
) as function
```


## Details

Pateikiamas dvejetainis formatas, kuris perskaito dvejetainę reikšmę.  Jei nurodyta <code>length</code>, dvejetainė reikšmė turės daug baitų.  Jei <code>length</code> nenurodyta, dvejetainė reikšmė turės likusius baitus.  <code>length</code> galima nurodyti arba kaip skaičių, arba kaip ilgio dvejetainį formatą, kuris nurodomas prieš dvejetainius duomenis.
