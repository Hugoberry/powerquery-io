# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Dvejetainis formatas, kuris perskaito 64 bitų sveikąjį skaičių be ženklo, užkoduotą naudojant 7 bitų kintamo ilgio kodavimą.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details
