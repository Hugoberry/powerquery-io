# BinaryFormat.7BitEncodedSignedInteger


## Description

Dvejetainis formatas, kuris perskaito 64 bitų sveikąjį skaičių su ženklu, užkoduotą naudojant 7 bitų kintamo ilgio kodavimą.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```


## Details
