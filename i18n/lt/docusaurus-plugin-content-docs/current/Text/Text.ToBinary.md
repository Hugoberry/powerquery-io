# Text.ToBinary


## Description

Tekstas koduojamas į dvejetainę formą.


## Syntax

```powerquery
Text.ToBinary(
    text as text,
    optional encoding as TextEncoding.Type,
    optional includeByteOrderMark as logical
) as binary
```


## Details

Nurodyta tekstinė reikšmė <code>text</code> koduojama į dvejetainę reikšmę naudojant nurodytą <code>encoding</code>.
