# Text.Reverse


## Description

Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Details

Nurodytas <code>text</code> pakeičiamas atvirkštine tvarka.


## Examples

### Example #1 
Tekstas 123 atvirkštine tvarka.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```

Text.Transformations
