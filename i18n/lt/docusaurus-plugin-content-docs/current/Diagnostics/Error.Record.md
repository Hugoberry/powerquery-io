# Error.Record


## Description

Pateikiamas klaidos įrašas iš pateiktų teksto reikšmių, skirtas priežasčiai, pranešimui ir informacijai.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```
