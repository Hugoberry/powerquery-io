# Cube.MeasureProperties


## Description

Vrátí tabulku obsahující sadu dostupných vlastností pro míry, které jsou v datové krychli rozbalené.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```
