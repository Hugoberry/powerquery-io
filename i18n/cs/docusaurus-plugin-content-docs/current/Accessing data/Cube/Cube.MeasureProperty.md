# Cube.MeasureProperty


## Description

Vrátí vlastnost míry (vlastnost buňky).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Details

Vrátí vlastnost <code>propertyName</code> atributu míry <code>measure</code>.

Cube
