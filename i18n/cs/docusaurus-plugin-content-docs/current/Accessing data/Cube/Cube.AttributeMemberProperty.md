# Cube.AttributeMemberProperty


## Description

Vrátí vlastnost atributu dimenze.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Details

Vrátí vlastnost <code>propertyName</code> atributu dimenze <code>attribute</code>.
