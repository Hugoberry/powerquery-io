# Cube.Transform


## Description

Použije seznam funkcí datové krychle.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Použije seznam funkcí datové krychle <code>transforms</code> na položku <code>cube</code>.
