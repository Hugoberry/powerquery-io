# DynamicsNav.Contents


## Description

Zadejte adresu URL koncového bodu služby Dynamics NAV OData.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Vrátí tabulku s odpovídajícími daty Dynamics NAV.
