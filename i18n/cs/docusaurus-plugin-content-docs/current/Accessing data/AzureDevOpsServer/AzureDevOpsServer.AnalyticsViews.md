# AzureDevOpsServer.AnalyticsViews


## Description

Zadejte názvy organizace a projektu.


## Syntax

```powerquery
AzureDevOpsServer.AnalyticsViews(
    url as text,
    project as text,
    optional options as record
) as table
```


## Details

Vrátí tabulku obsahující seznam zobrazení Analýzy, která nabízí služba Azure DevOps.
