# AzureDevOpsServer.Views


## Description

Zadejte názvy organizace a projektu.


## Syntax

```powerquery
AzureDevOpsServer.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Vrátí tabulku informačních kanálů OData nabízených službou Azure DevOps.
