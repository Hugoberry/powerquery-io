# Dynamics365BusinessCentral.ApiContentsWithOptions


## Description

Zadejte své prostředí a společnost ve službě Dynamics 365 Business Central.


## Syntax

```powerquery
Dynamics365BusinessCentral.ApiContentsWithOptions(
    optional environment as text,
    optional company as text,
    optional apiRoute as text,
    optional options as record
) as table
```


## Details

Vrátí tabulku s odpovídajícími daty Dynamics 365 Business Central.
