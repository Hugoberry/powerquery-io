# File.Contents


## Description

Vrátí obsah určeného souboru jako binární hodnotu.


## Syntax

```powerquery
File.Contents(
    path as text,
    optional options as record
) as binary
```


## Details

Vrátí obsah souboru <code>path</code>jako binární. Parametr <code>options</code> je aktuálně určen pouze pro interní použití.
