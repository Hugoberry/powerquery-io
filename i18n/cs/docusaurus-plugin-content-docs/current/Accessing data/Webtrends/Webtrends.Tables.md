# Webtrends.Tables


## Description

Zadejte ID profilu přidružené vašemu účtu Webtrends.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Details

Vrátí tabulku relevantních dat Webtrends.
