# Web.Page


## Description

Vrátí obsah dokumentu HTML rozložený do struktur, kterými je dokument tvořen, a reprezentaci celého dokumentu a jeho textu po odebrání značek.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
