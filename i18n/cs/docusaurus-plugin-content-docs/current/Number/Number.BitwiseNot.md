# Number.BitwiseNot


## Description

Vrátí bajt, ve kterém je každý bit protikladem vstupního bitu.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Details

Vrátí výsledek provedení bitové operace Not s hodnotou <code>number</code>.



## Category
Number.Bytes
