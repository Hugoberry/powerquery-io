# Number.Cos


## Description

Vrátí kosinus čísla.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Details

Vrátí kosinus hodnoty <code>number</code>.


## Examples

### Example #1 
Vrátí cosinus úhlu o velikosti 0 stupňů.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```

Number.Trigonometry
