# Number.BitwiseShiftLeft


## Description

Posune sadu bitů doleva.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Details

Vrátí výsledek bitového posunu hodnoty <code>number1</code> směrem doleva o zadaný počet bitů <code>number2</code>.



## Category
Number.Bytes
