# Number.BitwiseShiftRight


## Description

Posune sadu bitů doprava.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Details

Vrátí výsledek bitového posunu hodnoty <code>number1</code> směrem doprava o zadaný počet bitů <code>number2</code>.



## Category
Number.Bytes
