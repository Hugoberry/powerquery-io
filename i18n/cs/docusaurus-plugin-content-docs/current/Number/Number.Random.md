# Number.Random


## Description

Vrátí náhodné číslo.


## Syntax

```powerquery
Number.Random(

) as number
```


## Details

Vrátí náhodné číslo v rozsahu od 0 do 1.


## Examples

### Example #1 
Vrátí náhodné číslo.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
