# Number.BitwiseAnd


## Description

Vrátí výsledek provedení bitové operace And mezi dvěma vstupy.


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

Vrátí výsledek provedení bitové operace And mezi hodnotami <code>number1</code> a <code>number2</code>.



## Category
Number.Bytes
