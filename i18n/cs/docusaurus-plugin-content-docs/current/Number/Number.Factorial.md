# Number.Factorial


## Description

Vrátí faktoriál čísla.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Details

Vrátí faktoriál čísla <code>number</code>.


## Examples

### Example #1 
Vrátí faktoriál čísla 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
