# Number.Sin


## Description

Vrátí sinus čísla.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Details

Vrátí sinus hodnoty <code>number</code>.


## Examples

### Example #1 
Vrátí sinus úhlu 0 stupňů.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```




## Category
Number.Trigonometry
