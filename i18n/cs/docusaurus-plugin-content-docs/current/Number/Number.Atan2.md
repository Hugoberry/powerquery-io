# Number.Atan2


## Description

Vrátí arkus tangens podílu dvou čísel.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Vrátí arkus tangens podílu dvou čísel <code>y</code> a <code>x</code>. Podíl bude vypočten ve formátu <code>y</code>/<code>x</code>.



## Category
Number.Trigonometry
