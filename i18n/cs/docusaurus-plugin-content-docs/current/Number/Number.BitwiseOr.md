# Number.BitwiseOr


## Description

Vrátí výsledek provedení bitové operace Or mezi dvěma vstupy.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Details

Vrátí výsledek provedení bitové operace Or mezi hodnotami <code>number1</code> a <code>number2</code>.



## Category
Number.Bytes
