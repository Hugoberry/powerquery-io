# List.Count


## Description

Vrátí počet položek seznamu.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Vrátí počet položek seznamu <code>list</code>.


## Examples

### Example #1 
Zjistí počet hodnot v seznamu \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
