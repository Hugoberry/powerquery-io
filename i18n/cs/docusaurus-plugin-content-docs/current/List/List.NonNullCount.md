# List.NonNullCount


## Description

Vrátí počet položek seznamu, které nemají hodnotu null.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Details

Vrátí počet položek seznamu <code>list</code>, které nemají hodnotu null.

List.Information
