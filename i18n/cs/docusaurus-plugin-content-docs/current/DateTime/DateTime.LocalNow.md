# DateTime.LocalNow


## Description

Vrátí aktuální datum a čas v místním časovém pásmu.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Vrací hodnotu <code>datetime</code> nastavenou na aktuální datum a čas v systému.



## Category
DateTime
