# Text.Length


## Description

Vrátí počet znaků.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Vrátí počet znaků v textu <code>text</code>.


## Examples

### Example #1 
Zjistí, kolik znaků se nachází v textu „Hello World“.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
