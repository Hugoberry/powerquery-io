# Function.InvokeAfter


## Description

Vyvolá danou funkci po uplynutí zadané doby.


## Syntax

```powerquery
Function.InvokeAfter(
    function as function,
    delay as duration
) as any
```


## Details

Vrátí výsledek vyvolání funkce <code>function</code> po uplynutí doby <code>delay</code>.



## Category
Function
