# DateTimeZone.FixedUtcNow


## Description

Vrátí aktuální datum a čas v UTC (časové pásmo GMT). Tato hodnota je pevná a při následných voláních se nezmění.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
