# BinaryFormat.7BitEncodedUnsignedInteger


## Description

Binární formát, který umí číst 64bitové celé číslo bez znaménka kódované pomocí 7bitového kódování s proměnnou délkou


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```
