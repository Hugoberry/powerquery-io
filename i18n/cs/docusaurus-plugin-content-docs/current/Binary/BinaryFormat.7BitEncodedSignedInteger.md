# BinaryFormat.7BitEncodedSignedInteger


## Description

Binární formát, který umí číst 64bitové celé číslo se znaménkem kódované pomocí 7bitového kódování s proměnnou délkou


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
