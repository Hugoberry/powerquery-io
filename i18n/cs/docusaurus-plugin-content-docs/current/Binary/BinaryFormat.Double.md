# BinaryFormat.Double


## Description

Binární formát, který čte 8bajtovou hodnotu IEEE s pohyblivou desetinnou čárkou a s přesností na dvě desetinná čísla.


## Syntax

```powerquery
BinaryFormat.Double(
    binary as binary
) as any
```
