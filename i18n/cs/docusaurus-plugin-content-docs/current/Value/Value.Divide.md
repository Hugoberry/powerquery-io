# Value.Divide


## Description

Vrátí výsledek dělení první hodnoty druhou.


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

Vrátí výsledek dělení hodnoty <code>value1</code> hodnotou <code>value2</code>. Lze zadat volitelný parametr <code>precision</code>. Ve výchozím nastavení se používá hodnota <code>Precision.Double</code>.



## Category
Values.Arithmetic operations
