# Value.Alternates


## Description

Vyjadřuje alternativní plány dotazů.


## Syntax

```powerquery
Value.Alternates(
    alternates as list
) as any
```


## Details

Vyjadřuje alternativní plány dotazů v rámci výrazu plánu dotazů získaného pomocí Value.Expression(Value.Optimize(...)). Není určené pro jiná použití.



## Category
Expression
