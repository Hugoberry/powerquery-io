# Value.Equals


## Description

Zjistí, zda jsou si dvě hodnoty rovny.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Pokud je hodnota <code>value1</code> rovna hodnotě <code>value2</code>, vrátí hodnotu True. Jinak vrátí hodnotu False.
