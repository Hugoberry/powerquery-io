# Table.FirstValue


## Description

Vrátí první sloupec prvního řádku tabulky nebo zadanou výchozí hodnotu.


## Syntax

```powerquery
Table.FirstValue(
    table as table,
    optional default as any
) as any
```


## Details

Vrátí první sloupec prvního řádku tabulky <code>table</code> nebo zadanou výchozí hodnotu.

Table.Row operations
