# TeamsAnalytics.Contents



## Syntax

```powerquery
TeamsAnalytics.Contents(

) as table
```


## Details

Konektor analytiky Teams umožňuje získat přehledy o využití Teams. Data jsou k dispozici pro vaši osobní aktivitu v Teams i pro aktivitu v Teams, kde které jste členem. <br />Data jsou k dispozici za uplynulých 90 dnů. <br /> Podrobnosti o dostupných typech dat najdete v následující dokumentaci nápovědy: <br /><ul><li><b>Využití Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/teams-usage-report</li><li><b>Aktivita uživatele Teams:</b> https://docs.microsoft.com/en-us/microsoftteams/teams-analytics-and-reports/user-activity-report</li></ul>
