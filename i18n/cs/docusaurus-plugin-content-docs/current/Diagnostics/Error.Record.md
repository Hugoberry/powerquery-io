# Error.Record


## Description

Vrátí záznam chyby z poskytnutých textových hodnot obsahující informaci o příčině, zprávu a podrobnosti.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
