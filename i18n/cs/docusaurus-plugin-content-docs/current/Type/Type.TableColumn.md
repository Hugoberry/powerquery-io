# Type.TableColumn


## Description

Vrátí typ sloupce tabulky.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Vrátí typ sloupce <code>column</code> tabulky typu <code>tableType</code>.

Type
