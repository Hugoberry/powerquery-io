# Hdfs.Files


## Description

傳回表格，內含在 Hadoop 檔案系統的指定資料夾和子資料夾中找到的檔案的屬性和內容。


## Syntax

```powerquery
Hdfs.Files(
    url as text
) as table
```


## Details

傳回表格，內含在 Hadoop 檔案系統的資料夾 URL <code>url</code> 和子資料夾中找到的每個檔案各一列。每一列包含檔案的屬性，以及其內容的連結。
