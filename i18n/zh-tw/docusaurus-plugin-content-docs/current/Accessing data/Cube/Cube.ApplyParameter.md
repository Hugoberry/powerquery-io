# Cube.ApplyParameter


## Description

在對其套用參數之後，傳回一個 Cube。


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Details

在將含有 <code>arguments</code> 的 <code>parameter</code> 套用至 <code>cube</code> 之後，傳回一個 Cube。



## Category
Cube
