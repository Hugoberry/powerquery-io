# Value.Divide


## Description

傳回第一個值除以第二個值的結果。


## Syntax

```powerquery
Value.Divide(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as any
```


## Details

傳回 <code>value1</code> 除以 <code>value2</code> 的結果。可指定選擇性 <code>precision</code> 參數，預設使用 <code>Precision.Double</code>。

Values.Arithmetic operations
