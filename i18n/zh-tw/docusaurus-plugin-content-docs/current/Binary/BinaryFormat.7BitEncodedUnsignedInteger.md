# BinaryFormat.7BitEncodedUnsignedInteger


## Description

一種二進位格式，可讀取使用 7 位元可變長度編碼方式編碼的 64 位元不帶正負號的整數。


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Details

一種二進位格式，可讀取使用 7 位元可變長度編碼方式編碼的 64 位元不帶正負號的整數。
