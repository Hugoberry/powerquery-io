# BinaryFormat.UnsignedInteger32


## Description

一種二進位格式，可讀取 32 位元不帶正負號的整數。


## Syntax

```powerquery
BinaryFormat.UnsignedInteger32(
    binary as binary
) as any
```


## Details

一種二進位格式，可讀取 32 位元不帶正負號的整數。
