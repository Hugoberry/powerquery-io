# BinaryFormat.UnsignedInteger64


## Description

一種二進位格式，可讀取 64 位元不帶正負號的整數。


## Syntax

```powerquery
BinaryFormat.UnsignedInteger64(
    binary as binary
) as any
```


## Details

一種二進位格式，可讀取 64 位元不帶正負號的整數。
