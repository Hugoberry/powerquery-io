# Lines.ToText


## Description

將文字清單轉換成一個文字。指定的 lineSeparator 會附加至每一行。如果未指定，則使用歸位字元和換行字元。


## Syntax

```powerquery
Lines.ToText(
    lines as list,
    optional lineSeparator as text
) as text
```


## Details
