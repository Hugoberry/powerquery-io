# Number.BitwiseAnd


## Description

傳回在兩個輸入之間執行位元 &#34;And&#34; 運算的結果。


## Syntax

```powerquery
Number.BitwiseAnd(
    number1 as number,
    number2 as number
) as number
```


## Details

傳回在 <code>number1</code> 和 <code>number2</code> 之間執行位元 "And" 運算的結果。



## Category
Number.Bytes
