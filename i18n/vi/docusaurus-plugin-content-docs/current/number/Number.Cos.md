# Number.Cos


Trả về cosin của số.


## Syntax

```powerquery
Number.Cos(
    number as number
) as number
```


## Remarks

Trả về cosine của <code>number</code>.


## Examples

### Example #1 
Trả về cosin của góc 0.
```powerquery
Number.Cos(0)
```

Result: 
```powerquery
1
```
