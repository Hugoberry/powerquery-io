# Number.BitwiseShiftLeft


Chuyển dịch tập hợp bit sang trái.


## Syntax

```powerquery
Number.BitwiseShiftLeft(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Trả về kết quả thực hiện chuyển dịch theo bit về bên trái đối với <code>number1</code> theo số lượng bit được chỉ định <code>number2</code>.
