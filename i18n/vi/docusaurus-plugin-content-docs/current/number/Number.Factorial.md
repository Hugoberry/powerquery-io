# Number.Factorial


Trả về giai thừa của số.


## Syntax

```powerquery
Number.Factorial(
    number as number
) as number
```


## Remarks

Trả về giai thừa của số <code>number</code>.


## Examples

### Example #1 
Tìm giai thừa của số 10.
```powerquery
Number.Factorial(10)
```

Result: 
```powerquery
3628800
```
