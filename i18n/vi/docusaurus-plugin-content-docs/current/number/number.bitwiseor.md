# Number.BitwiseOr


Trả về kết quả thực hiện tác vụ &#34;Or&#34; theo bit giữa hai đầu vào.


## Syntax

```powerquery
Number.BitwiseOr(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Trả về kết quả thực hiện tác vụ "Or" theo bit giữa <code>number1</code> và <code>number2</code>.

Number.Bytes
