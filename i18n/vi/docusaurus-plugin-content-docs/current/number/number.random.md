# Number.Random


Trả về một số ngẫu nhiên.


## Syntax

```powerquery
Number.Random(

) as number
```


## Remarks

Trả về một số ngẫu nhiên nằm trong khoảng từ 0 đến 1.


## Examples

### Example #1 
Nhận một số ngẫu nhiên.
```powerquery
Number.Random()
```

Result: 
```powerquery
0.919303
```
