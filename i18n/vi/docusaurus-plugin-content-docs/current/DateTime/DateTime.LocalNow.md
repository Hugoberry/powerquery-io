# DateTime.LocalNow


## Description

Trả về ngày và giờ hiện tại trong múi giờ địa phương.


## Syntax

```powerquery
DateTime.LocalNow(

) as datetime
```


## Details

Trả về một giá trị <code>datetime</code> được đặt thành ngày và giờ hiện tại trên hệ thống.

DateTime
