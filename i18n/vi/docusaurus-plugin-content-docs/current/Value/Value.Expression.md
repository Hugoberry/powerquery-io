# Value.Expression


## Description

Trả về một cây cú pháp trừu tượng đại diện cho biểu thức của giá trị.


## Syntax

```powerquery
Value.Expression(
    value as any
) as record
```


## Details

Trả về một cây cú pháp trừu tượng (AST) đại diện cho biểu thức của giá trị
