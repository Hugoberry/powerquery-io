# Value.Equals


## Description

Trả về kết quả hai giá trị bằng nhau hay không bằng nhau.


## Syntax

```powerquery
Value.Equals(
    value1 as any,
    value2 as any,
    optional precision as Precision.Type
) as logical
```


## Details

Trả về true nếu giá trị <code>value1</code> bằng giá trị <code>value2</code>, false nếu ngược lại.
