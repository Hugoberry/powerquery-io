# BinaryFormat.7BitEncodedSignedInteger


## Description

Định dạng nhị phân đọc số nguyên có dấu 64 bit đã được mã hóa bằng mã hóa có độ dài biến thiên 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedSignedInteger(
    binary as binary
) as any
```
