# Binary.ViewError


## Description

Tạo bản ghi lỗi được sửa đổi không kích hoạt dự phòng khi được trình xử lý đã xác định trên dạng xem đưa ra (qua Binary.View).


## Syntax

```powerquery
Binary.ViewError(
    errorRecord as record
) as record
```


## Details

Tạo bản ghi lỗi được sửa đổi từ <code>errorRecord</code> không kích hoạt dự phòng khi được trình xử lý đã xác định trên dạng xem đưa ra (qua Binary.View).
