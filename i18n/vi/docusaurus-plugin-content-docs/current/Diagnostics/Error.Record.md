# Error.Record


## Description

Trả về một bản ghi lỗi từ các giá trị văn bản được cung cấp về lý do, nội dung và chi tiết.


## Syntax

```powerquery
Error.Record(
    reason as text,
    optional message as text,
    optional detail as any,
    optional parameters as list
) as record
```


## Details
