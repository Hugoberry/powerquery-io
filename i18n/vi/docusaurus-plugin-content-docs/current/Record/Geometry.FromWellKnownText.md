# Geometry.FromWellKnownText


Chuyển đổi văn bản biểu thị một giá trị hình học ở định dạng Văn bản quen thuộc (WKT) thành một bản ghi có cấu trúc.


## Syntax

```powerquery
Geometry.FromWellKnownText(
    input as text
) as record
```


## Remarks

Chuyển đổi văn bản biểu thị một giá trị hình học ở định dạng Văn bản quen thuộc (WKT) thành một bản ghi có cấu trúc. WKT là định dạng chuẩn được xác định bởi Open Geospatial Consortium (OGC) và là định dạng nối tiếp hóa điển hình được các cơ sở dữ liệu, trong đó có SQL Server, sử dụng.
