# List.Count


## Description

Trả về số lượng mục trong danh sách.


## Syntax

```powerquery
List.Count(
    list as list
) as number
```


## Details

Trả về số lượng mục trong danh sách <code>list</code>.


## Examples

### Example #1 
Tìm số lượng giá trị trong danh sách \{1, 2, 3}.
```powerquery
List.Count({1, 2, 3})
```

Result: 
```powerquery
3
```
