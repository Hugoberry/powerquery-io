# Type.TableColumn


## Description

Trả về loại của cột trong bảng.


## Syntax

```powerquery
Type.TableColumn(
    tableType as type,
    column as text
) as type
```


## Details

Trả về loại của cột <code>column</code> trong loại bảng <code>tableType</code>.

Type
