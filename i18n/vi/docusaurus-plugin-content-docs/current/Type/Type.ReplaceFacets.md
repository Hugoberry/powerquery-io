# Type.ReplaceFacets


Thay thế các khía cạnh của một loại.


## Syntax

```powerquery
Type.ReplaceFacets(
    type as type,
    facets as record
) as type
```


## Remarks

Thay thế các khía cạnh của <code>type</code> bằng các khía cạnh chứa trong bản ghi <code>facets</code>.
