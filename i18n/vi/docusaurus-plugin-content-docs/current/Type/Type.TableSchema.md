# Type.TableSchema


Trả về một bảng chứa mô tả về các cột (tức là sơ đồ) của loại bảng được chỉ định.


## Syntax

```powerquery
Type.TableSchema(
    tableType as type
) as table
```


## Remarks

Trả về bảng mô tả các cột của <code>tableType</code>.<br />Tham khảo tài liệu về <code>Table.Schema</code> để biết mô tả về bảng kết quả.<br />
