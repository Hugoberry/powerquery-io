# Text.Length


## Description

Trả về số lượng ký tự.


## Syntax

```powerquery
Text.Length(
    text as text
) as number
```


## Details

Trả về số lượng ký tự trong văn bản <code>text</code>.


## Examples

### Example #1 
Tìm số lượng ký tự trong văn bản &#34;Hello World&#34;.
```powerquery
Text.Length("Hello World")
```

Result: 
```powerquery
11
```
