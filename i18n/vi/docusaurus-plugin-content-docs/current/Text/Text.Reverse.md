# Text.Reverse


Text.Reverse


## Syntax

```powerquery
Text.Reverse(
    text as text
) as text
```


## Remarks

Đảo ngược <code>text</code> được cung cấp.


## Examples

### Example #1 
Đảo ngược văn bản &#34;123&#34;.
```powerquery
Text.Reverse("123")
```

Result: 
```powerquery
"321"
```
