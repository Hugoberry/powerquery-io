# Text.FromBinary


Mã hóa dữ liệu từ dạng nhị phân thành văn bản.


## Syntax

```powerquery
Text.FromBinary(
    binary as binary,
    optional encoding as TextEncoding.Type
) as text
```


## Remarks

Mã hóa dữ liệu <code>binary</code> từ giá trị nhị phân thành giá trị văn bản bằng cách sử dụng loại <code>encoding</code>.
