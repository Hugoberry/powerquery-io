# List.NonNullCount


Trả về số lượng các giá trị không rỗng trong danh sách.


## Syntax

```powerquery
List.NonNullCount(
    list as list
) as number
```


## Remarks

Trả về số lượng các giá trị không rỗng trong danh sách <code>list</code>.
