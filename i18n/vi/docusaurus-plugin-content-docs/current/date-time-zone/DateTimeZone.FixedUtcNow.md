# DateTimeZone.FixedUtcNow


Trả về ngày và giờ hiện tại theo giờ UTC (múi giờ GMT). Giá trị này được cố định và sẽ không thay đổi theo các lệnh gọi nối tiếp.


## Syntax

```powerquery
DateTimeZone.FixedUtcNow(

) as datetimezone
```
