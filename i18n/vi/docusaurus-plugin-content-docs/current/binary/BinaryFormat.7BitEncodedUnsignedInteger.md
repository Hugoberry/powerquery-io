# BinaryFormat.7BitEncodedUnsignedInteger


Định dạng nhị phân đọc số nguyên không dấu 64 bit đã được mã hóa bằng mã hóa có độ dài biến thiên 7 bit.


## Syntax

```powerquery
BinaryFormat.7BitEncodedUnsignedInteger(
    binary as binary
) as any
```


## Remarks
