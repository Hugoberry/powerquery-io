# Webtrends.Tables


Nhập ID Hồ sơ được liên kết với tài khoản Webtrends của bạn.


## Syntax

```powerquery
Webtrends.Tables(
    ProfileId as text,
    optional startDate as date,
    optional endDate as date
) as table
```


## Remarks

Trả về bảng bao gồm dữ liệu Webtrends có liên quan.
