# Dynamics365BusinessCentral.ApiContentsWithOptions


## Description

Nhập công ty và môi trường Dynamics 365 Business Central của bạn.


## Syntax

```powerquery
Dynamics365BusinessCentral.ApiContentsWithOptions(
    optional environment as text,
    optional company as text,
    optional apiRoute as text,
    optional options as record
) as table
```


## Details

Trả về bảng có dữ liệu liên quan về Dynamics 365 Business Central.
