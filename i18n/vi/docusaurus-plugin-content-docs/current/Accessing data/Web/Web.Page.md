# Web.Page


Trả về nội dung của tài liệu HTML được phân chia thành các cấu trúc hợp thành cũng như biểu diễn của toàn bộ tài liệu và văn bản của tài liệu đó sau khi loại bỏ thẻ.


## Syntax

```powerquery
Web.Page(
    html as any
) as table
```
