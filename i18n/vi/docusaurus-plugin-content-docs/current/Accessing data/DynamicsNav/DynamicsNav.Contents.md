# DynamicsNav.Contents


## Description

Nhập URL cho điểm cuối dịch vụ OData của Dynamics NAV.


## Syntax

```powerquery
DynamicsNav.Contents(
    url as text,
    optional company as text
) as table
```


## Details

Trả về bảng chứa dữ liệu Dynamics NAV có liên quan.
