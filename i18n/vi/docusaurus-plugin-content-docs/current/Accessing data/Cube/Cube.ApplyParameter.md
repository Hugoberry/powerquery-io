# Cube.ApplyParameter


Trả về khối cube sau khi áp dụng tham số.


## Syntax

```powerquery
Cube.ApplyParameter(
    cube as table,
    parameter as any,
    optional arguments as list
) as table
```


## Remarks

Trả về khối cube sau khi áp dụng <code>parameter</code> với <code>arguments</code> cho <code>cube</code>.

Cube
