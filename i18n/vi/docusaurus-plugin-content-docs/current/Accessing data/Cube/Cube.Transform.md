# Cube.Transform


## Description

Áp dụng danh sách các hàm cube.


## Syntax

```powerquery
Cube.Transform(
    cube as table,
    transforms as list
) as table
```


## Details

Áp dụng danh sách các hàm cube, <code>transforms</code>, trên <code>cube</code>.



## Category
Cube
