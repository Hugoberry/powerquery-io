# Cube.AttributeMemberProperty


Trả về tính chất của thuộc tính chiều.


## Syntax

```powerquery
Cube.AttributeMemberProperty(
    attribute as any,
    propertyName as text
) as any
```


## Remarks

Trả về tính chất <code>propertyName</code> của thuộc tính chiều <code>attribute</code>.

Cube
