# VSTS.Views


## Description

Nhập tên dự án và tổ chức.


## Syntax

```powerquery
VSTS.Views(
    url as text,
    project as text,
    areaPath as text,
    optional options as record
) as table
```


## Details

Trả về một bảng bao gồm các nguồn cấp dữ liệu OData do Azure DevOps cung cấp.
