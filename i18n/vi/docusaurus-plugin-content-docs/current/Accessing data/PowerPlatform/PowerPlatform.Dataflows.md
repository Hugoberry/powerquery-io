# PowerPlatform.Dataflows


## Description

Nhập dữ liệu từ luồng dữ liệu


## Syntax

```powerquery
PowerPlatform.Dataflows(
    optional options as record
) as table
```


## Details

Kết nối với tất cả các luồng dữ liệu mà bạn có quyền truy cập và chọn những thực thể bạn muốn sử dụng
