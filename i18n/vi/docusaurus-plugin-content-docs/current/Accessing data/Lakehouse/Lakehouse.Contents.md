# Lakehouse.Contents


Nhập dữ liệu từ kho hồ dữ liệu


## Syntax

```powerquery
Lakehouse.Contents(
    optional options as record
) as table
```


## Remarks

Kết nối với tất cả các Kho hồ dữ liệu mà bạn có quyền truy nhập và chọn những thực thể bạn muốn sử dụng
