# Marketo.Tables


## Description

Nhập URL của điểm cuối API REST Marketo được liên kết với tài khoản của bạn.


## Syntax

```powerquery
Marketo.Tables(
    url as text,
    QueryStart as date
) as table
```


## Details

Trả về bảng có dữ liệu Marketo liên quan.
