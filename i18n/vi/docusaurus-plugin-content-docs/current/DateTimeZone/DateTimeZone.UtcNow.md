# DateTimeZone.UtcNow


## Description

Trả về ngày và giờ hiện tại theo giờ UTC (múi giờ GMT).


## Syntax

```powerquery
DateTimeZone.UtcNow(

) as datetimezone
```


## Details

Trả về ngày và giờ hiện tại theo giờ UTC (múi giờ GMT).


## Examples

### Example #1 
Tải ngày và giờ hiện tại theo giờ UTC.
```powerquery
DateTimeZone.UtcNow()
```

Result: 
```powerquery
#datetimezone(2011, 8, 16, 23, 34, 37.745, 0, 0)
```
