# Number.Sin


Trả về sin của số.


## Syntax

```powerquery
Number.Sin(
    number as number
) as number
```


## Remarks

Trả về sin của <code>number</code>.


## Examples

### Example #1 
Trả về sin của góc 0.
```powerquery
Number.Sin(0)
```

Result: 
```powerquery
0
```
