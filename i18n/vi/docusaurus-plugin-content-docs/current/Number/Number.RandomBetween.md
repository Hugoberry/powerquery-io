# Number.RandomBetween


## Description

Trả về một số ngẫu nhiên giữa hai số.


## Syntax

```powerquery
Number.RandomBetween(
    bottom as number,
    top as number
) as number
```


## Details

Trả về một số ngẫu nhiên giữa <code>bottom</code> và <code>top</code>.


## Examples

### Example #1 
Nhận một số ngẫu nhiên giữa 1 và 5.
```powerquery
Number.RandomBetween(1, 5)
```

Result: 
```powerquery
2.546797
```
