# Number.Atan2


## Description

Trả về arctang của phép chia hai số.


## Syntax

```powerquery
Number.Atan2(
    y as number,
    x as number
) as number
```


## Details

Trả về arctang của phép chia hai số là <code>y</code> và <code>x</code>. Phép chia sẽ được thiết lập ở dạng <code>y</code>/<code>x</code>.
