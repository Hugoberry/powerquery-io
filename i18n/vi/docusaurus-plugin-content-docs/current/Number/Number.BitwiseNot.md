# Number.BitwiseNot


Trả về một byte trong đó mỗi bit là đối nghịch của đầu vào.


## Syntax

```powerquery
Number.BitwiseNot(
    number as any
) as any
```


## Remarks

Trả về kết quả thực hiện tác vụ "Not" theo bit đối với <code>number</code>.



## Category
Number.Bytes
