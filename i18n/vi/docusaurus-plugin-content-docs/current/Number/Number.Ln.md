# Number.Ln


## Description

Trả về logarit tự nhiên của số.


## Syntax

```powerquery
Number.Ln(
    number as number
) as number
```


## Details

Trả về giá trị logarit tự nhiên của số, <code>number</code>. Nếu <code>number</code> lànull <code>Number.Ln</code> trả lại null.


## Examples

### Example #1 
Nhận logarit tự nhiên của 15.
```powerquery
Number.Ln(15)
```

Result: 
```powerquery
2.70805020110221
```
