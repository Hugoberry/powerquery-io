# Number.Tan


## Description

Trả về arctang của số.


## Syntax

```powerquery
Number.Tan(
    number as number
) as number
```


## Details

Trả về the arctang của <code>number</code>.


## Examples

### Example #1 
Tìm tang của góc 1.
```powerquery
Number.Tan(1)
```

Result: 
```powerquery
1.5574077246549023
```

Number.Trigonometry
