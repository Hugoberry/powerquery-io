# Number.BitwiseShiftRight


Chuyển dịch tập hợp bit sang phải.


## Syntax

```powerquery
Number.BitwiseShiftRight(
    number1 as number,
    number2 as number
) as number
```


## Remarks

Trả về kết quả thực hiện chuyển dịch theo bit về bên phải đối với <code>number1</code> theo số lượng bit được chỉ định <code>number2</code>.
