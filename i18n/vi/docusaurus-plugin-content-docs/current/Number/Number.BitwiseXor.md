# Number.BitwiseXor


## Description

Trả về kết quả thực hiện &#34;XOR&#34; (Exclusive-OR) theo bit giữa hai đầu vào.


## Syntax

```powerquery
Number.BitwiseXor(
    number1 as number,
    number2 as number
) as number
```


## Details

Trả về kết quả thực hiện "XOR" (Exclusive-OR) theo bit giữa <code>number1</code> và <code>number2</code>.



## Category
Number.Bytes
