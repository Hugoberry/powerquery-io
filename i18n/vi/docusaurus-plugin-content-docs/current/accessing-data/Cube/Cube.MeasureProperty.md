# Cube.MeasureProperty


Trả về thuộc tính của giá trị đo (thuộc tính của ô).


## Syntax

```powerquery
Cube.MeasureProperty(
    measure as any,
    propertyName as text
) as any
```


## Remarks

Trả về thuộc tính <code>propertyName</code> của giá trị đo <code>measure</code>.



## Category
Cube
