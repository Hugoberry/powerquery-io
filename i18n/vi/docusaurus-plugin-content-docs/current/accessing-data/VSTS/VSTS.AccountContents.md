# VSTS.AccountContents


Nhập Url cho Dịch vụ phân tích Azure DevOps của bạn.


## Syntax

```powerquery
VSTS.AccountContents(
    url as text,
    optional options as record
) as binary
```


## Remarks

Trả về nội dung được tải xuống từ url Dịch vụ phân tích Azure DevOps dưới dạng giá trị nhị phân. Chức năng này sử dụng thông tin xác thực khác nhau cho các tổ chức khác nhau.
