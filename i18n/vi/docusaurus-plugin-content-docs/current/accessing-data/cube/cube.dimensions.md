# Cube.Dimensions


Trả về bảng chứa tập hợp kích thước khả dụng.


## Syntax

```powerquery
Cube.Dimensions(
    cube as table
) as table
```


## Remarks

Trả về bảng chứa tập hợp kích thước khả dụng trong <code>cube</code>. Mỗi kích thước là bảng chứa tập hợp các thuộc tính kích thước và mỗi thuộc tính kích thước được thể hiện dưới dạng cột trong bảng kích thước. Có thể mở rộng kích thước trong cube bằng Cube.AddAndExpandDimensionColumn.
