# Cube.MeasureProperties


Trả về một bảng có chứa tập hợp các thuộc tính giá trị đo có sẵn được mở rộng trong cube.


## Syntax

```powerquery
Cube.MeasureProperties(
    cube as table
) as table
```


## Remarks

Trả về một bảng có chứa tập hợp các thuộc tính có sẵn cho giá trị đo được mở rộng trong cube.
